### 拉普拉斯变换与拉普拉斯逆变换
#### 拉普拉斯变换的定义
+ 函数$ F(t) $的拉普拉斯变换定义为：

$ f(s) = \int_0^{+\infin} F(t)e^{-st}dt $

+ 记为：

$ \mathcal{L}[F(t)](s) = f(s) $

+ 或者记为：

$ F(t) \fallingdotseq f(s) $

#### 拉普拉斯逆变换的定义
+ $ f(s) $拉普拉斯逆变换可以通过下式进行计算：

$ F(t) = \frac{1}{2\pi i}\int_{c-i\infin}^{c+i\infin}f(s)e^{st}ds $

    - 其中，$ f(s) $在$ \mathrm{Re}(c)>0 $区域处处解析。
+ 记为：

$  \mathcal{L}^{-1}[f(s)](t) = F(t) $

+ 或者记为：

$ f(s) \risingdotseq F(t) $

### 拉普拉斯变换的简单性质
+ 线性性

$ \mathcal{L}[aF(t)+bG(t)] (s)= af(s)+bg(s) $

+ 伸缩定理

$ \mathcal{L}[F(at)](s) = \frac{1}{a}f(\frac{s}{a}) $

+ 卷积定理

$ \mathcal{L}[F_1*F_2](s) = f_1(s)f_2(s) $

    - 其中，拉普拉斯变换的卷积运算定义为：

$ [F_1*F_2](t) = \int_0^{t} F_1(\tau)F_2(t-\tau)d\tau $

### 拉普拉斯变换的复杂性质
#### XX的变换（正变换性质）
+ 导数的变换

$ \mathcal{L}[F'(t)](s) = sf(s) - F(0) $

+ n阶导数的变换

$ \mathcal{L}[F^{(n)}(t)](s) = s^nf(s) - s^{n-1}F(0) - s^{n-2}F'(0) - \cdots -sF^{(n-2)}(0)-F^{(n-1)}(0) $

+ 积分的变换

$ \mathcal{L}[\int_0^{t}F(\tau)d\tau](s) = \frac{f(s)}{s} $

    - 可以看成导数的变换一节中$ n=-1 $的结果。
+ 向右平移的变换

$ \mathcal{L}[F(t-b)H(t-b)](s) = e^{-bs}f(s),b>0 $

    - 其中$ H(t) $为$ \mathrm{Heaviside} $阶跃函数。
    - 此处需要引入$ \mathrm{Heaviside} $阶跃函数的原因是，$ F(t) $的图像整体向右平移了$ b $个单位长度，为了保证平移进来的长度为$ b $部分的影响，需要通过$ H(t-b) $删除掉该部分。

#### 变换的XX（逆变换性质）
+ 变换的导数

$ \mathcal{L}^{-1}[f'(s)](t) = (-t)F(t) $

+ 变换的n阶导数

$ \mathcal{L}^{-1}[f^{(n)}(s)](t) = (-t)^nF(t) $

+ 变换的积分

$ \mathcal{L}^{-1}[\int_s^{+\infin}f(x)dx](t) = \frac{F(t)}{t} $

    - 可以看成导数的变换一节中$ n=-1 $的结果，因为$ (-t) $中有负号，所以积分从$ s $积到$ +\infin $。
+ 变换的向右平移

$ \mathcal{L}^{-1}[f(s-a)](t) = e^{at}F(t),a>0 $

#### 对称性分析
+ 拉普拉斯变换与拉普拉斯逆变换具有非常强的对称性，但是往往相差一个负号
    - 导数的变换会乘以$ s^n $并引入初值，变换的导数会乘以$ (-t)^n $但不引入初值
    - 积分的变换积分范围从$ 0 $到$ t $，而变换的积分积分范围从$ s $到$ +\infin $
        * 积分的变换结果除以$ s $，而变换的积分结果除以$ t $
    - 向右平移的变换会需要乘以$ e^{-bs} $，而变换的向右平移需要乘以$ e^{at} $

### 一些常用函数的拉普拉斯变换
+ 常数

$ \mathcal{L}[1](s) = \frac{1}{s} $

+ 指数函数（变换的向右平移）

$ \mathcal{L}[e^{kt}](s) = \frac{1}{s-k} $

+ 双曲三角函数（指数函数的线性组合）

$ \mathcal{L}[\cosh (kt)](s) = \frac{s}{s^2-k^2} $

$ \mathcal{L}[\sinh (kt)](s) = \frac{k}{s^2-k^2} $

+ 三角函数（利用三角函数与双曲三角函数的关系）

$ \mathcal{L}[\cos (kt)](s) = \frac{s}{s^2+k^2} $

$ \mathcal{L}[\sin (kt)](s) = \frac{k}{s^2+k^2} $

+ 幂函数（利用$ \Gamma $函数的性质）

$ \mathcal{L}[t^n](s) =\frac{n!}{s^{n+1}} = \frac{\Gamma (n+1)}{s^{n+1}} $

### 拉普拉斯变换与傅立叶变换之间的对称性
+ 拉普拉斯变换的线性性、伸缩定理、卷积定理、导数、平移变换的结论都与傅立叶变换有很多相似之处。
+ 此外，拉普拉斯变换逆变换中的$ i $和特殊的积分路径可以视为傅立叶变换在“复平面上旋转90度”的结果。
+ Insight：之所以拉普拉斯变换可以用于计算积分，是因为拉普拉斯变换可以把带有指数成分（如指数、三角函数、双曲三角函数）的函数转化为分式，从而使计算更简便。另外拉普拉斯变换也是从实数域到实数域的变换，这是傅立叶变换不具有的优势。
