### 二阶线性齐次常微分方程的解
#### 方程的形式
+ 二阶线性齐次常微分方程的形式为：

$ y''(x) + P(x)y' +Q(x)y = 0 $

#### 线性无关解
+ 二阶线性齐次常微分方程有2个线性无关的齐次解：$ y_1 (x) $和$ y_2(x) $
+ 一般齐次解可以表示为：

$ y(x) = c_1 y_1 (x) + c_2 y_2 (x) $

    - 其中$ c_1 $和$ c_2 $为与$ x $无关的常数
+ 注意到，通解可以写为用$ P(x) $和$ Q(x) $表示的形式：

$ y(x) = y(P(x),Q(x)) $

#### 非齐次化
+ 考虑二阶线性非齐次常微分方程：

$ y''(x) + P(x)y' +Q(x)y = f(x) $

    - 假设非齐次方程有一特解$ y_p (x) $，则非齐次方程通解可以表示为：

$ y(x) = c_1 y_1 (x) + c_2 y_2 (x) + y_p(x) $

#### 解的关系与Wronski行列式
+ 定义二阶线性齐次常微分方程的解的$ \mathrm{Wronski} $行列式：

$ W = \begin{vmatrix} 
y_1(x) & y_2(x) \\y_1'(x) & y_2'(x)  \end{vmatrix}
= y_1 y_2' - y_2 y_1' $

+ 可证明：

$ W = \exp{[-\int^{x} P(x') dx']} $

    - 以及：

$ y_2(x) = y_1(x) \int^x \frac{W(x')}{[y_1 (x)]^2} dx' $

    - 上述表达式可以用于已知方程、一个解$ y_1(x) $求另外一个解$ y_2(x) $的情形。

### 奇点的分类
#### 有限远点的判定法则
1. $ x_0 $是方程的**常点**，如果$ P(x_0) $和$ Q(x_0) $有限
2. $ x_0 $是方程的**正则奇点**，若：
    1. $ P(x_0) $或$ Q(x_0) $发散
    2. 且$ P(x_0) $发散不快于$ \frac{1}{x-x_0} $（$ \mathrm{Laurent} $级数$ -2,-3... $次项系数为$ 0 $）
    3. 且$ 
Q(x_0) $发散不快于$ \frac{1}{(x-x_0)^2} $（$ \mathrm{Laurent} $级数$ -3,-4... $次项系数为$ 0 $）
3. 其他情形，$ x_0 $是方程的非正则奇点。
+ **注意：在讨论**$ P(x) $**和**$ Q(x) $**之前都需要将方程化为标准形式！**

#### 无穷远点的判定法则
+ 作变量替换$ t = 1/z $，$ y(z) = \tilde{y}(t) $，容易知道：

$ \frac{dy}{dz} 
= \frac{dt}{dz} \cdot \frac{d\tilde{y}}{dt} 
= (\frac{dz}{dt})^{-1} \cdot \frac{d\tilde{y}}{dt}
= -t^2 \cdot \frac{d\tilde{y}}{dt} $

$ \frac{d^2y}{dz^2} 
= \frac{d}{dz} (\frac{dy}{dz} )
= \frac{dt}{dz}\cdot\frac{d}{dt}(\frac{dy}{dz} )
= (\frac{dz}{dt})^{-1}\cdot\frac{d}{dt}(\frac{dy}{dz} )
= -t^2 \cdot \frac{d}{dt}[-t^2 \cdot \frac{d\tilde{y}}{dt}]
= 2t^3 \cdot \frac{d\tilde{y}}{dt} + t^4 \cdot \frac{d^2\tilde{y}}{dt^2} $

+ 代入原方程，得：

$ (2t^3 \cdot \frac{d\tilde{y}}{dt} + t^4 \cdot \frac{d^2\tilde{y}}{dt^2}) - t^2P(1/t)\cdot \frac{d\tilde{y}}{dt} +Q(1/t)\tilde {y} = 0 $

+ 整理得：

$  \frac{d^2\tilde{y}}{dt^2} + (\frac{2}{t} - \frac{P(1/t)}{t^2})\cdot \frac{d\tilde{y}}{dt} +\frac{Q(1/t)}{t^4}\cdot \tilde {y} = 0 $

+ 根据标准形式可知：

$ \tilde{P}(t) = \frac{2}{t} - \frac{P(1/t)}{t^2} $

$ \tilde{Q}(t) = \frac{Q(1/t)}{t^4} $

+ 考察$ t=0 $处方程的特征，即得$ z=\infin $处方程的解析性。

### 常点处的级数解
1. 常点：两个$ \mathrm{Taylor} $级数解：

$ y_1(x) = \sum_{n=0}^{\infin} a_n  (x-x_0)^n $

$ y_2(x) = \sum_{n=0}^{\infin} b_n  (x-x_0)^n $

### 正则奇点处的级数解
2. 正则奇点：
    1. 如果指标之差$ r_1-r_2 \notin \mathbb{Z} $，则有两个$ \mathrm{Frobenius} $解：

$ y_1(x) = (x-x_0)^{r_1}\sum_{n=0}^{\infin} a_n (x-x_0)^n $

$ y_2(x) =  (x-x_0)^{r_2}\sum_{n=0}^{\infin} b_n  (x-x_0)^n $

    2. 如果指标之差$ r_1-r_2 = 0 $，则有一个$ \mathrm{Frobenius} $解，第二解一定含有对数函数项。

$ y_1(x) =  (x-x_0)^{r}\sum_{n=0}^{\infin} a_n  (x-x_0)^n $

$ y_2(x) = y_1 \ln  (x-x_0) +  (x-x_0)^{r}\sum_{n=0}^{\infin} b_n  (x-x_0)^n $

    3. 如果指标之差$ r_1-r_2 = m \in \mathbb{Z}_+ $，则有一个$ \mathrm{Frobenius} $解，第二解可能含有对数函数项。

$ y_1(x) =  (x-x_0)^{r_1}\sum_{n=0}^{\infin} a_n  (x-x_0)^n $

$ y_2(x) = cy_1 \ln  (x-x_0) +  (x-x_0)^{r_2}\sum_{n=0}^{\infin} b_n  (x-x_0)^n $
