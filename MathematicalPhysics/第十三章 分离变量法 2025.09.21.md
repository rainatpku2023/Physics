### 分离变量法精髓概述
#### 步骤
分离变量法的精髓：求解本征值问题，用本征函数展开解。

使用本征函数的条件：齐次化的边界条件,，第123类齐次边界条件均可（[Sturm-Liouville理论](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/nsycge/zrpcbrrtvckx8zp3)）。

1 齐次方程、齐次边界条件：直接分离变量，求解本征值问题，证明本征函数正交性（需要用到齐次边界条件消去表面项），求出叠加系数，叠加出特解。

2 非齐次方程、齐次边界条件：

    1. 上策：选取齐次化函数（这个函数一定含有本征函数对应变量），待定系数，在保证边界条件仍然其次的情况下将方程化为齐次方程。转化为问题1。
    2. 下策：非齐次项按照对应非齐次方程的本征函数进行展开，代换，求叠加系数。

3 齐次方程、非齐次边界条件：

    1. 上策：选取齐次化函数（这个函数一定含有本征函数对应变量），待定系数，在保证方程仍然其次的情况下将边界条件化为齐次边界条件。转化为问题1。
    2. 下策：选取不太好的齐次化函数，边界条件齐次但是会让方程非齐次化，转化为2b。

#### 常用积分结果
这些是常用的积分结果，记忆后可以快速简化计算。

$ \int_0^l \sin \frac{n\pi x}{l} dx = 0 $

$ \int_0^l \sin \frac{n\pi x}{l}\sin \alpha x  dx = \frac{(-1)^{n+1} nπ l \sin(αl)} {n^2π^2-α^2 l^2 } $

$ \int_0^l x\sin \frac{n\pi x}{l}   dx = \frac{(-1)^{n+1}l^2}{n\pi} $

#### 半截Fourier级数 常用积分结果
+ 在高等数学中，常常定义对于周期为$ 2T $的函数$ f(x) $进行$ \mathrm{Fourier} $级数展开：

$ f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infin} a_n \cos\frac{n\pi}{T}x+\sum_{n=1}^{\infin}b_n \sin\frac{n\pi}{T}x $

    - 此时，展开系数的计算公式为：

$ a_n = \frac{1}{T}\int_{-T}^T f(x)\cos\frac{n\pi}{T}xdx,\; n=0,1,2,3... $

$ b_n = \frac{1}{T}\int_{-T}^T f(x)\sin\frac{n\pi}{T}xdx,\; n=1,2,3... $

+ 在分离变量法中，常常将函数定义在$ [0,l] $上的$ f(x) $在三角函数基上进行展开（半截$ \mathrm{Fourier} $级数）：
    - <font style="color:#DF2A3F;">之所以称为“半截”，是因为积分区间为</font>$ [0,l] $<font style="color:#DF2A3F;">而非</font>$ [-l,l] $，与高等数学常用的标准$ \mathrm{Fourier} $级数不同。

$ f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infin} a_n \cos\frac{n\pi}{l}x+\sum_{n=1}^{\infin}b_n \sin\frac{n\pi}{l}x $

    - 此时，展开系数$ a_0 $可以合并入$ n=1,2,3.... $中：

$ a_n = \frac{2}{l}\int_0^l f(x)\cos\frac{n\pi}{l}xdx,\; n=0,1,2,3... $

$ b_n = \frac{2}{l}\int_0^l f(x)\sin\frac{n\pi}{l}xdx,\; n=1,2,3... $

    - <font style="color:#DF2A3F;">注意，最后要计算</font>$ a_0/2 $<font style="color:#DF2A3F;">而非</font>$ a_0 $<font style="color:#DF2A3F;">！</font>
+ 特别地，当$ l = \pi $时，$ \mathrm{Fourier} $级数表达式为：

$ f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infin} a_n \cos nx+\sum_{n=1}^{\infin}b_n \sin nx $

    - 此时，展开系数的计算公式为：

$ a_n = \frac{2}{\pi}\int_0^\pi f(x)\cos nxdx,\; n=0,1,2,3... $

$ b_n = \frac{2}{\pi}\int_0^\pi f(x)\sin nxdx,\; n=1,2,3... $

### 情形1：齐次方程，齐次边界条件，直接应用分离变量法
#### 情景概述：以波动方程为例
$ \frac{\partial^2 u}{\partial x^2}-a^2 \frac{\partial^2 u}{\partial t^2} = 0 $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

$ u|_{t=0} = \phi(x) \;\;\; \frac{\partial u}{\partial t}|_{x=l} = \psi(x) $

#### 分离变量法的基本步骤
##### 分离变量
+ 希望特解具有分离变量的形式，即：

$ u(x,t) = X(x)T(t) $

+ 将上式代入齐次方程、齐次边界条件得：

$ X''(x)T(t)-a^2 X(x)T''(t) = 0 $

$ X(0)T(t) = 0 \;\;\; X(l)T(t) = 0 $

    - 考虑到不可能有$ T(t) = 0 $，否则会有$ u(x,t) = 0 $，所以边界条件可以改写为：

$ X(0)= 0 \;\;\; X(l) = 0 $

    - 此时选定$ X(x) $进行本征值问题的求解
    - 对于方程进行变量分离，引入常数$ \lambda $得：

$ \frac{X''(x)}{X(x)} = \frac{1}{a^2}\frac{T''(t)}{T(t)}  = -\lambda $

    - 此时得到$ X(x) $的本征方程以及由$ \lambda $相互关联的$ T(t) $的相关方程：

$ X''(x) + \lambda X(x) = 0 $

$ T''(t) + \lambda a^2T(t) = 0 $

##### 求解本征值问题
+ 对于$ X(x) $的本征方程$ X''(x) + \lambda X(x) = 0 $，可以先对本征值进行讨论，求解出来所有的本征值：
    - 情况1：$ \lambda = 0 $时，代入本征方程知：

$ X''(x) = 0 $

        * 由此确定$ X(x) $的形式为：

$ X(x) = A_0x+B_0 $

        * 结合齐次边界条件$ X(0)= 0 \;\;\; X(l) = 0 $可知：

$ A=B = 0 $

        * 代入知：

$ X(x) = 0 $

        * 但是在此种情形中$ u(x,t) = 0 $，没有讨论必要，因此$ 0 $并非一个本征值。
    - 情况2：$ \lambda \neq 0 $时，代入本征方程知：

$ X''(x) + \lambda X(x) = 0 $

        * 这是一个二阶线性齐次微分方程，有两个自由度，通解可以写为：

$ X(x) = A\sin\sqrt{\lambda}x+B\cos\sqrt{\lambda}x $

        * 结合齐次边界条件$ X(0)= 0 \;\;\; X(l) = 0 $可知：

$ X(0) = B \cos(0) = B = 0 $

$ X(l) = A\sin\sqrt{\lambda}l+B\cos\sqrt{\lambda}l = A\sin\sqrt{\lambda}l = 0 $

            + 此时必须有$ A \neq 0 $，否则$ X(x) =0 $，因此有：

$ \sin \sqrt{\lambda} l = 0 $

            + 由此**解出所有的本征值**：

$ \lambda_n = (\frac{n\pi}{l})^2, \; n = 1,2,3... $

            + 并**解出对应的所有本征函数**：

$ X(x) = \sin \frac{n\pi}{l}x, \; n = 1,2,3... $

            + 此处也应该考虑$ n = -1,-2,-3... $的情形，但是它们与$ n = 1,2,3... $的情形线性相关。
+ 综上，所有的本征值以及对应的本征函数为：

$ \lambda_n = (\frac{n\pi}{l})^2, \; n = 1,2,3... $

$ X_n(x) = \sin \frac{n\pi}{l}x, \; n = 1,2,3... $

+ 下面对于不同本征值的本征函数的正交性进行证明，并求解模方，为确定叠加系数做铺垫：
    - 先证明不同本征值的本征函数的正交性：

$ <X_n,X_m> = \int_0^l X_n^*(x)X_m(x)dx = \int_0^l \sin \frac{n\pi}{l}x\sin \frac{m\pi}{l}xdx $

$ =\int_0^l \frac{1}{2}\{\cos \frac{(n-m)\pi}{l}x - \cos \frac{(n+m)\pi}{l}x\}dx $

$ =\frac{1}{2} \{\frac{l}{(n-m)\pi}\sin \frac{(n-m)\pi}{l}x - \frac{l}{(n+m)\pi}\sin \frac{(n+m)\pi}{l}x\}_0^l $

$ =0, \: \mathrm{if} \: m\neq n $

        * 这种正交性的证明并不具有普遍性，普遍性的证明需要$ \mathrm{Sturm-Liouville} $理论，普遍性证明方法见后文。
    - 再求解模方：

$ ||X_n||^2 = <X_n,X_n> = \int_0^l X_n^*(x)X_n(x)dx = \int_0^l \sin \frac{n\pi}{l}x\sin \frac{n\pi}{l}xdx $

$ = \frac{1}{2}\int_0^{l} (1-\cos \frac{2n\pi}{l}x)dx = \frac{l}{2} $

        * 注意在复数域中，内积的定义为：

$ <X_n,X_m> = \int_0^l X_n(x)X_m^*(x)dx $

##### 求特解，并叠加出一般解
+ $ X(x) $的本征值问题求解完成后，求出$ X_n(x) $对应的$ T_n(t) $：
    - 应该用$ \lambda_n $代入$ T(t) $满足的方程：

$ T_n''(t) + (\frac{n\pi}{l})^2a^2T_n(t) = 0 $

    - 这是一个二阶线性齐次常微分方程，有两个自由度，不妨设解为：

$ T_n(t) = C_n \sin \frac{n\pi a}{l}t + D_n \cos \frac{n\pi a}{l}t $

+ 此时求出本征值$ \lambda_n $对应的整体方程的特解：

$ u_n(x,t) = X_n(x)T_n(t) =(C_n \sin \frac{n\pi a}{l}t + D_n \cos \frac{n\pi a}{l}t)\sin \frac{n\pi}{l}x $

+ 再叠加出整个方程的通解，可以重新规定$ C_n,D_n $意义将新增的叠加系数$ k_n $吸收进去，不换符号：

$ u(x,t) = \sum_{n=1}^{\infin}k_nu_n(x,t) =\sum_{n=1}^{\infin}(C_n \sin \frac{n\pi a}{l}t + D_n \cos \frac{n\pi a}{l}t)\sin \frac{n\pi}{l}x $

    - 这就是方程的一般解。

##### 利用本征函数的正交性定叠加系数
+ 方程的通解中仍然有两组自由度$ C_n,D_n $，需要用到两个边界条件进行确定：

$ u(x,0) = \sum_{n=1}^{\infin} D_n \sin \frac{n\pi}{l}x = \phi(x) $

$ u_t(x,0) = \sum_{n=1}^{\infin} C_n \frac{n\pi a}{l}\sin \frac{n\pi}{l}x = \psi(x) $

    - 注意，本例的形式决定了叠加系数形式会比较简洁，但是一般情况很可能更加复杂。
+ 根据本征函数的正交归“一”性：

$ <X_n,X_m> = \int_0^l X_n(x)X_m^*(x)dx = \int_0^l \sin \frac{n\pi}{l}x\sin \frac{m\pi}{l}xdx = \frac{l}{2}\delta_{nm} $

    - 前方叠加系数分别为：

$ D_n = \frac{2}{l}\int_0^l \phi(x) \sin \frac{n\pi}{l}xdx $

$ C_n \cdot \frac{n\pi a}{l}= \frac{2}{l}\int_0^l \phi(x) \sin \frac{n\pi}{l}xdx $

        * 注意，最终代入的值是$ C_n $而非$ C_n \cdot \frac{n\pi a}{l} $，所以：

$ C_n= \frac{2}{n\pi a}\int_0^l \phi(x) \sin \frac{n\pi}{l}xdx $

    - 求解出$ C_n,D_n $值后，将其代入通解$ u(x,t) $中，得到：

$ u(x,t) = \sum_{n=1}^{\infin}[(\frac{2}{n\pi a}\int_0^l \phi(x) \sin \frac{n\pi}{l}xdx) \sin \frac{n\pi a}{l}t +  (\frac{2}{l}\int_0^l \phi(x) \sin \frac{n\pi}{l}xdx) \cos \frac{n\pi a}{l}t]\sin \frac{n\pi}{l}x $

##### 补证：不同本征值的本征函数正交的一般性证明方法
:::tips
+ 要证：

$ <X_n,X_m> = \int_0^l X^*_n(x)X_m(x)dx = 0, \;\mathrm{if} \; m\neq n \;\mathrm{and}\; \lambda_m\neq \lambda_n
 $

+ 证明：
    - 根据本征方程以及对应的齐次边界条件：

$ X_n''(x) + \lambda_n X_n(x) = 0 $

$ X_n(0)= 0 \;\;\; X_n(l) = 0 $

$ X_m''(x) + \lambda_m X_m(x) = 0 $

$ X_m(0)= 0 \;\;\; X_m(l) = 0 $

        * 对$ \lambda_n $相关的本征方程和齐次边界条件求复共轭，并考虑到$ \mathrm{S-L} $算符为实数$ \lambda_n = \lambda_n^* $：

$ (X_n^*)''(x) + \lambda_n X_n^*(x) = 0 $

$ X_n^*(0)= 0 \;\;\; X_n^*(l) = 0 $

$ X_m''(x) + \lambda_m X_m(x) = 0 $

$ X_m(0)= 0 \;\;\; X_m(l) = 0 $

    - 将两个方程两边相减，分别乘以$ X_m(x),X_n^*(x) $再进行积分：

$ \int_0^l [(X_n^*)''(x)X_m(x)-X_m''(x)X_n^*(x)]dx + (\lambda_n-\lambda_m) \int_0^l X^*_n(x)X_m(x)dx = 0 $

    - 整理得：

$  (\lambda_n-\lambda_m) \int_0^l X^*_n(x)X_m(x)dx = -\int_0^l [(X_n^*)''(x)X_m(x)-X_m''(x)X_n^*(x)]dx  $

        * 代入齐次边界条件（并不一定是第一类，二三类均可）：

$  (\lambda_n-\lambda_m) \int_0^l X^*_n(x)X_m(x)dx = - [(X_n^*)'(x)X_m(x)-X_m'(x)X_n^*(x)]_{x=0}^{x=l} = 0 $

    - 结合前提假设$ \lambda_m \neq \lambda_n $：

$ <X_n,X_m> = \int_0^l X^*_n(x)X_m(x)dx = 0, \;\mathrm{if} \; m\neq n \;\mathrm{and}\; \lambda_m\neq \lambda_n
 $

    - 即不同本征值的本征函数正交。

$ \mathrm{Q.E.D.} $

:::

#### 另外一个情景与Insights：以稳定问题为例
$ \frac{\partial^2 u}{\partial x^2}+\frac{\partial^2 u}{\partial y^2} = 0 $

$ u|_{x=0} = 0 \;\;\; \frac{\partial u}{\partial x}|_{x=a} = 0 $

$ u|_{y=0} = \phi(x) \;\;\; \frac{\partial u}{\partial y}|_{y=b} = \psi(x) $

+ Insights：
    - 变量分离并不要求必须是齐次的第一类边界条件，齐次的二三类边界条件也可以。
        * 本例中$ \frac{\partial u}{\partial x}|_{x=a} = 0 $并不影响变量分离
    - 变量分离的选择和字母无关，主要是需要有齐次边界条件。
        * 本例中$ x $变量用于变量分离求解本征函数，$ y $变量用于叠加出特解。

#### 分叉：多于两个自变量的定解问题：多于一个本征方程
+ 假设存在一个$ n $个变量的二阶线性偏微分方程：

$ \hat{L}(\frac{\partial}{\partial x^1},\frac{\partial}{\partial x^2},...,\frac{\partial}{\partial x^n})u(x^1,x^2,...,x^n) = f(x^1,x^2,...,x^n) $

$ u^1|_{x^1=0}= \mu^1(x^1,x^2,...,x^n) \;\;\; u^1|_{x^1=l^1} = \nu^1(x^1,x^2,...,x^n) $

$ u^2|_{x^2=0}= \mu^2(x^1,x^2,...,x^n) \;\;\; u^2|_{x^2=l^2} = \nu^2(x^1,x^2,...,x^n) $

$ ...... $

$ u^{(n-1)}|_{x^{(n-1)}=0}(x^1,x^2,...,x^n)=\mu^{(n-1)}\;\;\; u^{(n-1)}|_{x^{(n-1)}=l^{(n-1)}} = \nu^{(n-1)}(x^1,x^2,...,x^n) $

$ u^n|_{x^n=0}= \mu^n(x^1,x^2,...,x^n) \;\;\; u^n|_{x^n=l^n} = \nu^n(x^1,x^2,...,x^n) $

    - <font style="color:#DF2A3F;">其中右上角的指标代表变量编号，而非次方数</font>
+ 用分离变量解法如下：
    - 先选定$ n-1 $个变量作为本征值问题待选变量

$ x^1,x^2,x^3,....,x^{(n-1)} $

    - 再将这$ n-1 $个变量对应的边界条件以及方程化为齐次，留下最后一个变量不做处理

$ \hat{L}(\frac{\partial}{\partial x^1},\frac{\partial}{\partial x^2},...,\frac{\partial}{\partial x^n})w(x^1,x^2,...,x^n) = 0 $

$ w^1|_{x^1=0}=0 \;\;\; w^1|_{x^1=l^1} = 0 $

$ w^2|_{x^2=0}= 0 \;\;\; w^2|_{x^2=l^2} = 0 $

$ ...... $

$ w^{(n-1)}|_{x^{(n-1)}=0}=0\;\;\; w^{(n-1)}|_{x^{(n-1)}=l^{(n-1)}} = 0 $

$ w^n|_{x^n=0}= \mu^{n*}(x^1,x^2,...,x^n) \;\;\; w^n|_{x^n=l^n} = \nu^{n*}(x^1,x^2,...,x^n) $

        * 其中$ \mu^{n*},\nu^{n*} $是$ n-1 $个变量对应的边界条件以及方程化为齐后第$ n $个边界条件的变化后的结果。
    - 求解$ n-1 $个本征值问题，得到$ n-1 $组本征值以及$ n-1 $组本征函数基，求模方证正交
        * 本征函数组：$ \{X_i^1(x^1)\},\{X_i^2(x^2)\},...,\{X_i^{(n-1)}(x^{(n-1)})\} $
        * 本征值组：$ \{\lambda_i^1\},\{\lambda_i^2\},...,\{\lambda_i^{(n-1)}\} $

$ <X_i^1,X_j^1> = P^1 \delta_{ij} $

$ <X_i^2,X_j^2> = P^2 \delta_{ij} $

$ ...... $

$ <X_i^{(n-1)},X_j^{(n-1)}> = P^{(n-1)} \delta_{ij} $

    - 用本征值组$ \{\lambda_i^1\},\{\lambda_i^2\},...,\{\lambda_i^{(n-1)}\} $表示$ X_i^n(x^n) $所满足的方程，求出其通解。
    - 用$ n-1 $个本征值表示未处理的变量的常微分方程，写出常微分方程通解，保留叠加系数

$ w_i(x^1,x^2,...,x^n) = X_{i^1}^1(x^1) X_{i^2}^2(x^2) \cdots X_{i^{(n-1)}}^{(n-1)}(x^{(n-1)}) X_{i^n}^n(x^{n}) $

    - 用特解组合出通解

$ w(x^1,x^2,...,x^n) = \sum_{i^1=1}^{\infin}\sum_{i^1=2}^{\infin}\cdots \sum_{i^{(n-1)}=1}^{\infin}\sum_{i^n=1}^{\infin}X_{i^1}^1(x^1) X_{i^2}^2(x^2) \cdots X_{i^{(n-1)}}^{(n-1)}(x^{(n-1)}) X_{i^n}^n(x^{n}) $

    - 利用$ n-1 $组本征函数基，代入第$ n $个边界条件/初始条件：

$ w(x^1,x^2,...,0) = \sum_{i^1=1}^{\infin}\sum_{i^1=2}^{\infin}\cdots \sum_{i^{(n-1)}=1}^{\infin}\sum_{i^n=1}^{\infin}X_{i^1}^1(x^1) X_{i^2}^2(x^2) \cdots X_{i^{(n-1)}}^{(n-1)}(x^{(n-1)}) X_{i^n}^n(0) = \mu^{n*}(x^1,x^2,...,l^n) $

$ w(x^1,x^2,...,l^n) = \sum_{i^1=1}^{\infin}\sum_{i^1=2}^{\infin}\cdots \sum_{i^{(n-1)}=1}^{\infin}\sum_{i^n=1}^{\infin}X_{i^1}^1(x^1) X_{i^2}^2(x^2) \cdots X_{i^{(n-1)}}^{(n-1)}(x^{(n-1)}) X_{i^n}^n(l^{n}) =  \nu^{n*}(x^1,x^2,...,l^n) $

        * 其中，$ A_{1,2,3...,(n-1)} $是叠加系数，他们组成$ n $阶张量。

$  X_{i^n}^n(0)A_{1,2,3...,(n-1)} = \frac{1}{P^1P^2\cdots P^n}\int \mu^{n*}(x^1,x^2,...,l^n) X_{i^1}^1(x^1) X_{i^2}^2(x^2) \cdots X_{i^{(n-1)}}^{(n-1)}(x^{(n-1)}) dx^1 dx^2\cdots dx^{(n-1)} $

            + 或者写为更加紧凑的形式：

$  X_{i^n}^n(0)A_{1,2,3...,(n-1)} = \frac{1}{\prod_{\alpha=1}^{n}P^\alpha} \int \mu^{n*}X_{i^\alpha}^1(x^\alpha) d\vec{r}^{(n-1)} $

### 情景2：非齐次方程
#### 方法1：将非齐次方程齐次化，同时保证边界条件齐次
##### 方法概述
$ \frac{\partial^2 u}{\partial x^2}-a^2 \frac{\partial^2 u}{\partial t^2} = f(x,t) $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

$ u|_{t=0} = \phi(x) \;\;\; \frac{\partial u}{\partial t}|_{t=0} = \psi(x) $

+ 总体上说，应该找一个齐次化函数$ v(x,t) $满足：

$ \frac{\partial^2 v}{\partial x^2}-a^2 \frac{\partial^2 v}{\partial t^2} = f(x,t) $

$ v|_{x=0} = 0 \;\;\; v|_{x=l} = 0 $

+ 这样，设$ u(x,t) = w(x,t) + v(x,t) $时，$ w(x,t) $满足：

$ \frac{\partial^2 w}{\partial x^2}-a^2 \frac{\partial^2 w}{\partial t^2} = 0 $

$ w|_{x=0} = -v|_{x=0} \;\;\; w|_{x=l} = -v|_{x=l} $

$ w|_{t=0} = \phi(x)-v|_{t=0} \;\;\; \frac{\partial w}{\partial t}|_{t=0} = \psi(x)-\frac{\partial v}{\partial t}|_{t=0} $

    - 这样问题就转化为了**齐次方程齐次边界条件（情景1）**的情形

##### 注意事项
+ 上述问题中$ v(x,t) $的选取比较考验技术，总体上分为三种情况：
    1. $ f(x,t) = f(x) $，即方程非齐次项不显含$ t $时，可以假定$ v(x,t) = v(x) $即不显含$ t $，因为齐次边界条件$ v|_{x=0} = 0 \;\;\; v|_{x=l} = 0 $应该被自然满足。

#### 方法2：按照相应齐次问题本征函数展开
+ 考虑定解问题：

$ \frac{\partial^2 u}{\partial t^2}-a^2 \frac{\partial^2 u}{\partial x^2} = f(x,t) $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

$ u|_{t=0} = \phi(x) \;\;\; \frac{\partial u}{\partial t}|_{t=0} = \psi(x) $

+ 对应的齐次问题为：

$ \frac{\partial^2 u}{\partial x^2}-a^2 \frac{\partial^2 u}{\partial t^2} =0 $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

    - 分离变量得到本征值问题：

$ X_n''(x) + \lambda_n X_n(x) = 0 $

$ X_n(0)= 0 \;\;\; X_n(l) = 0 $

    - 设其本征函数系为$ \{ X_n(x) \} $。
+ 将非齐次问题的方程**非齐次项**在本征函数系$ \{ X_n(x) \} $上进行展开:

$ f(x,t) = \sum_{n=1}^{\infin} g_n(t) X_n(x) $

    - 再将**特解**在本征函数系$ \{ X_n(x) \} $上进行展开：

$ u(x,t) = \sum_{n=1}^{\infin} T_n(t) X_n(x) $

+ 将上述两个展开式代入非齐次方程：

$ \sum_{n=1}^{\infin}T_n''(t)X_n(x)-a^2\sum_{n=1}^{\infin}T_n(t)X_n''(x) = \sum_{n=1}^{\infin}g_n(t)X_n(x) $

    - 将本征值问题的方程$ X_n''(x) + \lambda_n X_n(x) = 0 $代入：

$ \sum_{n=1}^{\infin}T_n''(t)X_n(x)+a^2\sum_{n=1}^\infin \lambda_nT_n(t)X_n(x) = \sum_{n=1}^{\infin}g_n(t)X_n(x) $

        * 此处本质上是将$ X''_n(x) $换成$ X_n(x) $，这样方便比较系数，找到$ T_n(t) $所满足的方程。
    - 根据本征函数正交性，可以逐项得到$ T_n(t) $所满足的常微分方程：

$ T_n''(t)+a^2\lambda_nT_n(t) =g_n(t) $

    - 上述常微分方程可以用拉普拉斯变换求解。
        * $ T_n(t),T_n'(t) $的值可以通过非齐次方程对应初值条件，结合叠加系数求法求出。
        * 因为用到了拉普拉斯变换，所以初始条件齐次最理想。
    - 更详细的论述，请参考：[再论按照相应齐次问题本征函数展开方法 2025.10.30](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/rczspq55v7e5ftkd)

### 情景3：非齐次边界条件
#### 基本策略
+ 要运用分离变量方法，一定要将非齐次边界条件齐次化
+ **上策是，在保证方程齐次的前提下，将非齐次边界条件齐次化。**
+ 非齐次边界条件的齐次化过程中，可能伴随着方程由齐次变为非齐次的现象。
    - 这个时候就要使用上文中提到的[方程非齐次项按照本征函数展开](#atu0A)的方式解决问题。

#### 方法概述
$ \frac{\partial^2 u}{\partial x^2}-a^2 \frac{\partial^2 u}{\partial t^2} = 0 $

$ u|_{x=0} = \mu(t) \;\;\; u|_{x=l} = \nu(t) $

$ u|_{t=0} = \phi(x) \;\;\; \frac{\partial u}{\partial t}|_{t=0} = \psi(x) $

+ 总体上说，应该找一个齐次化函数$ v(x,t) $满足：

$ v|_{x=0} = \mu(t) \;\;\; v|_{x=l} = \nu(t) $

    - 齐次化的原则也是，如果非齐次项仅显含$ t $但不显含$ x $，齐次化函数也应该假定同时显含$ x,t $，否则可能导致方程非齐次化。
+ 这样，设$ u(x,t) = w(x,t) + v(x,t) $时，$ w(x,t) $满足：

$ \frac{\partial^2 w}{\partial x^2}-a^2 \frac{\partial^2 w}{\partial t^2} = -(\frac{\partial^2 v}{\partial x^2}-a^2 \frac{\partial^2 v}{\partial t^2}) $

$ w|_{x=0} = 0 \;\;\; w|_{x=l} = 0 $

$ w|_{t=0} = \phi(x)-v|_{t=0} \;\;\; \frac{\partial w}{\partial t}|_{t=0} = \psi(x)-\frac{\partial v}{\partial t}|_{t=0} $

    - 这样问题就转化为了（非齐次方程）**齐次边界条件（情景2）**的情形
    - 此时不得不应用方程非齐次项按照本征函数展开方法。

#### 保证方程齐次的前提下，将非齐次边界条件齐次化（上策）
+ 如果能在**保证方程齐次的前提下，将非齐次方程边界条件齐次化**，即：

$ \frac{\partial^2 v}{\partial x^2}-a^2 \frac{\partial^2 v}{\partial t^2} = 0 $

$ v|_{x=0} = \mu(t) \;\;\; v|_{x=l} = \nu(t) $

+ 这个时候$ w(x,t) $的性质会更好，接下来的计算量更小：

$ \frac{\partial^2 w}{\partial x^2}-a^2 \frac{\partial^2 w}{\partial t^2} = 0 $

$ w|_{x=0} = 0 \;\;\; w|_{x=l} = 0 $

$ w|_{t=0} = \phi(x)-v|_{t=0} \;\;\; \frac{\partial w}{\partial t}|_{t=0} = \psi(x)-\frac{\partial v}{\partial t}|_{t=0} $

    - 这样问题就转化为了**齐次方程齐次边界条件（情景1）**的情形。
