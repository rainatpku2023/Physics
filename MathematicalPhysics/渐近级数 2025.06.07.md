### 前置知识
#### Cauchy定理
+ 下面给出吴崇试《数学物理方法（第三版）》中对于$ \mathrm{Cauchy} $定理的论述：
    - 设有有界区域$ G $，其边界$ C $为可求长曲线，如果函数$ f(z) $在闭区域$ \overline{G} $中解析，则一定有：

$ \oint_C f(z) dz = 0 $

+ $ \mathrm{Cauchy} $定理的另外一种表述形式为**变形定理**：
    - 若函数$ f(z) $在区域$ G $内解析，$ C $为$ G $中的简单闭合曲线，如果能将曲线$ C $在$ G $内连续地变形为曲线$ C' $，则必有：

$ \oint_C f(z) dz = \oint_{C'} f(z) dz $

    - 更准确地说，$ C $与$ C' $是同伦曲线。在实际的物理问题中，基本上可以认为形变过程不要跨过奇点就可以。

#### 解析函数常点处的Taylor展开
+ 设函数$ f(z) $在以$ a $为圆心的圆$ C $内以及在$ C $上解析，则对于园内任何$ z $点，$ f(z) $可以用幂级数展开为（或者说$ f(z) $可以在$ a $点处展开为幂级数）：

$ f(z) = \sum_{n=0}^{\infin} a_n (z-n)^n $

    - 其中：

$ a_n = \frac{1}{2\pi i} \oint_C \frac{f(\zeta)}{(\zeta-a)^{n+1}}dz = \frac{f^{(n)}(a)}{n!} $

+ $ \mathrm{Taylor} $展开的性质：
    - 唯一性：注意是相同点$ a $的$ \mathrm{Taylor} $系数相同
    - 收敛范围：半径可以取到$ a $到最近奇点的距离

#### Gauss积分公式
+ $ \mathrm{Gauss} $积分公式：

$ \int_{-\infin}^{+\infin} e^{-x^2}dx = \sqrt{\pi} $

+ 可以进一步拓展为：

$ \int_{-\infin}^{+\infin} e^{-\lambda x^2}dx = \sqrt{\frac{\pi}{\lambda}} $

    - 当$ \lambda>0 $。
+ 可以进一步拓展为，高斯型积分：

$ I = \int_{-\infin}^{+\infin} e^{i\alpha t^2}dt $

### 渐近级数
#### 渐近级数的概念
+ 收敛级数
    - 幂级数$ S_N (x) = \sum_{n=1}^{N} a_n (x) $收敛于$ f(x) $，当且仅当：

$ \lim_{N\rightarrow \infin} S_N (x) = f(x) $

+ 渐近级数
    - 幂级数$ S_N (x) $称为$ f(x) $在$ x \rightarrow x_0^+ $处的渐进展开，当且仅当：

$ \lim_{x \rightarrow x_0^+}|f(x)-S_N (x)| = 0,\forall N $

    - 渐进级数并不要求$ N \rightarrow \infin $，而是让$ x\rightarrow x_0^{+(-)} $时对于任意的$ N $，$ S_N (x) $对$ f(x) $都有很好的估计。
    - 注意渐近级数是**单边逼近**。
    - $ x_0 $可以取$ +\infin $，这也是渐近级数的一个重要应用。
+ 复数级数版本的渐进展开
    - 幂级数$ S_N (z) $称为$ f(x) $在$ z \rightarrow z_0 $处的渐进展开，当且仅当：

$ \lim_{z \rightarrow z_0}|f(z)-S_N (z)| = 0,\forall N $

        * 其中$ z\rightarrow z_0 $的过程应该是**在一个楔形内进行的**。

#### 为什么引入渐近级数？
+ 微扰展开
    - 考虑量子力学中的非简谐振子，势函数为：

$ V(x) = \frac{1}{2} kx^2 +\frac{1}{4}\lambda x^4,\lambda << 1 $

    - 对于$ \lambda > 0 $情形，能量取分离值，-粒子有无限个束缚态，不存在非束缚态/散射态。
    - 对于$ \lambda < 0 $情形，粒子在其中只有有限个束缚态，有无穷个非束缚态。
    - 从只有束缚态到出现非束缚态，这是一个质的改变，不能用收敛级数去讨论。

### 渐进级数的求解方法
#### 方法总汇与思路
| | 积分变量 | 积分形式 | 渐进方式 | 主导区域 | 图像 |
| --- | --- | --- | --- | --- | --- |
| 分部积分法 | 实数$ t $ | $ I(x) = \int_{a}^{b} f(t) e^{x\phi (t)}dt $ | $ x\rightarrow \infin $ | 边界上 |  |
| 拉普拉斯型积分 | 实数$ t $ | $ I(x) = \int_{a}^{b} f(t) e^{x\phi (t)}dt $ | $ x\rightarrow \infin $ | 中间的某个位置 | |
| 傅立叶型积分 | 实数$ t $ | $ I(x) = \int_{a}^{b} f(t) e^{ix\psi (t)}dt $ | $ x\rightarrow \infin $ | 边界上 | |
| 稳定相位法 | 实数$ t $ | $ I(x) = \int_{a}^{b} f(t) e^{ix\psi (t)}dt $ | $ x\rightarrow \infin $ | 中间的某个位置 | |
| 最速降线法/鞍点近似 | 复数$ z $ | $ I(x) = \int_{a}^{b} g(z) e^{x f(z)}dz $ | $ x\rightarrow \infin $ | 不一定 | |


+ 分部积分与拉普拉斯型积分，认为$ \phi (t) $取极大值时主导，因为$ e^{x \phi (t)} $增加了指数的增长行为。
+ 傅立叶型积分与稳定相位法，认为$ \phi (t) $取极值时主导，因为在其他区域$ e^{ix \phi (t)} $高频振荡相互抵消。
+ 分部积分法与傅立叶型积分，使用分离变量法，因为主导区域在边上，分离变量法可以将主导内容放到**表面项**$ uv|_a^b $中。
+ 拉普拉斯型积分与稳定相位法，使用高斯积分法，首先可以认为整个路径的积分值与极值处附近部分路径的积分值相近，这时还可以按照极值附近$ x\phi(t) $的行为，补全整个积分，以便用$ \mathrm{Gauss} $积分公式进行计算。
+ 鞍点近似法是最普遍的方法，可以解决复平面上任何路径的渐进展开。

#### 分部积分法
+ 适用积分形式：$ I(x) = \int_{a}^{b} f(t) e^{x\phi (t)}dt $，在边界上$ \phi'(t) = 0 $，考虑$ x\rightarrow + \infin $时的渐进展开
+ 计算：

$ I(x) =  \frac{f(t)}{x \phi'(t)}e^{x\phi (t)}|_a^b -\frac{1}{x} \int_a^b \frac{d}{dt} (\frac{f(t)}{\phi' (t)})e^{x\phi(t)}dt  $

    - 考虑到第二项较小，得到渐进展开：

$ \boxed{I(x)\sim \frac{f(t)}{x \phi'(t)}e^{x\phi (t)}|_a^b,x\rightarrow + \infin} $

#### 拉普拉斯型积分
+ 适用积分形式：$ I(x) = \int_{a}^{b} f(t) e^{x\phi (t)}dt $在中间某点$ c $处满足$ \phi '(c) = 0 $且$ \phi ''(c) <0 $，考虑$ x\rightarrow +\infin $时的渐进展开：
+ 计算：
    - 首先在积分号内对于$ f(t) $和$ \phi(t) $进行$ \mathrm{Taylor} $展开：

$ I(x) = \int_{a}^{b} [f(c)+f'(c)(t-c) + \frac{1}{2!}f''(c)(t-c)^2+\cdots] e^{x[\phi(c)+0 + \frac{1}{2!}\phi''(c)(t-c)^2+\cdots]}dt $

    - 结合$ e^{x\phi(t)} $在$ t=c $处极大且具有指数行为，整理并估计得：

$ I(x) \sim \int_{a}^{b} f(c)e^{x\phi(c) + \frac{1}{2}x\phi''(c)(t-c)^2}dt 
= f(c)e^{x\phi(c)} \int_{a}^{b} e^{\frac{1}{2}x\phi''(c)(t-c)^2}dt $

    - 再次结合$ e^{x\phi(t)} $在$ t=c $处极大且具有指数行为，可以将积分区间从$ (a,b) $扩展到$ (-\infin,+\infin) $，而不会对积分值进行在量级上改变：

$ I(x) \sim f(c)e^{x\phi(c)} \int_{-\infin}^{+\infin} e^{\frac{1}{2}x\phi''(c)(t-c)^2}dt $

    - 结合$ \mathrm{Gauss} $积分公式：

$ I(x) \sim f(c)e^{x\phi(c)} \sqrt{\frac{\pi}{-\frac{1}{2}x\phi''(c)}}
 $

    - 整理即得拉普拉斯型积分的渐进展开：

$ \boxed{I(x) \sim f(c)e^{x\phi(c)} \sqrt{\frac{2\pi}{-x\phi''(c)}}} $

#### 傅立叶型积分
+ 适用积分形式：$ I(x) = \int_{a}^{b} f(t) e^{ix\psi (t)}dt $，在边界上$ \psi'(t) = 0 $，考虑$ x\rightarrow + \infin $时的渐进展开
+ 计算：

$ I(x) =  \frac{f(t)}{xi \psi'(t)}e^{ix\psi (t)}|_a^b -\frac{1}{ix} \int_a^b \frac{d}{dt} (\frac{f(t)}{\psi' (t)})e^{ix\psi(t)}dt  $

    - 考虑到第二项较小，得到渐进展开：

$ \boxed{I(x)\sim \frac{f(t)}{ix \psi'(t)}e^{ix\psi (t)}|_a^b,x\rightarrow + \infin} $

#### 稳定相位法
+ 适用积分形式：$ I(x) = \int_{a}^{b} f(t) e^{ix\psi (t)}dt $在中间某点$ c $处满足$ \psi'(c) = 0 $，考虑$ x\rightarrow +\infin $时的渐进展开：
+ 计算：
    - 首先在积分号内对于$ f(t) $和$ \psi(t) $进行$ \mathrm{Taylor} $展开：

$ I(x) = \int_{a}^{b} [f(c)+f'(c)(t-c) + \frac{1}{2!}f''(c)(t-c)^2+\cdots] e^{ix[\psi(c)+0 + \frac{1}{2!}\psi''(c)(t-c)^2+\cdots]}dt $

    - 结合$ e^{ix\psi(t)} $在$ t=c $处稳定且具有振荡行为，整理并估计得：

$ I(x) \sim \int_{a}^{b} f(c)e^{ix\psi(c) + \frac{1}{2}ix\psi''(c)(t-c)^2}dt 
= f(c)e^{ix\psi(c)} \int_{a}^{b} e^{\frac{1}{2}ix\psi''(c)(t-c)^2}dt $

    - 结合$ \mathrm{Gauss} $型积分的结论，得到渐进展开：

$ \boxed{I(x) \sim f(c)e^{ix\psi(c)} e^{±i\frac{\pi}{4}}\sqrt{\frac{2\pi}{±x\psi''(c)}}} $

#### 最速下降法（鞍点近似）
##### 鞍点近似引入
+ 对于**解析函数**来说，在任意开区域内都无法取到模长的极大值。
+ 导函数为$ 0 $的点，实部和虚部都应该调和函数。
+ -> 平稳点都是**鞍点**。

##### 鞍点近似的具体推导和计算
+ 适用积分形式：$ I(x) = \int_{a}^{b} g(z) e^{x f(z)}dz $在复平面上某点$ z_0 $处满足$ f'(z_0) = 0 $，考虑$ x\rightarrow +\infin $时的渐进展开：
+ 计算：分别对$ g(z) $和$ f(z) $在$ z=z_0 $附近进行$ \mathrm{Taylor} $展开，得到：

$ I(x) = \int_{a}^{b} [g(z_0)+g'(z_0)(z-z_0)+\frac{1}{2}g''(z)(z-z_0)^2+\cdots] e^{x [f(z_0)+0+\frac{1}{2}f''(z)(z-z_0)^2+\cdots]}dz $

    - 渐进展开估计为：

$ I(x) \sim \int_{a}^{b} g(z_0) e^{x f(z_0)} e^{\frac{1}{2}xf''(z)(z-z_0)^2}dz
=  g(z_0) e^{x f(z_0)}\int_{a}^{b} e^{\frac{1}{2}xf''(z)(z-z_0)^2}dz $

    - 进行参数化：

$ f''(z_0) = \rho e^{i\theta} $

$ (z-z_0) = s e^{i\phi} $

    - 在这种参数化方案下，渐进展开可以写为：

$ I(x) \sim  g(z_0) e^{x f(z_0)}\int_{a}^{b} e^{\frac{1}{2}x(\rho e^{i\theta})(s e^{i\phi})^2}dz =g(z_0) e^{x f(z_0)}\int_{a}^{b} e^{\frac{1}{2}x\rho s^2  e^{i(\theta+2\phi)}}dz
 $

    - 注意到$ e^{i(\theta+2\phi)} = \cos (\theta+2\phi) + i \sin(\theta+2\phi) $，可知当$ \theta +2\phi = ±\pi $时有：

$ e^{i(\theta+2\phi)} = \cos (\theta+2\phi) + i \sin(\theta+2\phi) = -1 +0i = -1 $

    - 此时积分变为：

$ I(x) \sim  g(z_0) e^{x f(z_0)}\int_{a'}^{b'} e^{-\frac{1}{2}x\rho s^2  }ds

 $

        * 其中$ s $平面上的$ a' $与$ z $平面上的$ a $对应，$ s $平面上的$ b' $与$ z $平面上的$ b $对应。
    - 考虑到只有$ z=z_0 $处主导积分，所以可以扩展为整体：

$ I(x) \sim  g(z_0) e^{x f(z_0)}\int_{-\infin}^{+\infin} e^{-\frac{1}{2}x\rho s^2  }ds $

    - 结合$ \mathrm{Gauss} $型积分的结论，得到渐进展开：

$ \boxed{I(x) \sim  g(z_0) e^{x f(z_0)} \sqrt{\frac{2\pi}{x\rho}}} $

##### 鞍点近似与Cauchy定理的结合
+ 鞍点近似的本质是将复变函数形象地看做复平面上的地形，积分就是在复平面上运动并进行积分。
+ 在估计渐进展开时，只有积分值较大的区域主导，而穿过这种区域的方式是跨越鞍部
    - 而能较为准确计算的跨越鞍部的方式仅有穿过鞍点。
