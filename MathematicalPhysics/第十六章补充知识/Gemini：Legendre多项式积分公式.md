好的，勒让德多项式有几个非常重要的积分公式，它们在数学物理、工程和数值分析等领域有广泛应用。其中最核心的是**正交性关系**。下面我将介绍几个关键的积分公式并解释它们是如何得到的。

### 1. 勒让德多项式的正交性 (Orthogonality)
这是勒让德多项式最基本也是最重要的积分公式。

#### 公式
当 $ m $ 和 $ n $ 为非负整数时：

$ \int_{-1}^{1} P_m(x) P_n(x) dx = \frac{2}{2n+1} \delta_{mn} $

这里的 $ \delta_{mn} $ 是**克罗内克符号 (Kronecker Delta)**，定义为：

$ \delta_{mn} = \begin{cases} 1 & \text{如果 } m = n \\ 0 & \text{如果 } m \neq n \end{cases} $

这个公式表明：

+ 当 $ m \neq n $ 时，两个不同阶的勒让德多项式在区间 $ [-1, 1] $ 上是**正交**的，它们的乘积积分为 0。
+ 当 $ m = n $ 时，积分给出了多项式自身的“模方”或“范数平方”。

#### 如何得到 (推导过程)
这个性质通常是通过**罗德里格斯公式 (Rodrigues' formula)** 和**分部积分法**来证明的。

**罗德里格斯公式** 给出勒让德多项式的定义：

$ P_n(x) = \frac{1}{2^n n!} \frac{d^n}{dx^n} (x^2-1)^n $

令 $ u(x) = (x^2-1)^n $。在 $ x = \pm 1 $ 处，$ u(x) $ 及其直到 $ (n-1) $ 阶的导数都为零。

**情况一：**$ m \neq n $

假设 $ m < n $。我们需要证明 $ \int_{-1}^{1} P_m(x) P_n(x) dx = 0 $。

$ \int_{-1}^{1} P_m(x) P_n(x) dx = \frac{1}{2^n n!} \int_{-1}^{1} P_m(x) \left( \frac{d^n}{dx^n} (x^2-1)^n \right) dx $

我们对上式进行 $ n $ 次分部积分。注意到 $ P_m(x) $ 是一个 $ m $ 次多项式。

+ 进行第一次分部积分：

$ \int u \, dv = uv - \int v \, du $

令 $ u = P_m(x) $，$ dv = \frac{d^n}{dx^n} (x^2-1)^n dx $。

那么 $ du = P_m'(x) dx $，$ v = \frac{d^{n-1}}{dx^{n-1}} (x^2-1)^n $。

由于 $ (x^2-1)^n $ 在 $ x=\pm 1 $ 处的 $ (n-1) $ 阶导数均为零，边界项 $ uv|_{-1}^{1} $ 为 0。

积分变为：

$ \frac{1}{2^n n!} \left( -\int_{-1}^{1} P_m'(x) \left( \frac{d^{n-1}}{dx^{n-1}} (x^2-1)^n \right) dx \right) $

+ 重复这个过程 $ n $ 次。每次分部积分，边界项都为零。积分内部 $ P_m(x) $ 的导数阶数不断增加，而 $ (x^2-1)^n $ 的导数阶数不断降低。
+ 进行 $ n $ 次分部积分后，得到：

$ \int_{-1}^{1} P_m(x) P_n(x) dx = \frac{(-1)^n}{2^n n!} \int_{-1}^{1} \left( \frac{d^n}{dx^n} P_m(x) \right) (x^2-1)^n dx $



+ 因为 $ P_m(x) $ 是一个 $ m $ 次多项式，而我们假设了 $ m < n $，所以它的 $ n $ 阶导数必然为零：

$ \frac{d^n}{dx^n} P_m(x) = 0 \quad (\text{因为 } m < n) $

+ 因此，整个积分为零。

$ \int_{-1}^{1} P_m(x) P_n(x) dx = 0 \quad \text{for } m \neq n $

**情况二：**$ m = n $

当 $ m=n $ 时，进行 $ n $ 次分部积分后，$ \frac{d^n}{dx^n} P_n(x) $ 不再是零。$ P_n(x) $ 的最高次项是 $ \frac{(2n)!}{2^n (n!)^2} x^n $。对其求 $ n $ 阶导数，得到一个常数：

$ \frac{d^n}{dx^n} P_n(x) = \frac{(2n)!}{2^n (n!)^2} n! = \frac{(2n)!}{2^n n!} $

代入分部积分后的式子：

$ \int_{-1}^{1} [P_n(x)]^2 dx = \frac{(-1)^n}{2^n n!} \int_{-1}^{1} \frac{(2n)!}{2^n n!} (x^2-1)^n dx = \frac{(2n)!}{4^n (n!)^2} \int_{-1}^{1} (1-x^2)^n dx $

这个积分 $ \int_{-1}^{1} (1-x^2)^n dx $ 是一个经典的Beta函数积分。通过换元 $ x = \sin\theta $ 或者直接查表，可以得到：

$ \int_{-1}^{1} (1-x^2)^n dx = \frac{2^{2n+1} (n!)^2}{(2n+1)!} $

将此结果代回：

$ \int_{-1}^{1} [P_n(x)]^2 dx = \frac{(2n)!}{4^n (n!)^2} \cdot \frac{2^{2n+1} (n!)^2}{(2n+1)!} = \frac{(2n)!}{2^{2n}} \cdot \frac{2^{2n+1}}{(2n+1)!} = \frac{2}{2n+1} $

这样就完成了整个正交性关系的证明。

### 2. $ x^n $ 与 $ P_n(x) $ 的积分
这个公式在你上一个问题的解答中起到了关键作用。

#### 公式
$ \int_{-1}^{1} x^n P_n(x) dx = \frac{2^{n+1} (n!)^2}{(2n+1)!} $

#### 如何得到 (推导过程)
这个公式的推导利用了前面提到的正交性。

1. 任何一个次数小于 $ n $ 的多项式 $ Q_{n-1}(x) $ 都可以表示为勒让德多项式的线性组合：

```plain
$$Q_{n-1}(x) = c_0 P_0(x) + c_1 P_1(x) + \dots + c_{n-1} P_{n-1}(x)$$
根据正交性，$\int_{-1}^{1} Q_{n-1}(x) P_n(x) dx = 0$。
```

2. 考虑多项式 $ x^n $。我们可以把它写成 $ P_n(x) $ 和一个次数小于 $ n $ 的多项式 $ Q_{n-1}(x) $ 的组合。  
$ P_n(x) $ 的最高次项系数是 $ A_n = \frac{(2n)!}{2^n(n!)^2} $。  
所以我们可以写：

$ x^n = \frac{1}{A_n} P_n(x) + Q_{n-1}(x) $

其中 $ Q_{n-1}(x) $ 是一个次数小于 $ n $ 的多项式，用来消去 $ P_n(x) $ 中除最高次项外的其他项。

3. 现在计算积分：

$ \int_{-1}^{1} x^n P_n(x) dx = \int_{-1}^{1} \left( \frac{1}{A_n} P_n(x) + Q_{n-1}(x) \right) P_n(x) dx $

$ = \frac{1}{A_n} \int_{-1}^{1} [P_n(x)]^2 dx + \int_{-1}^{1} Q_{n-1}(x) P_n(x) dx $

根据正交性，第二项积分为 0。

4. 我们已经知道 $ \int_{-1}^{1} [P_n(x)]^2 dx = \frac{2}{2n+1} $，代入上式：

$ \int_{-1}^{1} x^n P_n(x) dx = \frac{1}{A_n} \left( \frac{2}{2n+1} \right) $

将 $ A_n = \frac{(2n)!}{2^n(n!)^2} $ 代入：

$ \int_{-1}^{1} x^n P_n(x) dx = \frac{2^n(n!)^2}{(2n)!} \cdot \frac{2}{2n+1} = \frac{2^{n+1}(n!)^2}{(2n+1)!} $

这就得到了该公式。

### 总结
勒让德多项式的积分公式是其理论的核心部分，特别是**正交性关系**，它不仅定义了这些多项式作为一个函数空间中的正交基，也是解决许多相关问题的基础。其他积分公式，如 $ \int x^n P_n(x) dx $ 的公式，通常都是基于正交性推导出来的。
