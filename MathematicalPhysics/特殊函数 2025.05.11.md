### Gamma函数
#### 定义
+ 第二类$ Euler $积分：

$ \boxed{\Gamma(z)=\int_0^\infin t^{z-1}e^{-t}dt} $

    - 积分内部自变量是$ t $，上面指数是$ z-1 $
+ 解析延拓到右半平面
+ 根据性质$ \Gamma(z+1) = z\cdot \Gamma(z) $延拓到左半平面

#### 性质
+ 性质1：初值

$ \boxed{\Gamma(1)=1} $

+ 性质2：递推公式

$ \boxed{\Gamma(z+1) = z\cdot \Gamma(z)} $

    - 推论：零点处留数，$ \mathrm{Res}[\Gamma,0] = 1 $，可以通过$ \Gamma(z) = \frac{\Gamma(z+1)}{z} $得出
    - 推论：负整数点处留数，$ \mathrm{Res}[\Gamma,-n] = \frac{(-1)^n}{n!} $，$ \Gamma(z) = \frac{\Gamma(z+n)}{z(z+1)\cdots(z+m-1)} $代入公式。
    - 推论：$ \Gamma(n+1) = n! $
+ 性质3：余元公式

$ \boxed{\Gamma(z)\Gamma(1-z) = \frac{\pi}{\sin \pi z}} $

    - 推论：$ \Gamma(\frac{1}{2}) = \sqrt{\pi} $，带入即得
+ 性质4：倍乘公式

$ \boxed{\Gamma(2z) = 2^{2z-1} \frac{\Gamma(z)\Gamma(z+\frac{1}{2})}{\Gamma(\frac{1}{2})}} $

+ 性质5：Stirling公式

#### Stirling公式：Gamma函数的渐近展开
+ 考虑函数$ \Gamma(x+1) $:

$ \Gamma(x+1)=\int_0^\infin t^{x}e^{-t}dt $

+ 把$ t^x $强行放到$ e $指数上面，并在$ t=x $附近进行泰勒展开，$ x $看做自变量：

$ =\int_0^\infin \exp(-t+x\ln t)dt 
= \int_0^\infin \exp[-x+x\ln x -\frac{(t-x)^2}{2x}+-\cdots]dt $

+ 注意到指数前两项都不含有$ t $，可以拿出积分号：

$ = e^{-x} x^x \int_0^\infin \exp[ -\frac{(t-x)^2}{2x}+-\cdots]dt $

+ 忽略高阶项：

$ ≈ e^{-x} x^x \int_0^\infin \exp[ -\frac{(t-x)^2}{2x}]dt
≈ e^{-x} x^x \int_{-\infin}^{+\infin} \exp[ -\frac{u^2}{2x}]du
= e^{-x} x^x \sqrt{2 \pi x} $

+ 猜测渐进展开式为：

$ \boxed{\Gamma(x+1)= x! = e^{-x} x^x \sqrt{2 \pi x} (1+\frac{A}{x}+\frac{B}{x^2}+\frac{C}{x^3}+\cdots)} $

+ 代入关系$ \Gamma(x+1) = x\Gamma(x) $....

$ A = \frac{1}{12},B = \frac{1}{288} $

+ 对估计取对数：

$ \ln \Gamma(x+1)= \ln x!= \ln (e^{-x} x^x \sqrt{2 \pi x}) =  -x + x\ln x +\frac{1}{2} \ln x + \frac{1}{2} \ln(2\pi) $

#### Gamma函数的Weierstrass无穷乘积表示
+ 表示方法2：更好地体现零点位置

$ \boxed{\frac{1}{\Gamma(z)} = \lim_{n \rightarrow \infin}\frac{n^z}{z(1+z)(1+\frac{z}{2})\cdots (1+\frac{z}{n})}} $

+ 表示方法3（Weistrass无穷乘积）：为了证明余元公式

$ \Gamma(z) = \lim_{N\rightarrow \infin}z \exp(-z\ln N+ \sum_{n=1}^{N} \frac{z}{n}) \prod_{n=1}^{N}[(1+\frac{z}{n})e^{-\frac{z}{n}}] =z e^\gamma \prod_{n=1}^{\infin}[(1+\frac{z}{n})e^{-\frac{z}{n}}] $

    - 发现：

$ \lim_{N\rightarrow \infin}(-\ln N+ \sum_{n=1}^{N} \frac{1}{n})= \gamma = 0.5772... $

    - 因此有：

$ \boxed{\Gamma(z)=z e^{\gamma z} \prod_{n=1}^{\infin}[(1+\frac{z}{n})e^{-\frac{z}{n}}]} $

#### 余元公式的证明
+ $ \sin \pi z $的Weierstrass无穷乘积形式为：

$ \sin (\pi z) = \pi z\prod_{n=1}^{\infin} (1-\frac{z^2}{n^2}) $ 

+ 所以：

$ \frac{\pi}{\sin (\pi z)}  = \frac{1}{z\prod_{n=1}^{\infin} (1-\frac{z^2}{n^2})} $ 

+ $ \Gamma(1-z) $的无穷乘积形式为：

$ \Gamma(1-z) = (-z)\Gamma(-z)

### Beta函数
#### 定义
+ 第一类$ Euler $积分：

$ \boxed{\Beta(p,q)=\int_0^1 t^{p-1}(1-t)^{q-1}dt} $

    - 定义在右半平面上。
+ 根据性质$ \Beta(p,q)= \frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)} $可以解析延拓到全平面。

#### 性质
+ 性质1：Beta函数的另外一种表示方法，进行变量替换$ t = \sin ^2 \theta $，得：

$ \Beta(p,q)= 2 \int_0^{\frac{\pi}{2}} \sin ^{2p-1}\theta \cos ^{2q-1}\theta d\theta $

+ 性质2：

$ \boxed{\Beta(p,q)= \frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)}} $

    - 先考虑$ \Gamma(p)\Gamma(q) $的积分，在积分号内分别作替换$ t \rightarrow x^2 $和$ t \rightarrow y^2 $，在自然对数的指数上进行平方和后进行三角换元$ (x,y) \rightarrow (r,\theta) $。



#### 应用：余元公式的另外一种证明
+ 根据$ \Beta(p,q)= \frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)} $可知：

$ \Gamma(z)\Gamma(1-z) = \Beta(z,1-z)\Gamma(1) = \int_0^1 t^{z-1}(1-t)^{-z}dt $

+ 进行变量替换$ x \rightarrow \frac{t}{1-t} $，即$ t \rightarrow \frac{x}{x+1} $得到：

$ = \int_0^{\infin} \frac{x^{z-1}}{1+x}dx $

+ 采用玦形围道，支点在$ x=0,\infin $处，奇点在$ x=-1 $处，割线选择$ x $轴正半轴，计分得：

$ = \frac{\pi}{\sin \pi z} $

+ 综上，得到余元公式：

$ \Gamma(z)\Gamma(1-z)= \frac{\pi}{\sin \pi z} $

### Digamma函数
#### 定义
+ Digamma函数用$ \psi $表示，它是Gamma函数的对数微商：

$ \boxed{\psi(z) = \frac{d\ln\Gamma(z)}{dz} = \frac{\Gamma'(z)}{\Gamma(z)}} $

    - 注意其中“对数微商”指的是对宗量$ z $的对数微商，所以一般来说有：

$ \psi(A(z)) = \frac{d\ln\Gamma(A)}{dA} \neq \frac{d\ln\Gamma(A)}{dz} $

#### 性质
+ 性质1：$ 0 $和负整数处留数都为$ -1 $，其他点都解析。

$ \mathrm{Res}[\psi,n] = -1,n=0,-1,-2,-3 $

+ 性质2：递推公式

$ \boxed{\psi(z+1) = \psi(z) + \frac{1}{z}} $

    - 证明：$ \psi(z+1) = \frac{d\ln \Gamma(z+1)}{z} = \frac{d\ln (z\Gamma(z))}{z}= \frac{d\ln (\Gamma(z))+d\ln z}{z} = \psi(z) + \frac{1}{z} $
    - 推论：

$ \boxed{\psi(z+n) = \psi(z) + \frac{1}{z}+ \frac{1}{z+1}+\cdots + \frac{1}{z+n-1}}  $

+ 性质3：余元公式

$ \boxed{\psi(1-z) = \psi(z) + \pi \cot \pi z} $

    - 证明：

$ \psi(1-z) = \frac{d\ln \Gamma(1-z)}{d(1-z)} = \frac{d\ln [\pi /(\sin\pi z\Gamma(z))]}{d(1-z)} $

$ = \frac{d[\ln\pi-\ln \Gamma(z)-\ln\sin\pi z]}{-dz}= \psi(z) + \pi \cot (\pi z ) $

+ 性质4：反射公式

$ \boxed{\psi(z)-\psi(-z) =  -\frac{1}{z}- \pi \cot \pi z} $

+ 性质5：倍乘公式

$ \boxed{\psi(2z) =  \frac{1}{2}\psi(z) + \frac{1}{2}\psi(z+\frac{1}{2}) +\ln 2} $



+ 性质6：特殊值

$ \boxed{\psi(1) = -\gamma} $

$ \boxed{\psi'(1) = \frac{\pi^2}{6}} $

+ 性质7：

$ \boxed{\lim_{n \rightarrow \infin} [\psi(z+n)-\ln n]=0 } $

    - 结合下列三个式子即证

$ \psi(1) = -\gamma $

$ \gamma =\lim_{N\rightarrow \infin}(-\ln N+ \sum_{n=1}^{N} \frac{1}{n})=  0.5772... $

$ \psi(z+n) = \psi(z) + \frac{1}{z}+ \frac{1}{z+1}+\cdots + \frac{1}{z+n-1}  $

#### 应用：计算无穷级数
+ 性质2的推论可以用于计算无穷级数：

$ \psi(z+n) - \psi(z) = \frac{1}{z}+ \frac{1}{z+1}+\cdots + \frac{1}{z+n-1}  $

+ 两边求导数

$ \psi'(z+n) - \psi'(z) = -\frac{1}{z^2}- \frac{1}{(z+1)^2}-\cdots - \frac{1}{(z+n-1)^2}  $

### Riemann Zeta函数
#### 定义
+ 在$ x>1 $实轴的定义为：

$ \boxed{\zeta(x) = \sum_{n=1}^{\infin}\frac{1}{n^x}} $

+ 通过下个式子可以解析延拓到$ \mathrm{Re} z>1 $区域：

$ \boxed{\zeta(z) = \frac{1}{\Gamma(z)}\int_0^{+\infin}\frac{t^{z-1}}{e^t-1}dt} $

    - 证明：

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1747021055536-aefce9dd-ac1b-4fe4-9301-061e10ad4741.png" width="662" title="" crop="0,0,1,1" id="u0a949ff7" class="ne-image">

+ 根据性质$ \zeta (z) = 2^{1-z} \pi^{-z} \Gamma(1-z)\cos \frac{\pi (1-z)}{2} \zeta (1-z) $，可以将$ \zeta (z) $函数解析延拓到$ \mathrm{Re} z<0 $区域，同时得到平凡零点：

$ z_0 = -2,-4,-6...-2n... $



#### 性质
+ 性质1：对映

$ \boxed {\zeta (1-z) = 2^{1-z} \pi^{-z} \Gamma(z)\cos \frac{\pi z}{2} \zeta (z)} $

$ \boxed{\zeta (z) = 2^{1-z} \pi^{-z} \Gamma(1-z)\cos \frac{\pi (1-z)}{2} \zeta (1-z)} $

+ 性质2：平凡零点位于

$ z_0 = -2,-4,-6...-2n... $

+ 性质3：非平凡零点位于临界带中，临界带为：$ 0<\mathrm{Re}(z)<1 $
