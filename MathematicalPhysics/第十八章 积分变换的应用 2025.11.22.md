2. <font style="color:#DF2A3F;">做题技巧：</font>
    1. <font style="color:#DF2A3F;">设线性无关解时，如果两边的都有边界条件，不妨设为</font>$ \cosh(\sqrt{\frac{s}{\kappa}}x) $<font style="color:#DF2A3F;">和</font>$ \cosh(\sqrt{\frac{s}{\kappa}}(l-x)) $
    2. $ f(s) $<font style="color:#DF2A3F;">一阶奇点</font>$ s_k $<font style="color:#DF2A3F;">的留数，如果分母上有三角函数，可以这么做：</font>
        1. <font style="color:#DF2A3F;">首先拆成分子分母部分：</font>

$ f(s) = \frac{N(s)}{D(s)} $

        1. <font style="color:#DF2A3F;">第一个等号是代入留数定理的定义式，第二个等号是运用洛必达法则：</font>

$ \mathrm{Res}[f(s),s] = \lim_{s\rightarrow s_k} \frac{N(s)}{D(s)}(s-s_k) = \frac{N(s_k)}{D'(s_k)}  $

#### Fourier变换的两种形式
+ 朱华星老师讲授《数学物理方法（上）》时，$ \mathrm{Fourier} $变换的正变换为：

$ \hat{f}(\omega) = \int_{-\infin}^{+\infin} f(t)e^{-i\omega t}dt $

    - $ \mathrm{Fourier} $逆变换定义为：

$ f(t) = \frac{1}{2\pi}\int_{-\infin}^{+\infin} \hat{f}(\omega)e^{i\omega t}d\omega $

    - 这种形式形式较为简洁。
+ 吴崇试老师的《数学物理方法》一书中使用的$ \mathrm{Fourier} $正变换约定为：

$ F(k) = \frac{1}{\sqrt{2\pi}}\int_{-\infin}^{+\infin} f(x)e^{-ikx}dx $

    - $ \mathrm{Fourier} $逆变换约定为：

$ f(x) = \frac{1}{\sqrt{2\pi}}\int_{-\infin}^{+\infin} F(k)e^{ikx}dx $

    - 这种形式对称性更好。

#### 一个关于Delta函数的有意思的推导
+ 考虑$ \mathrm{Fourier} $逆变换：

$ f(x) = \frac{1}{\sqrt{2\pi}}\int_{-\infin}^{+\infin} F(k)e^{ikx}dx $

    - 其中$ F(k) $可以写成$ f(x) $的$ \mathrm{Fourier} $正变换，但是也可以引入哑变量$ x' $写为：

$ F(k) = \frac{1}{\sqrt{2\pi}}\int_{-\infin}^{+\infin} f(x')e^{-ikx'}dx' $

+ 将这个式子代入上一个式子，得到：

$ f(x) = \frac{1}{\sqrt{2\pi}}\int_{-\infin}^{+\infin} [\frac{1}{\sqrt{2\pi}}\int_{-\infin}^{+\infin} f(x')e^{-ikx'}dx']e^{ikx}dx $

    - 进行简单的整理：

$ RHS = \frac{1}{2\pi} \int_{-\infin}^{+\infin}[\int_{-\infin}^{+\infin}e^{ik(x-x')}dk]f(x')dx' $

    - 又可以把左侧$ f(x) $改写为：

$ LHS = \int_{-\infin}^{+\infin} f(x')\delta (x'-x)dx' = \int_{-\infin}^{+\infin} f(x')\delta (x-x')dx'  $

+ 比较可知：

$ \boxed{\delta (x-x') = \frac{1}{2\pi} \int_{-\infin}^{+\infin}e^{ik(x-x')}dk} $

### Hankel变换
#### 从二维Fourier变换到Hankel变换
+ 使用对称形式的$ \mathrm{Fourier} $变换约定，正变换和逆变换分别为：

$ F(\vec{p}) = \frac{1}{2\pi} \int f(\vec{r})e^{-i\vec{p}\cdot \vec{r}}d^2 \vec{r} $

$ f(\vec{r}) = \frac{1}{2\pi} \int F(\vec{p})e^{i\vec{p}\cdot \vec{r}}d^2 \vec{p} $

    - 积分范围分别是全位形空间、全动量空间。
+ 动量空间转换到极坐标系$ (p,\phi) $，位形空间转换到极坐标系$ (r,\theta) $，上面两个式子可以改写为：

$ F(p,\phi) = \frac{1}{2\pi} \int_{r = 0}^{+\infin} \int_{\theta = 0}^{2\pi} f(r,\theta)e^{-ipr\cos(\theta -\phi)}rdrd\theta $

$ f(r,\theta) = \frac{1}{2\pi} \int_{p = 0}^{+\infin} \int_{\phi=0}^{2\pi} F(p,\phi)e^{ipr\cos(\theta -\phi)}pdpd\phi $

    - 平面波按照柱面波展开：

$ e^{ipr\cos(\theta -\phi)} = \sum_{n=-\infin}^{+\infin} J_n(pr)i^n e^{in(\theta - \phi)} $

$ e^{-ipr\cos(\theta -\phi)} =(e^{ipr\cos(\theta -\phi)})^*= ( \sum_{n=-\infin}^{+\infin} J_n(pr)i^n e^{in(\theta - \phi)})^*=\sum_{n=-\infin}^{+\infin} J_n(pr)(-i)^n e^{in(\phi-\theta)} $

        * $ J_n $是$ n $阶$ \mathrm{Bessel} $函数。
    - 代入得：

$ F(p,\phi) = \frac{1}{2\pi} \int_{r = 0}^{+\infin} \int_{\theta = 0}^{2\pi} f(r,\theta)\sum_{n=-\infin}^{+\infin} [J_n(pr)(-i)^n e^{in( \phi-\theta )}]rdrd\theta $

$  = \sum_{n=-\infin}^{+\infin}e^{in \phi} (-i)^n\frac{1}{2\pi} \int_{r = 0}^{+\infin}  f(r,\theta) J_n(pr) rdr $

    - 同理：

$ f(r,\theta) = \frac{1}{2\pi} \int_{p = 0}^{+\infin} \int_{\phi=0}^{2\pi} F(p,\phi)\sum_{n=-\infin}^{+\infin} [J_n(pr)i^n e^{in(\theta - \phi)}]pdpd\phi $

$ = \sum_{n=-\infin}^{+\infin} e^{in\theta}i^n\frac{1}{2\pi} \int_{p = 0}^{+\infin} F(p,\phi) J_n(pr) pdp $

+ 将$ F(p,\phi) $和$ f(r,\theta) $分别展开为$ \mathrm{Fourier} $级数：

$ F(p,\phi) = \sum_{n=-\infin}^{+\infin} F_n(p)e^{in\phi} $$ 系数：F_n(p) = \frac{1}{2\pi} \int_0^{2\pi} F(p,\phi)e^{-in\phi}d\phi $

$ f(r,\theta) = \sum_{n=-\infin}^{+\infin} f_n(r)e^{in\theta} $$ 系数：f_n(r) = \frac{1}{2\pi} \int_0^{2\pi} f(r,\theta)e^{-in\theta}d\theta $

    - 比较系数：

$ F_n(p) = (-i)^n\int_{r = 0}^{+\infin} J_n(pr)f_n(r)rdr $

$ f_n(r) = i^n \int_{p = 0}^{+\infin} J_n(pr)F_n(p)pdp $

#### Hankel变换
+ 令$ u(r) = (-i)^n f_n(r),U(p) = (-i)^n F_n(p) $：
+ 得到$ \mathrm{Hankel} $变换：

$ U(p) = \int_{r = 0}^{+\infin} J_n(pr)u(r)rdr（正变换） $

$ u(r) = \int_{r = 0}^{+\infin} J_n(pr)U(p)pdp（逆变换） $

### 关于积分变换的一般讨论
#### 变换核
+ 积分变换可以概括性地写为：

$ F(k) = \int_a^b K(k,x)f(x)dx $

+ 其中$ K(k,x) $称为积分变换的核：
    - $ \mathrm{Laplace} $变换：$ K(k,x) = e^{-kx}, 0\leq x < \infin $
    - $ \mathrm{Fourier} $变换：$ K(k,x) = e^{-ikx}, -\infin< x < \infin $
    - 正弦变换：$ K(k,x) = \sin kx, 0\leq x < \infin $
    - 余弦变换：$ K(k,x) = \cos kx, 0\leq x < \infin $
    - $ \mathrm{Hankel} $变换：$ K(k,x)=xJ_n(kx),0\leq x < \infin $
    - $ \mathrm{Mellin} $变换：$ K(k,x)=x^{k-1},0\leq x < \infin $

#### 选择变换核的原则
1. 所涉及的自变量的**变化区间**与该变换的要求一致。
    1. 例如Laplace变换就适合半无穷大空间。因此常常用来对时间进行变换。
2. 未知函数的该种积分变换存在。
3. 对于常系数方程，要求函数$ f(x) $及其导数$ f'(x),f''(x) $在该变换下有简单的代数关系。
4. 涉及的未知函数及其导数$ f'(x) $的特殊值正好由定解问题中的定解条件给出。
