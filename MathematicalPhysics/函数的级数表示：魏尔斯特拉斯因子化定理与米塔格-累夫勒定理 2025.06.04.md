### Weierstrass 因子化定理
#### 问题引入：复系数多项式的因子化
+ 对于一个$ n $次复系数多项式$ P_n(z) $如果有$ n $个零点$ z_1,z_2,...,z_n $（计入重数）那么它可以唯一地分解为线性因子的乘积，再乘上一个非零常数因子$ C $：

$ P_n(z) = C (z-z_1)(z-z_2)\cdots (z-z_n) = C \prod_{k=1}^{n} (z-z_k) $

+ 问：对于定义在复平面$ \mathbb{C} $上由**无穷多个**零点的函数$ f(z) $，是否能通过排上述方式给出因子化的表达式？

#### 问题引入：正弦函数的Euler无穷乘积展开
+ 无穷乘积的$ \mathrm{Euler} $无穷乘积展开为：

$ \frac{\sin \pi z}{\pi z} = \prod_{n=1}^{\infin}(1-\frac{z^2}{n^2}) $

#### Weierstrass 因子化定理内容
+ 对于定义在$ \mathbb{C} $上的全纯函数$ f(z) $
    - $ f(z) $在$ z=0 $处有一个$ m $阶零点，人为规定$ f(0) \neq 0 $时$ m=0 $
    - $ f(z) $的其他非零零点为$ z_1,z_2,z_3,... $（计入重数），总共有无穷个
        * 按照模长排序有：$ 0<|z_1|≤|z_2|≤|z_3|≤\cdots $
        * 而且有：$ \lim_{n\rightarrow \infin} |z_n| = \infin $
+ 则存在一个全纯函数$ g(z) $和一个非负整数序列$ \{p_n\}_{n=1}^{\infin} $使得：

$ \boxed{f(z) = z^m e^{g(z)} \prod_{n=1}^{\infin} E_{p_n} (\frac{z}{z_n})} $

    - 其中$ E_p(w) $称为$ \mathrm{Weierstrass} $基本因子，定义为：

$ E_p(w) = (1-w) \exp{(w+\frac{w^2}{2}+ \cdots +\frac{w^p}{p})},p≥1 $

    - 特殊地，有：

$ E_0(w) = 1-w $

+ $ \mathrm{Weierstrass} $基本因子的意义：
    - $ 1-w = 1-\frac{z}{z_n} = -\frac{z-z_n}{z_n} $用于描述零点。
    - $ \exp{(w+\frac{w^2}{2}+ \cdots +\frac{w^p}{p})} $用于约束防止级数发散。

### Mittag-Leffler 定理
#### M-L 定理内容
+ 设半纯函数$ f(z) $只有单极点（一阶极点）位于$ a_k $，其留数为$ b_k $，且绝对值满足：$ 0<|a_1|<|a_2|<\cdots<|a_n| <\cdots $
+ 且$ f(z) $的模长存在上界，即$ |f(z)|<M $
+ 则$ f(z) $可以写成如下的无穷求和形式：

$ \boxed{f(z) = f(0)+\sum_{k=1}^{\infin}(\frac{b_k}{z-a_k}+\frac{b_k}{a_k})} $

#### M-L 定理证明
+ 构建函数$ g(w) $：

$ g(w) = \frac{f(w)}{w(w-z)} $

+ 在$ w $平面上构建以原点为圆心的围道序列$ \{C_n\} $使得$ C_k $仅仅包含$ a_1 ... a_k $前$ k $个奇点。
+ 根据根据定义计算$ g(w) $在$ w $平面上各个奇点的留数：
    - 结合$ 0 $是$ f $的常点可知，$ w=0 $处留数为：

$ \mathrm{Res}(g,w=0) = \lim_{w \rightarrow 0} w \cdot \frac{f(w)}{w(w-z)} = \frac{f(0)}{-z} $

    - 结合$ z \neq a_k $是$ f $的常点可知，$ w=z $处留数为：

$ \mathrm{Res}(g,w=z) = \lim_{w \rightarrow z} (w-z) \cdot \frac{f(w)}{w(w-z)} = \frac{f(z)}{z} $

    - 结合$ a_k $是$ f $的奇点，其留数为$ b_k $可知，$ w=a_k $处的留数为：

$ \mathrm{Res}(g,w=a_k) = \lim_{w \rightarrow a_k} (w-a_k) \cdot \frac{f(w)}{w(w-z)} =\frac{1}{a_k(a_k-z)} \lim_{w \rightarrow a_k} (w-a_k) \cdot f(w) $

$  =\frac{1}{a_k(a_k-z)} \mathrm{Res}(f,w=a_k) = \frac{b_k}{a_k(a_k-z)} $

+ 使用留数定理计算$ C_k $围道上的积分值：

$ I_k = \frac{1}{2\pi i}\oint_{C_k} g(w)dw = \sum \mathrm{Res}(w_p) = \frac{f(0)}{-z}+\frac{f(z)}{z} + \sum_{j=1}^{k}\frac{b_j}{a_j(a_j-z)} $

+ 对其极限$ k\rightarrow \infin $，得：

$ I = \frac{1}{2\pi i}\oint_{C_\infin} g(w)dw = \frac{f(0)}{-z}+\frac{f(z)}{z} + \sum_{k=1}^{\infin}\frac{b_k}{a_k(a_k-z)} $

+ 考察积分的绝对值：

$ |I| = |\frac{1}{2\pi i}\oint_{C_\infin} \frac{f(w)}{w(w-z)}dw| ≤ \frac{1}{2\pi}\int_0^{2\pi} \frac{|f(w)|}{R(R-|z|)} Rd\theta ≤ \frac{M}{R-|z|} \rightarrow 0 $

+ 因此可知：

$ 0 = I = \frac{f(0)}{-z}+\frac{f(z)}{z} + \sum_{k=1}^{\infin}\frac{b_k}{a_k(a_k-z)} $

+ 整理即得：

$ f(z) = f(0) + \sum_{k=1}^{\infin} (\frac{b_k}{z-a_k}+\frac{b_k}{a_k}) $

$ \mathrm{Q.E.D.} $

### 辐角原理
#### 辐角原理内容
+ 存在半纯函数$ f(z) $在区域$ G $内含有$ Z $个零点（包含重数）和$ P $个极点（包含重数）
    - $ n $阶零点算是$ n $个
    - $ n $阶极点算是$ n $个
+ 那么$ f(z) $在$ z $平面上有积分表达式：

$ \boxed{\frac{1}{2\pi i} \oint_{C} \frac{f'(z)}{f(z)} dz = Z-P} $

    - 其中$ C $为沿着区域$ G $边界的正定向围道。

#### 辐角原理与绕数
+ 可以将左边的积分表达式整理一下：

$ LHS = \frac{1}{2\pi i} \oint_C \frac{f'(z)}{f(z)}dz = \frac{1}{2\pi i} \oint_C d (\ln (f(z)) $

+ 其中，$ \ln f(z) $可以进一步改写为：

$ \ln f(z) = \ln|f(z)| + i\arg f(z) $

+ 绕一圈后，第一部分的积分值为$ 0 $：

$ \frac{1}{2\pi i} \oint_C d (\ln (|f(z)|) = \frac{1}{2\pi i} \ln |f(z)||_{z_0}^{z_0} = 0 $

+ 因此：

$ LHS = \frac{1}{2\pi} \oint_C d(\arg(f(z)) = \frac{1}{2\pi} \Delta_C \arg(f(z)) $

    - 令$ w=f(z) $，则$  \frac{1}{2\pi} \Delta_C \arg(f(z)) $表明了$ z $在$ z $平面上沿着围道$ C $绕一圈的过程中，$ w $在$ w $平面上绕着原点$ w=0 $旋转的圈数，又称为绕数，即Winding number。

#### 辐角原理的证明
+ 考虑函数$ g(z) = f'(z)/f(z) $，容易看出$ g(z) $的极点只能出现在$ f(z) $的零点或者奇点处：
    - 出现在$ f(z) $零点：分母贡献留数
    - 出现在$ f(z) $极点：分子贡献留数
+ 设$ z_k $是$ f(z) $的一个$ a_k $阶零点，在$ z_k $附近可以将$ f(z) $改写为：

$ f(z) = (z-z_k)^{a_k} h(z) $

    - 其中$ h(z) $解析函数且$ h(z_k) \neq 0 $，对$ z $求导得：

$ f'(z) = a_k (z-z_k)^{a_k-1} h(z) + (z-z_k)^{a_k} h'(z)  $

    - 此时可以用$ h(z) $表示$ g(z) $：

$ g(z) = \frac{f'(z)}{f(z)} = \frac{a_k}{z-z_k}+\frac{h'(z)}{h(z)} $

    - 其中$ h'(z)/h(z) $在$ z_k $附近解析
    - $ f'(z)/f(z) $在$ z_k $处有一个单极点，留数为$ a_k $
+ 设$ z_m $是$ f(z) $的一个$ b_m $阶极点，在$ z_m $附近可以将$ f(z) $改写为：

$ f(z) = (z-z_m)^{-b_m} \phi(z) $

    - 其中$ \phi(z) $解析函数且$ \phi(z_m) \neq 0 $，对$ z $求导得：

$ f'(z) = b_k (z-z_m)^{-b_m-1} \phi(z) + (z-z_m)^{-b_m} \phi'(z)  $

    - 此时可以用$ h(z) $表示$ g(z) $：

$ g(z) = \frac{f'(z)}{f(z)} = \frac{-b_m}{z-z_m}+\frac{\phi'(z)}{\phi(z)} $

    - 其中$ \phi'(z)/\phi(z) $在$ z_m $附近解析
    - $ f'(z)/f(z) $在$ z_m $处有一个单极点，留数为$ -b_m $
+ 根据留数定理可知：

$ \frac{1}{2\pi i} \oint_{C} \frac{f'(z)}{f(z)} dz  = \sum \mathrm{Res}(\frac{f'}{f},z_{pole}) =  \sum \mathrm{Res}(\frac{f'}{f},z_k) +\sum \mathrm{Res}(\frac{f'}{f},z_m) $

$ = \sum a_k +\sum (-b_m) = Z-P $

$ \mathrm{Q.E.D.} $
