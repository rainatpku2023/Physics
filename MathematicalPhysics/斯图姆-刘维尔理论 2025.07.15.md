### 引入：非齐次常微分方程与Hermite矩阵
#### 非齐次线性微分方程解在Sturm-Liouville本征函数系上的展开
+ 考虑非齐次ODE：

$ a_2(x) y'' + a_1(x)y' + a_0(x)y = f(x) $

    - 定义微分算符：$ \hat{L} = a_2 \frac{d^2}{dx^2} + a_1 \frac{d}{dx} +a_0 $
    - 那么微分方程可以改写为：

$ \hat{L}y = f $

    - 下面寻找$ \hat{L} $的$ \mathrm{Sturm-Liouville} $本征函数$ \phi_n $满足：

$ \hat{L} \phi_n = -\lambda_n \phi_n (x) $

    - 此时可对$ y $进行本征值展开：

$ y = \sum_{n=1}^{\infin} c_n \phi_n(x) $

#### 从线性方程组到Hermite矩阵
+ 考虑线性方程组：

$ A\cdot \vec{Y} = \vec{F} $

    - 其中$ A $是$ n×n $的矩阵
    - $ A $具有$ \mathrm{Hermite} $性，即其$ \mathrm{Hermite} $共轭是本身：

$ A^{\dagger} = (A^*)^T = A $

+ 定义内积：$ (\vec{u},\vec{v}) = \vec{u}^{\dagger} \cdot \vec{v} $
    - 此时$ \mathrm{Hermite} $性可以等价地写为：

$ (\vec{u},A\vec{v}) 
= \vec{u}^{\dagger} A \vec{v} 
= \vec{u}^{\dagger} A^{\dagger} \vec{v}
= (\vec{u}A)^{\dagger} \vec{v}
= (A\vec{u},\vec{v})  $

#### Hermite矩阵的性质
+ **性质1：**$ \mathrm{Hermite} $**矩阵本征值是实数**
    - 证明：设$ \vec{v} $是$ \mathrm{Hermite} $矩阵的一个本征矢，即满足：

$ A\vec{v} = \lambda\vec{v} $

    - 则有：

$ \vec{v}^{\dagger} A \vec{v} = \lambda \vec{v}^{\dagger} \vec{v} = \vec{v}^{\dagger} A^{\dagger} \vec{v} = (A\vec{v})^{\dagger}\vec{v} = \lambda^* \vec{v}^{\dagger} \vec{v} $

    - 比较得：

$ \lambda = \lambda^* $

$ \mathrm{Q.E.D.} $

+ **性质2：**$ \mathrm{Hermite} $**矩阵对应不同本征值的本征矢彼此正交**
    - 证明：

$ \lambda_j (\vec{v}_i,\vec{v}_j) =  (\vec{v}_i,A\vec{v}_j) =  (A\vec{v}_i,\vec{v}_j) = \lambda_i^* (\vec{v}_i,\vec{v}_j) = \lambda_i (\vec{v}_i,\vec{v}_j) $

    - 比较前后得：

$ (\lambda_j - \lambda_i)\cdot(\vec{v}_i,\vec{v}_j) = 0 $

    - 考虑到本征值不同，即$ (\lambda_j - \lambda_i)≠ 0 $，有：

$ (\vec{v}_i,\vec{v}_j) = 0,i≠j $

$ \mathrm{Q.E.D.} $

    - 如果本征矢选取合适，可以实现正交归一：

$ (\vec{v}_i,\vec{v}_j) = \delta_{ij} $

#### 由Hermite矩阵的性质求线性方程组的特解
+ 先找$ A $的本征基$ \{ \vec{v}_1,\vec{v}_2, \cdots ,\vec{v}_n\} $
+ 对解$ y $组成的矢量$ \vec{Y} $的在$ A $的本征基上进行分解：

$ \vec{Y} = \sum_{i=1}^{n} c_i \vec{v}_i $

+ 再对非齐次部分$ f $组成的矢量$ \vec{F} $在$ A $的本征基上进行分解：

$ \vec{F} = \sum_{i=1}^{n} F_i \vec{v}_i $，其中$ F_i = (\vec{v}_i,\vec{F}) $

+ 分别代入左侧和右侧，得到：

$ \sum_{i=1}^{n} c_i A\vec{v}_i = \sum_{i=1}^{n} c_i \lambda_i \vec{v}_i =
\sum_{i=1}^{n} F_i \vec{v}_i $

+ 结合本征矢的正交性得：

$ c_i = \frac{F_i}{\lambda_i} = \frac{(\vec{v}_i,\vec{F})}{\lambda_i} $

    - 即得系数。

### Sturm-Liouville 理论
#### Sturm-Liouville本征值问题
+ 定义$ \mathrm{Sturm-Liouville} $方程微分算符：

$ \hat{L} = \frac{d}{dx}(p(x)\frac{d}{dx})+q(x) $

    - 则$ \mathrm{Sturm-Liouville} $型齐次方程可以改写为：

$ \hat{L}y=0 $

    - 即：

$ (py')' +qy = 0 $

    - 定义域为：$ [a,b] $
+ $ \mathrm{S-L} $型方程本征值问题为：

$ \hat{L} y= -\lambda w(x) y $

    - 补充条件：
    - $ p,p',q,w $四个函数都在$ (a,b) $上连续
    - 在$ [a,b] $上$ p(x)>0 $且$ w(x)>0 $
    - 满足齐次边界条件：

$ \begin{cases} \alpha_1 y(a)+\beta_1 y'(a) = 0 \\ \alpha_2 y(b) + \beta_2 y'(b) = 0 \end{cases} $

#### 【待检查】从任意二阶线性ODE到S-L型方程
+ 考虑方程：$ a_2(x) y'' + a_1(x)y' + a_0(x)y = f(x) $
    - 可以改写为：$ y'' + \frac{a_1(x)}{a_2(x)}y' + \frac{a_0(x)}{a_2(x)}y = 0 $
    - 此时选取满足$ \mu(x) \frac{a_1(x)}{a_2(x)} = \mu'(x) $的$ \mu(x) $，原方程方程可化为：

$ \mu(x)[y'' + \frac{a_1(x)}{a_2(x)}y' + \frac{a_0(x)}{a_2(x)}y] = 0 $

    - 根据$ \mu(x) $性质有：



$ \frac{d\mu}{\mu} = \frac{a_1}{a_2}dx $

$ \mu = \exp {[\int^{x}\frac{a_1(x')}{a_2(x')}dx']} $

        * 此时原方程左侧可化为：？？？

$ LHS = \frac{a_2(x)}{a_1(x)} \mu'(x) [y'' + \frac{a_1(x)}{a_2(x)}y' + \frac{a_0(x)}{a_2(x)}y] $

$ = \mu'(x) [\frac{a_2(x)}{a_1(x)}y'' + y' + \frac{a_0(x)}{a_1(x)}y] $

    - 此时在$ \mathrm{S-L} $方程中$ p = \mu $$ q = \mu \frac{a_0}{a_2} $。

#### 本征值问题
+ **本征值问题**的表述为：

$ \hat{L} y(x)= -\lambda w(x) y(x) $

    - 其中$ \lambda $称为本征值，$ w(x) $称为权函数
    - $ \hat{L} $的定义为：$ \hat{L} = \frac{d}{dx}(p(x)\frac{d}{dx})+q(x) $

#### 内积定义与算符具有Hermite性的证明
+ 定义**内积**：

$ (f,g) = \int_a^b f^*(x)g(x)dx $

    - 称$ \hat{L} $是$ \mathrm{S-L} $自伴的，当$ (f,\hat{L}g) = (\hat{L}f,g) $。
    - 此处给出的其实是$ \hat{L} $是$ \mathrm{Hermite} $算符的条件，推出是自伴算符还需要另外加条件。
+ 下面证明$ \hat{L} $具有$ \mathrm{Hermite} $性：
    - 根据定义，有：

$ (\hat{L}f,g) = \int_a^b \{ {\frac{d}{dx}[p(x)\frac{d}{dx}f^*(x)]+q(x)f^*(x)} \} g(x)dx $

    - 对于大括号中左侧第一项进行分部积分可知：

$ = p(\frac{d}{dx} f^*)g|^b_a - \int_a^b p(\frac{df^*}{dx})\frac{dg}{dx}dx + \int_a^b qf^*gdx  $

    - 对于上式第二项进行分部积分可知：

$ = p(\frac{d}{dx} f^*)g|^b_a -  p\frac{dg}{dx}f^*|_a^b + \int_a^b \{ f^*\frac{d}{dx}[p\frac{dg}{dx}]+f^*qg\}dx $

    - 注意到第三项可以改写为$ (f,\hat{L}g) $，式子可以化为：

$ = p(\frac{d}{dx} f^*)g|^b_a -  p\frac{dg}{dx}f^*|_a^b + (f,\hat{L}g) $

$ = g^2p\frac{d}{dx} (f^*g)|^b_a + (f,\hat{L}g) $

    - 代入齐次边界条件：

$ \begin{cases} \alpha_1 y(a)+\beta_1 y'(a) = 0 \\ \alpha_2 y(b) + \beta_2 y'(b) = 0 \end{cases} $

    - 表面项$ g^2p\frac{d}{dx}(f^*g)|^b_a $值为零，仅剩下积分项：

$ (\hat{L}f,g)= (f,\hat{L}g) $

    - 即$ \hat{L} $具有$ \mathrm{Hermite} $性。

$ \mathrm{Q.E.D.} $

#### 带权内积的定义
+ 定义带权内积：

$ (f,g)_w = \int_a^b f^* g w dx $

    - 其中$ w \in \mathbb{R} $，即权函数的函数值为实数。
    - 容易推知：

$ (f,g)_w = (f,gw) = (fw,g) $

#### Sturm-Liouville 算符的性质
+ **性质1：**$ \mathrm{S-L} $**算符的本征值是实数**
    - 证明：

$ \lambda(f,f)_w = (f,\lambda f)_w = (f,\lambda w f)= (f,\hat{L}f) = (\hat{L}f,f) = \lambda^*(f,f)_w $

    - 结合$ (f,f)_w \neq 0 $，可知：

$ \lambda = \lambda^* $

$ \mathrm{Q.E.D.} $

+ **性质2：**$ \mathrm{S-L} $**算符不同本征值对应的本征函数彼此正交**
    - 证明：

$ \lambda_j (f_i,f_j)_w = (f_i,\lambda_j w f_j) = (f_i,\hat{L}f_j) = (\hat{L}f_i,f_j) = \lambda_i(f_i,f_j)_w $

    - 移项相减得：

$ (\lambda_j -\lambda_i)(f_i,f_j)_w = 0 $

    - 对于不同的本征值$ \lambda_j -\lambda_i \neq 0 $，有：

$ (f_i,f_j)_w = 0 $

$ \mathrm{Q.E.D.} $

#### 一般的非齐次方程在Sturm-Liouville算符本征函数基上的展开
+ 考虑如何求解方程：

$ \hat{L}y = w(x)F(x) $

+ 首先求解本征值方程：

$ \hat{L}\phi_n = -\lambda_n w \phi_n $

+ 其中$ \phi_n $来自于正交归一的本征函数基$ \{\phi_1,\phi_2,\phi_3,...\} $。
+ 下面将目标方程的两侧分别在本征函数基上进行展开：

$ y = \sum_{n=1}^{\infin} c_n \phi_n $（目标是求解这个$ y $，所以$ c_n $其实是待定系数）

$ F(x) = \sum_{n=1}^{\infin} F_n \phi_n, \mathrm{where \space as \space}F_n = (\phi_n,F(x))_w $

+ 将上述展开分别代入等式两侧得：

$ \hat{L} y = \sum_{n=1}^{\infin} (-c_n \lambda_n) w \phi_n = w F = w \sum_{n=1}^{\infin} F_n \phi_n $

+ 整理得：

$ c_n = -\frac{F_n}{\lambda_n} = -\frac{(\phi_n,F)_w}{\lambda_n} $

#### Sturm-Liouville版本的Parseval恒等式
+ 已知函数$ f(x) $，其在$ \mathrm{Sturm-Liouville} $型微分算符的本征函数基上的展开为：

$ f(x) = \sum_{n=1}^{\infin} f_n \phi_n, \mathrm{where \space as \space}f_n = (\phi_n,f)_w $

+ 考虑$ f(x) $与自身的带权内积：

$ (f,f)_w = \int_a^b f^* f w dx =\int_a^b \sum_{n=1}^{\infin} f_n^* \phi_n^*
\sum_{m=1}^{\infin} f_m \phi_m dx $

$ = \sum_{n=1}^{\infin} \sum_{m=1}^{\infin} f_n^* f_m (\phi_n,\phi_m)_w
= \sum_{n=1}^{\infin} |f_n|^2 $

+ 综上：

$ (f,f)_w = \sum_{n=1}^{\infin}|f_n|^2 $

    - 左侧是$ f $与自身的带权内积，右侧是在$ \mathrm{S-L} $型微分算符的本征函数基上的展开系数。
    - 即$ \mathrm{Parseval} $恒等式。

### 函数逼近
+ 问题：给定$ f $定义在$ [a,b] $上，记有限项求和$ g = \sum_{i=1}^{n} c_i \phi_i $
    - 问：是否可能取不同于$ (f,\phi_i) = c_i $的$ c_i $，使其逼近效果更优？
    - 可证：不可能，必有$ c_i = (f,\phi_i) $。
+ 下面给出证明：
    - 逼近效果最好等价于需要最小化：

$ (f-g,f-g)_w = \int_a^b|f-g|^2wdx $

    - 下面对第$ k $个叠加系数$ c_k $求偏导，上一个命题的必要条件为：

$ \frac{\partial}{\partial c_k}(f-g,f-g)_w = 0  $

    - 展开得：

$ \frac{\partial}{\partial c_k}[(f,f)_w-(g,f)_w-(f,g)_w+(g,g)_w] = 0  $

    - 考虑到：$ g = \sum_{i=1}^{n} c_i \phi_i $，$ f = \sum_{i=1}^{n} f_i \phi_i $可知等式左侧仅有后面三项对偏导数有贡献。

$ 
(-f_k^* - f_k^* + 2 c_k^*)\phi_k = 0  $

    - 整理得：

$ c_k^* = f_k^* $

        * 即$ (c_1,c_2,c_3,...,c_k,...) $为稳定点。
    - 另外可证$ \frac{\partial^2}{\partial c_i \partial c_j} ≥ 0 $，即证$ (c_1,c_2,c_3,...,c_k,...) $为极小值点。
    - 此处并未严格论证这就是最小值点。

$ \mathrm{Q.E.D.} $
