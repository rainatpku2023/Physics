### Legendre 方程的解
#### Laplace 方程在球坐标中的变量分离
+ $ \mathrm{Legendre} $方程可以改写为：

$ \frac{d}{dx}[(1-x^2)\frac{dy}{dx}]+\lambda y = 0 $

+ 令$ \lambda = \nu(\nu+1) $，$ \mathrm{Legendre} $方程可以改写为：

$ (1-x^2)\frac{d^2 y}{dx^2}-2x\frac{dy}{dx}+\nu(\nu+1)=0 $

    - 易知，$ z=±1 $和$ z=\infin $都是正则奇点



#### x=0 处幂级数解
+ $ x=0 $是方程的常点，有两个线性无关的幂级数解：

$ y_1(x) = \sum_{n=0}^{\infin}\frac{2^{2n}}{(2n)!} \frac{\Gamma(n-\frac{\nu}{2})\Gamma(n+\frac{\nu+1}{2})}{\Gamma(-\frac{\nu}{2})\Gamma(\frac{\nu+1}{2})}x^{2n} $

$ y_2(x) = \sum_{n=0}^{\infin}\frac{2^{2n}}{(2n+1)!} \frac{\Gamma(n-\frac{\nu-1}{2})\Gamma(n+1+\frac{\nu}{2})}{\Gamma(-\frac{\nu-1}{2})\Gamma(1+\frac{\nu}{2})}x^{2n+1} $

#### x=1 处幂级数解
+ $ x=1 $是方程的正则奇点，第一解为泰勒级数：

$ P_\nu(x) = \sum_{n=0}^{\infin} \frac{1}{(n!)^2} \frac{\Gamma(\nu+n+1)}{\Gamma(\nu-n+1)}(\frac{x-1}{2})^n $

    - 这称为$ \nu $次第一类$ \mathrm{Legendre} $函数，因为两个指标差为$ 0 $，所以第二解一定含有对数函数项：

$ Q_\nu(z) = \frac{1}{2}P_\nu(z)[\ln \frac{z+1}{z-1} -2\gamma -2\psi(\nu+1)] + \sum_{n=0}^{\infin} \frac{1}{(n!)^2} \frac{\Gamma(\nu+n+1)}{\Gamma(\nu-n+1)}(1+\frac{1}{2}+\cdots +\frac{1}{n})(\frac{z-1}{2})^n $

    - 这称为$ \nu $次第二类$ \mathrm{Legendre} $函数。

### Legendre 多项式
#### Legendre 方程解的收敛性
+ $ \mathrm{Legendre} $方程+边界条件构成的本征值问题为：

$ (1-x^2)\frac{d^2 y}{dx^2}-2x\frac{dy}{dx}+\nu(\nu+1)=0 $

$ y(±1) 有界 $

+ 方程通解为：

$ y(x) = AP_\nu(x)+BQ_\nu(x) $

+ 因为$ y(1) $有界，而$ Q_\nu(x) $含有对数函数项，所以其系数必须为零，即：

$ y(x) = AP_\nu(x) $

+ 下面令$ y(-1) $有界，得必须有截断：

+ 综上，$ \mathrm{Legendre} $方程的解截断时的本征值为：

$ \lambda_l = l(l+1) $

    - 对应的本征函数为：

$ y_l(x) = P_l(x) $

    - 其中，表达式为：

$ P_l(x) =\sum_{n=0}^{l} \frac{1}{(n!)^2} \frac{\Gamma(l+n+1)}{\Gamma(l-n+1)}(\frac{x-1}{2})^n \\= \sum_{n=0}^{l} \frac{1}{(n!)^2} \frac{(l+n)!}{(l-n)!}(\frac{x-1}{2})^n  $

#### Legendre 多项式
+ $ l $次$ \mathrm{Legendre} $多项式的表达式为：

$ \boxed{P_l(x) =\sum_{n=0}^{l} \frac{1}{(n!)^2} \frac{\Gamma(l+n+1)}{\Gamma(l-n+1)}(\frac{x-1}{2})^n = \sum_{n=0}^{l} \frac{1}{(n!)^2} \frac{(l+n)!}{(l-n)!}(\frac{x-1}{2})^n } $

### Legendre 多项式的微分表示式
#### Rodrigue 公式
+ $ \mathrm{Legendre} $多项式的微分表示式为：

$ \boxed{P_l(x) = \frac{1}{2^l l!}\frac{d^l}{dx^l} (x^2-1)^l} $

+ 证明过程：

#### Rodrigue 公式的推论
+ 推论1：$ \mathrm{Legendre} $函数的奇偶性：

$ \boxed{P_l(-x) = (-1)^l P_l(x)} $

    - 证明：

$ P_l(-x) = \frac{1}{2^l l!} \frac{d^l}{d(-x)^l} ((-x)^2-1)^l 
= \frac{1}{2^l l!} \frac{d^l}{d(-x)^l} (x^2-1)^l 
= (-1)^l\frac{1}{2^l l!} \frac{d^l}{dx^l} (x^2-1)^l 
= (-1)^lP_l(x) $

+ 推论2：

$ \boxed{P_{2l}(0) = (-1)^l \frac{(2l)!}{2^{2l}l!l!}} $

$ \boxed{P_{2l+1}(0) = 0} $

    - 其实直接将$ x=0 $代入$ P_l(x) = \sum_{n=0}^{l} \frac{1}{(n!)^2} \frac{(l+n)!}{(l-n)!}(\frac{x-1}{2})^n  $即可证明。

### Legendre 多项式的正交完备性
#### Legendre 多项式正交性证明与模方求解
+ $ \mathrm{Legendre} $多项式的的正交归一关系可以统一写为：

$ \boxed{\int_{-1}^1 P_l(x)P_k(x)dx = \frac{2}{2l+1}\delta_{kl}} $

+ 下面给出证明：
    - 将$ \mathrm{Rodrigue} $公式代入积分表达式中：

$ \int_{-1}^1 P_l(x)P_k(x)dx = \int_{-1}^1 \frac{1}{2^ll!}\frac{d^l}{dx^l} (x^2-1)^l P_k(x)dx $

    - 对等式右侧进行分部积分：

$ =\frac{1}{2^ll!}P_k(x) \frac{d^{l-1}}{dx^{l-1}}(x^2-1)^l|_{-1}^{1}
- \frac{1}{2^ll!}\int_{-1}^1P_k'(x) \frac{d^{l-1}}{dx^{l-1}}(x^2-1)^ldx $

    - 注意到表面项微分次数不足以消掉所有的$ (x-1) $和$ (x+1) $成分，所以表面项为零：

$ =
- \frac{1}{2^ll!}\int_{-1}^1P_k'(x) \frac{d^{l-1}}{dx^{l-1}}(x^2-1)^ldx $

    - 将上述过程重复$ l $次，每一次表面项都是零：

$ = (-1)^l \frac{1}{2^l l!} \int_{-1}^1 P_k^{(l)}(x) (x^2-1)^ldx $

+ 回顾$ l $阶$ \mathrm{Legendre} $多项式形式：

$ P_l(x) =\sum_{n=0}^{l} \frac{1}{(n!)^2} \frac{(l+n)!}{(l-n)!}(\frac{x-1}{2})^n  $

    - 可知$ \mathrm{Legendre} $多项式最高次为$ l $次。即对$ P_l(x) $求导$ l+1 $次时刚好变成零：

$ P_l^{(l)}(x) \neq 0; \; P_l^{(l+1)}(x) =0 $

    - 因此，如果$ k<l $，$ P_k^{(l)}(x) = 0 $，有：

$ \int_{-1}^1 P_l(x)P_k(x)dx = 0 $

        * 同理可知如果$ k>l $内积也为零。**也就证明了正交性**。
    - 如果$ k=l $，那么有：

$ P_l(x) = \frac{(2l)!}{2^l(l!)^2}x^l + \cdots $

$ P_l^{(l)}(x) = \frac{(2l)!}{2^ll!} $

        * 可以**计算模方**：

$ ||P_l(x)||^2= (-1)^l \frac{1}{2^l l!} \int_{-1}^1 P_l^{(l)}(x) (x^2-1)^ldx
= (-1)^l \frac{1}{2^l l!} \int_{-1}^1 \frac{(2l)!}{2^ll!} (x^2-1)^ldx $

            + 积分得：

$ ||P_l(x)||^2=  \frac{2}{2l+1} $

#### 多项式按照 Legendre 多项式展开
+ 问题：$ x^k $按照$ \mathrm{Legendre} $多项式展开：

$ x^k = \sum_{l=0}^\infin c_l P_l(x) $

    - 展开系数为：

$ c_l = \frac{2l+1}{2}\int_{-1}^1 x^k P_l(x)dx $

+ 本节用到的最重要的一个性质就是奇偶性：

$ P_l(-x) = (-1)^l P_l(x) $

    - 因为它可以推出来如下结论：

$ \boxed{\int_{-1}^1 x^k P_l(x)dx = 0;\; \mathrm{if \; k+l \;is \;odd}} $

        * 即奇函数在对称区域$ [-1,1] $上积分值为零。
+ 根据$ \mathrm{Legendre} $函数的$ \mathrm{Rodrigue} $表达式，进行分部积分可知：

$ \boxed{\int_{-1}^1 x^k P_l(x)dx = 0;\; \mathrm{if \; k<l}} $

### Legendre 多项式的递推关系
#### 递推关系
+ 生成函数两端对$ t $求导数，得到第一个递推关系是：

$ (2l+1)xP_l(x) = (l+1)P_{l+1}(x)+lP_{l-1}(x) $

+ 生成函数两端对$ x $求导数，得到第二个递推关系为：

$ P_l(x) = P'_{l+1}(x) -2x P'_l(x) + P_{l-1}'(x) $

+ 生成函数对$ t,x $都求导，得到第四第五个递推关系为：

$ P'_{l+1}(x) = x P_l'(x) +(l+1)P_l(x) $

$ P'_{l-1}(x) = x P_l'(x) -lP_l(x) $

### 连带 Legendre 函数
#### 连带 Legendre 方程
+ $ \mathrm{Laplace} $方程经过分离变量后，得到关于方位角的定解问题为：

$ \frac{1}{\sin\theta}\frac{d}{d \theta}(\sin\theta \frac{d \Theta }{d \theta}) + (\lambda  - \frac{m^2}{\sin^2\theta})\Theta =0 $

$ \Theta(0)有界 $$ \Theta(\pi)有界 $

    - 分离变量过程可参考：[Laplace 方程在球坐标下的分离变量过程 2025.10.20](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/lwysf4awgk5e46g7#OvaOk)
+ 经过变量替换$ x = \cos \theta, y(x) = \Theta(\theta) $，得到连带$ \mathrm{Legendre} $方程和定解条件的形式为：

$ \frac{d}{dx} [(1-x^2) \frac{dy(x)}{dx}] + [\nu(\nu+1)-\frac{m^2}{1-x^2}]y(x) =0 $

$ y(±1)有界 $

#### m 阶 l 次连带 Legendre 函数
+ 当$ \nu = l = 0,1,2,3,... $时可以证明定解问题的一个解为：

$ y(x) = \boxed{P_l^m(x)=(-)^m (1-x^2)^{m/2} \frac{d^mP_l(x)}{dx^m }} $

    - 其中$ P_l^m(x) $称为$ m $阶$ l $次连带$ \mathrm{Legendre} $函数。
    - 对应的本征值为：

$ \boxed{\lambda_l = l(l+1),\; l=0,1,2,3,...} $

+ 此外，还有另外一个解$ P_l^{-m}(x) $，定义为：

$ P_l^{-m}(x) = (-)^2 \frac{(l-m)!}{(l+m)!} P_l^m(x) $

#### m 阶 l 次连带 Legendre 函数的正交性与模方
+ 如此规定$ m $阶$ l $次连带$ \mathrm{Legendre} $函数，可以将正交模方写成：

$ \boxed{\int_{-1}^1 P_l^m(x)P_k^m(x) dx = \frac{(l+m)!}{(l-m)!}\frac{2}{2l+1}\delta_{lk}} $

+ 如果将变量替换$ x = \cos\theta $恢复，正交模方可以写为：

$ \int_{0}^\pi P_l^m(\cos\theta)P_k^m(\cos\theta) \sin\theta d\theta = \frac{(l+m)!}{(l-m)!}\frac{2}{2l+1}\delta_{lk} $



### 球面调和函数（即球谐函数）
#### Laplace 方程的角向部分
+ $ \mathrm{Laplace} $方程经过分离变量后，得到关于方位角和极角的定解问题为：

$ \frac{1}{ \sin \theta}\frac{\partial}{\partial \theta}(\sin\theta \frac{\partial S }{\partial \theta}) + \frac{1}{\sin^2\theta} \frac{\partial^2S}{\partial \phi^2} + \lambda S = 0 $

$ S|_{\theta=0} 有界 $$ S|_{\theta=\pi} 有界 $

$ S|_{\phi=0} =S|_{\phi=2\pi}  $$ \frac{\partial S}{\partial \phi}|_{\phi=0} =\frac{\partial S}{\partial \phi}|_{\phi=2\pi}  $

    - 分离变量过程可参考：[Laplace 方程在球坐标下的分离变量过程 2025.10.20](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/lwysf4awgk5e46g7#OvaOk)

#### 球谐函数
+ $ \mathrm{Laplace} $方程角向部分的方程的一个解$ S_{lm}(\theta,\phi) $为球谐函数：
+ 物理学中常用的球谐函数的形式为：

$ S_{lm}(\theta,\phi) = P_l^{|m|}(\cos\theta) e^{im\phi}, \; m = 0,±1,±2,±3,... $

    - 推导过程可以参考：[Laplace 方程在球坐标下的分离变量过程 2025.10.20](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/lwysf4awgk5e46g7#zo5fX)
    - 需要注意是$ |m| $阶而非$ m $阶。

#### 球谐函数的正交性与模方
+ 结合连带$ \mathrm{Legendre} $函数的正交归一性和指数函数的归一性：

$ \int_{0}^\pi P_l^m(\cos\theta)P_k^m(\cos\theta) \sin\theta d\theta = \frac{(l+m)!}{(l-m)!}\frac{2}{2l+1}\delta_{lk} $

$ \int_0^{2\pi}e^{im\phi}(e^{in\phi})^* d\phi = \delta_{mn} $



+ 球谐函数的正交性和模方可以统一写为：

$ \int_0^\pi \int_0^{2\pi} S_{lm}(\theta,\phi) S^*_{kn}(\theta,\phi) \sin\theta d\theta d\phi = \frac{(l+|m|)!}{(l-|m|)!}\frac{4\pi}{2l+1}\delta_{lk}\delta_{mn} $

#### 归一化球谐函数
+ 希望球谐函数的模方为$ 1 $的版本，并且注意到：

$ ||S_{lm(\theta,\phi )}||^2 = \frac{(l+|m|)!}{(l-|m|)!}\frac{4\pi}{2l+1} $

+ 所以，如果将归一化球谐函数写为：

$ Y_{lm}(\theta,\phi) = \frac{ S_{lm(\theta,\phi)}}{||S_{lm(\theta,\phi )}||} = \sqrt{\frac{(l-|m|)!}{(l+|m|)!}\frac{2l+1}{4\pi}} S_{lm}(\theta,\phi) $

    - 需要注意根号里面的绝对值，以及负号在上正号在下。
+ 那么正交归一性可以写为：

$ \int_0^\pi \int_0^{2\pi} Y_{lm}(\theta,\phi) Y^*_{kn}(\theta,\phi) \sin\theta d\theta d\phi = \delta_{lk}\delta_{mn} $

    - 即模方为$ 1 $。
