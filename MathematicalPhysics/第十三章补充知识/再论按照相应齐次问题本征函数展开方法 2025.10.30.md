#### 方法2：按照相应齐次问题本征函数展开
+ 考虑定解问题：

$ \frac{\partial^2 u}{\partial t^2}-a^2 \frac{\partial^2 u}{\partial x^2} = f(x,t) $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

$ u|_{t=0} = \phi(x) \;\;\; \frac{\partial u}{\partial t}|_{t=0} = \psi(x) $

##### Step 1 求相应齐次问题本征函数
+ 对应的齐次问题为：

$ \frac{\partial^2 u}{\partial t^2}-a^2 \frac{\partial^2 u}{\partial x^2} =0 $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

    - 分离变量得到本征值问题：

$ X_n''(x) + \lambda_n X_n(x) = 0 $

$ X_n(0)= 0 \;\;\; X_n(l) = 0 $

    - 设其本征函数系为$ \{ X_n(x) \} $。
+ <font style="color:#DF2A3F;">对于更加复杂的问题，对于哪一个变量选为本征函数展开，可能有不止一种选择。</font>
    - <font style="color:#DF2A3F;">但是不同选择得到的本征函数问题不同，复杂度不同，可能在后续问题中产生不同的效果。</font>
    - <font style="color:#DF2A3F;">一般来说，选择最复杂的变量作为本征函数，剩下比较简单的部分之后成为叠加系数。</font>
        * <font style="color:#DF2A3F;">这是因为，最复杂的变量对应的本征函数一般书中有所论述</font>
        * <font style="color:#F1A2AB;">例如球坐标Poisson方程中，球谐函数常常作为本征函数，径向方程作为叠加系数。</font>[球函数×非齐次项按照相应齐次问题本征函数展开](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/lkc7sumlysb6kpxq)
        * <font style="color:#F1A2AB;">再如球坐标热传导方程中，球Bessel函数（径向）常常作为本征函数，而时间作为叠加系数。</font>
        * <font style="color:#F1A2AB;">如果把简单的部分当成了本征值问题，那么叠加系数会很难算。</font>
+ <font style="color:#DF2A3F;">为什么选择相应齐次问题的本征函数？</font>
    - <font style="color:#DF2A3F;">相应齐次问题的本征函数满足齐次边界条件。</font>
    - <font style="color:#DF2A3F;">用本征函数叠加出来的方程的解也自然满足齐次边界条件。</font>

##### Step 2 本征函数展开非齐次项和特解
+ 将非齐次问题的方程**非齐次项**在本征函数系$ \{ X_n(x) \} $上进行展开:

$ f(x,t) = \sum_{n=1}^{\infin} g_n(t) X_n(x) $

    - 再将待求**特解**在本征函数系$ \{ X_n(x) \} $上进行展开：

$ u(x,t) = \sum_{n=1}^{\infin} T_n(t) X_n(x) $

+ 将上述两个展开式代入非齐次方程：

$ \sum_{n=1}^{\infin}T_n''(t)X_n(x)-a^2\sum_{n=1}^{\infin}T_n(t)X_n''(x) = \sum_{n=1}^{\infin}g_n(t)X_n(x) $

+ <font style="color:#DF2A3F;">这个方程是关于</font>$ T_1(t),T_2(t),...,T_n(t) $<font style="color:#DF2A3F;">的微分方程，非常麻烦。</font>
    - <font style="color:#DF2A3F;">如果能实现脱耦合，就能得到仅仅关于</font>$ T_1(t),T_2(t),...,T_n(t) $<font style="color:#DF2A3F;">的</font>$ n $<font style="color:#DF2A3F;">个独立的微分方程。</font>
    - <font style="color:#DF2A3F;">下面利用本征函数正交性的就是一种脱耦合方法。</font>
    - <font style="color:#DF2A3F;">本征方程是将</font>$ X''_n(x) $<font style="color:#DF2A3F;">换成</font>$ X_n(x) $<font style="color:#DF2A3F;">。</font>

##### Step 3 代入本征方程，得到方程
    - 将本征值问题的方程$ X_n''(x) + \lambda_n X_n(x) = 0 $代入：

$ \sum_{n=1}^{\infin}T_n''(t)X_n(x)+a^2\sum_{n=1}^\infin \lambda_nT_n(t)X_n(x) = \sum_{n=1}^{\infin}g_n(t)X_n(x) $

        * 此处本质上是将$ X''_n(x) $换成$ X_n(x) $，这样方便比较系数，找到$ T_n(t) $所满足的方程。

##### Step 4 根据本征函数正交性脱耦合，得到每个叠加系数满足的独立的微分方程
    - 根据本征函数正交性，可以逐项得到$ T_n(t) $所满足的常微分方程：

$ T_n''(t)+a^2\lambda_nT_n(t) =g_n(t) $

    - 上述常微分方程可以用拉普拉斯变换求解。
        * $ T_n(t),T_n'(t) $的值可以通过非齐次方程对应初值条件，结合叠加系数求法求出。
        * 因为用到了拉普拉斯变换，所以初始条件齐次最理想。

##### Step 5 叠加出原来非齐次问题的解
+ 求得叠加系数$ T_n(t) $的具体形式后，叠加出原来非齐次问题的解：

$ u(x,t) = \sum_{n=1}^{\infin} T_n(t) X_n(x) $

+ <font style="color:#DF2A3F;">一个问题：为什么</font>$ f(x,t) $<font style="color:#DF2A3F;">在</font>$ x=l $<font style="color:#DF2A3F;">时不一定值为零，但用满足</font>$ X(l) = 0 $<font style="color:#DF2A3F;">的</font>$ X_n(x) $<font style="color:#DF2A3F;">叠加出的解却可以满足方程？</font>
+ <font style="color:#DF2A3F;">回答：级数解是一个点点收敛于真实解的方法。</font>
    - $ u|_{x=a} = 0 $<font style="color:#DF2A3F;">显然恒成立，这是选用了本征函数的结果。本征函数也是为了这个而构造的</font>
    - <font style="color:#DF2A3F;">无穷级数解在区间内部等于真实解，只在边界上有差异，但是边界测度为零。不会对实际物理问题产生影响。</font>

#### 关于非齐次项不满足齐次边界问题的讨论
+ 考虑定解问题：

$ \frac{\partial^2 u}{\partial t^2}-a^2 \frac{\partial^2 u}{\partial x^2} = C; \; C\neq 0 $

$ u|_{x=0} = 0 \;\;\; u|_{x=l} = 0 $

$ u|_{t=0} = \phi(x) \;\;\; \frac{\partial u}{\partial t}|_{t=0} = \psi(x) $

    - 为什么在$ X_n(0) = 0 $前提下，$ u(x,t) = \sum_{n=1}^{\infin} T_n(t) X_n(x) $能够满足上述方程？非齐次项$ C $不满足齐次边界条件。
+ 实际上级数解是点点收敛于真实解的，也就是说$ 0<x<l $的部分级数解都严格收敛于正确的$ u $。
+ 但是在$ 0,l $两点发散。但真正物理问题中这个现象可以忽略。
+ 可以构造一个函数$ C(x,\epsilon) $：

$ C(x,\epsilon) = \begin{cases} 
\frac{C}{\epsilon}x, 0<x<0+\epsilon \\
C, \; 0+\epsilon<x< l-\epsilon  \\
-\frac{C}{\epsilon}(x-l), l-\epsilon<x<l

\end{cases} $

+ 对应定解问题为：

$ \frac{\partial^2 u_\epsilon}{\partial t^2}-a^2 \frac{\partial^2 u_\epsilon}{\partial x^2} = C(x,\epsilon) $

$ u_\epsilon|_{x=0} = 0 \;\;\; u_\epsilon|_{x=l} = 0 $

$ u_\epsilon|_{t=0} = \phi(x) \;\;\; \frac{\partial u_\epsilon}{\partial t}|_{t=0} = \psi(x) $

    - $ u_\epsilon $的定解问题非齐次项$ C(x,\epsilon) $满足齐次边界条件，所以可以求出$ u_\epsilon $的级数解，而且级数解严格收敛于真实的$ u_\epsilon $，而且在边界上也是如此。
+ 当$ \epsilon \rightarrow 0 $时，$ 0+\epsilon<x< l-\epsilon $范围内两个级数解有$ u_\epsilon \rightarrow u $，最终$ \epsilon $推到$ 0 $时级数解就在$ 0<x<l $上都等于真实解了，只有边界上有问题。但是边界的测度为零。
+ 在波动方程语境下，非齐次项就是单位长度弦受到的力
    - 当然可以在弦的端点处施加不为零的力 -> 非齐次项不满足边界条件
    - 但是这对物理结果的影响趋于零 -> 级数解只在测度为零的边界不收敛到真实解。
