### 问题引入
+ 吴崇试《数学物理方法（第三版）》中，圆形区域内稳定问题的定解问题为：

$ \frac{\partial^2 u}{\partial x^2}+ \frac{\partial^2 u}{\partial y^2} = 0 $

$ u|_{x^2+y^2=a^2} = f $

    - 改为平面极坐标系后，加上周期性条件与有界条件，定解问题表述为：

$ \frac{1}{r}\frac{\partial}{\partial r}(r\frac{\partial u}{\partial r}) +\frac{1}{r^2}\frac{\partial^2 u}{\partial \phi^2} = 0 $

$ u(r,\phi)|_{\phi = 0}=u(r,\phi)|_{\phi = 2\pi} $

$ \frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 0}=\frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 2\pi} $

$ u(r,\phi)|_{r = 0}有界 $

$ u(r,\phi)|_{r = a}=f(\phi) $

+ 后文直接进行变量分离$ u(r,\phi) = R(r)\Phi(\phi) $，代入方程得到关于$ \Phi(\phi) $的本征值问题：

$ \frac{d^2 \Phi}{d\phi^2}+ \lambda \Phi = 0 $

$ \Phi(0) = \Phi(2\pi) \;\;\; \Phi'(0) = \Phi'(2\pi) $

    - 然后就直接得到本征值对应的本征函数：

$ \lambda_0 = 0, \; \Phi_0(\phi) = 1 $

$ \lambda_m = m^2, \; \Phi_{m1}(\phi) = \sin m\phi \:\:\:\ \Phi_{m2} = \cos m\phi $

    - 后文也证明了本征函数正交性。
    - 同时也证明了$ \Phi_{m1}(\phi),\Phi_{m2}(\phi) $的正交性，为之后直接用内积法求叠加系数做铺垫。
+ **问题在于，为什么认为**$ \Phi(0) = \Phi(2\pi) \;\;\; \Phi'(0) = \Phi'(2\pi) $**是齐次边界条件？**
    - **回到分离变量前的情形，为什么认为下述周期性条件在数学上可以认为是齐次边界条件？**

$ u(r,\phi)|_{\phi = 0}=u(r,\phi)|_{\phi = 2\pi} $

$ \frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 0}=\frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 2\pi} $

### 回答1：从数学上讲齐次
#### 对于齐次思考
+ 线性映射的定义为：对于自变量$ x $和$ y $来说，映射$ f $满足：

$ f(ax+by) = af(x)+bf(y) $

+ 上述两个边界条件可以改写为：

$ \hat{L_1}(u) =u(r,\phi)|_{\phi = 0}-u(r,\phi)|_{\phi = 2\pi} = 0 $

$ \hat{L_2}(u) = \frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 0}-\frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 2\pi} = 0 $

+ 显然$ \hat{L_1},\hat{L_2} $都是线性算符，应有：

$ \hat{L_1}(au_1+bu_2) = a\hat{L_1}(u_1)+b\hat{L_1}(u_2) $

$ \hat{L_2}(au_1+bu_2) = a\hat{L_2}(u_1)+b\hat{L_2}(u_2) $

    - 若$ \hat{L_1}(u_1) = 0 $且$ \hat{L_1}(u_2) = 0 $，则有：

$ \hat{L_1}(au_1+bu_2) = a\cdot0+b\cdot 0 = 0 $

    - 因此，$ \hat{L_1}(u) =0 $为齐次边界条件。$ \hat{L_2}(u) =0 $同理为齐次边界条件。
+ 即下述关系为齐次边界条件：

$ u(r,\phi)|_{\phi = 0}=u(r,\phi)|_{\phi = 2\pi} $

$ \frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 0}=\frac{\partial u(r,\phi)}{\partial \phi}|_{\phi = 2\pi} $

#### 一个不齐次的例子
$ u(r,\phi)|_{\phi = 0}=u(r,\phi)|_{\phi = 2\pi} +u_0 $

+ 这不是一个齐次边界条件，当$ u_0 \neq 0 $时。
+ 其条件可以改写为：

$ \hat{L_1}(u) =u(r,\phi)|_{\phi = 0}-u(r,\phi)|_{\phi = 2\pi} = u_0 $

+ 显然，若$ \hat{L_1}(u_1) = u_0 $且$ \hat{L_1}(u_2) = u_0 $，根据线性算符性质：

$ \hat{L_1}(au_1+bu_2) = a\cdot u_0+b\cdot  u_0= (a+b)u_0 \neq u_0 $

    - $ \hat{L_1}(a u_1+bu_2) = u_0 $不被满足，因此$ \hat{L_1}(u) = u_0 $并非齐次边界条件。
    - 即下述条件并非齐次边界条件：

$ u(r,\phi)|_{\phi = 0}=u(r,\phi)|_{\phi = 2\pi} +u_0 $

### 回答2：在实用上讲可以完成不同本征值对应的本征函数正交的证明
+ 回忆不同本征值的本征函数正交的一般方法：

[第十三章 分离变量法 2025.09.21](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/mkw0bmbrn3z4phff#mbhlT)

+ 其中最核心的一步是：

$  (\lambda_n-\lambda_m) \int_0^l X^*_n(x)X_m(x)dx = - [(X_n^*)'(x)X_m(x)-X_m'(x)X_n^*(x)]_{x=0}^{x=l} = 0 $

+ 在这个情境中的表达式为：

$  (\lambda_n-\lambda_m) \int_0^{2\pi} \Phi^*_n(\phi)\Phi_m(\phi)d\phi = - [(\Phi_n^*)'(\phi)\Phi_m(\phi)-\Phi_m'(\phi)\Phi_n^*(\phi)]_{\phi=0}^{\phi=2\pi}  $

+ 边界条件$ \Phi(0) = \Phi(2\pi) \;\;\; \Phi'(0) = \Phi'(2\pi) $
    - 一样可以使表面项$ -[(\Phi_n^*)'(\phi)\Phi_m(\phi)-\Phi_m'(\phi)\Phi_n^*(\phi)]_{\phi=0}^{\phi=2\pi} $等于$ 0 $，
    - 并完成正交性证明。
