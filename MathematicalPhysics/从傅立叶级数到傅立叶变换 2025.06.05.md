### Fourier级数
#### 问题引入：从Taylor级数到Fourier级数
+ 已知定义在$ \mathbb{R} $上的函数$ f(x) $可以展开成$ \mathrm{Taylor} $级数：

$ f(x) = \sum_{n=0}^{\infin} c_n x^n $

+ 更一般地，可以用函数项级数来展开任何一个函数：

$ f(x) = \sum_{n=0}^{\infin} a_n \omega_n(x) $

+ 考虑，是否可以使用周期函数来展开周期函数？

#### 共形映射与Fourier级数的导出
+ 先考虑定义在$ \mathbb{R} $上的周期函数，其中一个周期为$ [-\pi,\pi] $：
    - 如果能将其**解析延拓**到复平面$ \mathbb{C} $上，就可以使用解析函数的相关工具进行研究
    - 实轴$ \mathbb{R} $不具有复平面$ \mathbb{C} $那么好的性质，研究起来比较困难
+ 选取共形映射$ w=e^{iz} $，将$ z $平面上的函数$ f(z) $映射到$ w $平面上的函数

$ g(w) = f(\frac{\ln w}{i}) $

    - 虽然存在分支问题，但是$ f(z) $的周期是$ 2\pi $，因此在$ w $平面上问题也不会显现

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1749132145710-3ff7b18d-9449-4f98-abaa-c1426ed0d143.png" width="300" title="z平面" crop="0,0,1,1" id="u379ef0ec" class="ne-image"><img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1749132162400-10f388d4-52dc-42d9-9036-be2af60c081a.png" width="206" title="w平面" crop="0,0,1,1" id="ucde51216" class="ne-image">

    - 但是可以观察到，函数$ f(z) $无论在$ z $平面上如何延拓，对应的$ g(w) $在$ w $平面上总会在原点处缺乏定义，即成为一个奇点，此时不能在原点的实心邻域（圆域）中研究$ g(w) $，只能在**空心邻域（环域）**中进行研究。
    - 即需要使用$ \mathrm{Laurent} $级数。
+ 此时考虑在$ w $平面上对$ g(w) $进行$ \mathrm{Laurent} $级数展开，即：

$ g(w) = \sum_{n=-\infin}^{\infin} c_n w^n $

+ 根据$ \mathrm{Laurent} $系数的定义，在$ w $平面上应该有：

$ c_n = \frac{1}{2\pi i} \oint_{|\zeta|=1}\frac{g(\zeta)}{\zeta^{n+1}}d\zeta $

    - $ |\zeta| = 1 $是$ w $平面上的以原点为圆心，半径为$ 1 $的圆。
+ 代入参数化条件$ w=e^{iz} $可知，当$ \zeta $绕着圆在$ w $平面上运动时，$ t $在$ z $平面上运动轨迹与$ \zeta $的运动轨迹的关系为：

$ \zeta = e^{it} $

    - 即$ \zeta $绕着圆在$ w $平面上半径为$ 1 $的圆运动一周时，$ t $在$ z $平面上运动轨迹为从$ -\pi $到$ \pi $的直线。这条直线沿着实轴。
+ 将其代入$ \mathrm{Laurent} $级数，得到：

$ c_n = \frac{1}{2\pi i} \oint_{|\zeta|=1}\frac{g(\zeta)}{\zeta^{n+1}}d\zeta
=  \frac{1}{2\pi i} \int_{t=-\pi}^{\pi}\frac{g(e^{it})}{(e^{it})^{n+1}}ie^{it}dt $

+ 向$ g(w) = f(\frac{\ln w}{i}) $中代入$ w=e^{it} $可知：

$ g(e^{it}) = f(t) $

+ 根据上两个式子可以求得$ c_n $的表达式，即$ \mathrm{Fourier} $展开系数：

$ \boxed{c_n = \frac{1}{2\pi} \int_{-\pi}^{\pi}f(t)e^{-int}dt} $

+ 相应地，将$ w=e^{it} $和$  f(t)=g(e^{it}) $代入$ g(w) $的$ \mathrm{Laurent} $级数可知：

$ \boxed{f(t) = \sum_{n=-\infin}^{\infin}c_n e^{int}} $

    - 即$ \mathrm{Fourier} $级数。

#### Fourier级数的三角形式
+ 在高等数学中，常常将$ \mathrm{Fourier} $级数写成三角函数的形式：

$ f(x) = a_0 + \sum_{n=1}^{\infin}a_n \cos{(nx)} + \sum_{n=1}^{\infin}b_n \sin{(nx)} $

+ 其中系数$ a_0,a_n,b_n $与解析函数论得到的$ \mathrm{Fourier} $级数系数的关系为：

$ \begin{cases} a_n = c_n + c_{-n} \\ b_n = ic_n - ic_{-n} \end{cases} $

    - 特别地，$ a_0 = \frac{c_0+c_0}{2} = c_0 $。
    - 如果$ f(x) \in \mathbb{R} $，则有$ a_n,b_n $均为实数，$ c_n,c_{-n} $互为复共轭。
    - 证明：代入三角恒等式$ e^{inx} = \cos (nx) +i\sin(nx) $易得。
+ 上述系数也可以通过计算完成：

$ a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(t)dt $ 

$ a_n = \frac{1}{2\pi} \int_{-\pi}^{\pi}f(t)\cos{(nt)}dt $

$ b_n = \frac{1}{2\pi} \int_{-\pi}^{\pi}f(t)\sin{(nt)}dt $

#### Fourier级数重建的失真：Gibbs现象与Fourier定理
+ $ \mathrm{Gibbs} $现象：对于**任意有限项**$ \mathrm{Fourier} $级数，向左侧取极限都会比原来的函数高$ 17 \% $左右，而向右侧取极限都会低$ 17\% $左右。

+ $ \mathrm{Fourier} $定理：对于定义在$ (-\pi,\pi) $上的，仅有有限个极大值、极小值和间断点的函数$ f(x) $：
    - $ \mathrm{Fourier} $级数$ \sum_{n=-\infin}^{+\infin} c_ne^{inx} $收敛于$ \lim_{\epsilon\rightarrow 0} \frac{f(x+\epsilon)+f(x-\epsilon)}{2} $。

### 从DFT基导出Fourier级数
#### 复向量空间
+ 设$ N $维复线性空间$ \mathbb{C}^N $中的向量$ \vec{v} $在单位正交基的展开为：

$ \vec{v} = (v_0,v_1,v_2,...,v_{N-1}),v_i \in \mathbb{C} $

+ 可以定义内积：

$ <\vec{v},\vec{w}> = \sum_{n=0}^{N-1} v_n w_n ^* $

+ 内积具有如下性质：
    1. 线性性
    2. $ <\vec{v},\vec{w}> =<\vec{w},\vec{v}>^* $
    3. $ <\vec{v},\vec{v}> = 0 $，当且仅当$ \vec{v} = \vec{0} $

#### 希尔伯特空间
+ 定义：**带有内积的复向量空间**称为**希尔伯特空间（Hilbert Space）**，记为$ \mathbb{H} $。
+ 定义：范数/模长

$ ||\vec{v}|| = \sqrt{<\vec{v},\vec{v}>} $

+ 定义：正交性

$ \vec{v}与\vec{w}正交,当且仅当<\vec{v},\vec{w}> = 0 $

+ 定义：完备的正交归一基$ \vec{e_n},n = 0,1,2...N-1 $

$ <\vec{e_n},\vec{e_m}> = \delta_{nm} $

+ 定义：按照正交归一基的展开：

$ \vec{v}  = \sum_{n=0}^{N-1} c_n \vec{e_n} $

#### DFT基下函数的展开
+ 将函数$ f(x) $分立化，得到$ f(2\pi \frac{n}{N}) $，可以看成一个向量：

$ \vec{f} = (f(0),f(\frac{2\pi}{N}),...,f(\frac{2\pi(N-1)}{N})) $

+ 定义$ N $维$ \mathrm{DFT} $基：

$ w = e^{2\pi i/N} $

$ \vec{u_k} = (1,w^k,w^{2k},...,w^{(N-1)k}) $

+ 定义内积（$ N $维）：

$ <\vec{f},\vec{g}> = \frac{1}{N} \sum_{n=0}^{N-1} f_n g_n^* $

    - 可证明$ \mathrm{DFT} $基满足正交归一性：

$ <\vec{u_k},\vec{u_l}> = \delta_{kl} $

+ 函数$ f $按照$ \mathrm{DFT} $基的展开：

$ \vec{f} = \sum_{k=0}^{N-1} c_k \vec{u_k} = \sum_{k=0}^{N-1} c_k \left[\begin{matrix} e^{2\pi ik/N} \\\vdots \\ e^{2\pi i(N-1)k/N}\end{matrix}\right]  $

    - 其中展开系数$ c_k $定义为：$ c_k = <\vec{f},\vec{u_k}> $

$ =\frac{1}{N}\sum_{n=0}^{N-1} f_n (\vec{u_k})_n^* 
=\frac{1}{N}\sum_{n=0}^{N-1} f_n e^{-2\pi ikn/N} $

####  DFT基下的Parseval定理
+ 傅立叶级数$ \mathrm{Parseval} $定理为：

$ ||f||^2 = <\vec{f},\vec{f}> = \sum_{n=0}^{N-1} |c_n|^2 $

+ 证明：

$ LHS = <\sum_{k=0}^{N-1}c_k \vec{u_k},\sum_{l=0}^{N-1}c_l \vec{u_l}>
= \sum_{k=0}^{N-1} \sum_{l=0}^{N-1} c_k c_l ^*<\vec{u_k},\vec{u_l}> $

$ =\sum_{k=0}^{N-1} \sum_{l=0}^{N-1} c_k c_l ^* \delta_{kl} = \sum_{k=0}^{N-1} c_k c_k ^* =  \sum_{n=0}^{N-1} |c_n|^2 $

$ \mathrm{Q.E.D.} $

### 连续的DFT基
#### 连续极限与连续的DFT基
+ 内积的变化：$ {<f,g> = \frac{1}{N} \sum_{k=0}^{N-1} f_k g_k^*} \rightarrow \lim_{N \rightarrow\infin} \frac{1}{2\pi} \sum_{k=0}^{N-1}f_k g_k^* \frac{2\pi}{N} = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(x)g^*(x)dx $
    - 其中$ \frac{2\pi}{N} $相当于$ dx $，
+ 基的变化：

$ \vec{u_k} \rightarrow e_k (x) = e^{ikx} $

    - 基的正交归一性证明：

$ <e_m (x),e_n(x)> = \frac{1}{2\pi} \int_{-\pi}^{\pi} e^{imx} e^{-inx} dx = \delta_{nm} $

$ \mathrm{Q.E.D.} $

#### 2π 周期的傅立叶级数与Parseval定理
+ $ \mathrm{Fourier} $级数的变化：

$ f = \sum_{n=0}^{N-1} c_n \vec{u_n} \rightarrow \sum_{n=-\infin}^{+\infin} c_n e^{inx} $

    - 其中$ c_n $的定义为：

$ c_n = <f,e_n (x)> = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(x) e^{-inx} dx $

        * 可见，之后$ \mathrm{Fourier} $变换中$ e^{-inx} $的负号来自于复线性空间中内积其中一者需要取复共轭。
+ $ \mathrm{Parseval} $定理的变化：

$ \boxed{||f||^2 = \frac{1}{2\pi}\int_{-\pi}^{\pi} |f(x)|^2 dx 
=\sum_{n = -\infin}^{+\infin} |c_n|^2} $

#### Parseval定理的应用：求级数
+ $ \mathrm{Parseval} $定理形式如下：

$ \frac{1}{2\pi}\int_{-\pi}^{\pi} |f(x)|^2 dx 
=\sum_{n = -\infin}^{+\infin} |c_n|^2 $

    - 待求级数常常放在等式右侧，而用于计算的则是等式左侧。
    - 难点在于对于目标级数，如何找到$ f(x) $满足等式。

#### 2T 周期的傅立叶级数
+ 从$ \mathrm{DFT} $基入手，其变化为：
+ 则连续版本的基变化为：

$ e_n(x) = e^{inx} \rightarrow e^{i \frac{2\pi}{2T}nx} $

+ 模仿$ 2\pi $情形的展开，周期为$ 2T $的$ \mathrm{Fourier} $级数为：

$ \boxed{f(x) = \sum_{n=-\infin}^{+\infin}c_n e^{i \frac{2\pi}{2T} nx}} $

+ 其中系数为$ c_n = <f,e_n(x)> $，计算得：

$ \boxed {c_n = \frac{1}{2T}\int_{-T}^{T}f(x)e^{-i \frac{2\pi}{2T}nx}dx} $

#### ∞ 周期的傅立叶级数（即Fourier变换）的导出
+ 目标：使用$ \mathrm{Fourier} $级数处理非周期函数（可看作**无穷大周期函数**）。
+ 思路：将$ f(x) = \sum_{n=-\infin}^{+\infin}c_n e^{i \frac{2\pi}{2T} nx} $中$ (\frac{2\pi}{2T} n) $部分看成频率$ \omega_n $，然后让$ T\rightarrow \infin $。
+ 计算：

$ c_n \cdot 2T  = \hat{f}(\omega) = \int_{-T}^{T} f(x) e^{-i\omega x}dx $

    - 此处右侧取消了$ 1/2T $，否则$ T\rightarrow \infin $时$ \hat{f}(\omega) $的积分表达式为$ 0 $。
    - 这个式子就是$ \mathrm{Fourier} $正变换的表达式。
+ 下面尝试整理出来$ T\rightarrow \infin $时$ c_n $的表达式：
    - 则：

$ f(x) = \sum_{n = -\infin}^{+\infin} \frac{\hat{f}(\omega)}{2T} e^{i \omega_n x} $

    - 提出$ 2\pi $得：

$ = \frac{1}{2\pi} \sum_{n=-\infin}^{+\infin} \hat{f}(\omega_n)e^{i\omega_n x}\frac{2\pi}{2T} $

    - 根据$ \omega_n = \frac{2\pi}{2T}n $可知，$ \Delta \omega = \omega_{n+1} - \omega_n = \frac{2\pi}{2T} $，代入知：

$ = \frac{1}{2\pi} \sum_{n=-\infin}^{+\infin} \hat{f}(\omega_n)e^{i\omega_n x} \Delta \omega $

    - 取极限$ T\rightarrow \infin $：

$ \rightarrow  \frac{1}{2\pi} \int_{-\infin}^{+\infin} \hat{f}(\omega)e^{i\omega x} d \omega $

    - 这个式子就是$ \mathrm{Fourier} $逆变换的表达式。

### Fourier变换
#### Fourier正变换与逆变换
+ $ \mathrm{Fourier} $正变换为：

$ \boxed{\hat{f}(\omega) = \int_{-T}^{T} f(x) e^{-i\omega x}dx} $

+ $ \mathrm{Fourier} $逆变换为：

$ \boxed{f(x)  = \frac{1}{2\pi} \int_{-\infin}^{+\infin} \hat{f}(\omega)e^{i\omega x} d \omega} $

#### Fourier变换例题与留数定理应用
+ 选取大半圆弧围道补全时，需要注意让大半圆弧上的数值为高阶无穷小，这样才能用$ \mathrm{Jordan} $引理。
+ 然后使用**留数定理**。
