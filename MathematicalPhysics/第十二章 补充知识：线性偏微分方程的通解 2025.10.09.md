### 线性方程解的叠加性
#### 偏微分方程
+ 本文讨论线性偏微分方程的共性，因此引入线性算符$ \hat{L} $，将偏微分方程统一写成：

$ \hat{L}[u] = f $

    - 其中$ u $是待求的未知函数，$ f $是已知函数，称为**方程的非齐次项**。
    - 当$ f=0 $时，方程是**齐次的**；$ f \neq 0 $时，方程是**非齐次的**。

#### 线性算符
+ 一般而言，根据线性性定义，线性算符的一般性质为：

$ \hat{L}[c_1 u_1 + c_2 u_2] = c_1 \hat{L}[u_1]+c_2 \hat{L}[u_2] $

    - 其中$ c_1,c_2 $是常数。

#### 线性方程的性质
+ **性质1：齐次方程解的叠加性**
    - 如果$ u_1,u_2 $都是齐次方程$ \hat{L}[u]=0 $的解，即：

$ \hat{L}[u_1] = 0 \;\;\; \hat{L}[u_2] = 0 $

    - 那么他们的线性组合$ c_1 u_1 +c_2 u_2 $也是该齐次方程的解：

$ \hat{L}[c_1u_1+c_2u_2] = 0 $

+ **性质2：非齐次方程解的结构**
    - 非齐次方程的一个特解加上相应齐次方程的解仍然是原非齐次方程的解。
+ **性质3：非齐次方程解的“叠加性”**
    - 如果$ u_1,u_2 $分别满足非齐次方程：

$ \hat{L}[u_1] = f_1 \;\;\; \hat{L}[u_2] = f_2 $

    - 那么他们的线性组合$ c_1 u_1+c_2 u_2 $满足非齐次方程：

$ \hat{L}[c_1u_1+c_2u_2] = c_1f_1+c_2f_2 $

+ 在上述讨论中$ c_1,c_2 $都是常数。

#### 为了方便讨论引入的新记号
+ 引入新记号：

$ \hat{D}_x = \frac{\partial}{\partial x} \;\;\;
\hat{D}_y = \frac{\partial}{\partial y} $

    - 其效果等价于：

$ \hat{D}_x[u]= \frac{\partial u}{\partial x} \;\;\;
\hat{D}_y[u]= \frac{\partial u }{\partial y} $

+ 还有常数算符（这个看上去有点多余）：

$ P[u] = P·u $

    - 注意微分算符和常数算符满足的性质不同，例如对于乘积函数的作用：

$ \hat{D}_x [uv] = u \hat{D}_x [v] +v \hat{D}_x [u] $

$ P[uv] = P·uv $



+ 这种记号的方便之处在于可以将偏微分号进行“因式分解”：

### 常系数线性齐次偏微分方程的通解
#### 情况1：微分算符函数是偏微分号的齐次式
+ 当微分算符函数$ \hat{L} $是偏微分号$ \hat{D}_x,\hat{D}_y $的$ n $次齐次式时，偏微分方程可以写成如下形式：

$ (A_0\hat{D}_x^n+A_1\hat{D}_x^{n-1}\hat{D}_y +A_2\hat{D}_x^{n-2}\hat{D}_y^2 + \cdots A_n \hat{D}_y ^n)u = 0 $

    - 可以写成求和的形式：

$ (\sum_{i=0}^{n} A_i \hat{D}_x^{n-i} \hat{D}_y^{i})u = 0 $

    - 此时，可以将线性算符$ \hat{L}(\hat{D}_x,\hat{D}_y) $分解成$ n $个线性算符的乘积：

$ \hat{L}(\hat{D}_x,\hat{D}_y) = A_0(\hat{D}_x-\alpha_1 \hat{D}_y)(\hat{D}_x-\alpha_2 \hat{D}_y)\cdots (\hat{D}_x-\alpha_n \hat{D}_y) $

        * 其中$ \alpha_1,\alpha_2,\cdots,\alpha_n $都是常数，所以这些算符因子可以相互交换顺序而不改变结果。
        * 可以写成连乘的形式：

$ \hat{L}(\hat{D}_x,\hat{D}_y) = A_0\prod_{i=1}^n(\hat{D}_x-\alpha_i \hat{D}_y) $

+ 取一个**非零试探解**$ u = \phi(y+\alpha x) $，在微分号作用之后变为：

$ \hat{D}_x^k u = \alpha^k \phi^{(k)}(y+\alpha x) $

$ \hat{D}_y^k u =  \phi^{(k)}(y+\alpha x) $

$ \hat{D}_x^r\hat{D}_y^s u = \alpha^r \phi^{(r+s)}(y+\alpha x) $

    - 因此代入原方程得到：

$ (A_0\alpha^n+A_1 \alpha^{n-1}+\cdots +A_n)\phi^{(n)}(y+\alpha x) = 0 $

    - 以下方程称为**附加方程（auxilliary equation）**：

$ A_0\alpha^n+A_1 \alpha^{n-1}+\cdots +A_n = 0 $

    - 设其解为：$ \alpha_1,\alpha_2,\cdots,\alpha_n $。

##### 没有重根的情形
+ 如果$ \alpha_1,\alpha_2,\cdots,\alpha_n $两两不同，那么齐次偏微分方程的通解为：

$ u = \phi_1(y+\alpha_1 x) +\phi_2(y+\alpha_2 x)+\cdots +\phi_n(y+\alpha_n x) $

    - 其中$ \{\phi_i\} $是相互独立的任意$ n $次可微函数。

##### 有重根的情形
+ 可以参考高等数学中对于重根的解决方案。

#### 情况2：微分算符函数是偏微分号的非齐次式
+ 基本思路：
    - 将偏微分方程仍然因式分解，会发现有的因子里面不仅有$ \hat{D}_x,\hat{D}_y $，还会有常数算符。
    - 解决问题时，考虑在$ \beta = 0 $的解出来的结果上进行修改，将$ \phi(y+\alpha x) $改成$ f(x)\phi(y+\alpha x) $
    - 代入方程解出来$ f(x) $的具体形式，然后得到通解。

### 常系数线性非齐次偏微分方程的通解
#### 非齐次线性偏微分方程解的结构
+ 对于线性非齐次方程：

$ \hat{L}(\hat{D}_x,\hat{D}_y) u = f(x,y) $

+ 显然有：

$ \boxed{线性非齐次方程的通解 = 线性非齐次方程的任一特解+相应线性齐次方程的通解} $

    - 相应线性齐次方程的通解已经在[上文](#jgm6H)中部分论述。
    - 问题转化为找到线性非齐次方程的任意一个特解$ u_0 $。
+ 此时原来线性非齐次方程的特解可以形式地表示为：

$ u_0 = \frac{1}{\hat{L}(\hat{D}_x,\hat{D}_y)}f(x,y) $
