### Helmholtz方程在球坐标中的分离变量
+ $ \mathrm{Helmholtz} $方程：

$ \nabla^2 u +k^2 u = 0 $

+ 在柱坐标系$ (r,\theta,z) $中的形式为：

$ \frac{1}{r} \frac{\partial }{\partial r}(r\frac{\partial u}{\partial r}) + \frac{1}{r^2}\frac{\partial^2 u}{\partial \theta^2} + \frac{\partial^2 u}{\partial z^2} + k^2 u =0 $

+ 进行分离变量：

$ u(r,\theta,z) = R(r)\Theta(\theta)Z(z) $

+ 得到三个方程，其中两个对应本征值问题：

$ \frac{1}{r} \frac{d}{dr} (r\frac{dR}{dr}) + (k^2 -\lambda -\frac{\mu}{r^2})R = 0 $

$ \frac{d^2 \Theta}{d\theta^2} + \mu \Theta =0 $

$ \frac{d^2 Z}{dz^2} + \lambda Z =0 $

#### Bessel方程
+ 对于径向方程：

$ \boxed{\frac{1}{r} \frac{d}{dr} (r\frac{dR}{dr}) + (k^2 -\lambda -\frac{\mu}{r^2})R = 0} $

+ 作变量代换：

$ x = \sqrt{k^2-\lambda}r;\; y(x) = R(r); \; \mu = \nu^2 $

        * 前提要求$ k^2 - \lambda \neq 0 $。
    - 方程变为：

$ \boxed{\frac{1}{x} \frac{d}{dx} (r\frac{dy}{dx}) + (1 -\frac{\nu^2}{x^2})y(x) = 0} $

    - 这被称为$ \nu $<font style="color:#DF2A3F;">阶</font>$ \mathrm{Bessel} $<font style="color:#DF2A3F;">方程</font>。

#### Bessel方程的解
+ $ \nu \notin \Z $时，$ \mathrm{Bessel} $方程的两个正则解都不含有对数函数，二者线性无关：

$ J_{±\nu}(x) = \sum_{k=0}^{\infin} \frac{(-1)^k}{k!\Gamma(k±\nu+1)}(\frac{x}{2})^{2k±\nu} $

    - 这辈称为$ ±\nu $<font style="color:#DF2A3F;">阶</font>$ \mathrm{Bessel} $<font style="color:#DF2A3F;">函数</font>。
    - 关于常微分方程级数解法，可以参考：[【未完】常微分方程级数解 2025.06.25](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/nsycge/cku9ucwbunnrqnq1)
+ 当$ \nu \in \Z $时，第一节$ J_n(x) $仍然是$ \mathrm{Bessel} $函数，但是$ J_{-n}(x) $与之线性相关：

$ J_{-n}(x) = (-1)^n J_n(x) $

    - 因此第二解一定含有对数函数项，这个解被称为$ \mathrm{Neumann} $函数。

### Bessel函数
#### Bessel函数的定义
+ $ \nu $阶$ \mathrm{Bessel} $函数的定义为：

$ \boxed{J_{\nu}(x) = \sum_{k=0}^{\infin} \frac{(-1)^k}{k!\Gamma(k+\nu+1)}(\frac{x}{2})^{2k+\nu}} $

#### 整数阶Bessel函数的性质
1. 负整数阶$ \mathrm{Bessel} $函数

$ J_{-n}(x) = (-1)^n J_n(x) $

2. 奇偶性

$ J_n(-x) = (-1)^n J_n(x) $

3. 生成函数：整数阶$ \mathrm{Bessel} $函数的生成函数

$ \exp [\frac{x}{2}(t-\frac{1}{t})] = \sum_{n=-\infin}^{+\infin} J_n(x)t^n $

    - 注意求和范围是$ -\infin $到$ +\infin $。这是一个$ \mathrm{Laurent} $展开。
4. $ \mathrm{Bessel} $函数的积分表示。

$ J_n(x) = \frac{1}{2\pi}\int_{-\pi}^{\pi} \cos (x\sin\theta - n\theta)d\theta $

#### 任意阶Bessel函数的性质
1. 递推关系

$ \frac{d}{dx}[x^{\nu} J_\nu(x)] = x^\nu J_{\nu-1}(x) $

$ \frac{d}{dx}[x^{-\nu} J_\nu(x)] = -x^{-\nu}J_{\nu+1}(x) $

+ 根据这两个递推关系，还可以得到：

$ J_{\nu-1}(x)-J_{\nu+1}(x) = 2J'_\nu(x) $

$ J_{\nu-1}(x)+J_{\nu+1}(x) = \frac{2\nu}{x}J_\nu(x) $

    - 这说明，只需要知道$ J_0(x),J_1(x) $，任意整数阶$ \mathrm{Bessel} $函数$ J_n(0) $及其导数$ J_n'(0) $都可以用他们进行表示。
    - 这有一定的应用意义，因为$ \mathrm{\mathrm{Bessel}} $函数是一个无穷级数。
2. 渐进行为：

$ J_\nu (x) \sim \sqrt{\frac{2}{\pi x}} \cos (x-\frac{\nu \pi}{2} -\frac{\pi}{4}),\;x\rightarrow \infin $

$ J_\nu (x) = \frac{1}{\Gamma (\nu+1)}(\frac{x}{2})^\nu + O(x^{\nu+2}),\; x \rightarrow 0 $

3. $ \mathrm{Wronski} $行列式
4. 零点：**实数阶**$ \mathrm{Bessel} $函数的零点，当$ \nu>1 $时：
    1. $ J_\nu $有无穷多个零点。
    2. 它们全是实数，
    3. 对称地分布在实轴上。

### Neumann函数
#### Neumann函数的定义
+ $ \nu $阶$ \mathrm{Neumann} $函数的定义为：

$ N_\nu(x) = \frac{\cos \nu \pi J_\nu(x)-J_{-\nu}(x)}{\sin \nu \pi} $

+ 整数阶的$ \mathrm{Neumann} $函数的定义为：

$ N_n(x) = \lim_{\nu \rightarrow n}\frac{\cos \nu \pi J_\nu(x)-J_{-\nu}(x)}{\sin \nu \pi} $

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1761549529893-f57b1a2b-ed82-4eb5-9d7f-f7bc79a51bfb.png" width="401" title="" crop="0,0,1,1" id="ucdc9a222" class="ne-image">

#### Neumann函数的渐进行为
+ $ x \rightarrow 0 $时：

$ \mathrm{Re} \nu >0, N_\nu \sim -\frac{\Gamma(\nu)}{\pi} (\frac{x}{2})^{-\nu} $

$ \nu = 0, N_0(x) \sim \frac{2}{\pi} \ln \frac{x}{2} $

    - 可见$ x=0 $是$ \mathrm{Neumann} $函数的奇点。
+ $ x\rightarrow \infin $时：

$ N_\nu (x) \sim \sqrt{\frac{2}{\pi x}} \sin(x-\frac{\nu \pi}{2} - \frac{\pi}{4})，\; |\arg x|<\pi $

#### Neumann函数的递推关系
$ \frac{d}{dx}[x^{\nu} N_\nu(x)] = x^\nu N_{\nu-1}(x) $

$ \frac{d}{dx}[x^{-\nu} N_\nu(x)] = -x^{-\nu}N_{\nu+1}(x) $

### 柱函数
#### 柱函数的定义
+ 凡是满足下面的函数$ C_\nu $都是柱函数：

$ \frac{d}{dx}[x^{\nu} C_\nu(x)] = x^\nu C_{\nu-1}(x) $

$ \frac{d}{dx}[x^{-\nu} C_\nu(x)] = -x^{-\nu}C_{\nu+1}(x) $

+ $ \mathrm{Bessel} $函数和$ \mathrm{Neumann} $函数都是柱函数。

#### 柱函数的性质
+ 柱函数一定是$ \mathrm{Bessel} $方程的解。

#### Bessel函数的模方
+ $ \mathrm{Bessel} $函数的模方计算通常指计算下面的积分：

$ \int x J^2_\nu (x)dx $

    - 其中$ x $是权函数。
+ 最终积分结果为：

$ \int xJ_\nu^2(x)dx = \frac{1}{2}(xJ_\nu')^2 + \frac{1}{2}(x^2-\nu^2)J_\nu^2 + C $

    - 将$ J_\nu' $消掉，还可以得到：

$ \int xJ_\nu^2 dx = \frac{x^2}{2}(J_\nu^2 + J_{\nu+1}^2)-\nu x J_\nu J_{\nu+1}+C $

### Bessel方程的本征值问题
+ 具体习题可以参考ppt，此处只说重点。
    - 最后结果最好都能化为$ J_0(x),J_1(x) $的形式。
    - 经常使用$ J_n(x) $的零点与边界条件相互匹配，确定本征函数组$ J_n(\frac{\mu_{ni}}{a}r) $。
        * 其中$ n $是$ \mathrm{Bessel} $函数的阶数，由方程的次数决定。
        * $ i $是$ J_n $的第$ i $个零点，一般来说$ J_n(\frac{\mu_{ni}}{a}r) $和$ J_n(\frac{\mu_{nj}}{a}r) $相互正交。
    - 注意$ \mu_{ni} $是$ J_n $的第$ i $个零点，而不是其他$ J_m (m\neq n) $的零点。
    - 经常使用$ J_n(\mu_{ni})=0 $来简化分部积分的结果。

### Hankel函数
#### Hankel函数的定义
+ 第一类和第二类$ \mathrm{Hankel} $函数的定义为：

$ H_\nu^{(1)} = J_\nu (x) + iN_\nu (x) $

$ H_\nu^{(2)} = J_\nu (x) - iN_\nu (x) $

+ 第一类和第二类$ \mathrm{Hankel} $函数统称为第三类柱函数。

#### Hankel函数的渐进行为
+ 第一类和第二类$ \mathrm{Hankel} $函数在$ x\rightarrow \infin $时渐进行为：

$ H_\nu^{(1)}(x) \sim \sqrt{\frac{2}{\pi x}} \exp[i(x-\frac{\nu \pi}{2} - \frac{\pi}{4})] $

$ H_\nu^{(2)}(x) \sim \sqrt{\frac{2}{\pi x}} \exp[-i(x-\frac{\nu \pi}{2} - \frac{\pi}{4})] $

### 球Bessel方程
#### 球Bessel方程
+ **球坐标系**下的$ \mathrm{Helmholtz} $方程：

$ \nabla^2 u + k^2 u = 0 $

+ 分离变量后得到径向方程为：

$ \frac{1}{r^2} \frac{d}{dr} (r^2 \frac{dR}{dr})+ [k^2 - \frac{l(l+1)}{r^2}]R = 0 $

+ 作变量代换$ x=kr,y(x) = R(r) $得到方程：

$ \frac{1}{x^2} \frac{d}{dx} (x^2 \frac{dy}{dx}) +[1-\frac{l(l+1)}{x^2}]y=0 $

    - 这就是**<font style="color:#DF2A3F;">球</font>**$ \mathrm{Bessel} $**<font style="color:#DF2A3F;">方程</font>**。

#### 球Bessel方程转化为Bessel方程
+ 作变换$ y(x) = \frac{v(x)}{\sqrt{x}} $，得到$ \mathrm{Bessel} $方程的形式：

$ \frac{1}{x}\frac{d}{dx} (x\frac{dv}{dx}) +[1-\frac{(l+1/2)^2}{x^2}]v = 0 $

    - 这是$ l+1/2 $阶的$ \mathrm{Bessel} $函数，其解为：

$ J_{l+1/2}(x) $

$ J_{-l-1/2}(x) = (-1)^{l+1}N_{l+1/2}(x) $

#### 球Bessel函数和球Neumann函数
+ $ l $阶球$ \mathrm{Bessel} $方程的两个解可以取为：
    - $ l $阶球$ \mathrm{Bessel} $函数和$ l $阶球$ \mathrm{Neumann} $函数：

$ j_l(x) = \sqrt\frac{\pi}{2x}J_{l+1/2}(x) $

$ n_l(x) = \sqrt\frac{\pi}{2x}N_{l+1/2}(x) $

    - 这样的形式更加常用。
+ 球$ \mathrm{Bessel} $函数和球$ \mathrm{Neumann} $函数的递推公式为：

$ \frac{d}{dx}(x^{l+1} j_l) = x^{l+1}j_{l-1} $

$ \frac{d}{dx}(x^{-l} j_l) = x^{-l}j_{l+1} $

    - 由此可以推出球$ \mathrm{Bessel} $函数和球$ \mathrm{Neumann} $函数的一般表达式为：

$ j_l(x) = x^l (-\frac{1}{x}\frac{d}{dx})^l \{\frac{\sin x}{x}\} $

$ n_l(x) = x^l (-\frac{1}{x}\frac{d}{dx})^l \{-\frac{\cos x}{x}\} $

+ $ x\rightarrow \infin $时渐进行为为：

$ j_l (x)\sim \frac{1}{x} \sin (x-\frac{l\pi}{2}) $

$ n_l (x)\sim -\frac{1}{x} \cos (x-\frac{l\pi}{2}) $

#### 球Hankel函数
+ 球$ \mathrm{Hankel} $函数的定义为：

$ h_l^{(1)} = j_l(x)+in_l(x) $

$ h_l^{(2)} = j_l(x)-in_l(x) $

+ 球$ \mathrm{Hankel} $函数$ x\rightarrow \infin $时渐进行为为：

$ h_l^{(1)}\sim \frac{1}{ix} e^{i(x-\frac{l\pi}{2})} $

$ h_l^{(2)}\sim -\frac{1}{ix} e^{-i(x-\frac{l\pi}{2})} $
