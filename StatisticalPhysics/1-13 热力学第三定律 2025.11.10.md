### 能斯特定理
+ 能斯特定理：体系的**熵在等温过程中的改变**随绝对温度趋于零而趋于零。

$ \lim_{T\rightarrow 0} (\Delta S)_T = 0 $

+ 将熵看成温度$ T $以及另外一个参量$ y $的函数：

$ S = S(T,y) $

    - 对状态参量从$ y_1 $变为$ y_2 $过程，熵变为：

$ (\Delta S)_T = S(T,y_2)-S(T,y_1) $

    - 根据能斯特定理：

$  S(T=0,y_2) = S(T=0,y_1) $

### 热力学第三定律
+ 热三定律：不可能通过**有限步骤**使物体冷却到绝对零度/绝对零度不可能达到。
    - 注意此处说的是有限步骤不可达，<font style="color:#DF2A3F;">并没有说不能无限趋近于绝对零度</font>。

### 能斯特定理的推论（对于P-V-T系统）
#### 压强系数
+ 将熵看成温度和体积的函数：

$ S = S(T,V) $

+ 考虑等温过程中体积的微小变化中的熵变：

$ (\Delta S)_T = (\frac{\partial S}{\partial V})_T \Delta V =  (\frac{\partial p}{\partial T})_V \Delta V $

    - 第一个等号用到了等温过程的条件。
    - 第二个等号用到了麦克斯韦关系。
+ 让温度趋于零，得到：

$ (\Delta S)_T \rightarrow 0  $

$ (\frac{\partial p}{\partial T})_V \rightarrow 0 $

+ 可以推出**压强系数在温度趋于零时趋于零**：

$ \beta = \frac{1}{p}(\frac{\partial p}{\partial T})_V \rightarrow 0,\; T \rightarrow 0 $

#### 等压膨胀系数
+ 将熵看成温度和压强的函数：

$ S = S(T,p) $

+ 考虑等温过程中压强的微小变化中的熵变：

$ (\Delta S)_T = (\frac{\partial S}{\partial p})_T \Delta p =  -(\frac{\partial V}{\partial T})_p \Delta p $

    - 第一个等号用到了等温过程的条件。
    - 第二个等号用到了麦克斯韦关系。
+ 让温度趋于零，得到：

$ (\Delta S)_T \rightarrow 0  $

$ (\frac{\partial V}{\partial T})_p \rightarrow 0 $

+ 可以推出**等压膨胀系数在温度趋于零时趋于零**：

$ \alpha = \frac{1}{V}(\frac{\partial V}{\partial T})_p \rightarrow 0,\; T \rightarrow 0 $

#### 等压热容与等体热容
+ 考虑等压热容与等体热容的差值：

$ C_p - C_V = T (\frac{\partial p}{\partial T})_V (\frac{\partial V}{\partial T})_p $

    - 根据前文推导：

$ (\frac{\partial p}{\partial T})_V \rightarrow 0, \; T\rightarrow0 $

$ (\frac{\partial V}{\partial T})_p \rightarrow 0, \; T\rightarrow0 $

+ 可见：

$ \lim_{T\rightarrow 0} C_p =\lim_{T\rightarrow 0} C_V $

### 绝对熵
+ 体系的熵在等温过程中的改变随绝对温度趋于零而趋于零，如果定义此时熵的极限值为$ 0 $，则成为**普朗克熵**。

$ S_0 = 0 $

+ 热力学第三定律的另外一种表述：系统熵随着绝对温度趋于零而趋于零。

$ \lim_{T\rightarrow 0}S_0 = 0 $
