### 光子气体
+ 圆频率为$ \omega $，波矢为$ \vec{k} $的光子：
    - 能量$ \epsilon = \hbar \omega $
    - 动量$ \vec{p} = \hbar \vec{k} $
+ 光子是极端相对论性气体，且静质量为零，因此有：

$ p = \frac{\epsilon}{c} = \frac{\hbar \omega}{c} = \frac{h\omega}{2\pi c} $

+ 光子数不守恒，所以化学势规定为$ \mu = 0 $。
    - 光子是玻色子，量子态平均占据数函数为：

$ f(\epsilon) = \frac{1}{e^{\beta \epsilon}-1} $

+ 此外，光子有两个自旋方向，所以自旋简并度为$ 2 $。

### 光子的能量密度——普朗克公式
#### 态密度函数
+ 首先在动量空间计算量子态的数目：

$ \int g(p)dp = 2\int \frac{dxdydz}{h^3} dp_xdp_ydp_z = 2\frac{V}{h^3} 4\pi \int p^2dp  $

    - 即态密度函数为：

$ g(p)dp = \frac{8\pi V}{h^3}p^2dp $

+ 代入$ p $和$ \omega $的关系式，得到：

$ g(\omega)d\omega = \frac{V}{\pi^2c^3}\omega^2d\omega $

#### 单位体积的能量密度
+ 结合平均占据数，得到$ \omega \sim \omega + d\omega $范围内的辐射场的能量为：

$ U(\omega,T)d\omega = \hbar \omega f(\omega)g(\omega)d\omega =\hbar \omega\cdot \frac{1}{e^{\beta \epsilon}-1}\frac{V}{\pi^2c^3}\omega^2d\omega = 
 \frac{V}{\pi^2c^3} \frac{\hbar \omega^3}{e^{\hbar \omega/k_B T}-1}d\omega $

    - 单位体积的能量密度为：

$ u(\omega,T) = \frac{U(\omega,T)}{V} = \frac{1}{\pi^2c^3} \frac{\hbar \omega^3}{e^{\hbar \omega/k_B T}-1} $

### 普朗克公式与近似
#### 低频近似：瑞利金斯公式
+ 如果只考虑低频，即$ \frac{\hbar \omega}{k_B T}<<1 $时：

$ e^{\hbar \omega /k_B T} \approx 1+ \frac{\hbar \omega}{k_B T} $

    - 单位体积的能量密度可以近似为：

$ u(\omega,T) \approx \frac{1}{\pi^2c^3} \frac{\hbar \omega^3}{1+ \frac{\hbar \omega}{k_B T}-1} = \frac{\omega^2}{\pi^2c^3}  k_B T $

    - 这就是瑞利金斯公式。

#### 高频近似：维恩公式
+ 如果只考虑低频，即$ \frac{\hbar \omega}{k_B T}>>1 $时：

$ e^{\hbar \omega /k_B T}>>1 $

    - 单位体积的能量密度可以近似为：

$ u(\omega,T) \approx \frac{1}{\pi^2c^3} \frac{\hbar \omega^3}{e^{\hbar \omega /k_B T}} =\frac{1}{\pi^2c^3} \hbar \omega^3 \exp\left(-\frac{\hbar \omega}{k_B T}\right) $

    - 这就是维恩公式。

#### 平衡辐射内能
+ 对于圆频率进行积分：

$ U(T) = \int U(\omega,T)d\omega = \int_0^{\infin} 
 \frac{V}{\pi^2c^3} \frac{\hbar \omega^3}{e^{\hbar \omega/k_B T}-1}d\omega $

+ 做数变换$ x = \hbar \omega/k_B T $：

$ RHS = \frac{V\hbar}{\pi^2\omega^3} \left(\frac{k_B T}{\hbar}\right)^4\int_0^\infin \frac{x^3}{e^x-1}dx $

    - 查阅积分表：

$ \int_0^\infin \frac{x^3}{e^x-1}dx = \frac{\pi^4}{15} $

+ 代入数值得：

$ U(T) = \frac{\pi^2 k_B^4}{15c^3\hbar^3} VT^4 $

### 光子气体的巨配分函数及热力学函数
#### 光子气体的巨配分函数
+ 根据[玻色系统的巨配分函数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/iq54ttgxzfy5g8a9#E0vg4)可知，光子的巨配分函数$ \Xi $满足：

$ \ln \Xi = - \sum_l \omega_l \ln (1-e^{-\beta \epsilon_l}) $

    - 在能级准连续假设下，求和化为积分，能级简并度换为态密度函数：

$ RHS = -\frac{V}{\pi^2c^3}\int_0^{\infin}\omega^2\ln (1-e^{-\beta \epsilon})d\omega $

    - 做数变换$ x = \hbar \omega/k_B T $：

$ RHS = -\frac{V}{\pi^2c^3} \left( \frac{k_B T}{\hbar}\right)^3 \int_0^\infin x^2 \ln (1-e^{-x})dx $

+ 下面处理积分部分，进行分部积分：

$ \int_0^\infin x^2 \ln (1-e^{-x})dx = \left[ \frac{x^3}{3} \ln (1-e^{-x})\right]_0^{\infin} -\frac{1}{3} \int_0^{\infin} \frac{x^3dx}{e^x-1} = 0-\frac{1}{3}\cdot \frac{\pi^4}{15} = -\frac{\pi^4}{45} $

    - 代入得到：

$ \ln \Xi = \frac{\pi^2V}{45c^3}\left( \frac{k_B T}{\hbar}\right)^3 = \frac{\pi^2V}{45c^3}\left( \frac{1}{h \beta}\right)^3  $

#### 光子气体的热力学函数
+ 内能：

$ U = -\frac{\partial}{\partial \beta} \ln \Xi = \frac{\pi^2 V}{15c^3} \frac{1}{h^3} \frac{1}{\beta^4} =  \frac{\pi^2 V}{15c^3h^3} (k_B T)^4 $

+ 压强：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V} \ln \Xi = \frac{\pi^2}{45c^3} \frac{1}{h^3} \frac{1}{\beta^4} = \frac{\pi^2}{45c^3h^3} (k_B T)^4 $

    - 此时这也就验证了相对论性理想气体有：

$ p = \frac{1}{3}\frac{U}{V} = \frac{1}{3}u $

+ 熵：

$ S = k_B (\ln \Xi - \beta \frac{\partial}{\partial \beta} \ln \Xi) = k_B (\ln \Xi+\beta U) = \frac{4\pi^2 V}{45c^3h^3}k_B^3 T^4 $

+ 平均光子数：
    - 因为此时化学势为零，因此$ \alpha\equiv 0 $，改用态密度函数乘以平均占据数积分得到：

$ \bar{N} = \int_0^{\infin} \frac{V}{\pi^2c^3} \frac{ \omega^2}{e^{\hbar \omega/k_B T}-1}d\omega $

    - 仍然做数变换$ x = \hbar \omega/k_B T $：

$ RHS = \frac{V}{\pi^2 c^3}\left(\frac{k_B T}{\hbar}\right)^3 \int_0^{\infin}\frac{x^2}{e^x-1}dx $

    - 其中：

$ \int_0^{\infin}\frac{x^2}{e^x-1}dx = 2 \zeta(3) \approx 2 \times 1.202 $

        * 其中$ \zeta $为$ \mathrm{Riemann-zeta} $函数，代入得：

$ \bar{N} = 16 \pi \zeta(3) V \left(\frac{k_B T}{hc}\right)^3 \sim VT^3 $
