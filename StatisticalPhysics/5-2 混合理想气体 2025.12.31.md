### 混合理想气体的配分函数
#### 情景设定
+ 假设存在$ k $个组元，$ j = 1,2,3,...,k $。
    - 其中第$ j $个组元的粒子数为$ N_j $。

#### 混合理想气体配分函数
+ 首先按照定义写出混合气体的配分函数：

$ Z = \sum_S e^{-\beta E_S} $

    - 求和，可以按照每种组分分别求和；能量也可以拆成每种组分求和：

$ RHS = \sum_{S_1}\cdots \sum_{S_j}\cdots \sum_{S_k}e^{-\beta E_1} \cdots e^{-\beta E_j}\cdots e^{-\beta E_k} $

    - 可以进一步化为：

$ Z = \prod_{j=1}^{k} Z_{j} $

        * 其中$ Z_j $是第$ j $中组元的配分函数。
+ 对于$ Z_j $：

$ Z_j = \sum_{S_j}e^{-\beta E_j} = \sum_{S_{j1}}\cdots\sum_{S_{ji}}\cdots\sum_{S_{jN_j}} e^{-\beta \epsilon_{j1}}\cdots e^{-\beta \epsilon_{ji}}\cdots e^{-\beta \epsilon_{jN_j}} = z_j^{N_j} $

    - 其中$ z_j $是第$ j $中组元单个分子的子系配分函数。
+ 在经典极限下，考虑全同粒子：

$ Z_j = \frac{z_j^{N_j}}{N_j!} $

### 再论吉布斯佯谬
+ 全同粒子组成的理想气体的熵为：

$ S_{全同} = \frac{3}{2} Nk_B \ln T + Nk_B \ln \frac{V}{N} + \frac{3}{2} Nk_B (\frac{5}{3}+ \ln \frac{2\pi m k_B}{h^2}) $

+ 可以进行整理：

$ RHS = \frac{3}{2} Nk_B \ln T + Nk_B \ln \frac{V}{N} + \frac{3}{2} Nk_B (\frac{5}{3}-2 \ln \lambda_T-2\ln T) $

    - 注意对数的运算规则：

$ S_{全同} = Nk_B\left[ \frac{5}{2} + \ln \left(\frac{V}{N\lambda_T^3}\right)\right] $

#### 全同粒子假设下，不同气体混合
+ 初态，组元$ A $和$ B $的熵为：

$ S_{A}^i = N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A}{N_A\lambda_{AT}^3}\right)\right] $

$ S_{B}^i = N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_B}{N_B\lambda_{BT}^3}\right)\right] $

    - 总熵为：

$ S^i = S_A^i +S_B^i =N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A}{N_A\lambda_{AT}^3}\right)\right]+ N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_B}{N_B\lambda_{BT}^3}\right)\right] $

+ 末态，组元$ A $和$ B $的熵为：

$ S_{A}^f = N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A+V_B}{N_A\lambda_{AT}^3}\right)\right] $

$ S_{B}^f = N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_A+V_B}{N_B\lambda_{BT}^3}\right)\right] $

$ S^f = S_A^f +S_B^f =N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A+V_B}{N_A\lambda_{AT}^3}\right)\right]+ N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_A+V_B}{N_B\lambda_{BT}^3}\right)\right] $

+ 等压混合$ \frac{V_A}{N_A} = \frac{V_B}{N_B} $整个过程的熵变为：

$ \Delta S = S^f -S^i = -Nk_B (x_A\ln x_A+x_B\ln x_B) $

    - 其中$ N $是总粒子数，$ x_A = \frac{N_A}{N},x_B = \frac{N_B}{N} $。

#### 全同粒子假设下，同种气体混合
+ 初态，组元$ A $和$ B $（其实是同种，分别处于两个不同的容器中）的熵为：

$ S_{A}^i = N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A}{N_A\lambda_{AT}^3}\right)\right] $

$ S_{B}^i = N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_B}{N_B\lambda_{BT}^3}\right)\right] $

    - 总熵为：

$ S^i = S_A^i +S_B^i =N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A}{N_A\lambda_{AT}^3}\right)\right]+ N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_B}{N_B\lambda_{BT}^3}\right)\right] $

+ 因为是同种气体混合，所以末态熵为：

$ S^{f} = (N_A+N_B)k_B\left[ \frac{5}{2} + \ln \frac{V_A+V_B}{(N_A+N_B)\lambda_{AT}^3}\right] $

    - 如果是等压混合$ \frac{V_A}{N_A} = \frac{V_B}{N_B} = \frac{V_A+V_B}{N_A+N_B} $，那么可以拆成：

$ S^{f} = N_Ak_B\left[ \frac{5}{2} + \ln \left(\frac{V_A}{N_A\lambda_{AT}^3}\right)\right] +N_Bk_B\left[ \frac{5}{2} + \ln \left(\frac{V_B}{N_B\lambda_{BT}^3}\right)\right] = S_A^i +S_B^i= S^i $

    - 即：

$ \Delta S = S^f -S^i = 0 $

+ **<font style="color:#DF2A3F;">综上，在全同粒子假设下，同种理想气体混合熵变为零，不同种理想气体熵变为正。</font>**
    - **<font style="color:#DF2A3F;">注意表达式推到要求两瓶气体压强相等，同时假设最终总容器的体积等于混合前两个容器体积之和。</font>**
