### 模型简介
#### 电子自旋与体系能量
+ 电子自旋产生磁矩，一块固体内有很多个自旋。
+ 对于自旋分布为$ \{\sigma_i\} $的情形，体系的总能量为：

$ E\{\sigma_i\} = -J\sum_{<i,j>} \sigma_i \sigma_j - \mu B \sum_i \sigma_i $

+ 其中$ \sigma_i $的取值可以为$ +1,-1 $，分别对应自旋向上和自旋向下的情形。
+ $ J $是耦合系数。
    - 对于铁磁系统$ J>0 $，即自旋按照相同的方向排列有利于能量降低。
+ $ \sum_{<i,j>} $代表临近求和，也就是说只有相邻的自旋才会对本地的自旋产生力的作用。

#### 配分函数
+ 选用正则系综，配分函数为：

$ Z = \sum_{\{\sigma_i\}} e^{-\beta E\{\sigma_i\}} = \sum_{\sigma_1 = \pm 1}\sum_{\sigma_2 = \pm 1}\cdots \sum_{\sigma_N = \pm 1} e^{\beta J\sum_{<i,j>} \sigma_i \sigma_j + \beta\mu B \sum_i \sigma_i} $

    - 这个配分函数非常难求解。

### 平均场近似
+ 为了解决配分函数难以求解的问题，使用**平均场近似**：
+ 首先将能量进行形式的改写：

$ E\{\sigma_i\} = -J\sum_{<i,j>} \sigma_i \sigma_j - \mu B \sum_i \sigma_i
 = - \sum_i \mu \sigma_i (B + \frac{J}{\mu} \sum_{<i,j>} \sigma_j) $

+ 再进行平均场近似，用配位数$ n $乘以平均自旋$ \bar{\sigma} $代替自旋临近求和：

$ \sum_{<i,j>} \sigma_j \approx n\bar{\sigma} $



+ 配位数指的是一个粒子附近直接相邻的粒子数。
    - 对于一维来说$ n=2 $，二维$ n=4 $，三维$ n=6 $。
+ 平均自旋是对整个体系指定的，而非仅仅对于被研究粒子的周边。所以这个才被称为平均场近似。
    - 如果只是对周边，则并没有近似。



+ 此时能量可以近似为：

$ E\{\sigma_i\} = - \sum_i \mu \sigma_i (B + \frac{J}{\mu} n\bar{\sigma}) = \sum_i \mu \sigma_i \bar{B} $

    - 观察形式可以引入等效磁场：

$ \bar{B} = \bar{B}_i = B + \frac{1}{\mu} Jn\bar{\sigma} $

### 配分函数及磁化强度
#### 配分函数的改写
+ 进行平均场近似后，可以对配分函数进行改写：

$ Z = \sum_{\{\sigma_i\}} e^{-\beta E\{\sigma_i\}} = \sum_{\sigma_1 = \pm 1}\sum_{\sigma_2 = \pm 1}\cdots \sum_{\sigma_N = \pm 1} e^{\beta \sum_i \mu \sigma_i \bar{B}} = \sum_{\sigma_1 = \pm 1}\sum_{\sigma_2 = \pm 1}\cdots \sum_{\sigma_N = \pm 1} \prod_{i=1}^{N}e^{\beta \mu \sigma_i \bar{B}} $

    - 换一种遍历方式：

$ Z = \prod_{i=1}^{N} \sum_{\sigma_i = \pm 1} e^{\beta \mu \sigma_i \bar{B}} = z^N $

+ 其中$ z $为子系配分函数，具体表达式为：

$ z = e^{\beta \mu \bar B}+ e^{+\beta \mu \bar B} = 2\cosh(\beta \mu \bar{B}) =
2\cosh \left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right) $

#### 自由能与磁化强度
+ 首先根据正则系综公式计算自由能：

$ F = -k_B T \ln Z = -Nk_B T\left[\ln 2+ \ln \cosh\left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right)\right] $

+ 计算偏导数：

$ M = -\frac{\partial F}{\partial B} = N k_B T \frac{\sinh 
 \left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right)}{\cosh 
 \left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right)}  \frac{\mu}{k_B T}= N \mu \tanh 
 \left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right)
 $

$ = N \mu \frac{e^{\beta \mu \bar{B}}-e^{-\beta \mu \bar{B}}}{e^{\beta \mu \bar{B}}+e^{-\beta \mu \bar{B}}}= N\mu \frac{\sum \sigma e^{\beta \mu \bar{B}}}{z} $

    - 注意到分数正是自旋平均值的定义式，所以有：

$ M =  N \mu \tanh 
 \left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right)
 = N \mu \bar{\sigma} $

### 自发磁化
#### 自发磁化
+ 根据前面的论述，我们可以得出：

$ \tanh 
 \left( \frac{\mu B}{k_BT} +\frac{nJ \bar{\sigma}}{k_B T}\right)
 = \bar{\sigma} $

+ 在不存在外磁场$ B=0 $时有：

$ \tanh 
 \left(\frac{nJ \bar{\sigma}}{k_B T}\right)
 = \bar{\sigma} $

+ 根据函数性质可知：
    - $ \frac{nJ}{k_B T} <1 $时只有一个解，$ \bar{\sigma} = 0 $，磁化强度$ M = N\mu \bar{\sigma} = 0 $。
    - $ \frac{nJ}{k_B T} >1 $时只有三个解
        * 其中$ \bar{\sigma} = 0 $的解对应自由能极大，舍弃。
        * $ \bar{\sigma} = \pm \bar{\sigma}_0 $对应自由能极小，磁化强度$ M = \pm N\mu \bar{\sigma}_0 $。这时处于铁磁态。

#### 临界温度
+ 临界温度满足：

$ \frac{nJ}{k_B T_c} = 1 \Rightarrow T_c = \frac{nJ}{k_B} $

+ 一维晶格，平均场近似$ T_c = \frac{2J}{k_B} $，严格解$ T_c = 0 $。
+ 二维晶格，平均场近似$ T_c = \frac{4J}{k_B} $，严格解$ T_c = \frac{2.3J}{k_B} $。
+ 二维晶格，平均场近似$ T_c = \frac{6J}{k_B} $，数值解$ T_c = \frac{6J}{k_B} $。
