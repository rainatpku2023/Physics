### 玻色系统热力学量的统计表达式
#### 粒子数、内能与广义力
+ 引入[玻色系统的巨配分函数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/iq54ttgxzfy5g8a9#E0vg4)：

$ \Xi_{BE} =\prod_l[1-e^{-(\alpha+\beta \epsilon_l)}] ^{-\omega_l} $

    - 其对数为：

$ \ln \Xi_{BE} =-\sum_l\omega_l\ln [1-e^{-(\alpha+\beta \epsilon_l)}] $

+ 根据[巨正则系综热力学量的统计表达式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/gmqwy7tivky6a1pa#bFWmw)，可以计算平均粒子数$ \bar{N} $，内能$ U $，广义力$ Y $：

$ \bar{N} = -\frac{\partial}{\partial \alpha} \ln \Xi_{BE} = \sum_l \omega_l \frac{e^{-\alpha-\beta \epsilon_l}}{1-e^{-\alpha-\beta \epsilon_l}} = \sum_l \frac{\omega_l}{e^{\alpha+\beta\epsilon_l}-1} $

$ U = \bar{E} = -\frac{\partial}{\partial \beta} \ln \Xi_{BE} = \sum_l \epsilon_l\omega_l \frac{e^{-\alpha-\beta \epsilon_l}}{1-e^{-\alpha-\beta \epsilon_l}} = \sum_l \frac{\omega_l \epsilon_l}{e^{\alpha+\beta\epsilon_l}-1} $

$ Y = -\frac{1}{\beta} \frac{\partial}{\partial y}\ln \Xi_{BE} 
= \sum_l \omega_l \frac{\partial \epsilon_l}{\partial y} \frac{e^{-\alpha-\beta \epsilon_l}}{1-e^{-\alpha-\beta \epsilon_l}} = \sum_l  \frac{\partial \epsilon_l}{\partial y} \frac{\omega_l }{e^{\alpha+\beta\epsilon_l}-1} $

    - 特别地，对于压强$ p $：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V}\ln \Xi_{BE} 
= -\sum_l \omega_l \frac{\partial \epsilon_l}{\partial V} \frac{e^{-\alpha-\beta \epsilon_l}}{1-e^{-\alpha-\beta \epsilon_l}} = -\sum_l  \frac{\partial \epsilon_l}{\partial V} \frac{\omega_l }{e^{\alpha+\beta\epsilon_l}-1} $

#### 熵、亥姆霍兹自由能、吉布斯自由焓、巨势
$ S = k_B (\ln \Xi_{BE} - \alpha \frac{\partial }{\partial \alpha}\ln \Xi_{BE}- \beta \frac{\partial }{\partial \beta}\ln \Xi_{BE}) $

$ F = U-TS = -k_B T (\ln \Xi_{BE} - \alpha \frac{\partial}{\partial \alpha }\Xi_{BE}) $

$ G= \bar{N}\mu = -\bar{N}k_B T \alpha= k_B T \alpha \frac{\partial}{\partial \alpha} \ln \Xi_{BE} $

$ \Psi = F - \bar{N}\mu = -k_B T \ln \Xi_{BE} $

#### 熵的统计意义
+ 在巨正则系综一章，我们推出过：

$ S = k_B (\ln\Xi + \alpha \bar{N} +\beta U) $

+ 在[玻色系统最概然分布](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/qr3cuico1i1olwne#r2tEY)一章，我们推出过：

$ \ln \Omega_{BE} = \sum_l[(\omega_l + a_l)\ln(\omega_l + a_l) -a_l \ln a_l -\omega_l\ln\omega_l]  $

$ = \sum_l [\omega_l \ln(\frac{\omega_l +a_l}{\omega_l})+a_l \ln(\frac{\omega_l +a_l}{a_l})] $

    - 再考虑到：

$ a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}-1} $

    - 可知：

$ \frac{\omega_l +a_l}{\omega_l} = \frac{e^{\alpha + \beta \epsilon_l}}{ e^{\alpha + \beta \epsilon_l}-1} = \frac{1}{1-e^{-\alpha - \beta \epsilon_l}} $

$ \frac{\omega_l +a_l}{a_l} = e^{\alpha+\beta \epsilon_l} $

+ 代入巨配分函数表达式：

$ \ln \Omega_{BE} = \sum_l[-\omega_l \ln (1-e^{-\alpha - \beta \epsilon _l })+a_l (\alpha + \beta \epsilon_l)]  = \ln\Xi_{BE} + \alpha \bar{N} +\beta U $

+ 因此，在玻色系统中：

$ S = k_B \ln \Omega_{BE} $



### 费米系统热力学量的统计表达式
#### 粒子数、内能与广义力
+ 引入[费米系统的巨配分函数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/iq54ttgxzfy5g8a9#M8srd)：

$ \Xi_{FD} =\prod_l[1+e^{-(\alpha+\beta \epsilon_l)}] ^{\omega_l} $

    - 其对数为：

$ \ln \Xi_{FD} =\sum_l\omega_l\ln [1+e^{-(\alpha+\beta \epsilon_l)}] $

+ 根据[巨正则系综热力学量的统计表达式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/gmqwy7tivky6a1pa#bFWmw)，可以计算平均粒子数$ \bar{N} $，内能$ U $，广义力$ Y $：

$ \bar{N} = -\frac{\partial}{\partial \alpha} \ln \Xi_{FD} = \sum_l \omega_l \frac{e^{-\alpha-\beta \epsilon_l}}{1+e^{-\alpha-\beta \epsilon_l}} = \sum_l \frac{\omega_l}{e^{\alpha+\beta\epsilon_l}+1} $

$ U = \bar{E} = -\frac{\partial}{\partial \beta} \ln \Xi_{FD} = \sum_l \epsilon_l\omega_l \frac{e^{-\alpha-\beta \epsilon_l}}{1+e^{-\alpha-\beta \epsilon_l}} = \sum_l \frac{\omega_l \epsilon_l}{e^{\alpha+\beta\epsilon_l}+1} $

$ Y = -\frac{1}{\beta} \frac{\partial}{\partial y}\ln \Xi_{FD} 
= \sum_l \omega_l \frac{\partial \epsilon_l}{\partial y} \frac{e^{-\alpha-\beta \epsilon_l}}{1+e^{-\alpha-\beta \epsilon_l}} = \sum_l  \frac{\partial \epsilon_l}{\partial y} \frac{\omega_l }{e^{\alpha+\beta\epsilon_l}+1} $

    - 特别地，对于压强$ p $：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V}\ln \Xi_{FD} 
= -\sum_l \omega_l \frac{\partial \epsilon_l}{\partial V} \frac{e^{-\alpha-\beta \epsilon_l}}{1+e^{-\alpha-\beta \epsilon_l}} = -\sum_l  \frac{\partial \epsilon_l}{\partial V} \frac{\omega_l }{e^{\alpha+\beta\epsilon_l}+1} $

#### 熵、亥姆霍兹自由能、吉布斯自由焓、巨势
$ S = k_B (\ln \Xi_{FD} - \alpha \frac{\partial }{\partial \alpha}\ln \Xi_{FD}- \beta \frac{\partial }{\partial \beta}\ln \Xi_{FD}) $

$ F = U-TS = -k_B T (\ln \Xi_{FD} - \alpha \frac{\partial}{\partial \alpha }\Xi_{FD}) $

$ G= \bar{N}\mu = -\bar{N}k_B T \alpha= k_B T \alpha \frac{\partial}{\partial \alpha} \ln \Xi_{FD} $

$ \Psi = F - \bar{N}\mu = -k_B T \ln \Xi_{FD} $

#### 熵的统计意义
+ 类比[玻色系统](#tYps0)，在费米系统中也有：

$ S = k_B \ln \Omega_{FD} $
