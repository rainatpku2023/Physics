### 相关数学知识
#### 偏导数与全微分
+ 考虑函数$ f $是独立变量$ x,y $的函数$ f = f(x,y) $，那么偏导数的概念为：
    - $ f $对$ x $的偏导数：

$ (\frac{\partial f}{\partial x})_y = \lim_{\Delta x\rightarrow 0} \frac{f(x+ \Delta x,y)-f (x,y)}{\Delta x} $

    - $ f $对$ y $的偏导数：

$ (\frac{\partial f}{\partial y})_x = \lim_{\Delta x\rightarrow 0} \frac{f(x,y+ \Delta y)-f (x,y)}{\Delta y} $

+ 物理学中常常使用$ (\frac{\partial f}{\partial x})_y $而非$ \frac{\partial f}{\partial x} $，是因为不同问题中选择的独立变量不同
    - 这样标记更不容易混淆。
+ 全微分：

$ df = (\frac{\partial f}{\partial x})_ydx+(\frac{\partial f}{\partial y})_xdy $

    - 全微分存在的条件：

$ \Delta f = f(x+\Delta x,y+\Delta y) - f(x,y) = A\Delta x+B\Delta y + o(\rho) $

        * 中$ A,B $的值与$ (\Delta x,\Delta y) $趋近于$ (0,0) $的路径无关。

#### 隐函数与循环关系
+ 考虑一个以$ x,y,z $为变量的常函数$ F $：

$ F(x,y,z) = C $

    - 对两侧求全微分：

$ dF = (\frac{\partial F}{\partial x})_{y,z}dx+(\frac{\partial F}{\partial y})_{z,x}dy+(\frac{\partial F}{\partial z})_{x,y}dz = 0 $

    - 可见，常函数$ F(x,y,z) = C $实际上确定了$ x,y,z $的隐函数关系。
        * 即$ x,y,z $中只有$ 2 $个变量能独立变化。
+ 根据隐函数与变量的关系：

$ (\frac{\partial x}{\partial y})_z = -(\frac{\partial F}{\partial y})_{z,x}/({\frac{\partial F}{\partial x}})_{y,z} $

$ (\frac{\partial y}{\partial z})_x = -(\frac{\partial F}{\partial z})_{x,y}/({\frac{\partial F}{\partial y}})_{z,x} $

$ (\frac{\partial z}{\partial x})_y = -(\frac{\partial F}{\partial x})_{y,z}/({\frac{\partial F}{\partial z}})_{x,y} $

    - 三个式子乘在一起，得到循环关系：

$ \boxed{(\frac{\partial x}{\partial y})_z (\frac{\partial y}{\partial z})_x (\frac{\partial z}{\partial x})_y = -1} $

    - 之所以称为循环关系，是因为每一项$ x,y,z $都往后推了一个。
    - $ x,y,z \rightarrow y,z,x \rightarrow z,x,y $

#### 复合函数与链式法则
+ 考虑一个函数$ f $是关于$ x,y $的函数，同时$ x,y $又是$ u,v $的函数，即：

$ f = f(x(u,v),y(u,v)) $

    - 那么函数$ f $关于底层变量$ u,v $的导数为：

$ \frac{\partial f}{\partial u} = \frac{\partial f}{\partial x} \frac{\partial x}{\partial u}+\frac{\partial f}{\partial y} \frac{\partial y}{\partial u} $

$ \frac{\partial f}{\partial v} = \frac{\partial f}{\partial x} \frac{\partial x}{\partial v}+\frac{\partial f}{\partial y} \frac{\partial y}{\partial v} $

#### 勒让德变换
+ 勒让德变换的一种表述形式为：

$ xdy = d(xy) - ydx $

+ 完整论述可以参考：[【空白】勒让德变换的进一步讨论](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/tukfgpy9rfkrubqn?toc_node_uuid=UqoTZWuCYvmmJzhl&view=doc_embed)

### 热力学方程，基本热力学函数与麦克斯韦关系
#### 热力学基本微分方程
+ 热力学第一定律为：

$ dU = đQ + đW $

+ 根据热力学第二定律定义熵，对于小可逆过程：

$ đQ = TdS $

+ 根据体积功的定义：

$ đW = -pdV $

+ 三个式子联立得到热力学基本微分方程：

$ \boxed{dU = TdS - pdV} $

#### 内能（U）
+ 内能是最先引入的热力学函数。
    - 它的微分表达式就是热力学基本微分方程：

$ dU = TdS - pdV $

    - 考察微分式，注意到：

$ p = -(\frac{\partial U}{\partial V})_S $$ T = (\frac{\partial U}{\partial S})_V $

    - 相内部，内能$ U $应该是二阶可导的函数：

$ \frac{\partial^2 U}{\partial S \partial V}  =  \frac{\partial^2 U}{\partial V \partial S} $

+ 代入表达式后得到麦克斯韦关系：

$ (\frac{\partial T}{\partial V})_S = -(\frac{\partial p}{\partial S})_V $

#### 焓（H）
+ 对内能$ U $进行勒让德变换：

$ H = U+pV $

    - 计算得到焓$ H $的微分表达式：

$ dH = d(U+pV) = TdS - pdV + Vdp +pdV $

        * 合并后得到：

$ dH = TdS+Vdp $

    - 考察微分式，注意到：

$ V = (\frac{\partial H}{\partial p})_S $$ T = (\frac{\partial H}{\partial S})_p $

    - 相内部，焓$ H $应该是二阶可导的函数：

$ \frac{\partial^2 H}{\partial S \partial p} = \frac{\partial^2 H}{\partial p \partial S}  $

+ 代入后得到麦克斯韦关系：

$ (\frac{\partial T}{\partial p})_S = (\frac{\partial V}{\partial S})_p $

#### 亥姆霍兹自由能（F）
+ 对内能$ U $进行勒让德变换：

$ F = U-TS $

    - 计算得到亥姆霍兹自由能$ F $的微分表达式：

$ dF = -SdT -pdV $

    - 考察微分式，注意到：

$ p = -(\frac{\partial F}{\partial V})_T $$ S = -(\frac{\partial F}{\partial T})_V $

    - 相内部，亥姆霍兹自由能$ F $应该是二阶可导的函数：

$ \frac{\partial^2 F}{\partial T \partial V} = \frac{\partial^2 F}{ \partial V  \partial T} $

+ 带入后得到麦克斯韦关系：

$ (\frac{\partial S}{\partial V})_T = (\frac{\partial p}{\partial T})_V $

#### 吉布斯自由焓（G）
+ 对亥姆霍兹自由能$ F $进行勒让德变换：

$ G = F+pV $

    - 计算得到吉布斯自由焓$ G $的微分表达式：

$ dG = -SdT +Vdp $

    - 考察微分式，注意到：

$ V = (\frac{\partial G}{\partial p})_T $$ S = -(\frac{\partial G}{\partial T})_p $

    - 相内部，吉布斯自由焓$ G $应该是二阶可导的函数：

$ \frac{\partial^2G}{\partial T \partial p}=\frac{\partial^2G}{\partial p \partial T} $

+ 带入后得到麦克斯韦关系：

$ (\frac{\partial S}{\partial p})_T = -(\frac{\partial V}{\partial T})_p $

#### 麦克斯韦关系的适用条件
+ 对于一个**<font style="color:#DF2A3F;">二阶可导</font>**的函数$ f(x,y) $，那么有：

$ (\frac{\partial}{\partial x})_y(\frac{\partial f}{ \partial y})_x = (\frac{\partial}{\partial y})_x(\frac{\partial f}{ \partial x})_y $

    - 即求导结果与求导顺序无关。
+ 如果$ A = \frac{\partial f}{\partial x}, B = \frac{\partial f}{\partial y} $：

$ (\frac{\partial B}{\partial x})_y = (\frac{\partial A}{\partial y})_x $

    - 这就是麦克斯韦关系。
+ 注意，这要是二阶可导的函数。
    - 一般来说，在相内，热力学函数都是二阶可导的，可以用这个关系。
    - 在相变过程中，热力学函数性质没有那么好，不能得出麦克斯韦关系。

### 小结
| 热力学函数 | 全微分 | 麦克斯韦关系 |
| :---: | :---: | :---: |
| $ U = U(S,V) $ | $ dU = TdS - pdV $ | $ (\frac{\partial T}{\partial V})_S = -(\frac{\partial p}{\partial S})_V $ |
| $ H = H(S,p) $ | $ dH = TdS + Vdp $ | $ (\frac{\partial T}{\partial p})_S = (\frac{\partial V}{\partial S})_p $ |
| $ F = F(T,V) $ | $ dF = -SdT -pdV $ | $ (\frac{\partial S}{\partial V})_T = (\frac{\partial p}{\partial T})_V $ |
| $ G = G(T,p) $ | $ dG = -SdT + Vdp $ | $ (\frac{\partial S}{\partial p})_T = -(\frac{\partial V}{\partial T})_p $ |
