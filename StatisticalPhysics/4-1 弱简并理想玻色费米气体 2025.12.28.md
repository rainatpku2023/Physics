### 非简并、弱简并与强简并
#### 非简并
+ 弱简并的定义是：$ e^\alpha >>1 $或者$ e^{-\alpha}<<1 $
    - 具体表现为化学势$ \mu $很低（很负），也说明粒子数很少。
    - 玻色分布和费米分布退化为玻尔兹曼分布：

$ a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}\mp 1} \rightarrow a_l = \omega_l e^{-\alpha-\beta \epsilon_l} $

    - 状态数也发生相应的变化：

$ a_l <<\omega_l \Rightarrow \Omega_{B.E.}=\Omega_{F.D.}=\frac{\Omega_{M.B.}}{N!} $

+ 对于氮原子理想气体来说：

$ e^{\alpha} = \frac{V}{N}(\frac{2\pi m k_B T}{h^2}) >>1 $

    - 考虑到$ \lambda = h (\frac{1}{2\pi m k_B T})^{1/2} $，可以化为：

$ n\lambda^3<<1 $

    - 即：

$ d>>\lambda $

        * 分子间隔远大于热波长。

#### 弱简并
+ $ e^{-\alpha} $是小量但是不可以忽略。
+ 需要使用玻色系统/费米系统处理。￥￥强简并
+ $ e^{-\alpha} $不是小量。

### 弱简并理想玻色气体
+ **先用能级准连续将求和化为对相空间的积分，计算出态密度函数。**
+ **态密度函数代替简并度。**

#### 计算态密度
+ 单个粒子的能量为：

$ \epsilon = \frac{1}{2m} (p_x^2+p_y^2 + p_z^2) = \frac{1}{2m}p^2 $

+ 根据能级准连续：

$ \sum \omega_l \rightarrow \int \frac{dxdydzdp_xdp_ydp_z}{h^3} = \frac{4\pi V}{h^3} \int_0^{\infin}p^2dp = \frac{2\pi V (2m)^{3/2}}{h^3} \int_0^{\infin}\epsilon^{1/2}d\epsilon $

    - 可见**态密度函数**为：

$ D(\epsilon)d\epsilon  = \frac{2\pi V (2m)^{3/2}}{h^3} \epsilon^{1/2}d\epsilon $

#### 理想玻色气体的巨配分函数
+ 根据[玻色分布的巨配分函数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/iq54ttgxzfy5g8a9#E0vg4)，设巨配分函数为$ \Xi $，则：

$ \ln \Xi = -\sum_l \omega_l \ln [1-e^{-\alpha -\beta \epsilon_l}] $

+ 根据能级准连续可知：

$ \ln \Xi = -\frac{2\pi V (2m)^{3/2}}{h^3} \int_0^{\infin}\ln [1-e^{-\alpha -\beta \epsilon}]\epsilon^{1/2}d\epsilon  $

+ 做变数变换$ x = \beta \epsilon $：

$ RHS = -\frac{2\pi V}{h^3}(\frac{2m}{\beta})^{3/2}\int_0^{\infin} \ln (1-e^{-\alpha - x})x^{1/2}dx $

#### 近似过程与级数解法
+ 根据弱简并条件：$ e^{-\alpha}<1 \Rightarrow e^{-\alpha-x}<1 $，将$ e^{-\alpha - x} $视为整体，进行泰勒展开：

$ \ln(1-e^{-\alpha-x}) = -\sum_{j=1}^\infin \frac{e^{-j(\alpha+x)}}{j} $

+ 代入上面的积分式：

$ RHS = \frac{2\pi V}{h^3}(\frac{2m}{\beta})^{3/2}\int_0^{\infin} \sum_{j=1}^\infin \frac{e^{-j(\alpha+x)}}{j}x^{1/2}dx $

+ 交换求和和积分的顺序，将$ x $与$ j $配对形成$ jx $：

$ RHS = \frac{2\pi V}{h^3}(\frac{2m}{\beta})^{3/2}\sum_{j=1}^{\infin} \frac{e^{-j\alpha}}{j}\frac{1}{j^{3/2}} \int_0^{\infin}e^{-jx}(jx)^{1/2}d(jx) $

    - 注意到积分正是$ \Gamma(\frac{3}{2}) = \frac{\sqrt{\pi}}{2} $，代入并整理得：

$ RHS = \frac{ V}{h^3}(2\pi mk_BT)^{3/2}\sum_{j=1}^{\infin} \frac{e^{-j\alpha}}{j^{5/2}} $

+ 定义**热波长**：

$ \lambda_T = \frac{h}{(2\pi mk_B T)^{1/2}} $

    - 以及**逸度**（化学势越大，逸度越大）：

$ z = e^{-\alpha} $

    - 代入后得到：

$ \ln \Xi = \frac{V}{\lambda_T^3}\sum_{j=1}^{\infin}\frac{z^j}{j^{5/2}} = \frac{V}{\lambda_T^3} g_{5/2}(z) $

#### 热力学量与状态方程
+ 根据[巨正则系综的热力学公式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/gmqwy7tivky6a1pa#bFWmw)可得：
    - 压强：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V}\ln \Xi = \frac{k_B T}{\lambda_T^3}  g_{5/2}(z) $

    - 内能：

$ U = -\frac{\partial}{\partial \beta}\ln \Xi = \frac{3}{2}k_B T \frac{V}{\lambda_T^3}g_{5/2}(z) $

        * 需要注意$ \beta $包含于$ \lambda_T $中。
    - 平均粒子数：

$ \bar{N} = -\frac{\partial}{\partial \alpha}\ln \Xi = -\frac{V}{\lambda_T}^3 \frac{\partial}{\partial \alpha}(\sum_{j=1}^{\infin}\frac{e^{-j\alpha}}{j^{5/2}})
=
\frac{V}{\lambda_T^3}\sum_{j=1}^{\infin} \frac{e^{-j\alpha}}{j^{3/2}} = \frac{V}{\lambda_T^3}\sum_{j=1}^{\infin}\frac{z^j}{j^{3/2}}
= \frac{V}{\lambda_T^3} g_{3/2}(z) $

+ 由此可以得到状态方程：

$ \frac{pV}{Nk_B T}= \frac{g_{5/2}(z)}{g_{3/2}(z)} =\frac{z+\frac{1}{2^{5/2}}z^2+\cdots }{z+\frac{1}{2^{3/2}}z^2+\cdots} = 1-\frac{1}{2^{5/2}}z+\cdots  = 1-\frac{1}{2^{5/2}}n\lambda_T^3+\cdots  $

    - 其中第三个等号是用分子凑出分母的结果，所以三个等号后面第二项是减号。
    - 第四个等号用到了$ z = e^{-\alpha} = n \lambda_T^3 $，这个式子会在[补充推导](#cZ9MW)中给出证明。
+ 进一步给出内能的近似表达式：

$ U = \frac{3}{2} k_B T \frac{V}{\lambda_T^3}g_{3/2}(z) \cdot \frac{g_{5/2}(z)}{g_{3/2}(z)} = \frac{3}{2} N k_B T  \cdot \frac{g_{5/2}(z)}{g_{3/2}(z)} \approx \frac{3}{2} N k_B T(1-\frac{1}{z^{5/2}}z)  $

$ = \frac{3}{2}N k_B T(1-\frac{1}{z^{5/2}}n\lambda_T^3) $

#### 补充推导
+ 先考虑**经典粒子系统**平均粒子数的表达式：

$ N = \sum_l \bar{a}_l = e^{-\alpha}\sum_l \omega_l e^{-\beta \epsilon_l} = e^{-\alpha}Z $

    - 因此：

$ z = e^{-\alpha} = \frac{N}{Z} $

+ 结合**经典粒子正则系综**配分函数$ Z $的表达式：

$ Z = \frac{V}{h^3}\int e^{-\beta(p_x^2+p_y^2+p_z^2)/2m}dxdydzdp_xdp_ydp_z
= \frac{V}{h^3}(2\pi m k_B T)^{3/2} = \frac{V}{\lambda_T^3} $

    - 可见：

$ z = e^{-\alpha} = \frac{N}{Z} = \frac{N}{V}\lambda_T^3 = n\lambda_T^3 $

### 弱简并理想费米气体
#### 理想费米气体的巨配分函数
+ 费米体系的巨配分函数为：

$ \ln \Xi = \sum_l \omega_l \ln [1+e^{-\alpha -\beta \epsilon_l}] $

+ 结合能级准连续，关注到费米子自旋可以向上和向下：

$ \ln \Xi = 2\cdot \frac{2\pi V (2m)^{3/2}}{h^3} \int_0^{\infin}\ln [1+e^{-\alpha -\beta \epsilon}]\epsilon^{1/2}d\epsilon  $

+ 做变数变换$ x = \beta \epsilon $：

$ RHS = \frac{4\pi V}{h^3}(\frac{2m}{\beta})^{3/2}\int_0^{\infin} \ln (1+e^{-\alpha - x})x^{1/2}dx $

#### 近似过程与级数解法
+ 根据弱简并条件：$ e^{-\alpha}<1 \Rightarrow e^{-\alpha-x}<1 $，将$ e^{-\alpha - x} $视为整体，进行泰勒展开：

$ \ln(1+e^{-\alpha-x}) = \sum_{j=1}^\infin \frac{(-1)^{j-1}e^{-j(\alpha+x)}}{j} $

+ 代入上面的积分式：

$ RHS = 2\frac{V}{h^3}(2\pi mk_BT)^{3/2}\sum_{j=1}^{\infin} \frac{(-1)^{j-1}e^{-j\alpha}}{j^{5/2}} $

+ 定义**热波长**：

$ \lambda_T = \frac{h}{(2\pi mk_B T)^{1/2}} $

    - 以及**逸度**（化学势越大，逸度越大）：

$ z = e^{-\alpha} $

    - 代入后得到：

$ \ln \Xi = \frac{V}{\lambda_T^3}\sum_{j=1}^{\infin}\frac{(-1)^{j-1}z^j}{j^{5/2}} = \frac{V}{\lambda_T^3} f_{5/2}(z) $

#### 热力学量与状态方程
+ 根据[巨正则系综的热力学公式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/gmqwy7tivky6a1pa#bFWmw)可得：
    - 压强：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V}\ln \Xi = \frac{2k_B T}{\lambda_T^3}  f_{5/2}(z) $

    - 内能：

$ U = -\frac{\partial}{\partial \beta}\ln \Xi = 3k_B T \frac{V}{\lambda_T^3}f_{5/2}(z) $

        * 需要注意$ \beta $包含于$ \lambda_T $中。
    - 平均粒子数：

$ \bar{N} = -\frac{\partial}{\partial \alpha}\ln \Xi = -\frac{V}{\lambda_T}^3 \frac{\partial}{\partial \alpha}(\sum_{j=1}^{\infin}\frac{e^{-j\alpha}}{j^{5/2}})
= \frac{2V}{\lambda_T^3} f_{3/2}(z) $

+ 由此可以得到状态方程：

$ \frac{pV}{Nk_B T}= \frac{g_{5/2}(z)}{g_{3/2}(z)} =\frac{z-\frac{1}{2^{5/2}}z^2+\cdots }{z-\frac{1}{2^{3/2}}z^2+\cdots} = 1+\frac{1}{2^{5/2}}z+\cdots  = 1+\frac{1}{2^{5/2}}\frac{n}{2}\lambda_T^3+\cdots  $

    - 其中第三个等号是用分子凑出分母的结果，所以三个等号后面第二项是减号。
    - 第三个等号出现$ 1/2 $因子是因为费米子的计算简并度为$ 2 $。
+ 进一步给出内能的近似表达式：

$ U = \frac{3}{2} k_B T \frac{V}{\lambda_T^3}f_{3/2}(z) \cdot \frac{f_{5/2}(z)}{f_{3/2}(z)} = \frac{3}{2} N k_B T  \cdot \frac{f_{5/2}(z)}{f_{3/2}(z)} \approx \frac{3}{2} N k_B T(1+\frac{1}{z^{5/2}}\frac{z}{2})  $

$ =  \frac{3}{2} N k_B T(1+\frac{1}{z^{5/2}}\frac{n}{2}\lambda_T^3)  $

### 统计关联
+ 根据上文可知，两种气体的状态方程为：

$ \frac{pV}{Nk_B T} = 1\pm \frac{1}{2^{5/2}g}n\lambda_T^3 $

    - 对于弱简并理想费米气体，取$ + $号，自旋简并度$ g=2 $。
    - 对于弱简并理想玻色气体，取$ - $号，自旋简并度$ g=1 $。
+ 类似地，内能表达式为：

$ U = \frac{3}{2}Nk_B T (1\pm \frac{1}{2^{5/2}g}n\lambda_T^3) $

    - 可见费米气体存在有效排斥，而玻色气体存在有效吸引。
    - “有效”的英文是“effective”，意思是等效的、表观的，本质上这是量子统计效应，而非力学效应。
