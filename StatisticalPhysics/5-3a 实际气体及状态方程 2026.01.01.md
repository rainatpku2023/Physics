### 模型
#### 模型假设
+ 系统的能量为：

$ E = \sum_{i=1}^N \frac{p_i^2}{2m} +\sum_{i<j} \phi(r_{ij}) $

+ **假设1：无外场，气体稀薄，只有两体相互作用，忽略三体以上的相互作用。**
+ **假设2：满足经典极限，无量子统计关联。**

#### 势能模型
+ **莱纳德-琼斯势**（又称6-12势）：

$ \phi(r) = \phi_0\left[\left(\frac{r_0}{r}\right)^{12}-2\left(\frac{r_0}{r}\right)^{6}\right] $

+ 为了简化计算，使用**硬球模型**：

$ \phi(r) = \begin{cases}+\infin,\; r<r_0 \\ -\phi_0 \left(\frac{r_0}{r}\right)^6,\; r\geq r_0 \end{cases} $

### 配分函数与位形积分
#### 配分函数与位形积分的引入
+ 根据正则系综的定义，写出配分函数：

$ Z = \frac{1}{N! h^{3N}}\int d^{3}r_1\cdots d^{3}r_N \int d^{3}p_1\cdots d^{3}p_N e^{-\beta E} $

+ 代入总能量的表达式：

$ RHS = \frac{1}{N! h^{3N}}\int d^{3}p_1\cdots d^{3}p_N e^{-\beta \sum_{i=1}^N \frac{p_i^2}{2m}}\int d^{3}r_1\cdots d^{3}r_N e^{-\beta \sum_{i<j} \phi(r_{ij})}  $

+ 第一个积分可以使用高斯积分公式积分出来，第二项被称为位形积分$ Q $。

$ Z = \frac{1}{N!} \left( \frac{2\pi m}{\beta h^2}\right)^{\frac{3N}{2}}\cdot Q $

+ 位形积分的表达式为：

$ \boxed{Q = \int d^{3}r_1\cdots d^{3}r_N e^{-\beta \sum_{i<j} \phi(r_{ij})}} $

#### 引入辅助函数f
+ 对于一般的势能来说，在无穷远处势应该衰减到零，而越靠近中心势会增长到正无穷。
+ 此时定义函数：

$ f_{ij} = e^{-\beta \phi(r_{ij})}-1 $

    - 这个函数有如下比较好的性质：

$ r_{ij}\rightarrow 0 \Rightarrow \phi(r_{ij})\rightarrow +\infin \Rightarrow f_{ij} \rightarrow -1 $

$ r_{ij}\rightarrow +\infin \Rightarrow \phi(r_{ij})\rightarrow 0 \Rightarrow f_{ij} \rightarrow 0 $

#### 位形积分的计算
+ 考虑位形积分：

$ Q = \int d^{3}r_1\cdots d^{3}r_N e^{-\beta \sum_{i<j} \phi(r_{ij})}
= \int d^{3}r_1\cdots d^{3}r_N \prod_{i<j}e^{-\beta\phi(r_{ij})} $

+ 代入$ f_{ij} $的表达式：

$ RHS = \int d^{3}r_1\cdots d^{3}r_N \prod_{i<j}(1+f_{ij})
 $

$ =\int d^{3}r_1\cdots d^{3}r_N (1+\sum_{i<j}f_{ij} + \sum_{i<j}\sum_{i'<j'}f_{ij}f_{i'j'}+\cdots) $

    - 这个表达式其实就是**迈耶集团展开**。
+ 在气体浓度不太高的状况下，可以只保留第一级修正：

$ RHS \approx\int d^{3}r_1\cdots d^{3}r_N (1+\sum_{i<j}f_{ij})
 $

+ 分别考虑两项，注意第二项实际上可以拆成$ \frac{N(N+1)}{2} $项：

$ RHS=\int d^{3}r_1\cdots d^{3}r_N+ \frac{N(N-1)}{2}  \int d^{3}r_1\cdots d^{3}r_N f_{12}(r_{12}) $

$ = V^N + \frac{N(N-1)}{2} V^{N-2}\int d^{3}r_1 d^{3}r_2 f_{12}(r_{12}) $

$ \approx V^N + \frac{N^2}{2} V^{N-2}\int d^{3}r_1 d^{3}r_2 f_{12}(r_{12}) $

    - 第二项中的积分，实际上可以进一步化简。定义相对坐标和质心坐标：

$ \begin{cases} \vec{r} = \vec{r}_1 - \vec{r}_2 = \vec{r}_{12} \\
\vec{R} = \frac{\vec{r}_1+\vec{r}_2}{2}\end{cases} $

    - 变换的$ \mathrm{Jacobi} $行列式为$ 1 $，所以：

$ \int d^{3}r_1 d^{3}r_2 f_{12}(r_{12}) = \int d^3 R d^3 r f_{12}(r) = V\int d^3 r f_{12}(r) $

    - 代入位形积分的表达式后：

$ RHS = V^N + \frac{N^2}{2} V^{N-1}\int d^{3}r  f_{12}(r) $

+ 由此就可以计算出**位形积分**：

$ \boxed{Q = V^N \left[1+\frac{N^2}{2V} \int d^3 r f_{12}(r)\right]} $

#### 热力学量与状态方程
+ 首先给出位形积分的对数：

$ \ln Q = N\ln V +\ln \left[1+\frac{N^2}{2V} \int d^3 r f_{12}(r)\right] $

    - 修正应该只是很小的，所以可以用泰勒展开得到：

$ \ln Q = N\ln V +\frac{N^2}{2V} \int d^3 r f_{12}(r） $

+ 根据正则系综热力学公式计算**压强**：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V} \ln Z =  \frac{1}{\beta} \frac{\partial}{\partial V} \ln Q = \frac{Nk_B T}{V}\left[ 1-\frac{N}{2V}\int d^3 r f_{12}(r) \right] $

    - 第二个等号是因为动量空间部分的积分不显含$ V $。



+ 实际气体状态方程的位力展开为：

$ pV = N k_B T \left(1+\frac{B_2}{V}+\cdots \right) $

    - 其中$ B_2 $称为第二位力系数。
    - 对照表达式可知，**第二位力系数**的表达式为：

$ \boxed{B_2 = -\frac{N}{2} \int f_{12}(r)d^3r} $

#### 附录：位力展开
+ 一般来说，实际气体状态方程的位力展开为：

$ pV = N k_B T \left(1+\frac{B_2}{V}+\frac{B_3}{V^3}+\cdots \right) $

    - 此处只讨论最低阶修正。

### 第二位力系数的计算
#### 使用硬球模型在高温状态计算第二位力系数
+ 前面提到，硬球模型的势为：

$ \phi(r) = \begin{cases}+\infin,\; r<r_0 \\ -\phi_0 \left(\frac{r_0}{r}\right)^6,\; r\geq r_0 \end{cases} $

    - 对应地：

$ f_{12}(r) = \begin{cases}-1,\; r<r_0 \\ e^{ \beta \phi_0 \left(\frac{r_0}{r}\right)^6}-1,\; r\geq r_0 \end{cases} $

    - 在高温情形下，$ \beta $很小，使用泰勒展开：

$ f_{12}(r) = \begin{cases}-1,\; r<r_0 \\ \beta \phi_0 \left(\frac{r_0}{r}\right)^6,\; r\geq r_0 \end{cases} \; T很大 $



+ 代入第二位力系数表达式：

$ B_2 = -\frac{N}{2} \int f_{12}(r)d^3r = -\frac{N}{2}\int_0^\infin  f_{12}(r) 4\pi r^2d r = -2\pi N \int_0^\infin f_{12}(r)  r^2d r $

    - 分段代入：

$ B_2 = -2\pi N \left[\int_0^{r_0} -  r^2d r+ \int_{r_0}^\infin \beta\phi_0 \frac{r_0^6}{r^4}dr \right]
= 2\pi N \left[\frac{r_0^3}{3}-\frac{\phi_0}{3} \frac{r_0^3}{k_B T}\right]
= Nb - \frac{Na}{k_B T} $

    - 其中引入的常数的表达式为：

$ b = \frac{2}{3}\pi r_0^3,\; a = \frac{2}{3}\pi r_0^3 \phi_0 $

#### 范德瓦耳斯方程的导出
+ 将位力系数表达式代入压强表达式：

$ p = \frac{N k_B T}{V} \left[1+\frac{1}{V}\left(Nb - \frac{Na}{k_B T}\right) \right]
= \frac{Nk_B T}{V}\left(1+\frac{Nb}{V}\right) - \frac{N^2a}{V^2} $

    - 如果修正不太大：

$ 1+\frac{Nb}{V} \approx \frac{1}{1-\frac{Nb}{V}} $

    - 可以将体积修正项从分子搬到分母上：

$ p  = \frac{Nk_B T}{V\left(1-\frac{Nb}{V}\right)}-\frac{N^2a}{V} $

+ 简单整理后就可以得到**范德瓦耳斯方程**：

$ \boxed{\left(p+\frac{N^2a}{V^2}\right) \left(V-Nb\right)= Nk_B T} $

+ 可见，$ a $对应压强修正项，$ b $对应体积修正项。
