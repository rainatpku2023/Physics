### 前言
+ 在[3-1b 近独立粒子系统的最概然分布](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/qr3cuico1i1olwne)一节中，已经不严格地推出了玻尔兹曼分布、费米分布和玻色分布的形式。	
    - 不严格之处在于，假定了$ \omega_l > >a_l $，能级简并度远大于能级上的粒子数，即粒子分布足够稀疏。
    - 这个条件常常难以满足。
+ 因此，在本节将用正则系综理论严格推导玻尔兹曼系统的最概然分布，[下一节](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/iq54ttgxzfy5g8a9)将用巨正则系综严格推导费米系统以及玻色系统的最概然分布。	
    - 我们将看到，不严格推导过程得到了正确的结果。
    - 因此，由此得到的[热力学公式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/qoko1ox06e22c68d)也都是正确的。

### 问题概述
+ 希望用正则系综理论推导出玻尔兹曼分布
+ 设：
    - 单粒子能级为$ \epsilon_1,\epsilon_2,..,\epsilon_l,... $
    - 能级简并度为$ \omega_1,\omega_2,..,\omega_l,... $
    - 粒子按能级分布为$ a_1,a_2,...,a_l,... $

### 近独立子系的配分函数可以用子系配分函数乘积表示
#### 系统的配分函数
+ 正则系综配分函数的定义为：

$ Z = \sum_{S} e^{-\beta E_S} $

+ 假设系统中总共存在$ N $个可分辨粒子，那么：

$ e^{-\beta E_S} = e^{-\beta \epsilon_1}e^{-\beta \epsilon_2}\cdots e^{-\beta \epsilon_k}\cdots e^{-\beta \epsilon_N} = \prod_{k=1}^N e^{-\beta \epsilon_k} $

+ 带入得：

$ Z = \sum_{所有情形} \prod_{k=1}^N e^{-\beta \epsilon_k} = \prod_{k=1}^{N} \sum_{第k个粒子所有情形}e^{-\beta \epsilon_k}  $

    - 对于第$ k $个粒子的状态，按照能级进行遍历：

$ RHS = \prod_{k=1}^{N} \sum_{l}\omega_l e^{-\beta \epsilon_l} = z^N $

+ 即：

$ Z = z^N $

#### 子系配分函数
+ 其中$ z $为子系配分函数，表达式为：

$ z = \sum_{l}\omega_l e^{-\beta \epsilon_l} $

+ <font style="color:#DF2A3F;">此处之所以能将系统的配分函数拆成子系配分函数的乘积，是因为粒子之间没有相互作用。</font>
+ <font style="color:#DF2A3F;">对于玻色系统和费米系统，因为统计规律要求，玻色子之间以及费米子之间会产生表观的相互作用。此时总的配分函数不能拆解成子系配分函数的乘积。</font>
    - <font style="color:#8A8F8D;">例如费米子就表现为有“斥力”，玻色子表现为有“引力”。可以不严谨地看成哈密顿量中有相互作用项。</font>
    - <font style="color:#8A8F8D;">对于一个简并度为2的能级</font>
        * <font style="color:#8A8F8D;">如果有两个经典（可分辨）粒子，总共有4种情况，其中2种情况下两个粒子处于同一量子态。根据等概率原理，观测到处于相同量子态的概率为1/2。</font>
        * <font style="color:#8A8F8D;">如果有两个费米子（不可分辨），总共有1种情况，其中0种情况下两个粒子处于同一量子态。根据等概率原理，观测到处于相同量子态的概率为0。这体现了有效吸引作用。</font>
        * <font style="color:#8A8F8D;">如果有两个玻色子（不可分辨），总共有3种情况，其中2种情况下两个粒子处于同一量子态。根据等概率原理，观测到处于相同量子态的概率为2/3。这体现了有效吸引作用。</font>
    - <font style="color:#DF2A3F;">此时可以按照能级去遍历，总的配分函数等于能级配分函数的乘积，这是因为能级之间的粒子没有表观相互作用。</font>
    - <font style="color:#DF2A3F;">详情可以参考：</font>[3-3b 由巨正则系综推导玻色分布和费米分布 2025.11.23](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/iq54ttgxzfy5g8a9)

### 玻尔兹曼分布
#### 玻尔兹曼分布的配分函数
+ 能量为$ E_S $的状态出现的概率大小为：

$ \rho_S = \frac{1}{Z} e^{-\beta E_S} $

+ 对于玻尔兹曼系统，则粒子分布$ \{a_l\} $的概率大小为：

$ \rho_{\{a_l\}} = \frac{1}{Z} \Omega(\{a_l\})e^{-\beta \sum_l a_l\epsilon_l} $

+ 其中：

$ \Omega(\{a_l\}) = \frac{N!}{\prod_l a_l!} \prod_l \omega_l^{a_l} = N!\prod_l \frac{\omega_l^{a_l}}{a_l!} $

$ Z = \sum_{\{a_l\}} \Omega(\{a_l\})e^{-\beta\sum_l a_l\epsilon_l} = \sum_{\{a_l\}} \frac{N!}{\prod_l a_l!} \prod_l \omega_l^{a_l}e^{-\beta\sum_l a_l\epsilon_l} $

#### 根据配分函数求玻耳兹曼分布
+ 第$ k $个能级上平均粒子数为：

$ \bar{a}_k = \sum_{\{a_l\}} a_k \rho _{ \{a_l\}} $

+ 代入具体表达式：

$ \bar{a}_k = \sum_{\{a_l\}} a_k \rho _{ \{a_l\}} = \frac{1}{Z} \sum_{\{a_l\} }a_kN!\prod_l \frac{\omega_l^{a_l}}{a_l!}e^{-\beta \sum_l a_l\epsilon_l} $

    - 使用偏微分表达式：

$ RHS = -\frac{1}{\beta}\frac{1}{Z} \frac{\partial }{\partial \epsilon_k}(\sum_{\{a_l\} }N!\prod_l \frac{\omega_l^{a_l}}{a_l!}e^{-\beta \sum_l a_l\epsilon_l})  $

    - 注意到等号右侧就是配分函数：

$ RHS =  -\frac{1}{\beta}\frac{1}{Z} \frac{\partial Z}{\partial \epsilon_k} = -\frac{1}{\beta}\frac{\partial}{\partial \epsilon_k}\ln Z $

    - 代入$ Z = z^N $：

$ RHS = -\frac{N}{\beta}\frac{\partial}{\partial \epsilon_k}\ln z = -\frac{N}{\beta}\frac{1}{z} \frac{\partial z}{\partial \epsilon_k} $

    - 代入$ z = \sum_{l}\omega_l e^{-\beta \epsilon_l} $：

$ RHS = \frac{N}{z} \omega_k e^{-\beta \epsilon_k} $

    - 引入$ e^{-\alpha} = \frac{N}{z} $：

$ RHS = \omega_k e^{-\alpha-\beta \epsilon_k} $

+ 综上，**玻尔兹曼分布**为：

$ \boxed{\bar{a}_l = e^{-\alpha -\beta \epsilon_l}} $

    - 其中：

$ e^{-\alpha} = \frac{N}{z} $
