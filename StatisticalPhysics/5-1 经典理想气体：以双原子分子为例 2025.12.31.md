### 双原子分子的配分函数
#### 双原子分子的能量
+ 双原子分子的能量近似可以分解成平动+转动+振动，即：

$ \epsilon = \epsilon^t +\epsilon^r + \epsilon^v $

#### 子系配分函数
+ 代入子系配分函数定义式：

$ z = \sum_l \omega_l e^{-\beta \epsilon_l} = \sum_t\sum_r\sum_v \omega^t \omega^r \omega^v e^{-\beta(\epsilon^t +\epsilon^r + \epsilon^v)} $ 

    - 进行拆分：

$ RHS = \left(\sum_t \omega^t e^{-\beta \epsilon^t}\right)\left(\sum_r \omega^r e^{-\beta \epsilon^r}\right)\left(\sum_v \omega^v e^{-\beta \epsilon^v}\right) = z^t\cdot z^r\cdot z^v $

+ 其中$ z^t, z^r, z^v $分别为平动（translational）、转动（rotational）和振动（vibrational）的子系配分函数。

##### 热力学函数
+ 内能：

$ U = -N\frac{\partial }{\partial \beta}\ln z = -N \frac{\partial}{\partial \beta} (\ln z^t+\ln z^r+\ln z^v) = U^t + U^r + U^v $

+ 定容热容：

$ C_V = (\frac{\partial U}{\partial T})_V = C_V^t+C_V^r+C_V^v $

#### 平动、转动、振动的量子模型
+ 对于平动，在宏观尺度上量子效应并不明显，因而还可以使用：

$ \epsilon^t = \frac{1}{2m}(p_x^2+p_y^2+p_z^2) $

+ 对于转动，经典模型为：

$ \epsilon^r = \frac{1}{2I}(p_\theta^2 +\frac{1}{\sin^2\theta}p_\phi^2) $

    - 量子模型的能量与其对应的简并度为：

$ \epsilon^r = \frac{l(l+1)\hbar^2}{2I},\; n=0,1,2,... $

$ \omega_l = 2l+1 $

+ 对于振动，经典模型为：

$ \epsilon^v = \frac{1}{2m_v}p_r^2 + u(r) $

    - 量子模型的能量与其对应的简并度为：

$ \epsilon_n = (n+\frac{1}{2})\hbar \omega,\; n=0,1,2,... $

$ \omega_n = 1 $

### 平动
#### 平动子系配分函数
+ 平动子系配分函数为：

$ z^t = \int \frac{dxdydz dp_xdp_ydp_z}{h^3}e^{-\frac{\beta}{2m}(p_x^2+p_y^2+p_z^2)} = \frac{V}{h^3} \left( \frac{2m}{\beta}\right)^{3/2} $

#### 平动对应的内能和热容
+ 对应的内能和定容热容为：

$ U^t = -\frac{\partial}{\partial \beta} \ln z_\beta^t = \frac{3N}{2\beta} = \frac{3}{2}Nk_B T $

$ C_V^t = \left(\frac{\partial U^t}{\partial t}\right)_V = \frac{3}{2}Nk_B $

### 振动
#### 振动子系配分函数
+ 线性谐振子近似下，振动的子系配分函数为：

$ z^v = \sum_{n=0}^{\infin}e^{-\beta\hbar \omega(n+\frac{1}{2})} = e^{-\frac{1}{2}\beta \hbar \omega} \sum_{n=0}^{\infin}e^{-\beta\hbar\omega n} = \frac{e^{\frac{\beta\hbar\omega}{2}}}{1-e^{-\beta \hbar \omega}} $

#### 振动对应的内能和热容
+ 对应的内能和定容热容为：

$ U^v = -\frac{\partial}{\partial \beta} \ln z_\beta^v = \frac{N\hbar\omega}{2}+\frac{N\hbar\omega}{e^{\beta\hbar\omega}-1} $

$ C_V^v = \left(\frac{\partial U^v}{\partial t}\right)_V = N k_B \left(\frac{\hbar\omega}{k_B T}\right)^2 \frac{e^{\hbar\omega/k_BT}}{(e^{\hbar\omega/k_BT}-1)^2} $

#### 特征温度与近似
+ 引入振动特征温度$ \theta_v $满足：

$ k_B \theta_v = \hbar\omega $

+ 对于双原子分子来说，$ \theta_v \sim 10^3\mathrm{K} $。
+ 所以在常温下$ \frac{\theta_v}{T}>>1 $，内能和热容可以取近似：

$ U^v \approx \frac{Nk_B\theta_v}{2}+Nk_B \theta_ve^{-\frac{\theta_v}{T}} $

$ C_V^v \approx Nk_B \left(\frac{\theta_v}{T}\right)^2 e^{-\frac{\theta_v}{T}} $

    - 可见常温范围内，振动自由度对于内能贡献恒定，而对定容热容没有多少贡献。

### 转动
#### 转动子系配分函数
+ 根据量子力学模型，注意到简并度，转动的子系配分函数为：

$ z^r = \sum_{l=0}^{\infin} (2l+1)e^{-\frac{l(l+1)\hbar^2}{2Ik_B T}} $

#### 特征温度
+ 定义转动特征温度$ \theta_r $满足：

$ k_B\theta_r = \frac{\hbar^2}{2I} $

#### 常温下转动子系配分函数与热力学函数
+ 常温下，温度足够高，有$ \frac{\theta_r}{T}<<1 $，可以认为能级准连续：

$ z^r = \sum_{l=0}^{\infin} (2l+1)e^{-\frac{l(l+1)\hbar^2}{2Ik_B T}} \approx \int_0^\infin (2l+1)e^{-\frac{l(l+1)\theta_r}{T}}dl =  \frac{T}{\theta_r}\int_0^{\infin}e^{-x}dx = \frac{T}{\theta_r} = \frac{2I}{\beta\hbar^2} $

    - 其中用到了$ (2l+1)dl = d[l(l+1)] $。
+ 对应的内能和热容为：

$ U^r = -\frac{\partial}{\partial \beta} \ln z_\beta^r = Nk_B T $

$ C_V^r = \left(\frac{\partial U^r}{\partial t}\right)_V = Nk_B $

#### 同核双原子分子的情形：以氢气为例
> 由于氢原子核是费米子，所以两个氢原子核组成的系统的波函数必须是反对称的。这个系统的波函数可以写成自旋波函数和轨道波函数（也就是两个氢原子核构成的转子的波函数）的乘积。如果氢原子核的自旋相互平行，那么系统的自旋波函数就是对称的，这就要求波轨道波函数是反对称的，也就要求转动量子数$ l $必须是奇数。
>

+ 正氢，自旋平行，$ l $为奇数，其转动配分函数为：

$ z_o^r = \sum_{l=1,3...}(2l+1)e^{-\frac{l(l+1)\theta_r}{T}} $

+ 仲氢，自旋反平行，$ l $为偶数，其转动配分函数为：

$ z_p^r = \sum_{l=0,2...}(2l+1)e^{-\frac{l(l+1)\theta_r}{T}} $

+ 考虑到正氢占$ 3/4 $，仲氢占$ 1/4 $，一团分子数为$ N $的氢气的内能应该为：

$ U^r = -\frac{3}{4} N \frac{\partial}{\partial \beta}\ln z_o^r-\frac{1}{4} N \frac{\partial}{\partial \beta}\ln z_p^r $

+ 在高温极限下：

$ \sum_{l=0,2...}(2l+1)e^{-\frac{l(l+1)\theta_r}{T}}\approx\sum_{l=1,3...}(2l+1)e^{-\frac{l(l+1)\theta_r}{T}}\approx\frac{1}{2}\sum_{l=0}^{\infin}(2l+1)e^{-\frac{l(l+1)\theta_r}{T}} $

#### 低温极限
+ 低温极限下，求和不能化成积分，所以级数求和。数值求解。

### 电子贡献
+ 电子基态与激发态的能级之差为$ \mathrm{eV} $量级 ，换算到温度是$ 10^4\sim10^5 \mathrm{K} $。
    - 一般温度难以将电子从基态激发到激发态。
+ 电子冻结于基态，对内能贡献恒定，对热容无贡献。
