### 热力学函数的统计表达式
#### 子系配分函数
+ **<font style="color:#DF2A3F;">子系配分函数函数将一个粒子看成一个系统，其配分函数称为子系配分函数。</font>**
    - 之前推导玻尔兹曼系统用的是系统的配分函数，而非子系配分函数。
+ 玻尔兹曼系统的最概然分布为：

$ a_l = \omega_l e^{-\alpha- \beta \epsilon_l} $

    - 粒子数和内能分别为：

$ N = \sum_l a_l = \sum_l \omega_l e^{-\alpha -\beta \epsilon_l} = e^{-\alpha} \sum_l \omega_l e^{-\beta \epsilon_l} $

$ U = \sum_l a_l \epsilon_l = \sum_l \epsilon_l\omega_l e^{-\alpha -\beta \epsilon_l} = e^{-\alpha} \sum_l \epsilon_l\omega_l e^{-\beta \epsilon_l} $

+ 引入**子系配分函数**：

$ z = \sum_l \omega_l e^{-\beta \epsilon_l} $

    - 粒子数和内能可以分别表示为：

$ N = e^{-\alpha} z $

$ U = e^{-\alpha} \sum_l \epsilon_l \omega_l e^{-\beta \epsilon_l}= e^{-\alpha} (-\frac{\partial}{\partial \beta})\sum_l  \omega_l e^{-\beta \epsilon_l}
= \frac{N}{z} [-\frac{\partial}{\partial \beta}z] = -N\frac{\partial}{\partial \beta}\ln z $

#### 内能
+ 上面已经推出，内能的热力学公式：

$ \boxed{U = -N\frac{\partial}{\partial \beta}\ln z} $

#### 广义力
+ 设与广义力$ Y $对应的广义坐标为$ y $，那么外界施加在$ \epsilon_l $能级的一个粒子的广义力为$ \frac{\partial \epsilon_l}{\partial y} $。
    - 那么总的广义力为：

$ Y = \sum_l\frac{\partial \epsilon_l}{\partial y}a_l $

    - 代入粒子数分布后：

$ RHS = \sum_l \frac{\partial \epsilon_l}{\partial y} \omega_l e^{-\alpha -\beta \epsilon_l}=e^{-\alpha}\sum_l \frac{\partial \epsilon_l}{\partial y} \omega_l e^{ -\beta \epsilon_l} = e^{-\alpha} (-\frac{1}{\beta} \frac{\partial}{\partial y})\sum_l \omega_l e^{-\beta \epsilon_l}= e^{-\alpha} (-\frac{1}{\beta} \frac{\partial}{\partial y})z $

    - 考虑到$ N = e^{-\alpha} Z $，可得：

$ \boxed{Y = -\frac{N}{\beta}\frac{\partial }{\partial y} \ln z} $

+ 特别地，对于压强：

$ \boxed{p = \frac{N}{\beta}\frac{\partial }{\partial V} \ln z} $

#### 熵
+ 根据普遍的推导，可以得到：

$ \boxed{S = N k_B (\ln z - \beta \frac{\partial}{\partial \beta} \ln z)} $

    - 其中$ N = e^{-\alpha}z $。

#### 亥姆霍兹自由能
+ 根据$ F = U - TS $，可以推出：

$ \boxed{F = -Nk_B T \ln Z} $

### 熵的统计意义
+ 熵的表达式为：

$ S = N k_B (\ln z - \beta \frac{\partial}{\partial \beta} \ln z) $

+ 考虑到下列两个关系式：

$ U = -N\frac{\partial}{\partial \beta}\ln z $

$ N = e^{-\alpha} z \Rightarrow \ln z = \ln N +\alpha $

+ 熵的表达式可以化为：

$ S = k_B (N\ln N + \alpha N + \beta U)
= k_B (N\ln N + \sum_l \alpha a_l  + \sum_l \beta a_l \epsilon_l) $

$ = k_B [N\ln N + \sum_l a_l (\alpha + \beta \epsilon_l)] $

+ 根据$ a_l = \omega_l e^{-\alpha- \beta \epsilon_l} $可知$ \alpha + \beta \epsilon_l = \ln \frac{\omega_l}{a_l} $，带入得：

$ S = k_B (N\ln N +\sum_l a_l \ln \omega_l - \sum_l a_l \ln a_l) $

+ 根据[玻尔兹曼系统微观状态数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/rx6x9vuxr04cfhxb#dmxoy)：

$ \Omega_{MB} = \frac{N!}{\prod_l a_l!}\prod_l \omega_l^{a_l} $

    - 求对数：

$ \ln \Omega_{MB} = N\ln N -\sum a_l \ln a_l + \sum_l a_l \ln \omega_l $

    - 对比其与$ S $的表达式，由此可以推出：

$ \boxed{S = k_B \ln \Omega_{MB}} $

### 可分辨粒子与全同粒子系统
+ 上文中讨论的玻尔兹曼系统是可分辨粒子组成的系统。
    - 这可以从微观状态数中看出：

$ \Omega_{MB} = \frac{N!}{\prod_l a_l!}\prod_l \omega_l^{a_l} $

        * 此处并没有对于每一个粒子进行去重，保留了$ N! $因子，因此仍然是粒子可分辨系统。

#### 微观状态数
+ 对于全同粒子系统，在经典极限下：

$ \Omega_{全同} = \Omega_{BE/FD} = \frac{\Omega_{MB}}{N!} $

#### 熵
+ 对两侧取对数：

$ S_{全同} = k_B \ln \Omega_{全同} = k_B \ln \frac{\Omega_{MB}}{N!}
= Nk_B (\ln z - \beta \frac{\partial}{\partial \beta} \ln z) - k_B \ln N! $

    - 其中$ z $沿用了子系配分函数的定义。
    - 这也侧面说明了，在有量子统计关联的系统中（如玻色、费米系统），系统的配分函数不能通过子系配分函数相乘得到。

#### 亥姆霍兹自由能
$ F = -Nk_B T \ln z + k_B T \ln N! $

### 能级准连续
#### 能级准连续的概念
+ 能级准连续的要求是：

$ \beta \Delta \epsilon = \frac{\Delta \epsilon}{k_B T} <<1 $

    - 其中$ \Delta \epsilon $为相邻能级之间的能量差，即能级间隔。

#### 能级准连续下的配分函数改写
+ 考虑子相空间$ \{q_1,q_2,q_3,p_1,p_2,p_3\} $中$ \Delta \epsilon $间隔内的点都具有相同的能量，相应相空间体积元为$ \Delta \omega $，那么简并度可以改写为：

$ \omega_l \rightarrow \frac{\Delta \omega}{h^3} $

+ 那么能级上粒子数分布可以改写为：

$ a_l \rightarrow e^{-\alpha - \beta \epsilon_l }(\frac{\Delta \omega}{h^3}) $

+ 配分函数可以改写为：

$ \sum_l \omega_l e^{-\beta \epsilon_l} \rightarrow \sum_l (\frac{\Delta \omega_l}{h^3})e^{-\beta \epsilon_l} \rightarrow \int \frac{d\omega}{h^3} e^{-\beta \epsilon_l} $

    - 用笛卡尔座标$ d\omega $可以写为：

$  d \omega =  d^3 \vec{q}  d^3 \vec{p} =  dxdydz dp_x dp_y dp_z $

+ 这些论述都是针对三维系统讨论的，因此写为$ h^3 $。
