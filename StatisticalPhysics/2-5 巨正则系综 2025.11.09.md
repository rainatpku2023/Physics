### 回顾：统计物理的基本假设
+ **假设1（吉布斯假设）：系统宏观量是相应微观量的系综平均值。**
    - 假设有一个物理量$ B $，微观量为$ B(q,p) $，相空间中系综概率密度为$ \rho(q,p,t) $，那么宏观量测量结果应该为：

$ \bar{B} = \int_{全相空间} B(q,p)\rho(q,p,t)dqdp $

    - 一般来说$ B(q,p) $可以直接写出，问题就转化为求解$ \rho(q,p,t) $的形式。
    - 这个表达式适用于相空间维度不变（即粒子数不变）的情形，如果维度改变应该做如下修正：

$ \bar{B} = \sum_{N = 0}^{\infin} P_N\int_{粒子数为N的全相空间} B_N(q,p)\rho_N(q,p,t)dqdp $

        * 其中$ P_N $是粒子数为$ N $的概率大小。
+ **假设2（等概率原理）：处于****<font style="color:#DF2A3F;">平衡态</font>****的****<font style="color:#DF2A3F;">孤立系统</font>****，各个可能的****<font style="color:#DF2A3F;">微观状态</font>****出现概率相同。**
    - 前提条件有：平衡态、孤立系统。
        * 说明等概率原理在平衡态下才起作用，不能讨论非平衡态。
        * 如果不是孤立系统，需要将选择更大的范围，使研究的范围孤立。正则系综的理论也是建立在微正则系综不上的。
    - 关于微观态，需要注意在非定域情形下，交换全同粒子不会产生新的微观态。

### 巨正则系综
+ 巨正则系综中的系统与大热源、大粒子源接触，因此化学势$ \mu $，体积$ V $以及温度$ T $恒定。
    - 巨正则系综中的系统不是孤立系统，孤立系统应该是$ N,V,E $恒定。

### 巨正则系综的理论建立
#### 构建孤立的复合系统
+ $ \mu,V,T $系统$ s $与大热源大粒子源$ r $接触并达到热平衡以及化学平衡，共同组成一个孤立的复合系统$ o $。
    - 系统$ s $和源$ r $之间有能量交换和粒子数交换，但是二者体积保持不变。
+ 此时能量关系为：

$ E_s + E_r = E^{(0)} $

$ E_s << E_r \approx E^{(0)} $

    - 第一个式子代表复合系统$ o $是一个孤立系统，其总能量保持不变。
    - 第二个式子表示源的能量占绝大多数。这就保持了$ \mu,V,T $系统的$ T $保持稳定。
+ 粒子数关系为：

$ N_s + N_r = N^{(0)} $

$ N_s << N_r \approx N^{(0)} $

    - 第一个式子代表复合系统$ o $是一个孤立系统，其粒子数保持不变。
    - 第二个式子表示源的粒子数占绝大多数。这就保持了$ \mu,V,T $系统的$ \mu $保持稳定。

#### 整体进行分析
+ 对于系统的一个粒子数为$ N $的特定的微观态$ S $，粒子数为$ N $能量为$ E_S $，微观态数为$ 1 $。
    - 同时源的粒子数为$ N^{(0)}-N_S $能量为$ E^{(0)}-E_S $，可以取到的微观态数为$ \Omega_r(N^{(0)}-N_S,E^{(0)}-E_S) $。
    - 因此复合系统粒子数为$ N^{(0)} $能量为$ E^{(0)} $，根据乘法原理复合系统的微观态数为$ 1 \cdot \Omega_r(N^{(0)}-N_S,E^{(0)}-E_S) $。
+ 因此
    - 与源$ r $接触的系统$ s $正好处于$ N,S $这一状态的概率$ \rho_{N,S} $等于复合系统处于上述状态的概率$ \rho_{N,S}^{(0)} $，即：
    - 复合系统是一个平衡态下的孤立系统，因此可以用等概率原理计算出复合系统在这个状态的概率$ \rho_{N,S}^{(0)} $：

$ \rho_{N,S} = \rho_{N,S}^{(0)} = \frac{1 \cdot \Omega_r(N^{(0)}-N_S,E^{(0)}-E_S)}{\Omega^{(0)}(N^{(0)},E^{(0)})} $

#### 泰勒展开
+ 为了具体分析$ N,E_S $大小对于系统$ s $处于微观态$ N,S $的概率大小$ \rho_{N,S} $的影响，可以将式子改写为：

$ \rho_{N,S}= \frac{1 }{\Omega^{(0)}(N^{(0)},E^{(0)})}\cdot \Omega_r(N^{(0)}-N_S,E^{(0)}-E_S)
\\=  \frac{1 }{\Omega^{(0)}(N^{(0)},E^{(0)})} \exp [\ln \Omega_r(E^{(0)}-E_S)\Omega_r(N^{(0)}-N_S,E^{(0)}-E_S)] $

    - 这是一个二元函数，对对数部分进行二元函数泰勒展开：

$ \ln \Omega_r(N^{(0)}-N_S,E^{(0)}-E_S) = \ln \Omega_r(N^{(0)},E^{(0)}) + \frac{\partial \ln \Omega_r}{\partial N}|_{N = N^{(0)}} (-N_S)+\frac{\partial \ln \Omega_r}{\partial E}|_{E = E^{(0)}} (-E_S)\\+ \cdots  $

    - 因为热源极大，所以可以只保留到一阶项：

$ \rho_{N,S} =  \frac{1 }{\Omega^{(0)}(N^{(0)},E^{(0)})} \exp  [\ln \Omega_r(N^{(0)},E^{(0)}) -\frac{\partial \ln \Omega_r}{\partial N}|_{N = N^{(0)}} N-\frac{\partial \ln \Omega_r}{\partial E}|_{E = E^{(0)}} E_S] $

$ =  \frac{\Omega_r(N^{(0)},E^{(0)}) }{\Omega^{(0)}(N^{(0)},E^{(0)})} \exp  [ -\frac{\partial \ln \Omega_r}{\partial N}|_{N = N^{(0)}} N -\frac{\partial \ln \Omega_r}{\partial E}|_{E = E^{(0)}} E_S] $

    - 依照之前选定的：

$ \alpha = \frac{\partial \ln\Omega_r}{\partial N} |_{N = N^{(0)}} \approx \frac{\partial \ln\Omega^{(0)}}{\partial N}|_{N = N^{(0)}}  = -\frac{\mu}{k_BT} $

$ \beta = \frac{\partial \ln\Omega_r}{\partial E} |_{E = E^{(0)}} \approx \frac{\partial \ln\Omega^{(0)}}{\partial E}|_{E = E^{(0)}}  = \frac{1}{k_BT} $

        * 可以表示为：

$ \boxed{\rho_{N,S}=  \frac{\Omega_r(N^{(0)},E^{(0)}) }{\Omega^{(0)}(N^{(0)},E^{(0)})} e^{-\alpha N-\beta E_S}} $

### 巨正则系综的配分函数
#### 巨正则系综配分函数的量子版本
+ 根据系统$ s $所有状态出现概率的总和应该为$ 1 $，模仿正则系综的配分函数，巨正则系综的配分函数$ \Xi $形式应该为：

$ \Xi = \sum_{N=0}^{\infin} \sum_{S}  e^{-\alpha N - \beta E_S} $

    - 某一个状态$ \rho_{N,S} $出现的概率为：

$ \rho_{N,S} = \frac{e^{-\alpha N - \beta E_S}}{\Xi} $

+ 值得注意的是，当粒子数$ N $改变时，能级分布也可能发生改变，所以更加严谨的写法为：

$ \boxed{\Xi = \sum_{N=0}^{\infin} \sum_{S}  e^{-\alpha N - \beta E_{N,S}}} $

    - 概率：

$ \boxed{\rho_{N,S} = \frac{e^{-\alpha N - \beta E_{N,S}}}{\Xi}} $

#### 巨正则系综配分函数的经典版本
+ **在粒子数为**$ N $**的前提下**，相空间中$ q,p $附近的概率密度为$ \rho_N(q,p) $，那么$ q,p $附近相空间区域$ dqdp $中的相空间体积元个数为：

$ \frac{dqdp}{h^{Nr}} $

    - $ q,p $附近相空间区域$ dqdp $内总概率为：

$ \rho_N(q,p)dqdp = \frac{1}{N! h^{Nr}} \frac{e^{-\beta E(q,p)}}{Z_N}dqdp $

        * 其中$ Z_N $是粒子数为$ N $时相应正则系综的配分函数。
    - 希望实现归一化：

$ \sum_{N}P_N\int_{全相空间}\rho_N(q,p)dqdp = 1 $

        * 其中$ P_N $是粒子数为$ N $的概率大小。
    - 此时配分函数为：

$ \boxed{\Xi =\sum_{N} \frac{e^{-\alpha N}}{N! h^{Nr}}\int_{全相空间} e^{-\beta E(q,p)}dqdp} $

+ <font style="color:#DF2A3F;">需要注意的是，相空间的维度与粒子数挂钩，所以粒子数变换后相空间的维度也会相应发生变化。不同粒子数的系统状态的相空间维度数不同，不可以混在一块积分。</font>

#### 巨正则系综的配分函数与正则系综的配分函数的关系
+ 巨正则系综的配分函数的形式为：

$ \Xi = \sum_{N=0}^{\infin} \sum_{S}  e^{-\alpha N - \beta E_{N,S}} $

    - 可以进行如下改写：

$ \Xi = \sum_{N=0}^{\infin} \sum_{S}  e^{-\alpha N}e^{-\beta E_{N,S}} = \sum_{N=0}^{\infin} e^{-\alpha N}\sum_{S}e^{-\beta E_{N,S}} = \sum_{N=0}^{\infin} e^{-\alpha N} Z_N $

    - 即：

$ \boxed{\Xi = \sum_{N=0}^{\infin} e^{-\alpha N} Z_N} $

    - 其中$ Z_N $是粒子数为$ N $时相应正则系综的配分函数：

$ Z_N = \sum_{S}e^{-\beta E_{N,S}} $

### 巨正则系综的热力学公式
+ 因为：

$ \Xi = \sum_{N=0}^{\infin} \sum_{S}  e^{-\alpha N - \beta E_{N,S}}, E_{N,S} =  E_{N,S}(y,N) $

+ 其中$ N $已经被求和。所以配分函数$ \Xi $是$ \alpha,\beta,E_S $的函数，也可以写为$ \alpha,\beta,\{y_i\} $的函数：

$ \Xi = \Xi(\alpha,\beta,E_{N,S})= \Xi(\alpha,\beta,E_{N,S}(y)) $

+ 因此有全微分关系：

$ d\ln \Xi = \frac{\partial \ln \Xi}{\partial \beta}  d\beta +\frac{\partial \ln \Xi}{\partial \alpha}  d\alpha +\sum_i\frac{\partial\ln \Xi}{\partial y_i}  dy_i $

#### 平均粒子数
+ 平均粒子数$ \bar{N} $是粒子数$ N $的系综平均值：

$ \boxed{\bar{N}= -\frac{\partial}{\partial \alpha}\ln \Xi} $

    - 推导过程可以类比参考[正则系综的热力学公式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/kg5tyr0l3ngs463c#xFCFP)。

#### 内能
+ 内能$ U $（即$ \bar{E}_{N,S} $）是能量$ E_{N,S} $的系综平均值：

$ \boxed{ U= -\frac{\partial}{\partial \beta}\ln \Xi} $

#### 广义力
+ 广义力$ Y $是能量对相应广义位移偏导$ \frac{\partial E_{N,S}}{\partial y} $的系综平均值：

$ \boxed{Y = -\frac{1}{\beta} \frac{\partial}{\partial y} \ln \Xi} $

#### 熵
+ 考虑到：

$ \beta(dU - \sum_{i} Y_i dy_i + \frac{\alpha}{\beta} d\bar{N}) = -\beta d(\frac{\partial \ln \Xi}{\partial\beta}) + \sum_i\frac{\partial \ln \Xi}{\partial y_i}  dy_i - \alpha d(\frac{\partial\ln \Xi}{\partial\alpha}) $

    - 此处使用$ \bar{N} $而非$ N $是为了强调这是宏观的粒子数平均值。
    - 以及：

$ d\ln \Xi = \frac{\partial \ln \Xi}{\partial \beta}  d\beta +\frac{\partial \ln \Xi}{\partial \alpha}  d\alpha +\sum_i\frac{\partial\ln \Xi}{\partial y_i}  dy_i $

    - 进行两组勒让德变换：

$ \beta(dU - \sum_{i} Y_i dy_i + \frac{\alpha}{\beta} d\bar{N}) = d (\ln \Xi - \alpha \frac{\partial \ln \Xi}{\partial \alpha}- \beta \frac{\partial \ln \Xi}{\partial \beta}) $

+ 再注意到：

$ dS = \frac{1}{T}(dU - \sum_{i} Y_i dy_i + \frac{\alpha}{\beta} d\bar{N}) = k_B \beta(dU - \sum_{i} Y_i dy_i + \frac{\alpha}{\beta} dN) $

    - 得到：

$ dS =  k_Bd (\ln \Xi - \alpha \frac{\partial \ln \Xi}{\partial \alpha}- \beta \frac{\partial \ln \Xi}{\partial \beta}) $

    - **两侧积分，并****<font style="color:#DF2A3F;">选定</font>**$ S_0 =0 $**得**：

$ \boxed{S =  k_B(\ln \Xi - \alpha \frac{\partial \ln \Xi}{\partial \alpha}- \beta \frac{\partial \ln \Xi}{\partial \beta})} $

    - 还有一个有用的表达式：

$ S= k_B(\beta U +\alpha \bar{N}+\ln \Xi) $

#### 巨势
+ 巨势的定义为：

$ \Psi = U-TS +\mu N $

    - 代入表达式为：

$ \Psi = U-TS+\mu N=
 -\frac{\partial}{\partial \beta}\ln\Xi-k_B T(\ln \Xi - \alpha \frac{\partial \ln \Xi}{\partial \alpha}- \beta \frac{\partial \ln \Xi}{\partial \beta}) - \frac{\alpha}{\beta } \frac{\partial }{\partial \alpha}\ln\Xi
= - k_BT\ln \Xi $

        * 其中用到了$ \beta = \frac{1}{k_B T} $，$ \alpha = -\frac{\mu}{k_B T} $，$ \mu= - \frac{\alpha }{\beta} $。
    - 即：

$ \boxed{\Psi 
= - k_BT\ln \Xi} $

### 再论广义熵
+ 我们下面证明，我们在[正则系综](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/kg5tyr0l3ngs463c#LGc79)理论中推导出的广义熵的表达式进行简单修改后仍然成立，即：

$ \boxed{S = -k_B\overline{\ln \rho_{N,S}} = -k_B\sum_{N}\sum_{S} \rho_{N,S} \ln \rho _{N,S}} $

+ 证明：
    - 我们之前推出，在巨正则系综中$ N,E_{N,S} $状态出现的概率大小为：

$ \rho_{N,S} = \frac{e^{-\alpha N - \beta E_{N,S}}}{\Xi} $

    - 两侧取对数：

$ \ln \rho_{N,S} = -\alpha N - \beta E_{N,S} -\ln \Xi $

    - 考察$ \ln \rho_{N,S} $的系综平均值：

$ \overline{\ln \rho_{N,S}} = \sum_{N}\sum_{S} \rho _{N,S} \ln \rho _{N,S} \\= \sum_{N}\sum_{S} \rho _{N,S}(-\alpha N - \beta E_{N,S} -\ln \Xi) =-\alpha \bar{N}-\beta \bar{E}_{N,S} - \ln \Xi  $

    - 考虑到$ E_{N,S} $的平均值$ \bar{E}_{N,S} $就是内能$ U $：

$ RHS = -\alpha \bar{N}-\beta U - \ln \Xi = -(\alpha \bar{N}+\beta U + \ln \Xi) $

    - 对比熵函数的表达式：

$ S= k_B(\beta U +\alpha N+\ln \Xi) $

    - 可见：

$ S = -k_B\overline{\ln \rho_{N,S}} = -k_B\sum_{N}\sum_{S} \rho_{N,S} \ln \rho _{N,S} $

$ \mathrm{Q.E.D.} $

### 巨正则系综的涨落
#### 回顾：物理量的涨落
+ 对于一个物理量$ B $，涨落大小一般用偏差平方平均值来描述。
    - 而且可以推出下述公式：

$ \boxed{\overline{(B - \bar{B})^2} = \bar{B^2} - \bar{B}^2} $

    - 推导过程可以参考[【未完】2-4 正则系综 2025.11.08](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/kg5tyr0l3ngs463c#lbXRA)。

#### 巨正则系综的粒子数涨落
+ 巨正则系综粒子数涨落的的衡量方式为$ \overline{(N-\bar{N})^2} $。
+ 首先考虑：

$ \frac{\partial \bar{N}}{\partial \alpha} 
= \frac{\partial }{\partial \alpha} \frac{\sum_N \sum_S N e^{-\alpha N-\beta E_{S,N}}}{\sum_N \sum_S e^{-\alpha N-\beta E_{S,N}}}
= - \frac{\sum_N \sum_S N^2 e^{-\alpha N-\beta E_{S,N}}}{\sum_N \sum_S e^{-\alpha N-\beta E_{S,N}}} +\frac{(\sum_N \sum_S N e^{-\alpha N-\beta E_{S,N}})^2}{(\sum_N \sum_S e^{-\alpha N-\beta E_{S,N}})^2} \\ $

    - 其中第一项是对分子求偏导，第二项是对分母求偏导。
    - 不难注意到：

$ RHS = -\bar{N^2}+ \bar{N}^2 = -(\bar{N^2}-\bar{N}^2) = -\overline{(N-\bar{N})^2} $

+ 因此：

$ \overline{(N-\bar{N})^2} = -(\frac{\partial \bar{N}}{\partial \alpha})_{\beta,\{y_i\}} = k_BT(\frac {\partial \bar{N}}{\partial \mu})_{T,V} $

    - 此处用到了$ \alpha = -\frac{\mu}{k_B T} $。
+ 因此相对涨落可以写为：

$ \frac{\sqrt{\overline{(N-\bar{N})^2}}}{\bar{N}} = \frac{\sqrt{k_B T}}{\bar{N}} \sqrt{(\frac{\partial N}{\partial \mu})_{T,V}} \propto \frac{1}{\sqrt{N}} $
