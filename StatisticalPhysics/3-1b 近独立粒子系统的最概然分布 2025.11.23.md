### 推导最概然分布的方法
#### 基本问题与记号约定
+ 希望求出最概然分布$ \{\bar{a}_l\} $。
+ 设：
    - 单粒子能级为$ \epsilon_1,\epsilon_2,..,\epsilon_l,... $
    - 能级简并度为$ \omega_1,\omega_2,..,\omega_l,... $
    - 粒子按能级分布为$ a_1,a_2,...,a_l,... $

#### 孤立系的粒子数和能量约束
+ 孤立系是不与外界交换能量，不与外界交换粒子的系统。
    - 系统的能量和粒子数守恒，即：

$ \delta N = \sum_{l}\delta a_l \equiv 0 $

$ \delta E = \sum_{l}\epsilon_l\delta a_l \equiv 0 $

    - 这说明$ \delta a_l $之间不是相互独立的。

#### 斯特林公式
+ 根据[Gamma函数的渐进展开](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/nsycge/dh1fmwrdeg85cgxn#ld90r)，可以知道：
+ 当$ m $很大时，有：

$ \ln m! \approx m\ln m - m $

### 三种最概然分布
#### 玻尔兹曼分布
+ [玻尔兹曼系统的微观状态数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/rx6x9vuxr04cfhxb#dmxoy)为：

$ \Omega_{MB} = \frac{N!}{\prod_l a_l!}\prod_l \omega_l^{a_l} $

+ 因此：

$ \ln \Omega_{MB} = \ln N! - \sum_l \ln a_l! +\sum_l a_l \ln \omega_l $

    - 代入斯特林公式：

$ \ln \Omega_{MB} = N\ln N - N - \sum_l a_l\ln a_l + \sum_l a_l +\sum_l a_l \ln \omega_l $

+ 极值点应该对应满足$ \delta  \ln \Omega_{MB} = 0 $的点：
    - 首先求变分：

$ 0=\delta \ln \Omega_{MB} = - \sum_l \delta a_l\ln a_l- \sum_l \delta a_l+ \sum_l \delta a_l +\sum_l \delta a_l \ln \omega_l= -\sum_l \delta a_l \ln(\frac{a_l}{\omega_l}) $

    - 问题在于$ \delta a_l $之间不是相互独立的，有两个约束（能量约束、粒子数约束）。
        * 引入拉格朗日乘子$ \alpha,\beta $：

$ 0 = 
\delta \ln \Omega_{MB} - \alpha \delta N - \beta \delta E
 $

        * 代入$ \delta N $和$ \delta E $的具体表达式：

$ RHS = -\sum_l [\ln(\frac{a_l}{\omega_l}) + \alpha +\beta \epsilon_l ]\delta a_l $

        * 引入乘子后$ \delta a_l $之间就相互独立了，前面的系数应该均为零：

$ \ln(\frac{a_l}{\omega_l}) + \alpha +\beta \epsilon_l = 0 $

    - 解得：

$ \boxed{a_l = \omega_l e^{-\alpha-\beta \epsilon_l}} $

        * 这就是**玻尔兹曼分布**。
    - 其中$ \alpha,\beta $通过如下等式确定：

$ N = \sum_l a_l = \sum_l \omega_l e^{-\alpha-\beta \epsilon_l} $

$ E = \sum_l a_l \epsilon_l = \sum_l \omega_l \epsilon_l e^{-\alpha-\beta \epsilon_l} $

+ 根据二级变分小于零可知这个稳定值对应极大值：

$ \delta^2 \ln \Omega_{MB} = -\delta [\sum_l \ln (\frac{a_l}{\omega_l})\delta a_l] = -\sum_l \frac{(\delta a_l)^2}{a_l}<0 $

    - 这里用到了粒子数大于零$ a_l>0 $的前提。

#### 玻色-爱因斯坦分布
+ [玻色系统的微观状态数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/rx6x9vuxr04cfhxb#jwx74)为：

$ \Omega_{BE} = \prod_l \frac{(\omega_l + a_l -1)!}{a_l! (\omega_l-1)!} $

+ 类似地，对两侧取对数，代入斯特林公式：

$ \ln \Omega_{BE} = \sum_l[(\omega_l + a_l -1)\ln(\omega_l + a_l -1) -a_l \ln a_l -(\omega_l-1)\ln(\omega_l-1)] $

    - 此处作一个假设$ \omega_l>>a_l $，那么有：

$ \ln \Omega_{BE} = \sum_l[(\omega_l + a_l)\ln(\omega_l + a_l) -a_l \ln a_l -\omega_l\ln\omega_l] $

+ 对两侧求变分，极大值应该对应$ \delta  \ln \Omega_{BE} = 0 $的点：

$ 0=\delta \ln \Omega_{BE} = \sum_l \ln (\frac{\omega_l+a_l}{a_l})\delta a_l $

    - 为了解除约束约束，使$ \delta a_l $相互独立，引入拉格朗日乘子$ \alpha,\beta $：

$ 0 = 
\delta \ln \Omega_{BE} - \alpha \delta N - \beta \delta E
 $

    - 代入$ \delta N $和$ \delta E $的具体表达式：

$ RHS =  \sum_l [\ln (\frac{\omega_l+a_l}{a_l}) - \alpha -\beta \epsilon_l ]\delta a_l $

    - 此时$ \delta a_l $相互独立，前面的系数应该均为零：

$ \ln (\frac{\omega_l+a_l}{a_l}) - \alpha -\beta \epsilon_l = 0 $

+ 整理得：

$ \boxed{a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}-1}} $

    - 这就是**玻色-爱因斯坦分布**。
+ 其中$ \alpha,\beta $通过如下等式确定：

$ N = \sum_l a_l = \sum_l \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}-1} $

$ E = \sum_l a_l \epsilon_l = \sum_l   \frac{\omega_l\epsilon_l}{e^{\alpha+\beta \epsilon_l}-1} $

#### 费米-狄拉克分布
+ [费米系统的微观状态数为](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/rx6x9vuxr04cfhxb#Dqp4b)为：

$ \Omega_{FD} = \prod_l \frac{\omega_l!}{a_l!(\omega_l-a_l)!} $

+ 对两侧取对数，代入斯特林公式：

$ \ln \Omega_{FD} = \sum_l[\omega_l\ln\omega_l  -a_l \ln a_l -(\omega_l-a_l)\ln(\omega_l-a_l)] $

+ 对两侧求变分，极大值应该对应$ \delta  \ln \Omega_{FD} = 0 $的点：

$ 0=\delta \ln \Omega_{FD} = \sum_l \ln (\frac{\omega_l-a_l}{a_l})\delta a_l $

    - 为了解除约束约束，使$ \delta a_l $相互独立，引入拉格朗日乘子$ \alpha,\beta $：

$ 0 = 
\delta \ln \Omega_{FD} - \alpha \delta N - \beta \delta E
 $

    - 代入$ \delta N $和$ \delta E $的具体表达式：

$ RHS =  \sum_l [\ln (\frac{\omega_l-a_l}{a_l}) - \alpha -\beta \epsilon_l ]\delta a_l $

    - 此时$ \delta a_l $相互独立，前面的系数应该均为零：

$ \ln (\frac{\omega_l-a_l}{a_l}) - \alpha -\beta \epsilon_l = 0 $

+ 整理得：

$ \boxed{a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}+1}} $

    - 这就是**费米-狄拉克分布**。
+ 其中$ \alpha,\beta $通过如下等式确定：

$ N = \sum_l a_l = \sum_l \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}+1} $

$ E = \sum_l a_l \epsilon_l = \sum_l   \frac{\omega_l\epsilon_l}{e^{\alpha+\beta \epsilon_l}+1} $

### 三种分布的关系
+ 根据上文，费米-狄拉克分布和玻色-爱因斯坦分布可以同一写为：

$ a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}\mp1} $

    - 玻色分布对应减号，费米分布对应加号。
    - $ e^\alpha>>1 $时，有：

$ a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}\mp1} 
\approx \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}} 
= \omega_l e^{-\alpha-\beta \epsilon_l} $

    - 可见，在经典极限条件下费米分布和玻色分布过渡为玻尔兹曼分布。
+ 但是，因为费米子和玻色子受到全同性的影响，微观状态数上还是有差异：

$ \Omega_{BE} \approx \Omega_{FD} \approx \frac{\Omega_{MB}}{N!} $

    - 这个$ 1/N! $的倍数就是全同粒子造成的。之后会看到，正是这个因子保证了熵严格是广延量。
