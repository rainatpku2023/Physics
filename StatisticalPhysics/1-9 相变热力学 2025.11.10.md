### 平衡相变
+ 平衡相变：系统**缓慢地**从外界吸收或者放出热量，从一个相变为另外一个相，而**始终保持平衡态**。
+ 根据[单元复相系的平衡条件](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/ngr167yhcg1kbdga)，相平衡曲线/相变曲线上：温度、压强、化学势相等。

$ T^\alpha = T^\beta $

$ p^\alpha = p^\beta $

$ \mu^\alpha = \mu^\beta $

### 一级相变：克拉伯龙方程
#### 克拉伯龙方程的导出
+ 相平衡曲线上，应该处处满足化学势相等。
    - 化学势$ \mu $可以看成温度$ T $以及压强$ p $的函数。

$ \mu^\alpha(T,p)=\mu^\beta(T,p) $

$ \mu^\alpha(T+dT,p+dp)=\mu^\beta(T+dT,p+dp) $

+ 两边相减得到：

$ d\mu^\alpha=d\mu^\beta $

+ 将化学势的全微分写为：

$ d\mu = -sdT + vdp $

+ 那么：

$ -s^\alpha dT + v^\alpha dp = -s^\beta dT + v^\beta dp $

+ 移项并相除：

$ \frac{dp}{dT} = \frac{s^\beta-s^\alpha}{v^\beta-v^\alpha} $

+ 温度确定后，相变潜热可以表示为：

$ \boxed{L_{\alpha\beta} = T(s^\beta - s^\alpha)} $

+ 代入后得到克拉伯龙方程：

$ \boxed{\frac{dp}{dT} = \frac{L_{\alpha\beta}}{T(v^\beta-v^\alpha)}} $

#### 克拉伯龙方程的应用
+ 应用1：物质发生溶解、蒸发、生化吸热$ L_{\alpha\beta}>0 $。
    - 一般地，$ \Delta v_{\alpha\beta} = v^\beta - v^\alpha >0 $，此时有$ \frac{dp}{dT}>0 $。
    - 对于冰的熔化过程，$ \Delta v_{\alpha\beta} = v^\beta - v^\alpha <0 $，此时有$ \frac{dp}{dT}<0 $。
        * <font style="color:#8A8F8D;">例子：滑冰冰刀下出现水印，这是因为压强增大，相平衡温度下降，冰变为液态。</font>
+ 应用2：饱和蒸气压方程。
    - 饱和蒸气：与液相或者固相达到平衡的蒸汽。
    - 推导思路：为了推导$ p-T $的函数关系，将$ L_{\alpha\beta} $看成不变的参数。
    - 因为液相的摩尔体积远小于气相$ v_\beta >> v_\alpha $，气态用理想气体近似$ pv^\beta = RT $，代入知：

$ \frac{dp}{dT} = \frac{L_{\alpha\beta}}{Tv^\beta} = \frac{pL_{\alpha\beta}}{RT^2} $

        * 移项得：

$ \frac{dp}{p} = \frac{L_{\alpha\beta}dT}{RT^2} $

        * 两侧积分得：

$ \ln p = -\frac{L_{\alpha\beta}}{RT} + A $

### 相变分类
#### 相变分类
+ 一级相变：化学势连续，化学势的一级偏导不连续。
+ 二级相变：化学势和化学势的以及偏导连续，二阶偏导不连续。
+ $ n $**级相变：化学势和化学势的前**$ n-1 $**级偏导数连续，**$ n $**级偏导不连续。**
+ 连续相变：二级及以上的相变。

#### 一级相变
+ 化学势的表达式为：

$ d\mu = -sdT + vdp $

    - 因此：

$ s = -(\frac{\partial \mu}{\partial T})_p,\; v =( \frac{\partial \mu}{\partial p})_T  $

    - 一级相变化学势一节偏导数不相同：

$ s^\alpha \neq s^\beta $

$ v^\alpha \neq v^\beta $

    - 可见一级相变有体积突变，且存在相变潜热：

$ \Delta v_{\alpha\beta} = v^\beta - v^\alpha \neq 0 $

$ L_{\alpha\beta} = T(s^\beta - s^\alpha) \neq 0 $

#### 二级相变
+ 二级相变，化学势的一级偏导数连续，因此没有体积突变，也没有相变潜热：

$ \Delta v_{\alpha\beta} = v^\beta - v^\alpha = 0 $

$ L_{\alpha\beta} = T(s^\beta - s^\alpha) = 0 $

+ 可以证明，定压比热、定压膨胀系数、等温压缩系数存在跃变：

$ c_p = T (\frac{\partial s}{\partial T})_p = -T \frac{\partial^2\mu}{\partial T^2},\;c_p^\alpha \neq c_p^\beta $

$ \alpha = \frac{1}{v} (\frac{\partial v}{\partial T})_p = \frac{1}{v} \frac{\partial^2\mu}{\partial p \partial T}, \; \alpha^{\alpha} \neq \alpha^\beta $

$ \kappa_T = -\frac{1}{v}(\frac{\partial v}{\partial p})_T = -\frac{1}{v} \frac{\partial^2u}{\partial p^2},\; \kappa_T^\alpha \neq \kappa_T^\beta $

### 二级相变：埃伦费斯特方程
#### 埃伦费斯特方程的导出
+ 我们在[克拉伯龙方程的推导过程](#twU3x)中导出：

$ \frac{dp}{dT} = \frac{s^\beta-s^\alpha}{v^\beta-v^\alpha} $

+ 在二级相变中，因为化学势的一级偏导（例如摩尔熵$ s $和摩尔体积$ v $连续），因此这个比值是$ \frac{0}{0} $的未定式。考虑使用**洛必达法则**：



+ 对温度求偏导：

$ \frac{dp}{dT} = \frac{(\frac{\partial s^\beta}{\partial T})_p-(\frac{\partial s^\alpha}{\partial T})_p}{(\frac{\partial v^\beta}{\partial T})_p-(\frac{\partial v^\alpha}{\partial T})_p} = \frac{c_p^\beta-c_p^\alpha}{Tv(\alpha^\beta-\alpha^\alpha)} = \frac{\Delta c_p}{Tv\Delta\alpha} $



+ 对压强求偏导：

$ \frac{dp}{dT} = \frac{(\frac{\partial s^\beta}{\partial p})_T-(\frac{\partial s^\alpha}{\partial p})_T}{(\frac{\partial v^\beta}{\partial p})_T-(\frac{\partial v^\alpha}{\partial p})_T} = \frac{\alpha^\beta -\alpha^\alpha}{\kappa_T^\beta-\kappa_T^\alpha} = \frac{\Delta \alpha}{\Delta \kappa_T} $

#### 埃伦费斯特方程的推论
+ 根据方程自洽，$ dp/dT $应该只能取一个值，即：

$ \frac{\Delta c_p}{Tv\Delta\alpha} = \frac{\Delta \alpha}{\Delta \kappa_T} $
