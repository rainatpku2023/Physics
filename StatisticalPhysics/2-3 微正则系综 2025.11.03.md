### 微正则系综
+ 微正则系综中含有大量结构相同（哈密顿量表达式相同），孤立的系统。

### 微正则系综的理论建立
+ 考虑一个孤立系统$ A^{(0)} $，它由微弱相互作用的两个系统$ A_1 $和$ A_2 $构成。
    - $ \Omega_1(N_1,E_1,V_1) $：$ A_1 $在粒子数为$ N_1 $，能量为$ E_1 $，体积为$ V_1 $时的微观状态数。
    - $ \Omega_2(N_2,E_2,V_2) $：$ A_2 $在粒子数为$ N_2 $，能量为$ E_2 $，体积为$ V_2 $时的微观状态数。

#### 子系统只有能量交换的情形 与 热平衡
+ 考虑$ A_1,A_2 $只有能量交换的情形，此时孤立系统$ A^{(0)} $的微观状态数可以由两个子系统微观状态数的乘积表达：

$ \Omega^{(0)}(E_1,E_2) = \Omega_1(E_1)\Omega_2(E_2)
 $

+ 考虑到总系统是孤立系统，两个子系统的能量总和不变：

$ E_1 + E_2 = E^{(0)} $

    - 可以将$ \Omega^{(0)} $进行改写：

$ \Omega^{(0)}(E_1,E^{(0)}-E_1) = \Omega_1(E_1)\Omega_2(E^{(0)}-E_1) $

    - 这个孩子表明，对于给定的总能量$ E^{(0)} $，系统的总状态数$ \Omega^{(0)} $取决于能量$ E^{(0)} $在两个子系统$ A_1,A_2 $之间的分配。
+ 根据等概率原理，平衡态下孤立系统一切可能的微观状态出现的概率都相等。
    - 因此，当$ E_1 $达到平衡态$ \bar{E}_1 $时，$ \Omega^{(0)} $应该取到极大值。
    - 此时我们对（1）两端对$ E_1 $求全导数，使之为零：

$ \frac{\partial \Omega^{(0)}}{\partial E_1} = 0 $

$ \frac{\partial \Omega_1(E_1)}{\partial E_1} \Omega_2(E_2) +\Omega_1(E_1) \frac{\partial \Omega_2(E_2)}{\partial E_2} \frac{\partial E_2}{\partial E_1} = 0 $

+ 整个式子用$ \Omega_1(E_1)\Omega_2(E_2) $，并考虑到$ \frac{\partial E_2}{\partial E_1}=-1 $，得到：

$ \boxed{
[\frac{\partial \ln \Omega_1(E_1)}{\partial E_1}]_{N_1,V_1} = [\frac{\partial \ln \Omega_2(E_2)}{\partial E_2}]_{N_2,V_2}} $

    - 这个式子确定了平衡时的关系。
+ 此时引入$ \beta $物理量来进行描述：

$ \boxed{\beta = [\frac{\partial \ln \Omega(N,E,V)}{\partial E}]_{N,V}} $

    - 此时热平衡条件可以描述为：

$ \beta_1 = \beta_2 $

#### 子系统只有粒子交换的情形 与 化学平衡
+ 同理考虑$ A_1,A_2 $只有粒子交换的情形，此时孤立系统$ A^{(0)} $的微观状态数可以由两个子系统微观状态数的乘积表达：

$ \Omega^{(0)}(N_1,N_2) = \Omega_1(N_1)\Omega_2(N_2)
 $

+ 按照相似的理论，可以得到平衡时的表达式：

$ \boxed{
[\frac{\partial \ln \Omega_1(N_1)}{\partial N_1}]_{E_1,V_1} = [\frac{\partial \ln \Omega_2(N_2)}{\partial N_2}]_{E_2,V_2}} $

    - 这个式子确定了平衡时的关系。
+ 此时引入$ \alpha $物理量来进行描述：

$ \boxed{\alpha = [\frac{\partial \ln \Omega(N,E,V)}{\partial N}]_{E,V}} $

    - 此时化学平衡条件可以描述为：

$ \alpha_1 = \alpha_2 $

#### 子系统只有体积变化的情形 与 力学平衡
+ 同理考虑$ A_1,A_2 $只有体积变化的情形，此时孤立系统$ A^{(0)} $的微观状态数可以由两个子系统微观状态数的乘积表达：

$ \Omega^{(0)}(V_1,V_2) = \Omega_1(V_1)\Omega_2(V_2)
 $

+ 按照相似的理论，可以得到平衡时的表达式：

$ \boxed{
[\frac{\partial \ln \Omega_1(V_1)}{\partial V_1}]_{E_1,N_1} = [\frac{\partial \ln \Omega_2(V_2)}{\partial V_2}]_{E_2,N_2}} $

    - 这个式子确定了平衡时的关系。
+ 此时引入$ \gamma $物理量来进行描述：

$ \boxed{\gamma = [\frac{\partial \ln \Omega(N,E,V)}{\partial V}]_{E,N}} $

    - 此时力学平衡条件可以描述为：

$ \gamma_1 = \gamma_2 $

### 从微观到宏观：热力学函数的计算
#### 熵函数
+ 根据内能的全微分：

$ dU = TdS - pdV +\mu dN $

    - 可以得到：

$ (\frac{\partial S}{\partial U})_{N,V} = \frac{1}{T} $

+ 考虑到对于微观状态数对数的计算：

$ \beta = [\frac{\partial \ln \Omega(N,E,V)}{\partial E}]_{N,V} $

+ 可见$ \beta $和$ 1/T $应该成正比，设比值为$ 1/k_B $，那么得到：

$ \boxed{
\beta = \frac{1}{k_B T}} $

    - 其中$ k_B $是**玻尔兹曼常数**。
    - 可见熵可以用微观状态数的对数进行表示：

$ \boxed{S = k_B\ln\Omega} $

#### 压强、温度
+ 可见，熵具有特别地意义，下面写出熵的全微分：

$ dS = \frac{dU}{T} + \frac{p}{T}dV-\frac{\mu}{T}dN $

    - 并考虑$ \ln \Omega $的全微分：

$ \boxed{d\ln \Omega =  \beta dE +\gamma dV + \alpha dN} $

    - 这样就确定了$ \alpha,\gamma $的物理意义：

$ \boxed{\gamma = \frac{p}{k_B T}} $$ \boxed{\alpha = -\frac{\mu}{k_B T}} $

+ 根据统计平衡条件：$ \beta_1 = \beta_2,\alpha_1 = \alpha_2,\gamma_1 = \gamma_2 $
+ 也可以推出热力学角度的平衡条件：$ T_1 = T_2, p_1 = p_2, \mu_1 = \mu_2 $
