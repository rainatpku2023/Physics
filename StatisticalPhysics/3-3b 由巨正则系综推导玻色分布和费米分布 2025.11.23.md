### 问题概述
+ 希望用巨正则系综理论推导出玻色分布和费米分布。
+ 设：
    - 单粒子能级为$ \epsilon_1,\epsilon_2,..,\epsilon_l,... $
    - 能级简并度为$ \omega_1,\omega_2,..,\omega_l,... $
    - 粒子按能级分布为$ a_1,a_2,...,a_l,... $

### 能级简并度
+ 玻色系统、费米系统、不可分辨的玻尔兹曼系统的状态数分别为：

$ \Omega_{BE} = \prod_l \frac{(\omega_l +a_l-1)!}{a_l ! (\omega_l-1)!} $

$ \Omega_{FD} = \prod_l \frac{\omega_l !}{a_l ! (\omega_l-a_l)!} $

+ 由此，得到玻色系统、费米系统、不可分辨的玻尔兹曼系统第$ i $个能级的状态数分别为：

$ W_{l,BE}(a_l,\omega_l) = \frac{(\omega_l +a_l-1)!}{a_l ! (\omega_l-1)!} $

$ W_{l,FD}(a_l,\omega_l) = \frac{\omega_l !}{a_l ! (\omega_l-a_l)!} $

### 巨配分函数
#### 引入能级的巨配分函数
+ 想写出一个巨正则系综的配分函数，需要遍历所有的情形。
    - 如果已经确定了一个能级$ \epsilon_l $上的粒子数为$ a_l $，能级简并度为$ \omega_l $，那么这个能级的状态数为$ W_l $，未归一化概率为：

$ W_l e^{-(\alpha+\beta \epsilon_l)a_l} $

    - 如果已知粒子在能级$ \{\epsilon_l\} $上的分布为$ \{a_l\} $，那么这种分布的总的未归一化概率密度为：

$ \prod_l[W_l e^{-(\alpha+\beta \epsilon_l)a_l}] $

    - 巨正则系综的配分函数需要遍历所有的分布，因此巨正则系综的巨配分函数为：

$ \boxed{
\Xi = \sum_{\{a_l\}} \prod_l[W_l e^{-(\alpha+\beta \epsilon_l)a_l}]} $

+ 注意到巨正则系综中粒子数$ a_1,a_2,...,a_l,... $均可以独立变化，因此可以改为逐个遍历所有的能级：

$ 
\Xi = \sum_{a_1} \sum_{a_2}\cdots  \sum_{a_l}\cdots  [W_1 e^{-(\alpha+\beta \epsilon_1)a_1}][W_2 e^{-(\alpha+\beta \epsilon_2)a_2}]\cdots [W_l e^{-(\alpha+\beta \epsilon_l)a_l}] \cdots  $

$ =[\sum_{a_1}W_1 e^{-(\alpha+\beta \epsilon_1)a_1}][\sum_{a_2}W_2 e^{-(\alpha+\beta \epsilon_2)a_2}]
\cdots [\sum_{a_l}W_l e^{-(\alpha+\beta \epsilon_l)a_l}]\cdots  $

$ =
 \prod_l[\sum_{a_l}W_l e^{-(\alpha+\beta \epsilon_l)a_l}]
= \prod_l \Xi_l $

    - 此处引入了能级的巨配分函数：

$ \boxed{
\Xi_l =\sum_{a_l} [W_l e^{-(\alpha+\beta \epsilon_l)a_l}]} $

#### 玻色系统的能级巨配分函数
+ 代入玻色分布，结合玻色子不遵循泡利不相容原理，能级上粒子数可以到正无穷大：

$ \Xi_{l,BE} =\sum_{a_l} W_l e^{-(\alpha+\beta \epsilon_l)a_l} = \sum_{a_l=0}^{\infin} \frac{(\omega_l +a_l-1)!}{a_l ! (\omega_l-1)!} e^{-(\alpha+\beta \epsilon_l)a_l}  $

    - 考虑下面的数学关系：

$ f_m(x) = (1-x)^{-m} = [1+(-x)]^{-m} = \sum_{n=0}^{\infin} \frac{f_m^{(n)}(1)}{n!}(-x)^n = \sum_{n=0}^{\infin} \frac{(m+n-1)!}{n!(m-1)!}x^n $

+ 可以立即得到玻色系统的能级巨配分函数：

$ \boxed{\Xi_{l,BE} =[1-e^{-(\alpha+\beta \epsilon_l)}] ^{-\omega_l}} $

    - 玻色系统的巨配分函数：

$ \boxed{\Xi_{BE} =\prod_l[1-e^{-(\alpha+\beta \epsilon_l)}] ^{-\omega_l}} $

#### 费米系统的能级巨配分函数
+ 代入费米分布，结合费米子遵循泡利不相容原理，能级上粒子数不可能超过能级简并度的结论：

$ \Xi_{l,FD} =\sum_{a_l} W_l e^{-(\alpha+\beta \epsilon_l)a_l} = \sum_{a_l=0}^{\omega_l} \frac{\omega_l !}{a_l ! (\omega_l-a_l)!} e^{-(\alpha+\beta \epsilon_l)a_l}  $

    - 注意到等号右边是二项分布：

$ RHS = \sum_{a_l=0}^{\omega_l} \begin{pmatrix}\omega_l \\a_l\end{pmatrix} e^{-(\alpha+\beta \epsilon_l)a_l} = [1+e^{-(\alpha+\beta \epsilon_l)}] ^{\omega_l} $

+ 因此费米系统的能级巨配分函数为：

$ \boxed{\Xi_{l,FD} =[1+e^{-(\alpha+\beta \epsilon_l)}] ^{\omega_l}} $

    - 费米系统的巨配分函数为：

$ \boxed{\Xi_{FD} =\prod_l[1+e^{-(\alpha+\beta \epsilon_l)}] ^{\omega_l}} $

### 能级平均粒子数
#### 根据能级巨配分函数计算能级平均粒子数
+ 使用巨配分函数进行计算平均粒子数，应该遍历所有的总能量、所有的总粒子数。
+ 此处改为依次遍历所有的分布情形，然后再在每一个分布情形内遍历能量和粒子数：

$ \Xi = \sum_{\{a_l\}} \prod_l[W_l e^{-(\alpha+\beta \epsilon_l)a_l}] $

$ \bar{a}_k = \frac{1}{\Xi}\sum_{\{a_l\}} a_k\prod_l[W_l e^{-(\alpha+\beta \epsilon_l)a_l}] $

    - 每个分布的概率按照能级进行遍历。
+ 展开后：

$ \bar{a}_k = \frac{\sum_{\{a_l\}} a_k\prod_l[W_l e^{-(\alpha+\beta \epsilon_l)a_l}]}{\sum_{\{a_l\}} \prod_l[W_l e^{-(\alpha+\beta \epsilon_l)a_l}]}

=\frac{[\sum_{a_k}a_kW_k e^{-(\alpha+\beta \epsilon_k)a_k}]\prod_{l \neq k}[\sum_{a_l}W_l e^{-(\alpha+\beta \epsilon_l)a_l}]}{[\sum_{a_k}W_k e^{-(\alpha+\beta \epsilon_k)a_k}]\prod_{l \neq k}[\sum_{a_l}W_l e^{-(\alpha+\beta \epsilon_l)a_l}]} $

    - 注意到不涉及$ k $能级的部分可以消掉：

$ RHS=\frac{\sum_{a_k}a_kW_k e^{-(\alpha+\beta \epsilon_k)a_k}}{\sum_{a_k}W_k e^{-(\alpha+\beta \epsilon_k)a_k}} $

+ 考虑到：

$ 
\Xi_l =\sum_{a_l} [W_l e^{-(\alpha+\beta \epsilon_l)a_l}] $

    - 可以进行进一步改写：

$ RHS=\frac{1}{\Xi_k}(-\frac{\partial}{\partial \alpha} \sum_{a_k}W_k e^{-(\alpha+\beta \epsilon_k)a_k}) = -\frac{\partial}{\partial \alpha}\ln \Xi_k $

+ 也就得到了第$ k $个能级的分布粒子数表达式：

$ \boxed{
\bar{a}_k= -\frac{\partial}{\partial \alpha}\ln \Xi_k} $

#### 玻色-爱因斯坦分布
+ 考虑到玻色系统的能级的巨配分函数为：

$ \Xi_{l,BE} =[1-e^{-(\alpha+\beta \epsilon_l)}] ^{-\omega_l} $

+ 代入平均粒子数表达式后得到玻色分布：

$ 
\bar{a}_{k,BE}
= -\frac{\partial}{\partial \alpha}\ln \Xi_{k,BE} 
= -\frac{\partial}{\partial \alpha}\ln \{[1-e^{-(\alpha+\beta \epsilon_k)}] ^{-\omega_k}\} $

$ =-\frac{\partial}{\partial \alpha}\{-\omega_k\ln [1-e^{-(\alpha+\beta \epsilon_k)}]\}
= \frac{\omega_k e^{-(\alpha + \beta\epsilon_k)}}{1-e^{-(\alpha + \beta\epsilon_k)}}

= \frac{\omega_k }{e^{\alpha + \beta\epsilon_k}-1} $

    - 玻色-爱因斯坦分布：

$ \boxed{\bar{a}_{l,BE}
= \frac{\omega_l }{e^{\alpha + \beta\epsilon_l}-1}} $

#### 费米-狄拉克分布
+ 考虑到费米系统的能级的巨配分函数为：

$ \Xi_{l,FD} =[1+e^{-(\alpha+\beta \epsilon_l)}] ^{\omega_l} $

+ 代入平均粒子数表达式后得到玻色分布：

$ 
\bar{a}_{k,FD}
= -\frac{\partial}{\partial \alpha}\ln \Xi_{k,FD} 
= -\frac{\partial}{\partial \alpha}\ln \{[1+e^{-(\alpha+\beta \epsilon_k)}] ^{\omega_k}\} $

$ =-\frac{\partial}{\partial \alpha}\{\omega_k\ln [1+e^{-(\alpha+\beta \epsilon_k)}]\}
= \frac{\omega_k e^{-(\alpha + \beta\epsilon_k)}}{1+e^{-(\alpha + \beta\epsilon_k)}}

= \frac{\omega_k }{e^{\alpha + \beta\epsilon_k}+1} $

    - 费米-狄拉克分布：

$ \boxed{\bar{a}_{l,FD}
= \frac{\omega_l }{e^{\alpha + \beta\epsilon_l}+1}} $

### 总结
+ 用正则系综推导玻耳兹曼分布，将总的配分函数分解成了各个粒子子系配分函数的乘积，前提是粒子之间没有相互作用（包括力学相互作用，以及量子统计效应两部分）。
+ 用巨正则系综推导玻色分布和费米分布，将总的巨配分函数分解成各个能级的巨配分函数，用到了能级之间没有力学相互作用以及量子统计等效相互作用的结论。



+ 求系统配分函数的基本思路是将其化成子系统配分函数的乘积
    - 子系统可以选定为粒子，此时找子系配分函数。
    - 子系统可以选定为能级，此时找能级的巨配分函数。
    - 前提都是子系统之间相互独立，在力学相互作用上、在统计效应上都是如此。
