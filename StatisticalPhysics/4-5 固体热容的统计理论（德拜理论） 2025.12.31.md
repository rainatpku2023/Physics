### 理论图像
+ 低温极限下，[固体热容的爱因斯坦理论](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/uozgwdcrskhkol1s#LELLE)描述的热容趋于零行为不正确。
    - 错误原因是将$ N $个原子看成了相互独立的$ 3N $个振子，实际上在低温下这些振子并不是独立振动的。
+ 德拜模型将振动看成固体中的驻波。
    - 在低温极限下，低频、长周期振动起主导作用，此时会给出与实验更加契合的$ C_V\propto T^3 $。
    - 在高温极限下，高频、短周期振动起主导作用，热容会回到杜隆珀替定律。

### 态密度函数
#### 周期性边界条件
+ 一维模型，取周期性边界条件，那么当固体驻波有$ n_x $个周期时，其波数为：

$ k_x = \frac{2\pi n_x}{L} $

    - 那么对于波矢在$ \Delta k $范围内的状态数为：

$ \Delta n_x = \frac{\Delta k_x}{2\pi/L} = \frac{L}{2\pi}\Delta k_x $

    - 如果连续性足够好，可以将微小变化化为微分：

$ d n_x = \frac{d k}{2\pi/L} = \frac{L}{2\pi}d k_x $

+ 三维模型，取周期性边界条件：

$ d n_xd n_yd n_z  = \left(\frac{L}{2\pi}d k_x\right)\left(\frac{L}{2\pi}d k_y\right)\left(\frac{L}{2\pi}d k_z\right) = \frac{L}{8\pi^3}dk_xdk_ydk_z $

    - 在波矢空间内，先对极角与方位角进行积分，得到：

$ g(k)dk = \frac{L}{8\pi^3}4\pi k dk = \frac{V}{2\pi^2}k^2dk $

#### 考虑到横波和纵波
+ 对于一个给定的波矢$ \vec{k} $，三维固体中的声波可以有三个不同的偏振方向：
    - 一个纵偏振和两个横偏振。
    - 二者的传播速度不同，速度分别设为$ c_l,c_t $，其表达式为：

$ \omega = c_l k,\; \omega = c_t k $

+ 借鉴黑体辐射简正振动数目为$ \frac{4\pi Vdk}{(2\pi)^3} $，两种声子的总量子态数为：

$ g(\omega)d\omega = \frac{V}{2\pi^2} \left( \frac{1}{c_l^3}+\frac{2}{c_t^3}\right)\omega ^2d\omega $

    - 其中$ \omega $为圆频率，和时间频率$ f $的关系为$ \omega = 2\pi f $。

### 德拜截止频率
+ 固体总振动的自由度不是无穷大，应该等于三维固体总的振动自由度$ 3N $，所以必须有：

$ \int_0^{\omega_D} g(\omega)d\omega = 3N $

    - 其中$ \omega_D $是德拜引入的截止频率，称为**德拜截止频率**。
+ 设：

$ g(\omega)d\omega = \frac{V}{2\pi^2} \left( \frac{1}{c_l^3}+\frac{2}{c_t^3}\right)\omega ^2d\omega = B\omega^2d\omega $

    - 代入积分：

$ \int_0^{\omega_D} B\omega^2d\omega = 3N $

    - 积分得：

$ \frac{1}{3}B\omega^3 = 3N $

    - 整理得：

$ \omega_D^3 = \frac{9N}{B} $

+ 态密度函数可以进一步改写为：

$ g(\omega)d\omega = \frac{9N}{\omega_D^3}\omega^2 d\omega $

### 内能的统计表达式
+ 结合[振子的能量公式](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/uozgwdcrskhkol1s#B4NmL)，算出配分函数。
    - 进一步可以得出单个振子的能量期望：

$ \bar{\epsilon} = \frac{1}{2}\hbar \omega +\frac{\hbar{\omega}}{e^{\frac{\hbar \omega}{k_B T}}-1} $

+ 总内能的表达式为：

$ U = \sum_{i=1}^{3N} \frac{1}{2}\hbar \omega_i +\sum_{i=1}^{3N}\frac{\hbar{\omega_i}}{e^{\frac{\hbar \omega_i}{k_B T}}-1} $

    - 其中$ i $是指第$ i $个简正振动模式，而非第$ i $小的圆频率。
    - 此处是对简正振动模式进行遍历，而非对于振动频率大小进行遍历。
+ 在能级准连续的假设下，求和化为积分：

$ U = \int_0^{\omega_D} \frac{\hbar\omega}{2} g(\omega)d\omega + \int_0^{\omega_D}\frac{\hbar{\omega}}{e^{\frac{\hbar \omega}{k_B T}}-1}g(\omega)d\omega $

+ 代入$ g(\omega) $具体的表达式：

$ RHS = \frac{9N\hbar}{\omega_D^3}\int_0^{\omega_D}\frac{\omega}{2}\omega^2d\omega
+\frac{9N\hbar}{\omega_D^3}\int_0^{\omega_D}\frac{\omega}{e^{\hbar\omega/k_BT}-1}\omega^2d\omega $

+ 第一项直接积分出来，第二项作变数变换：$ x= \hbar\omega/k_B T,x_D= \hbar\omega_D/k_B T $：

$ RHS = \frac{9N\hbar}{\omega_D^3}\frac{\omega_D^4}{8}+ \frac{9N\hbar}{\omega_D^3}\frac{(k_B T)^4}{\hbar^4} \int_0^{x_D} \frac{x^3}{e^x-1}dx $

$ =\frac{9}{8}N\hbar \omega_D+3Nk_B T\left[3\left(\frac{k_BT}{\hbar \omega_D}\right)^3\int_0^{x_D}\frac{x^3}{e^x-1}dx\right] $

+ 定义**德拜温度**$ \theta_D $：

$ \theta_D = \frac{\hbar\omega_D}{k_B} $

    - 以及**德拜函数**$ D(x) $：

$ D(x) = \frac{3}{x^3}\int_0^x \frac{y^3}{e^y-1}dy $

+ 那么内能可以写为：

$ \boxed{U = U_0 + 3Nk_B T D \left(\frac{\theta_D}{T} \right)} $

    - 其中零点内能$ U_0 $的表达式为：

$ U_0 = \frac{9}{8}N\hbar \omega_D $

    - 德拜函数：

$ D \left(\frac{\theta_D}{T} \right) = 3 \left(\frac{\theta_D}{T} \right)^{-3} \int_0^{\theta_D/T}\frac{x^3}{e^x-1}dx $

#### 内能的高温极限和低温极限
+ 在**高温极限**下$ \frac{\theta}{T}<<1 $：

$ D \left(\frac{\theta_D}{T} \right) = 3 \left(\frac{\theta_D}{T} \right)^{-3} \int_0^{\theta_D/T}\frac{x^3}{e^x-1}dx \approx3 \left(\frac{\theta_D}{T} \right)^{-3} \int_0^{\theta_D/T}\frac{x^3}{x}dx = 1 $

    - 内能：

$ U \approx U_0 + 3Nk_B T  $

    - 定容热容：

$ C_V = \left( \frac{\partial U}{\partial T}\right)_V \approx 3Nk_B $

+ 在**低温极限**下$ \frac{\theta}{T}>>1 $：

$ D \left(\frac{\theta_D}{T} \right) = 3 \left(\frac{\theta_D}{T} \right)^{-3} \int_0^{\theta_D/T}\frac{x^3}{e^x-1}dx \approx 3\left(\frac{\theta_D}{T} \right)^{-3} \int_0^{\infin}\frac{x^3}{e^x-1}dx= 3\left(\frac{\theta_D}{T} \right)^{-3} \frac{\pi^4}{15} $

    - 内能：

$ U \approx U_0 + 3Nk_B  \frac{\pi^4}{5} \frac{T^4}{\theta_D^3} $

    - 定容热容：

$ C_V = \left( \frac{\partial U}{\partial T}\right)_V \approx \frac{12\pi^4}{5}N k_B \left( \frac{T}{\theta_D}\right)^3 $

    - 这就是德拜$ T^3 $律，在低温时热容近似以$ \propto T^3 $的规律趋近于零。
