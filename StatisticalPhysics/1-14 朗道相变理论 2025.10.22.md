### 序参量和对称性破缺
+ 二级相变的本质是对称性发生了自发破缺
    - 朗道理论中，用于描述系统对称性的变量称为序参量。
    - 本文中用$ \eta $表示序参量。
+ 一般地，认为：
    - 高温相：高对称相，更加无序，$ \eta = 0 $
    - 低温相：低对称相，比较有序，$ \eta \neq 0 $

### 二级相变的朗道理论
#### 热力学函数对序参量的展开
+ 在二级相变附近，序参量$ \eta $是小量，考虑热力学函数$ \Omega $：

$ \Omega(T,\Delta,\eta) = \Omega_0 + \frac{a(T,\Delta)}{2}\eta^2 + \frac{b(T,\Delta)}{4}\eta^4 + \cdots $

    - $ \Delta $是其他状态参量，例如$ p,V,\mathcal{H} $
    - 热力学函数一般选为自由能$ F $或者是自由焓$ G $。
    - 这个式子中假定了系统具有$ \eta \rightarrow -\eta $的对称性，因此只有偶数次幂。
    - 注意前面的系数与$ T,\Delta $有关，也就是说$ \Omega \sim \eta $的函数关系与温度有关。

#### 参数a,b的形式
+ $ T<T_c $时，平衡的系统处于低对称相，序参量$ \bar{\eta} \neq 0 $
+ $ T>T_c $时，平衡的系统处于高对称相，序参量$ \bar{\eta} = 0 $
+ $ T = T_c $是临界状态。
+ <font style="color:#DF2A3F;">如果平衡时，热力学函数</font>$ \Omega $<font style="color:#DF2A3F;">应该取极小值（自由能、自由焓都是如此），那么可以认为：</font>

$ a(T,\Delta) = a_0(\Delta) (T-T_c),\; a_0(\Delta)>0 $

    - <font style="color:#DF2A3F;">这说明相变发生在</font>$ T_c $<font style="color:#DF2A3F;">处。</font>

$ b(T,\Delta) = b_0 >0 $

    - <font style="color:#DF2A3F;">如果这个条件不被满足，热力学函数的最小值会变为负无穷，这显然不合理。</font>
    - <font style="color:#DF2A3F;background-color:#FBDE28;">（此处是最重要的推导假设）</font>
+ 在这种情况下，热力学函数变为：

$ \Omega(T,\Delta,\eta) = \Omega_0 + \frac{a_0(\Delta)(T-T_0)}{2}\eta^2 + \frac{b(T,\Delta)}{4}\eta^4 + \cdots $

#### 热力学函数Ω在各个温区的数值
+ 当热力学函数有这种形式时，可以对$ \Omega $求$ \eta $的偏导数：

$ \frac{\partial \Omega}{\partial \eta} = \eta (a+b^2 \eta) =\eta (a_0 (T-T_c)+b_0 \eta)  $

$ \frac{\partial^2 \Omega}{\partial \eta^2} = a + 3b\eta ^2 =  a_0 (T-T_c) + 3b_0\eta ^2 $

+ 稳定平衡的条件有：

$ \frac{\partial \Omega}{\partial \eta} = \eta (a+b^2 \eta) =\eta (a_0 (T-T_c)+b_0 \eta) =0 （平衡条件） $

$ \frac{\partial^2 \Omega}{\partial \eta^2} = a + 3b\eta ^2 =  a_0 (T-T_c) + 3b_0\eta ^2>0（稳定条件） $

+ 在这个形式下：
    - $ T>T_c $时，$ \frac{\partial \Omega}{\partial \eta}>0 $在$ \eta>0 $时恒成立
        * 稳定平衡点在$ \bar\eta = 0 $处，即高对称相。此处$ \bar\eta $为平衡时的序参量，与自变量序参量$ \eta $相区分。
        * 对应的平衡时的热力学函数为：

$ \bar{\Omega}(T,\Delta) = \Omega_0(T,\Delta) $

    - $ T<T_c $时，$ \frac{\partial\Omega}{\partial\eta} $在$ \eta>0 $时先负后正
        * 稳定平衡点在$ \bar\eta = ±\sqrt{-\frac{a}{b}} = ±\sqrt{-\frac{a_0 (T-T_c)}{b_0}} $处，即低对称相。
        * 对应平衡时的热力学函数为：

$ \bar{\Omega}(T,\Delta) = \Omega_0(T,\Delta) -\frac{a_0^2(\Delta)}{4b_0}(T-T_c^2) $
