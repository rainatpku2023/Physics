### 量子力学的基本规律
+ 波粒二象性、不确定性关系、量子化、全同性原理

#### 量子化
+ 一个粒子的状态可以用几个量子数来描述。
+ 简并度：处于同一个能级的不同量子态的数量。

#### 全同性原理
+ 全同粒子的**内禀性质**完全相同，例如两个质子完全相同、两个电子完全相同、两个光子完全相同。
+ 在两个全同粒子波函数相互重叠（非定域子系）时，全同粒子交换不引起新的系统量子态，
+ 但是在全同粒子波函数不相互重叠的时候，虽然不能通过内禀性质区分，但是可以通过“位置”区分。
    - 这种情况称为“定域子系”。

#### 费米子与玻色子
+ 子相空间中费米子交换导致波函数变成原来相反数，玻色子交换导致波函数不变。

$ \psi(\vec{r}_2,\vec{r}_1) = \begin{cases} -\psi(\vec{r}_1,\vec{r}_2),费米子 \\
\psi(\vec{r}_1,\vec{r}_2),玻色子\end{cases} $

+ 泡利不相容原理：不允许两个全同费米子处在同一个量子态。
    - 数学解释：假如两个全同费米子处于同一个位置$ \vec{r}_1=  \vec{r}_2 = \vec{r} $：

$ \psi(\vec{r},\vec{r}) = -\psi(\vec{r},\vec{r}) \Rightarrow \psi(\vec{r},\vec {r}) =0 $

    - 波函数为零，说明不可能存在这种情形。
    - 玻色子就没有这种问题：

$ \psi(\vec{r},\vec{r}) = -\psi(\vec{r},\vec{r})  $

### 粒子运动状态的量子描述
#### 例子：一维谐振子
+ 一维谐振子的状态可以用一个量子数$ n $确定，其能级为：

$ \epsilon_n = \hbar \omega(n+\frac{1}{2}), \; n = 0,1,2,3... $

    - 能级简并度为$ 1 $，即无简并现象。

#### 例子：转子
+ 对于一个转动惯量为$ I $，总角动量为$ M $的转子，**经典描述**能量为：

$ \epsilon = \frac{M^2}{2I}, \; M可以取连续值 $

+ 量子结果为：
    - 角动量只能取分立值：

$ M^2 = l(l+1)\hbar^2,\; l = 0,1,2,3... $

        * $ l $又称为角量子数。
    - 角动量在$ z $轴上的分量由$ l,m_z $共同决定：

$ M_z = m_z \hbar,\; m_z = -l,-l+1,...,0,...,l-1,l $

        * $ m_z $又称为磁量子数。
    - 能量由一个量子数$ l $决定：

$ \epsilon = \frac{l(l+1)\hbar^2}{2I}, \; l  =0,1,2,3... $

        * 对于$ l $相同但是$ m_z $不同的量子态，能量相等。
        * 因此角量子数为$ l $能级简并度为$ 2l+1 $

#### 例子：一维自由粒子
+ 量子力学中，常常将以为自由粒子放在长度为$ L $的一维容器中。
+ 根据周期性边界条件：
    - 可以引入一个量子数$ n_x $来描述粒子的运动状态，其取值为：

$ n_x = 0,±1,±2,±3... $

    - 德布罗意波长$ \lambda $为：

$ \lambda = \frac{L}{|n_x|}, \; |n_x| = 0,1,2,3... $

    - 动量$ p_x $为：

$ p_x = \frac{2\pi \hbar}{L} n_x, \; n_x = 0,±1,±2,±3... $

    - 能量$ \epsilon_{nx} $为：



$ \epsilon_{nx} = \frac{\pi^2 \hbar^2}{2m}\cdot\frac{n_x^2}{L^2}, \; n_x = 0,1,2,3... $

        * 可见，基态$ n_x = 0 $的能级简并度为$ 1 $，其他能级的简并度为$ 2 $。

#### 例子：三维自由粒子
+ 简单起见，也是为了之后讨论简并度问题，假设自由粒子被放在了边长为$ L $的正方体容器中。
    - 容器体积显然为$ V = L^3 $。
+ 对于三维自由粒子，在$ x,y,z $方向都能作上述讨论，即：
    - 引入三个量子数$ n_x,n_y,n_z $d对粒子的状态进行描述：

$ n_x = 0,±1,±2,±3... $

$ n_y = 0,±1,±2,±3... $

$ n_x = 0,±1,±2,±3... $

    - 根据上述量子数，动量可以表达为：

$ \vec{p} = (p_x,p_y,p_z) = \frac{2\pi \hbar}{L}(n_1,n_2,n_3) $

    - 能量可以表示为：

$ \epsilon = \frac{1}{2m} \vec{p} \cdot \vec{p} = \frac{2\pi\hbar^2}{mL^2} (n_x^2+n_y^2+n_z^2) $

+ 由此可见，下述能量的能级简并度为$ 8 $：

$ \epsilon = \frac{2\pi\hbar^2}{mL^2} (1+1+1) = \frac{6\pi\hbar^2}{mL^2} $

    - 对应的量子数取值为：

$ (n_x,n_y,n_z) = (±1,±1,±1),\;三个±1相互独立，不必取同号 $

### 三维自由粒子量子态数、态密度
#### 能级准连续假设
+ 当温度对应的能量大于能级间隔时，能量可以视为准连续，即：

$ \Delta \epsilon << k_BT $

+ 在此条件下引入态密度概念：单位能级间隔内，可能的状态数。
    - 态密度的定义为：

$ D(\epsilon) = \frac{dN(\epsilon \sim \epsilon + d\epsilon)}{d\epsilon} $

        * 其中$ dN $是能量处于$ \epsilon \sim \epsilon + d\epsilon $范围内的量子态数。
    - 下面以三维自由粒子为例，计算态密度函数$ D(\epsilon) $。

#### 三维自由粒子的密度
+ 在动量为$ p_x \sim p_x + dp_x $范围内量子态数目为：

$ dn_x = \frac{L}{2\pi \hbar} dp_x $

    - 在体积元$ dV = d^3 \vec{p} = dp_x dp_y dp_z $范围内，量子态数目为：

$ dn_xdn_ydn_z = (\frac{L}{2\pi \hbar})^3 dp_xdp_ydp_z $

+ 在动量空间作坐标变换：

$ p_x = p \sin \theta \cos \phi $

$ p_y = p \sin \theta \sin \phi $

$ p_z = p \cos \theta  $

    - 空间体积元的放大率应该由$ \mathrm{Jacobi} $行列式给出：

$ \frac{dp_x dp_y dp_z}{dp d\theta d\phi} = \frac{\partial(p_x,p_y,p_z)}{\partial(p,\theta,\phi)} = p^2\sin\theta  $

+ 因此：

$ dn_xdn_ydn_z = (\frac{L}{2\pi \hbar})^3 p^2\sin\theta dp d\theta d\phi = \frac{V}{h^3}p^2\sin\theta dp d\theta d\phi $

+ 对$ \theta,\phi $进行积分，得到动量大小为$ p \sim p + dp $范围内的量子态数：

$ dN(p\sim p+dp) = \int_0^\pi \sin \theta d\theta \int_0^{2\pi} d\phi \frac{V}{h^3} p^2 dp = \frac{4\pi V}{h^3}p^2dp $

+ 根据把$ p $看成能量$ \epsilon $的函数，
    - 代入公式$ p(\epsilon)= \sqrt{2m\epsilon} $，估计对应的$ \epsilon \sim \epsilon + d\epsilon $范围内的量子态数：

$ dN(\epsilon\sim \epsilon+d\epsilon) = \frac{4\pi V}{h^3}p^2dp = \frac{4\pi V}{h^3} 2m\epsilon d\sqrt{2m\epsilon} = \frac{2\pi V}{h^3} (2m)^{3/2}\epsilon^{1/2}d\epsilon $

+ 因此，态密度为：

$ D(\epsilon) = \frac{dN}{d\epsilon} = \frac{2\pi V}{h^3}(2m)^{3/2}\epsilon^{1/2} $

### 粒子状态与子相空间体积元的对应关系
#### 从经典描述到量子描述
+ 三维经典粒子：子相空间中的一个点$ (x,y,z,p_x,p_y,p_z) $。
+ 三维粒子的量子描述，由不确定性关系：

$ \Delta x\cdot \Delta p_x \sim h $

$ \Delta y\cdot \Delta p_y \sim h $

$ \Delta z\cdot \Delta p_z \sim h $

    - 因此用一个大小为$ h^3 $的正方体代表一个量子态。

#### 子相空间体积元
+ 在一个宏观小微观大的体积元$ dxdydz dp_x dp_y dp_z $内部，量子态数为：

$ \frac{dxdydz dp_x dp_y dp_z}{h^3} $

+ 位形体积$ V $内部的量子态数为：

$ \frac{V dp_x dp_y dp_z}{h^3} $
