### 化学平衡
#### 一个例子
+ 考虑一个化学反应：

$ \mathrm{2H_2 +O_2 \rightleftharpoons 2H_2 O} $

+ 在热力学中，通常将其写为：

$ \mathrm{2H_2 O-2H_2 - O_2 = 0}   $

#### 单相化学反应的数学表述
+ 更普遍地，对于单相的化学反应，可以总结成更加一般的形式：

$ \sum_i \nu_i\mathrm{A}_i = 0 $

    - 其中$ \mathrm{A}_i $是反应物的分子式，$ \nu_i $是化学反应中组元$ i $的系数。
    - 生成物的系数$ \nu_i>0 $，反应物的系数$ \nu_i<0 $。

#### 再论例子
$ \nu_1 = 2,\; \mathrm{A}_1 = \mathrm{H_2 O} $

$ \nu_2 = -2,\; \mathrm{A}_2 = \mathrm{H_2} $

$ \nu_3 = -1,\; \mathrm{A}_3 = \mathrm{O_2} $

+ 考虑$ dN $量级的反应，那么这个过程中物质的量的变化：

$ dN_{H_2 O}:dN_{H_2}:dN_{O_2}:dN = 2:-2:-1:1 $

### 化学平衡条件
#### 考虑dN量级的反应
+ 考虑进行了$ dN $量级的反应，那么底$ i $个组元数量的变化量为：

$ dN_i = \nu_i dN $

+ 假设组元$ i $的化学势是$ \mu_i $
    - 体系的总吉布斯自由焓为：

$ G = \sum_i  \mu_i N_i  $

    - 其中$ N_i $是组元$ i $的物质的量。
    - 那么$ dN $量级的反后，总吉布斯自由焓变化量为：

$ dG = \sum_i  \mu_i dN_i =\sum_i  \mu_i \nu_i dN $

#### 化学反应平衡条件
+ 等温等压条件下，平衡时吉布斯自由焓应该极小。一阶变分为零。

$ \delta G = \sum_i  \mu_i \delta N_i = \sum_i  \mu_i \nu_i \delta N = 0 $

    - 其中$ \delta N $虚变动是可以随意变化的，所以平衡时前面的系数应该为零：

$ \sum_i  \mu_i \nu_i = 0 $

#### 未达到平衡时化学反应进行方向
+ 平衡条件未满足时，反应进行的方向应该使吉布斯自由焓减小：

$ dG = \sum_i  \mu_i dN_i =\sum_i  \mu_i \nu_i dN <0 $

+ 那么可以得出下列结论：

$ 若\sum_i  \mu_i \nu_i <0 ,则 dN>0 , 即反应正向进行 $

$ 若\sum_i  \mu_i \nu_i >0 ,则 dN<0 , 即反应逆向进行 $

### 混合理想气体的化学平衡及质量作用定律


+ 考虑混合理想气体组成的体系，体系的总吉布斯自由焓为：

$ G = \sum_i  \mu_i N_i  $

+ 之前计算过纯理想气体的化学势有如下形式：

$ \mu_i = RT[\phi_i(T)+\ln p_i] = RT[\phi_i(T)+\ln (x_i p)]  $

+ 结合[化学反应平衡条件](#AyBUe)可得：

$ RT\sum_i \nu_i[\phi_i(T)+\ln p_i] = 0 $

+ 此时定义：

$ \ln K_p = -\sum_i \nu_i \phi_i(T) = \sum_i \ln p_i^{\nu_i} = \ln[\prod_i p_i^{\nu_i}] $

    - 由此可知：

$ \boxed{K_p(T) = \prod_i p_i^{\nu_i}} $

    - 这就是质量作用定律。

#### 化学反应进行方向
$ 若\sum_i  \mu_i \nu_i <0 ,\prod_i p_i^{\nu_i}<K_p(T),则dN>0,即反应正向进行 $

$ 若\sum_i  \mu_i \nu_i >0 ,\prod_i p_i^{\nu_i}>K_p(T),则dN<0,即反应逆向进行 $

$ 若\sum_i  \mu_i \nu_i =0 ,\prod_i p_i^{\nu_i}=K_p(T),则dN=0,即反应平衡 $
