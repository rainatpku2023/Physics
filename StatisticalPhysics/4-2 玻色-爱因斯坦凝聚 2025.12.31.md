### 理想玻色气体的化学势
+ 之前推出过，玻色分布为：

$ a_l = \frac{\omega_l}{e^{\alpha+\beta \epsilon_l}} = \frac{\omega_l}{\exp(\frac{\epsilon_l-\mu}{k_B T})-1} $

+ 对于任意能级，都应该有粒子数非负：

$ a_l \geq 0\Rightarrow \epsilon_l \geq \mu $

+ 对于基态来说，能量为$ \epsilon_0 $：

$ \mu \leq \epsilon_0 $

    - 此时规定$ \epsilon_0 = 0 $，则：

$ \mu \leq 0 $

$ z = e^{-\alpha} = e^{\frac{\mu}{k_B T}}\leq 1 $

+ 对于$ N,V,T $玻色系统，化学势$ \mu $满足约束：

$ n = \frac{N}{V} = \frac{1}{V}\sum_l a_l = \frac{1}{V} \sum_l \frac{\omega_l}{e^{\frac{\epsilon-\mu}{k_B T}}-1} $

    - 对于给定的粒子数密度$ n = \frac{N}{V} $，化学势应该随着温度降低而上升，即：

$ \mu < 0, T \rightarrow T_c^+, |\mu|\rightarrow 0 $

    - 假设存在一个特征温度$ T_c $，那么：

$ T \rightarrow T_c, \mu \rightarrow 0^-,z = e^{-\alpha} \rightarrow 1 $

### 强简并理想玻色气体
#### 理想玻色气体的巨配分函数
+ 一般来说，玻色体系的巨配分函数为：

$ \ln \Xi = -\sum_l \omega_l \ln[1-e^{-\alpha - \beta \epsilon_l}] $

#### 凝聚态不可以忽略的情形：强简并理想玻色气体
+ 对于强简并情形，凝聚相基态上存在宏观数目的粒子，不可忽略。
    - 但是如果直接积分会发现基态的态密度为$ 0 $，因此需要单独拿出来讨论。
    - 可以分别写成：

$ \ln \Xi= - \ln[1-e^{ - \beta \epsilon_l}]-\sum_{l=1}^{\infin} \omega_l \ln[1-e^{-\alpha - \beta \epsilon_l}] $

    - 此处用到了基态能量为$ 0 $，以及简并度为$ 1 $。
+ 根据准连续近似，关于能量的态密度函数可以写为：

$ D(\epsilon)d\epsilon = \frac{2\pi V}{h^3}(2m)^{3/2}\epsilon^{1/2}d\epsilon $

    - 代入巨配分函数的表达式，可以得到：

$ \ln \Xi = -\ln[1-e^{-\alpha}] - \int_0^{\infin}\ln(1-e^{-\alpha -\beta \epsilon})D(\epsilon)d\epsilon $

    - 此处用态密度函数$ D(\epsilon) $代替了能级简并度$ \epsilon $，然后求和化为积分。

#### 强简并理想玻色气体的热力学函数
+ 平均粒子数：

$ \bar{N}=-\frac{\partial}{\partial \alpha}\ln \Xi = \frac{1}{e^{\alpha}-1}+ \int_0^{\infin} \frac{D(\epsilon)d\epsilon}{e^{\alpha+\beta \epsilon}-1} = \bar{N}_0 + \bar{N}_{exc} $

    - 考虑到下列关系：

$ z = e^{-\alpha} $

$ \lambda_T = \frac{h}{(2\pi mk_B T)^{1/2}} $

$ g_n(z) = \sum_{j=1}^{\infin}\frac{z^{j}}{j^{n}} $

+ 类比[理想玻色气体的巨配分函数](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/vicyo3euqyor14ph#r2Uyg)，使用变数变换并进行泰勒展开，可以得到：

$ \bar{N}_0 = \frac{z}{1-z} $

$ \bar{N}_{exc} = \frac{V}{\lambda_T^3}g_{3/2}(z) = \frac{V}{h^3}(2\pi m k_B T)^{3/2}g_{3/2}(z) $

    - 巨配分函数：

$ \ln \Xi = -\ln[1-e^{-\alpha}] +\frac{V}{\lambda_T^3}g_{5/2}(z) $

+ 压强和内能：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V}\ln \Xi = \frac{1}{\beta \lambda_T^3}g_{5/2}(z) $

$ U = -\frac{\partial}{\partial \beta} \ln \Xi = \frac{3}{2}k_B T \frac{V}{\lambda_T^3}g_{5/2}(z) $

    - 可见，基态$ \epsilon_0=0 $对于压强$ p $内能$ U $均没有贡献。

### 玻色-爱因斯坦凝聚（BEC）
#### 定性分析
+ 下面对于温度与化学势的关系进行讨论：

$ T>T_c \Rightarrow \mu<0(z<1)\Rightarrow 弱简并\Rightarrow \bar{N}_0可忽略 $

$ T\rightarrow T_c \Rightarrow \mu \rightarrow 0^-(z\rightarrow 1)\Rightarrow强简并\Rightarrow  \bar{N}_0不可忽略 $

$ T\leq T_c \Rightarrow \mu=0(z=1)\Rightarrow 强简并\Rightarrow \bar{N}_0不可忽略，BEC $



+ 当温度**低于临界温度**$ T \leq T_c $时：

$ \mu = 0, z=1 $

$ \bar{N}_{exc} = \frac{V}{h^3}(2\pi m k_B T)^{3/2}g_{3/2}(1) $

    - 当温度下降$ \bar{N}_{exc} $以$ T^{3/2} $行为下降。

#### 临界温度的表达式
+ 临界温度$ T_c $及以下化学势$ \mu $为$ 0 $，逸度$ z $为$ 1 $。
+ 临界温度$ T_c $处$ BEC $刚刚开始，所以$ T = T_c $时全部粒子数等于激发态粒子数，且逸度为$ 1 $：

$ \bar{N} = \bar{N}_{exc}(T_c) = \frac{V}{h^3}(2\pi m k_B T_c)^{3/2}g_{3/2}(1) $

    - 由这个式子可以反解出来：

$ T_c = \frac{2\pi}{g_{3/2}(1)^{3/2}}\frac{\hbar^2}{mk_B}(\frac{N}{V})^{2/3} $

    - 可见，临界温度$ T_c $与总粒子数$ N $与系统总体积$ V $的比值有关，即与粒子的数密度$ n $有关。

#### 用临界温度和总粒子数表示激发态、基态粒子数
+ 临界温度$ T_c $及以下，激发态的平均粒子数为：

$ \bar{N}_{exc} = (\frac{T}{T_c})^{3/2} N $

$ \bar{N}_0 = N - \bar{N}_{exc} = N[1-(\frac{T}{T_c})^{3/2}] $

+ 需要注意的是，玻色-爱因斯坦凝聚是动量空间的凝聚，表现为玻色子聚集在基态（动量为零的状态），而非在位形空间中聚集成团。

#### BEC凝聚体的热力学函数
+ 前文中已经推出了强简并玻色气体的热力学函数表达式：

$ p = \frac{1}{\beta} \frac{\partial}{\partial V}\ln \Xi = \frac{1}{\beta \lambda_T^3}g_{5/2}(z) $

$ U = -\frac{\partial}{\partial \beta} \ln \Xi = \frac{3}{2}k_B T \frac{V}{\lambda_T^3}g_{5/2}(z) $

    - 对于小于等于临界温度$ T_c $的情形，逸度为$ 1 $，上述表达式可以推出：

$ p = \frac{1}{\beta \lambda_T^3}g_{5/2}(1),\; T\leq T_c $

$ U =\frac{3}{2}k_B T \frac{V}{\lambda_T^3}g_{5/2}(1),\; T\leq T_c $

+ 前文还推出：

$ \bar{N} = \bar{N}_{exc}(T_c) = \frac{V}{h^3}(2\pi m k_B T_c)^{3/2}g_{3/2}(1)
= \frac{V}{\lambda_{T_c}^3} g_{3/2}(1) $

    - 据此，可以反解出来体系的体积：

$ V = \frac{\bar{N}}{g_{3/2}(1)}\lambda_{T_c}^3 $

    - 代入**内能**的表达式：

$ U =\frac{3}{2}k_B T \frac{V}{\lambda_T^3}g_{5/2}(1) = \frac{3}{2}Nk_B T (\frac{T}{T_c})^{3/2} \frac{g_{5/2}(1)}{g_{3/2}(1)},\; T\leq T_c $

        * 其中$ N $为体系的总粒子数
        * 这更清楚地表明，只有激发态上的$ N_{exc} = N (\frac{T}{T_c})^{3/2} $个粒子对内能有贡献。
        * 而这个内能比经典粒子的内能要小，是$ \frac{g_{5/2}(1)}{g_{3/2}(1)} $倍，这是因为玻色气体存在量子统计效应带来的有效吸引。
+ 进一步，可以给出**热容**：

$ C_V = (\frac{\partial U}{\partial T})_V = (\frac{\partial}{\partial T})_V[\frac{3}{2}Nk_B T (\frac{T}{T_c})^{3/2} \frac{g_{5/2}(1)}{g_{3/2}(1)}] = \frac{15}{4} Nk_B (\frac{T}{T_c})^{3/2} \frac{g_{5/2}(1)}{g_{3/2}(1)} $

    - 可见，在$ T\leq T_c $的范围内，热容有两个来源：
        * 激发态的粒子能量升高。
        * 基态的粒子被激发到激发态上，导致激发态的粒子数增多。
