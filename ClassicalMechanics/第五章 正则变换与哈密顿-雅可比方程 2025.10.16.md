### 1 全微分的数学基础
#### 1-1 定义全微分的前提
+ 考虑$ s $个变量$ q_1,q_2,...,q_s $组成的$ s $维空间。
+ 设$ f_1(q_1,q_2,...,q_s),f_2(q_1,q_2,...,q_s),...,f_s(q_1,q_2,...,q_s) $为$ q_1,q_2,...,q_s $的函数。
+ 那么当满足下列任一条件时可以写为全微分：

$ dF(q_1,q_2,...,q_s) = f_1dq_1+f_2dq_2+\cdots f_s dq_s $

+ 条件1：

$ \oint_C \sum_\alpha^s(f_\alpha d q_\alpha) = 0,对于任意闭合路径C $

+ 条件2：

$ \int_A^B \sum_\alpha^s(f_\alpha d q_\alpha),对于任意始末点确定的路径,积分值与路径无关 $

+ 条件3：（这个有点废话了，但是后面轮转证法要用）

$ \sum_\alpha^s f_\alpha d q_\alpha 可以写为全微分 $

+ 条件4：$ f_\alpha $的所有交叉微商相等，即：

$ \frac{\partial f_\alpha}{\partial q_\beta} = \frac{\partial f_\beta}{\partial q_\alpha},\; \forall \alpha,\beta \in \{1,2,...,s\} $

### 2 正则变换与生成函数
#### 2-1 哈密顿正则方程回顾
+ 在一般语境下，认为$ q,p $是**正则变量**，$ H(q,p,t) $是**哈密顿量**。
+ 哈密顿正则方程表述为：

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha} $

$ \dot q_\alpha = \frac{\partial H}{\partial p_\alpha} $

#### 2-2 正则变换的概念
+ 希望找到另外一组正则变量$ Q,P $和另外一个哈密顿量$ K(Q,P,t) $，使得正则方程仍然满足，即：

$ \dot P_\alpha = -\frac{\partial K}{\partial Q_\alpha} $

$ \dot Q_\alpha = \frac{\partial K}{\partial P_\alpha} $

+ 从$ q,p $到$ Q,P $的变换称为**正则变换**。

#### 2-3 正则变换与生成函数：以第一类为例
+ 下面考虑如何确定正则变换关系以及$ K(Q,P,t) $形式的方法，即找到：

$ Q_\alpha = Q_\alpha (q,p,t) $

$ P_\alpha = P_\alpha (q,p,t) $

$ K = K(Q,P,t) $

+ 根据与哈密顿正则方程等效的[修正的哈密顿原理](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/hntfv5ccctxdkh2m#ssW9t)：
    - 由$ q,p $满足哈密顿正则方程可知：

$ \delta \int_{t_1}^{t_2} [\dot q_\alpha p_\alpha - H (q,p,t)]dt = 0 $

    - 同理，由$ Q,P $满足哈密顿正则方程可知：

$ \delta \int_{t_1}^{t_2} [\dot Q_\alpha P_\alpha - K (q,p,t)]dt = 0 $

+ 上面的两个式子相减，得到：

$ \delta \int_{t_1}^{t_2} [\dot q_\alpha p_\alpha - \dot Q_\alpha P_\alpha + (K-H)]dt = 0 $

    - 观察积分号内容：

$ [\dot q_\alpha p_\alpha - \dot Q_\alpha P_\alpha + (K-H)]dt = p_\alpha dq_\alpha -P_\alpha dQ_\alpha + (K-H)dt $

    - 等式右端只含有$ q,Q,t $，因此可以令：

$ \boxed{dF_1(q,Q,t) = p_\alpha dq_\alpha -P_\alpha dQ_\alpha + (K-H)dt} $



+ 其中$ F_1 = F_1(q,Q,t) $就称为上述正则变换的**生成函数**，
    - 每一个特定的$ F_1 $都对应着一个特定的正则变换。
    - 这种**指定变换后广义坐标进行的正则变换**，称为第一类正则变换，对应的生成函数被称为第一类生成函数

#### 2-4 已知变换后广义坐标，根据生成函数确定变换后广义动量和哈密顿量
+ 如果已知生成函数$ F_1 = F_1(q,Q,t) $的形式：
    - 可以先选取变换后的广义坐标$ Q_\alpha $，确定其变换关系为$ Q_\alpha = Q_\alpha (q,p,t) $。
    - 此时可以结合生成函数$ F_1 $，根据全微分$ dF_1 $的形式，确定**变换后的广义动量**$ P_\alpha $形式：

$ \boxed{P_\alpha = -\frac{\partial F_1(q,Q,t)}{\partial Q_\alpha} = P_\alpha(q,p,t)} $

+ 也可以通过生成函数$ F_1 $算出变换前的广义动量：

$ \boxed{p_\alpha =\frac{\partial F_1(q,Q,t)}{\partial q_\alpha}} $

+ 根据$ dF_1 $的形式还可以确定**变换后的哈密顿量**$ K $的形式：

$ \boxed{K(Q,P,t) = [H(q,p,t)+ \frac{\partial F_1 (q,Q,t)}{\partial t}]_{q\rightarrow q(Q,P,t);p\rightarrow p(Q,P,t)}} $

    - 其中角标$ {q\rightarrow q(Q,P,t);p\rightarrow p(Q,P,t)} $的意思是将$ q,p $写成$ Q,P $的形式，这样$ K = K(Q,P,t) $就以$ Q,P,t $为独立变量了。
    - <font style="color:#8A8F8D;">可以写成更加累赘的形式：</font>

$ K(Q,P,t) = H(q(Q,P,t),p(Q,P,t),t)+ \frac{\partial F_1 (q(Q,P,t),Q,t)}{\partial t} $

        * <font style="color:#8A8F8D;">或者可以写成不太严谨的形式：</font>

$ K(Q,P,t) = H+ \frac{\partial F_1 (q,Q,t)}{\partial t} $

    - 如果第一类生成函数$ F_1 $不显含时间，则第一类变换前后的哈密顿量相等。

$ F_1 = F_1(q,Q) \Leftrightarrow \frac{\partial F_1}{\partial t}= 0\Rightarrow K = H $

    - 如果第一类生成函数$ F_1 $显含时间，则第一类变换前后哈密顿量之间不等：

$ F_1 = F_1(q,Q,t) \Leftrightarrow \frac{\partial F_1}{\partial t}\neq0 \Rightarrow K \neq H $

#### 2-5 第二类正则变换
+ 定义第二类生成函数：

$ F_2(q,P,t) = F_1(q,Q,t) +Q_\alpha  P_\alpha $

$ dF_2(q,P,t) = dF_1(q,Q,t) +d(Q_\alpha  P_\alpha) = dF_1(q,Q,t) +P_\alpha dQ_\alpha+Q_\alpha dP_\alpha   $

+ 回顾第一类生成函数性质：

$ dF_1(q,Q,t) = p_\alpha dq_\alpha -P_\alpha dQ_\alpha + (K-H)dt $

+ 根据勒让德变换可知：

$ \boxed{dF_2(q,P,t) = p_\alpha dq_\alpha +Q_\alpha dP_\alpha + (K-H)dt} $

+ 可以知：
    - 如果指定变换前的广义坐标$ q_\alpha $和变换后的广义动量$ P_\alpha $，可以求得变换后的广义坐标$ Q_\alpha $和哈密顿量$ K $：

$ \boxed{p_\alpha = \frac{\partial F_2 (q,P,t)}{\partial q_\alpha}} $

$ \boxed{Q_\alpha = \frac{\partial F_2 (q,P,t)}{\partial P_\alpha} = Q_\alpha(q,p,t)} $

$ \boxed{K(Q,P,t) = [H(q,p,t)+ \frac{\partial F_2 (q,P,t)}{\partial t}]_{q\rightarrow q(Q,P,t);p\rightarrow p(Q,P,t)}} $

#### 2-7 正则变换的物理学意义
+ <font style="color:#DF2A3F;">正则变换下，坐标和动量失去了原先的力学意义，二者相互等价。</font>
+ 此时称它们为一堆**共轭变量**，更为普适。
+ 此外，新动量和新坐标的形式可能非常奇怪，但是仍然满足量纲的积与作用量相同。
    - 这也为寻找更多的共轭变量提供可能。
    - [【空白】正则变换的一些例子](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/qrogl82vnyd1hvfu)

#### 2-8 正则变换的判定
+ 一个变换是正则变换，需要保证变换后的新坐标和新动量仍然共轭，即仍然为正则变量
+ **第一个判据**是变换后坐标仍然满足哈密顿正则方程：

$ \dot Q_\alpha = \frac{\partial K}{\partial P_\alpha} $

$ \dot P_\alpha = -\frac{\partial K}{\partial Q_\alpha} $

+ **第二个判据**是变换后仍然满足泊松括号：

$ [Q_\alpha,Q_\beta]_{q,p} = 0 $

$ [P_\alpha,P_\beta]_{q,p} = 0 $

$ [Q_\alpha,P_\beta]_{q,p} = \delta_{\alpha\beta} $

    - 数学上的可以证明，第二个判据更加严格。范围更为狭窄。

### 3 哈密顿-雅可比方程
#### 3-1 基本思路
+ 假设已经找到一个第二类生成函数$ F_2 (q,P,t) $，可以使正则变换后哈密顿量恒为零：$ K\equiv 0 $。
    - <font style="color:#DF2A3F;">此时</font>$ K $<font style="color:#DF2A3F;">的表达式</font>**<font style="color:#DF2A3F;">不显含</font>**<font style="color:#DF2A3F;">任何广义坐标、广义动量。</font>
+ 根据哈密顿正则方程的泊松括号形式，可知新坐标和动量都是常数：

$ \dot Q_\alpha = [Q_\alpha,K] \equiv[Q_\alpha,0] = 0 \Rightarrow Q_\alpha \equiv b_\alpha $

$ \dot P_\alpha \equiv [P_\alpha,K] =[P_\alpha,0] = 0 \Rightarrow P_\alpha = a_\alpha $



+ 已知广义坐标$ q_\alpha $，根据[第二类正则变换的性质1、2](#hmmPB)：

$ p_\alpha = \frac{\partial F_2 (q,P,t)}{\partial q_\alpha}= \frac{\partial F_2 (q,a,t)}{\partial q_\alpha} $

$ Q_\alpha = \frac{\partial F_2(q,P,t)}{\partial P_\alpha}=\frac{\partial F_2(q,a,t)}{\partial a_\alpha} \equiv b_\alpha $

    - 其中$ a,b $是可以自己确定的常数。
    - 这时运动的解可以写为：

$ q_\alpha = q_\alpha(Q,P,t) = q_\alpha(b,a,t) $

$ p_\alpha = p_\alpha (Q,P,t) = p_\alpha(b,a,t) $

    - 这就是运动方程。





+ 所以，$ q,p $描述的运动方程的求解问题转化为了如何求解出第二类生成函数$ F_2 (q,P,t) $。

#### 3-2 哈密顿-雅可比方程
+ 限制第二类生成函数，的方程为[第二类正则变换的性质3](#hmmPB)：

$ K = H(q,p,t)+\frac{\partial F_2 (q,P,t)}{\partial t} \equiv 0 $

    - 将$ P_\alpha = a_\alpha $以及$ p_\alpha =  \frac{\partial F_2 (q,a,t)}{\partial q_\alpha} $代入上式得到：

$ H(q,\frac{\partial F_2 (q,a,t)}{\partial q},t)+\frac{\partial F_2 (q,a,t)}{\partial t} = 0 $

+ 设其解为$ S $，可以简单记为：

$ \boxed{H(q,\frac{\partial S}{\partial q},t)+\frac{\partial S (q,a,t)}{\partial t} = 0} $

    - 这就是**哈密顿-雅可比方程**。

#### 3-3 哈密顿主函数
+ 满足哈密顿-雅可比方程的第二类生成函数$ S = S(q,P=a,t) $被称为**哈密顿主函数**。

#### 3-4 哈-雅方程的分离变量
+ 观察哈密顿-雅可比方程：

$ H(q,\frac{\partial S}{\partial q},t)+\frac{\partial S (q,a,t)}{\partial t} = 0 $

    - 因为$ a $是我们事先认为选定的新动量常量$ P=a $，所以实际上$ S $只是$ q,t $的函数：

$ S = S^a(q,t) $

        * 增加上标$ a $是为了说明$ S^a(q,t) $是一个二元函数，而$ S (q,a,t) $在形式上仍然是一个三元函数，二者在形式上有区别。
    - 哈密顿-雅可比方程可以写成如下形式：

$ H(q,\frac{\partial S^a}{\partial q},t)+\frac{\partial S^a (q,t)}{\partial t} = 0 $

+ 这是一个关于$ S^\alpha $的二元一阶偏微分方程，可能为非线性。
    - 这种方程只要凑出来一个解满足初始条件就结束了。



+ 我们作试探分离变量解，分离广义坐标与时间：

$ \boxed{S^a(q,t) = T(t)+W(q)} $

    - 如果最终由此解出来了$ S^a $的形式，那么就是解。
    - 分离变量有时能成功，有时成功不了。



+ 将分离变量形式代入哈密顿-雅可比方程：

$ H(q,\frac{d W}{d q},t)+ \frac{dT}{dt} = 0 $

+ 为了讨论简单，只讨论哈密顿量不显含时间的情形：

$ H(q,\frac{d W}{d q})+ \frac{dT}{dt} = 0 $

    - 发现左边一项之和位置有关，而右边的只和时间有关，因此二者应该为互为相反数的常数。

$ H(q,\frac{d W}{d q})=E $

$  \frac{dT}{dt} = -E $

    - 代入$ H(q,p = \frac{d W}{d q}) $，可以求得$ W= W(q) $的数学表达式。
    - 第二个式子可得$ T = -Et+C $，其中$ C $为常数。
+ 综合上述论述，我们就求得了满足哈密顿-雅可比方程的第二类生成函数，即哈密顿主函数：

$ S = S^a(q,t) = -W(q)(Et-C) = S (q,a,t) $

    - 在例题中我们会看到，$ C $是否为零并不重要，因为我们最终用的是$ S $的偏微商。



+ <font style="color:#8A8F8D;">根据上文，将其代入：</font>

$ p_\alpha = \frac{\partial S (q,P,t)}{\partial q_\alpha}= \frac{\partial S (q,a,t)}{\partial q_\alpha} $

$ Q_\alpha = \frac{\partial S(q,P,t)}{\partial P}=\frac{\partial S(q,a,t)}{\partial a_\alpha} \equiv b_\alpha $

    - <font style="color:#8A8F8D;">这也就说明了</font>$ C $<font style="color:#8A8F8D;">是否为零并不影响结果，因为求了一次偏微商后就消失了。</font>
    - <font style="color:#8A8F8D;">就求出了运动方程：</font>

$ q_\alpha= q_\alpha(b,a,t) $

$ p_\alpha = p_\alpha(b,a,t) $

#### 3-5 哈密顿特性函数
+ 哈密顿-雅可比方程分离变量过程中，将哈密顿主函数拆成了广义坐标部分$ W(q) $和时间部分$ T(t) $的加和。
    - 哈密顿主函数广义坐标部分$ W(q) $被称为**哈密顿特性函数**。
    - 哈密顿特性函数在周期问题 作用量变量与角变量 过程中有重要意义。

#### 3-6 周期问题：作用量变量与角变量

    - 例如，角度可以作为广义坐标，相应的“总角动量”就是广义动量。

$ T = \frac{p^2}{2m} $，这是一个物理定义、一种数学映射

$ \dot T = 0,\; \dot p = 0 $，这是一个特殊情境

$ T = T_0, p = p_0 $，这是一个特殊情境

        * 但是动能、动量关系仍然成立，即：

$ T = \frac{p^2}{2m} $

$ \frac{\partial T}{\partial p} = \frac{p}{m} $

+ 在周期问题中，总能量守恒，哈密顿量等于总的机械能$ H = E $。
    - 我们已知哈密顿函数的形式$ H = H(q,p) $。
    - 希望求出运动的周期$ T $。



+ 可以证明，选定如下的作用量$ J_\alpha $：

$ \boxed{J_\alpha = \oint_{一个周期} p_\alpha dq_\alpha} $

    - 那么周期运动的周期可以表示为：

$ \boxed{T_\alpha = \frac{\partial E}{\partial J_\alpha}} $



+ 更严格地，希望选用哈密顿特征函数$ W = W(q_1,q_2,...,q_s;E,C_2,C_3,...,C_s) $来处理问题。
    - 假设$ W = \sum_{\alpha=1}^sW_\alpha(q_\alpha;E,C_2,C_3,...,C_s) $，即系统完全可分离的情形。
+ 定义**作用量变量**：

$ J_\alpha = \oint_{一个周期} p_\alpha dq_\alpha = \oint_{一个周期} \frac{\partial W_\alpha(q_\alpha;E,C_2,C_3,...,C_s)}{\partial q_\alpha} dq_\alpha $

    - 这样的式子总共有$ s $个。
    - 将上述方程视为关于$ J_1,J_2,...,J_s $的方程组，$ E,C_2,C_3,...,C_s $是已知量，可以反解出来用$ J_1,J_2,...,J_s $表示$ W_\alpha $的形式，即：

$ W = \sum_{\alpha=1}^sW_\alpha(q_\alpha;J_1,J_2,...,J_s) $



+ 用$ W $作为正则变换的生成函数，$ J_\alpha $作为变换后的动量$ P_\alpha $ 
    - 那么变换后的坐标$ Q_\alpha $的量纲应该为单位$ 1 $。
    - 这和角度的量纲相同，因此记为$ \omega_\alpha $，并称之为**角变量**。
+ 根据[第二类正则变换的性质2](#hmmPB)：

$ Q_\alpha = \frac{\partial F_2(q,P,t)}{\partial P_\alpha} $

    - 得新广义坐标-角变量表达式：

$ \omega_\alpha = \frac{\partial W}{\partial J_\alpha},\; \alpha = 1,2,...,s $

+ 根据[第二类正则变换的性质3](#hmmPB)：

$ K(Q,P,t) = [H(q,p,t)+ \frac{\partial F_2 (q,P,t)}{\partial t}]_{q\rightarrow q(Q,P,t);p\rightarrow p(Q,P,t)} $

    - 得新哈密顿量表达式：

$ K = H + \frac{\partial W}{\partial t} = H = E = E(J_1,J_2,...,J_s) $

        * <font style="color:#DF2A3F;">此时</font>$ E $<font style="color:#DF2A3F;">虽然是常数（不随时间变化</font>$ dE/dt = 0 $<font style="color:#DF2A3F;">）</font>
        * <font style="color:#DF2A3F;">但是其表达式</font>**<font style="color:#DF2A3F;">仍然显含</font>**$ J_1,J_2,...,J_s $<font style="color:#DF2A3F;">的全部或者部分。</font>
+ 变换后的结果满足哈密顿正则方程：

$ \dot Q_\alpha = \frac{\partial K}{\partial P_\alpha} $

    - 得角变量对时间导数：

$ \dot \omega_\alpha = \frac{\partial K}{\partial J_\alpha} = \frac{\partial E}{\partial J_\alpha} = \frac{\partial E(J_1,J_2,...,J_s)}{\partial J_\alpha} $

    - 因此$ \omega_\alpha $的表达式为：

$ \omega_\alpha = \frac{\partial E}{\partial J_\alpha}\cdot t +\omega_{\alpha 0} $



+ 考虑$ q_\alpha $在一个周期$ T_\alpha $内引起的角变量$ \omega_\alpha $的增量：

$ \Delta_{T_\alpha} \omega_\alpha = \oint_{一个周期} d\omega_\alpha 
= \oint \frac{\partial \omega_\alpha}{\partial q_\alpha}dq_\alpha  $

    - 代入角变量的表达式并交换偏微分次序：

$ RHS
= \oint \frac{\partial}{\partial q_\alpha }(\frac{\partial W}{\partial J_\alpha})dq_\alpha
= \frac{\partial}{\partial J_\alpha} \oint \frac{\partial W}{\partial q_\alpha}dq_\alpha
 $

    - 注意到积分号内被积函数就是角变量：

$ RHS =\frac{\partial}{\partial J_\alpha} \oint p_\alpha dq_\alpha
= \frac{\partial J_\alpha}{\partial J_\alpha}
= 1 $

    - 可见，在$ q_\alpha $在一个周期$ T_\alpha $内引起的角变量$ \omega_\alpha $的增量为$ 1 $。由此：

$ f_\alpha =\frac{\partial E}{\partial J_\alpha} = \frac{\partial E(J_1,J_2,...,J_s)}{\partial J_\alpha} $

$ T_\alpha = (\frac{\partial E}{\partial J_\alpha})^{-1} $



+ 需要注意的是：
    - 虽然每个自由度各自做周期运动，但他们的频率$ f_1,f_2,...,f_s $可能不同，系统整体周期可能不和任何一个周期相同。
    - 如果$ f_1,f_2,...,f_s $其中两个比值为无理数，那么系统整体不具有周期性。
