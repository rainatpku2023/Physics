### 勒让德变换
#### 基本设定：老变量与老函数，新变量与新函数
+ 考虑一系列**老变量**$ x_1,x_2,...,x_n $，以及其**老函数**$ F = F(x_1,x_2,...,x_n) $。
    - 定义一系列**新变量**$ u_1,u_2,...,u_n $，依次与$ x_1,x_2,...,x_n $相对应。在数值上满足：

$ \boxed{u_i = \frac{\partial F}{\partial x_i}, i = 1,2,...,n} $

    - 并定义一个**新函数**$ G = G(u_1,u_2,...,u_n) $，在数值上满足：

$ \boxed{G(u_1,u_2,...,u_n) = \sum_{i=1}^{n} x_i u_i -F(x_1,x_2,...,x_n)} $

#### 用两种方法求全微分并比较
+ 求$ G $的全微分。
+ 一方面，根据$ G = G(u_1,u_2,...,u_n) $，可以得到全微分的第一个表达式：

$ dG = \sum_{i=1}^{n}\frac{\partial G}{\partial u_i}du_i $

+ 另一方面，根据$ G= \sum_{i=1}^{n} x_i u_i -F $可以推导全微分的第二个表达式：

$ dG = \sum_{i=1}^n u_i dx_i + \sum_{i=1}^n x_i du_i  - \sum_{i=1}^{n} \frac{\partial F}{\partial x_i}dx_i $

    - 根据$ u_i = \frac{\partial F}{\partial x_i} $可知：

$ RHS = \sum_{i=1}^n u_i dx_i + \sum_{i=1}^n x_i du_i  - \sum_{i=1}^{n} u_idx_i = \sum_{i=1}^n x_i du_i $

    - 最终得到全微分的第二个表达式：

$ \boxed{dG = \sum_{i=1}^n x_i du_i} $

+ 对比全微分的两种表达式可知：

$ \boxed{x_i = \frac{\partial G}{\partial u_i}, i = 1,2,...,n} $

#### 对称性
+ 这样，我们就得到了两组非常对称的关系：

$ dF = \sum_{i=1}^n u_i dx_i $$ dG = \sum_{i=1}^n x_i du_i $

$ u_i = \frac{\partial F}{\partial x_i}, i = 1,2,...,n $$ x_i = \frac{\partial G}{\partial u_i}, i = 1,2,...,n $

+ 它们由最初的规定联系起来：

$ G(u_1,u_2,...,u_n) = \sum_{i=1}^{n} x_i u_i -F(x_1,x_2,...,x_n) $

或写为：

$ G(u_1,u_2,...,u_n) +F(x_1,x_2,...,x_n)= \sum_{i=1}^{n} x_i u_i  $

### 推广的勒让德变换
#### 基本设定：引入不参加变换的参数
+ 考虑一系列**老变量**$ x_1,x_2,...,x_n $，一系列**参数**$ y_1,y_2,...,y_m $以及其**老函数**$ F = F(x_1,x_2,...,x_n;y_1,y_2,...,y_m) $
    - 定义一系列**新变量**$ u_1,u_2,...,u_n $，依次与$ x_1,x_2,...,x_n $相对应。在数值上满足：

$ u_i = \frac{\partial F}{\partial x_i}, i = 1,2,...,n $

    - 并定义一个**新函数**$ G = G(u_1,u_2,...,u_n;y_1,y_2,...,y_m) $，在数值上满足：

$ \boxed{G(u_1,u_2,...,u_n;y_1,y_2,...,y_m) = \sum_{i=1}^{n} x_i u_i -F(x_1,x_2,...,x_n;y_1,y_2,...,y_m)} $

#### 考察老函数、新函数对于变量的偏微商
+ 参数的有无并不影响论述，因此对于变量还会得到：

$ dF = \sum_{i=1}^n u_i dx_i $$ dG = \sum_{i=1}^n x_i du_i $

$ u_i = \frac{\partial F}{\partial x_i}, i = 1,2,...,n $$ x_i = \frac{\partial G}{\partial u_i}, i = 1,2,...,n $

#### 考察老函数、新函数对于参数的偏微商
+ 考虑第$ j $个参数$ y_j $。考虑新函数$ G $对于参数$ y_j $的偏微商：

$ \frac{\partial G}{\partial y_j} = \frac{\partial}{\partial y_j}[\sum_{i=1}^{n} x_i u_i -F(x_1,x_2,...,x_n;y_1,y_2,...,y_m)] $

    - 使用复合函数的求导法则：

$ RHS = \sum_{i=1}^{n}u_i \frac{\partial x_i}{\partial y_j} - \sum_{i=1}^n \frac{\partial F}{\partial x_i}\frac{\partial x_i}{\partial y_j}- \frac{\partial F}{\partial y_j} $ 

    - 再考虑到$ u_i = \frac{\partial F}{\partial x_i} $，代入知：

$ RHS = \sum_{i=1}^{n}u_i \frac{\partial x_i}{\partial y_j} - \sum_{i=1}^n u_i\frac{\partial x_i}{\partial y_j}- \frac{\partial F}{\partial y_j} = - \frac{\partial F}{\partial y_j} $ 

    - 我们也就得到了老函数、新函数对于参数的偏微商关系：

$ \boxed{\frac{\partial G}{\partial y_j} = - \frac{\partial F}{\partial y_j}} $

+ 这也就说明，不进行勒让德变换的变量（即参数），新老函数对其的偏导互为相反数。

#### 总结
$ dF = \sum_{i=1}^n u_i dx_i + \sum_{j=1}^{m}\frac{\partial F}{\partial y_j}dy_j $

$  dG = \sum_{i=1}^n x_i du_i+\sum_{j=1}^{m}\frac{\partial G}{\partial y_j}dy_j = \sum_{i=1}^n x_i du_i-\sum_{j=1}^{m}\frac{\partial F}{\partial y_j}dy_j $

$ u_i = \frac{\partial F}{\partial x_i}, i = 1,2,...,n $$ x_i = \frac{\partial G}{\partial u_i}, i = 1,2,...,n $

$ \frac{\partial G}{\partial y_j} = - \frac{\partial F}{\partial y_j}, j= 1,2,...,m $

### 根据推广的勒让德变换引入哈密顿力学
#### 拉格朗日量与拉格朗日方程
+ 假设系统有$ s $个自由度，拉格朗日量$ L $是广义坐标$ q_1,q_2,...,q_s $，广义速度$ \dot q_1,\dot q_2,...,\dot q_s $以及时间$ t $的函数：

$ L = L(q_1,q_2,...,q_s;\dot q_1,\dot q_2,...,\dot q_s ;t) $

+ 物理在拉格朗日力学中表述为第二类拉格朗日方程：

$ \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} = Q_\alpha^{耗散}, \alpha =1,2,...,s $

    - 加入不存在耗散力，那么可以写成更加简单的形式：

$ \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} =0 , \alpha =1,2,...,s $

#### 哈密顿量的引入
+ 仿照[推广的勒让德变换](#tcqh2)，我们如此看待：
    - 老函数：拉格朗日量$ L = L(q_1,q_2,...,q_s;\dot q_1,\dot q_2,...,\dot q_s ;t) $
    - 老变量：广义速度$ \dot q_1,\dot q_2,...,\dot q_s $
    - 参数：广义坐标$ q_1,q_2,...,q_s $以及时间$ t $。
    - 新函数：哈密顿量$ H = H(q_1,q_2,...,q_s;p_1,p_2,...,p_s ;t) $
    - 新变量：广义动量$ p_1,p_2,...,p_s $
+ 为了实现勒让德变换，我们设置**变换**为：

$ H(q_1,q_2,...,q_s;p_1,p_2,...,p_s;t) = \sum_{\alpha=1}^{s} \dot q_\alpha p_\alpha -L(q_1,q_2,...,q_s;\dot q_1,\dot q_2,...,\dot q_s ;t) $

    - 或者简写地写为：

$ H(q,p,t) = \sum_{\alpha=1}^{s} \dot q_\alpha p_\alpha -L(q,\dot q ,t) $

    - 设置**新老变量的关系**为：

$ p_\alpha = \frac{\partial L}{\partial \dot q_\alpha}, \alpha = 1,2,...,s $

        * 这正是我们在[拉格朗日力学中引入广义动量](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/in7z7g1ogm3bl9ma#PcdOk)的定义。

#### 哈密顿正则方程的导出
+ 由此，我们可以得到：
    - 新老函数对于参数的偏微商：

$ \frac{\partial H}{\partial q_\alpha} = - \frac{\partial L}{\partial q_\alpha},\alpha = 1,2,...,s $

$ \boxed{\frac{\partial H}{\partial t} = - \frac{\partial L}{\partial t}} $

    - 新老函数对于新老变量的偏微商：

$ \frac{\partial H}{\partial p_\alpha} = \frac{\partial L}{\partial \dot q_\alpha},\alpha = 1,2,...,s $

    - 新老变量的变换关系：

$ p_\alpha = \frac{\partial L}{\partial \dot q_\alpha}, \alpha = 1,2,...,s $

$ \boxed{\dot q_\alpha = \frac{\partial H}{\partial p_\alpha}, \alpha = 1,2,...,s} $

    - 其中最后一个式子就是正则方程的第一个式子。



+ 考察下面三个式子：

$ \frac{\partial H}{\partial q_\alpha} = - \frac{\partial L}{\partial q_\alpha},\alpha = 1,2,...,s $

$ p_\alpha = \frac{\partial L}{\partial \dot q_\alpha}, \alpha = 1,2,...,s $

$ \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} =0 , \alpha =1,2,...,s $

    - 可以推知：

$ \dot p_\alpha = \frac{d}{dt} p_\alpha = \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} = \frac{\partial L}{\partial q_\alpha} +Q_\alpha^{耗散}= -\frac{\partial H}{\partial q_\alpha}+Q_\alpha^{耗散} $

    - 如果耗散力$ Q_\alpha^{耗散} $为零，这也就是正则方程的第二个式子：

$ \boxed{\dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}, \alpha = 1,2,...,s} $

#### 总结
+ 哈密顿正则方程的第一个式子描述的是勒让德变换的**新老变量变换关系**：

$ \dot q_\alpha = \frac{\partial H}{\partial p_\alpha}, \alpha = 1,2,...,s $

+ 哈密顿正则方程的第二个式子是**参数变换关系**与**物理**（第二类拉格朗日方程）的结合：

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}, \alpha = 1,2,...,s $

+ 哈密顿量对于时间偏导与拉格朗日量对时间偏导式子是**参数变换关系**：

$ \frac{\partial H}{\partial t} = - \frac{\partial L}{\partial t} $

#### 带有耗散力的哈密顿正则方程
+ 带有耗散力的拉格朗日方程为：

$ \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} = Q_\alpha^{耗散}, \alpha =1,2,...,s $

+ 经过勒让德变换后，得到带有耗散力的哈密顿正则方程：

$ \dot q_\alpha = \frac{\partial H}{\partial p_\alpha}, \alpha = 1,2,...,s $

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}+Q^{耗散}_\alpha, \alpha = 1,2,...,s $
