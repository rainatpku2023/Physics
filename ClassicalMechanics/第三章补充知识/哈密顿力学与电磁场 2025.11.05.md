#### 问题概述
+ 考虑一个在电场$ \vec{E} $和磁场$ \vec{B} $中运动的电量为$ e $的带电粒子，希望用泊松括号方法求解其运动方程。

#### 拉格朗日量的计算
+ 根据电动力学理论，可以引入标量势$ \phi $和矢量势$ \vec{A} $对电场$ \vec{E} $和磁场$ \vec{B} $进行表示：

$ \vec{E} = -\nabla \phi - \frac{\partial \vec{A}}{\partial t} $

$ \vec{B} = \nabla \times \vec{A} $

+ 此时广义势可以写为：

$ U = e(\phi -\vec{v} \cdot \vec{A}) $

+ 因此拉格朗日量可以写为：

$ L = T-U = \frac{1}{2}mv^2-e(\phi -\vec{v} \cdot \vec{A}) $

    - 如果使用坐标$ \vec{r} = x \vec{i} +y \vec{j} + z \vec{k} $表述，可以将拉格朗日量写为：

$ L = \frac{1}{2}m(\dot x^2+\dot y^2 +\dot z^2)-e(\phi -\dot x A_x-\dot y A_y - \dot z A_z) $

#### 勒让德变换与哈密顿量的求解
+ 根据[勒让德变换](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/tukfgpy9rfkrubqn)，希望：
    - 将拉格朗日量$ L $（老函数）换为哈密顿量$ H $（新函数）
    - 将广义速度$ \dot x,\dot y,\dot z $（老变量）换为广义动量$ p_x,p_y,p_z $（新变量）
    - 保持广义坐标$ x,y,z $（参数）不变。



+ 我们首先求解广义动量$ p_x,p_y,p_z $，：

$ p_x = \frac{\partial L}{\partial \dot x} = m\dot x +eA_x $

$ p_y = \frac{\partial L}{\partial \dot y} = m\dot y +eA_y $

$ p_z = \frac{\partial L}{\partial \dot z} = m\dot z +eA_z $

    - 不太严谨地写成矢量形式：

$ \vec{p} = m \vec{v} + e\vec{A} $

        * 发现广义动量总共有两项，第一项$ m\vec{v} $等于带电粒子的动量，这是粒子动量。
        * 第二项$ e\vec{A} $与电磁场有关，这时**场动量**。
        * 这也就说明了根据定义求解广义动量的重要性。

$ p_\alpha = \frac{\partial L}{\partial \dot q_\alpha} $



+ 再来求解哈密顿量：

$ H = \sum \dot x_i p_i - \bar L
 $

$ = (\sum \dot x_i p_i -\sum\frac{1}{2}m\dot x_i -e\sum \dot x_i A_i +e\phi)|_{用x_i,p_i,t表示} $

$ = \sum (\frac{p_i - eA_i}{m}) \cdot p_i - \sum \frac{1}{2m}(p_i - eA_i)^2-e\sum(\frac{p_i - eA_i}{m}) A_i +e\phi $

$ =\sum \frac{1}{2m}(p_i-eA_i)^2 + e\phi $

    - 哈密顿量为：

$ \boxed{H = \frac{1}{2m}(p_x-eA_x)^2 +\frac{1}{2m}(p_y-eA_y)^2 +\frac{1}{2m}(p_z-eA_z)^2 + e\phi} $

#### 关于哈密顿量意义的讨论
+ 我们可以将哈密顿量不严谨地写为：

$ H = \frac{1}{2}mv^2 + e\phi \neq T+U $

    - 不等号有点奇怪，我们来仔细讨论一下。
+ 在进行勒让德变换时，我们用到了[欧拉定理](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/in7z7g1ogm3bl9ma#OsLcp)：
    - 对于$ n $次齐次函数$ f $有：

$ \sum_{i=1}^{s}x_i \frac{\partial f}{\partial x_i} = nf $

    - 对于这种情景中的拉格朗日量：

$ L = \frac{1}{2}m(\dot x^2+\dot y^2 +\dot z^2)-e(\phi -\dot x A_x-\dot y A_y - \dot z A_z) $

    - 动能只有广义速度$ \dot x_i $的二次齐次项：

$ T = T_2=\frac{1}{2}m(\dot x^2+\dot y^2 +\dot z^2) $

    - 势能只有广义速度$ \dot x_i $的零次齐次项和一次齐次项：

$ U = U_0+U_1 $

$ U_0 = e\phi $

$ U_1 = e(\dot x A_x+\dot y A_y + \dot z A_z) $

    - 代入变换式：

$ H = \sum \dot x_i \frac{\partial L}{\partial \dot x_i} - L
 $

        * 代入拉格朗日量表达式$ L= T-U $：

$ RHS=\sum \dot x_i \frac{\partial (T-U)}{\partial \dot x_i} - (T-U) $

        * 将动能按照广义速度$ \dot x_i $的齐次式次数展开：

$ RHS=\sum \dot x_i \frac{\partial (T_2-U_0-U_1)}{\partial \dot x_i} - (T_2-U_0-U_1) $

        * 代入欧拉定理：

$ RHS = (2T_2 - 0U_0 - 1U_1)-(T_2-U_0-U_1) $

        * 合并同类项：

$ H = T_2 +U_0 \neq T+U $

#### 哈密顿量的一般表达式
+ **我们还可以作更一般的讨论**：
    - 将动能项按照广义速度$ \dot x_i $齐次式次数展开：

$ T = T_0 +T_1 +T_2 + \cdots = \sum_{n=1}^\infin T_n $

    - 将广义势按照广义速度$ \dot x_i $齐次式次数展开：

$ U = U_0 +U_1 +U_2 + \cdots = \sum_{n=1}^\infin U_n $

    - 拉格朗日量的表示式变为：

$ L = T-U=\sum_{i=1}^\infin T_n-\sum_{i=1}^\infin U_n $

    - 作勒让德变换：

$ H = \sum \dot x_i \frac{\partial L}{\partial \dot x_i} - L
 $

        * 将拉格朗日量代入：

$ RHS=\sum \dot x_i \frac{\partial}{\partial \dot x_i}(\sum_{n=1}^\infin T_n-\sum_{n=1}^\infin U_n) - (\sum_{n=1}^\infin T_n-\sum_{n=1}^\infin U_n) $

        * 代入欧拉定理：

$ RHS=(\sum_{n=1}^\infin nT_n-\sum_{n=1}^\infin nU_n) - (\sum_{n=1}^\infin T_n-\sum_{n=1}^\infin U_n) $

        * 合并同类项：

$ \boxed{H=\sum_{n=1}^\infin (n-1)T_n-\sum_{n=1}^\infin (n-1)U_n} $

#### 使用泊松括号求解运动方程
+ 泊松括号形式的哈密顿正则方程为：

$ \dot x_i = [x_i,H] $

$ \dot p_i = [p_i,H] $



+ 以求解$ \dot x $的表达式为例：

$ \dot x = [x,\frac{1}{2m}(p_x-eA_x)^2 +\frac{1}{2m}(p_y-eA_y)^2 +\frac{1}{2m}(p_z-eA_z)^2 + e\phi] $

    - 考虑到$ \phi,A_x,A_y,A_z $表达式中都不含有$ p_x $，因此和他们的泊松括号为零，式子可以迅速简化为：

$ RHS = \frac{1}{2m}[x,(p_x-eA_x)^2 ] = \frac{1}{2m}[x,(p_x-eA_x)(p_x-eA_x) ] $

    - 利用$ [f,gh] = g[f,h]+[f,g]h $：

$ RHS = \frac{1}{2m}\cdot 2(p_x-eA_x)[x,(p_x-eA_x) ] = \frac{1}{m}(p_x-eA_x)[x,(p_x-eA_x) ] $

    - 考虑到$ A_x $表达式中都不含有$ p_x $：

$ RHS = \frac{1}{2m}\cdot 2(p_x-eA_x)[x,(p_x-eA_x) ] = \frac{1}{m}(p_x-eA_x)[x,p_x] = \frac{1}{m}(p_x-eA_x) $

    - 即：

$ \dot x = \frac{1}{m}(p_x-eA_x) $

    - 这与哈密顿正则方程求解结果相同。



+ 再求解一下$ \dot p_x $：

$ \dot p_x = [p_x,\frac{1}{2m}(p_x-eA_x)^2 +\frac{1}{2m}(p_y-eA_y)^2 +\frac{1}{2m}(p_z-eA_z)^2 + e\phi] $

    - 先讨论一部分：

$ [p_x,\frac{1}{2m}(p_x-eA_x)^2] $

        * 利用$ [f,gh] = g[f,h]+[f,g]h $：

$ [p_x,\frac{1}{2m}(p_x-eA_x)^2] = \frac{1}{m}(p_x-eA_x)[p_x,(p_x-eA_x)] $

        * **<font style="color:#DF2A3F;">需要注意，</font>**$ A_x = A_x(x,y,z,t) $**<font style="color:#DF2A3F;">含有广义坐标</font>**$ x $**<font style="color:#DF2A3F;">：</font>**

$ RHS = \frac{1}{m} (p_x-eA_x)(-e) [p_x,A_x(x,y,z,t)] $

        * 即：

$ [p_x,\frac{1}{2m}(p_x-eA_x)^2]=-\frac{e}{m} (p_x-eA_x) [p_x,A_x(x,y,z,t)] $

            + 同理需要注意$ \phi = \phi(x,y,z,t) $**含有广义坐标**$ x $**。**
    - 因此整体表达式为

$ \dot p_x = -\frac{e}{m}\sum_i  (p_x-eA_i) [p_x,A_i(x,y,z,t)] + e[p_x,\phi(x,y,z,t)] $

        * 如果想要继续求解，必须知道$ \phi = \phi (x,y,z,t) $和$ \vec{A} = \vec{A}(x,y,z,t) $的表达式。



+ 如何理解$ [p_x,A_i]\neq 0 $？
    - 可以将$ A_i(x,y,z,t) $进行多元函数泰勒展开，里面会含有$ x $的项。
    - 而且$ [x,p_x]=1 $。
