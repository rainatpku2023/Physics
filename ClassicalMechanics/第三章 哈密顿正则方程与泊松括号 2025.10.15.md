### 1 哈密顿正则方程
#### 1-1 拉格朗日力学
+ 自由度为$ s $的保守系统的拉格朗日方程为：

$ \frac{d}{dt} \frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha}= 0,\;\alpha = 1,2,...,s $

    - **值得注意的是，在微分和求导的过程中**$ q_\alpha,\dot{q}_\alpha $**被视作独立变量。**
+ 拉格朗日力学中，广义动量$ p_\alpha $的定义为：

$ p_\alpha = \frac{\partial L}{\partial \dot{q}_\alpha} $

+ 据此，拉格朗日方程可以改写为：

$ \dot{p}_\alpha = \frac{\partial L}{\partial q_\alpha} $

+ 本章希望通过广义坐标$ q_\alpha $和其对应的广义动量$ p_\alpha $书写运动方程。

#### 1-2 勒让德变换
+ 勒让德变换是上述推导过程中要用到的数学工具，其内容如下。
+ 考虑二元函数$ F = F(x,y) $的微分：

$ dF = \frac{\partial F}{\partial x}dx+\frac{\partial F}{\partial y}dy = udx+vdy $

    - 右边只包含$ dx,dy $两个微分，就是将$ x,y $视为独立变量。
+ 我们想要构造一个函数$ G = G(x,v) $，即以$ x,v $为独立变量的函数
    - 可以如此构造：

$ dG(x,v) = d[F(x,y) - yv] = udx-ydv $

    - 这就是勒让德变换。
+ 勒让德变换还可以用更加紧凑的形式写出：

$ ydx = d(xy) - xdy $

#### 1-3 哈密顿量的引入
+ 考虑拉格朗日函数$ L = L(q,\dot{q},t) $的全微分：

$ -dL = -\sum_{\alpha=1}^s\frac{\partial L}{\partial q_\alpha} dq_\alpha -\sum_{\alpha=1}^s\frac{\partial L}{\partial \dot{q}_\alpha} d\dot{q}_\alpha - \frac{\partial L}{\partial t}dt $

    - 代入1-1中推出的两个数学表达式$ p_\alpha = \frac{\partial L}{\partial \dot{q}_\alpha} $和$ \dot{p}_\alpha = \frac{\partial L}{\partial q_\alpha} $：

$ -dL = -\sum_{\alpha=1}^s\dot{p}_\alpha dq_\alpha -\sum_{\alpha=1}^sp_\alpha d\dot{q}_\alpha - \frac{\partial L}{\partial t}dt $

    - 两边都加上$ \sum_{\alpha=1}^s d(p_\alpha\dot{q}_\alpha) $，得到以$ q_\alpha,p_\alpha $为独立变量的函数：

$ d(\sum_{\alpha=1}^s p_\alpha\dot{q}_\alpha-L) = -\sum_{\alpha=1}^s\dot{p}_\alpha dq_\alpha +\sum_{\alpha=1}^s\dot{q}_\alpha d p_\alpha - \frac{\partial L}{\partial t}dt $

        * 此处用到了勒让德变换：

$ \dot{q}_\alpha dp_\alpha = d(p_\alpha\dot{q}_\alpha) - p_\alpha d\dot{q}_\alpha $

+ 记微分号中的函数为$ H $，称为**哈密顿量**：

$ \boxed{H = \sum_{\alpha=1}^s p_\alpha\dot{q}_\alpha-L|_{只用q,p,t表示}} $

+ 需要注意：<font style="color:#DF2A3F;">哈密顿量</font>$ H $<font style="color:#DF2A3F;">应该写为义</font>$ q,p,t $<font style="color:#DF2A3F;">为独立变量的形式，不能出现广义速度</font>$ \dot q $。
    - 所以需要将广义速度$ \dot q_\alpha $改写为广义动量$ p_\alpha $，同时将$ L $中的广义速度也换成广义动量。
    - 具体方法是根据$ p_\alpha = \frac{\partial L(q,\dot q,t)}{\partial \dot{q}_\alpha} $反解出来：

$ \dot q_\alpha = \dot q_\alpha(q,p,t) $

    - 据此我们也可以将$ L $换成广义动量的表示形式：

$ L = L(q,\dot q(q,p,t),t) $

    - <font style="color:#DF2A3F;">需要注意：</font>$ p_\alpha = m \dot q_\alpha $<font style="color:#DF2A3F;">一般不成立，我们将在</font>[【空白】哈密顿力学与电磁场](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/wmysr6cf4dg6l7c9)<font style="color:#DF2A3F;">中看到这一点。</font>
        * <font style="color:#DF2A3F;">电磁场带电粒子的运动动量分为两部分，粒子动量和场动量，</font>$ m \dot q_\alpha $<font style="color:#DF2A3F;">只能描述前者。</font>

#### 1-4 哈密顿正则方程的导出
    - 微分形式：

$ dH = -\sum_{\alpha=1}^s\dot{p}_\alpha dq_\alpha +\sum_{\alpha=1}^s\dot{q}_\alpha d p_\alpha - \frac{\partial L}{\partial t}dt $

+ 考虑哈密顿量$ H(q,p,t) $的全微分：

$ dH = \sum_{\alpha=1}^s\frac{\partial H}{\partial q_\alpha} dq_\alpha +\sum_{\alpha=1}^s\frac{\partial H}{\partial p_\alpha} dp_\alpha + \frac{\partial H}{\partial t}dt $

+ 比较上面的两个式子，得到：

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}; \;\dot q_\alpha = \frac{\partial H}{\partial p_\alpha}; \; \frac{\partial H}{\partial t} = -\frac{\partial L}{\partial t},\alpha =1,2,...,s $  

+ 提取出前面的广义坐标$ q_\alpha $和其对应的广义动量$ p_\alpha $的相关方程：

$ \boxed{\dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}; \;\dot q_\alpha = \frac{\partial H}{\partial p_\alpha},\alpha =1,2,...,s} $ 

    - 这就是**哈密顿正则方程**。

#### 1-5 由拉格朗日量求哈密顿量
+ 如果将拉格朗日函数$ L $按照广义速度$ \dot q_\alpha $的齐次式展开：

$ L = L_0 + L_1 + L_2 $

+ 那么：

$ H = \sum_{\alpha=1}^s p_\alpha\dot{q}_\alpha-L
= \sum_{\alpha=1}^s \frac{\partial L}{\partial \dot q_\alpha}\dot{q}_\alpha-L = (2L_2+L_1)-L = (L_2-L_0)|_{用p_\alpha表示} $

### 2 初积分
#### 2-1 能量积分与广义能量积分
+ 考虑哈密顿函数$ H $对时间$ t $的全导数：

$ \frac{dH}{dt} = \sum_{\alpha=1}^s \frac{\partial H}{\partial q_\alpha} \dot q_\alpha+\sum_{\alpha=1}^s \frac{\partial H}{\partial p_\alpha} \dot p_\alpha+ \frac{\partial H}{\partial t} $

    - 考虑哈密顿正则方程：

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}; \;\dot q_\alpha = \frac{\partial H}{\partial p_\alpha}; \; \frac{\partial H}{\partial t} = -\frac{\partial L}{\partial t},\alpha =1,2,...,s $  

+ 可以得到：

$ \frac{dH}{dt} = \frac{\partial H}{\partial t} = -\frac{\partial L}{\partial t} $

+ 如果拉格朗日函数$ L $不显含$ t $，或者哈密顿函数$ H $不显含$ t $：

$ \frac{dH}{dt} = \frac{\partial H(q,p)}{\partial t} = -\frac{\partial L(q,\dot q)}{ \partial t} = 0 $

$ H(q,p) = \mathrm{const.} $

    - 即哈密顿量不随时间变化，这个时候称这个积分值为**广义能量积分**。
+ 之前在[拉格朗日力学 哈密顿量与机械能](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/in7z7g1ogm3bl9ma#Bzxju)中提到过，如果：
    - 稳定约束
    - 或者坐标变换方程不显含时间
    - 或者拉格朗日量不显含时间
    - 或者哈密顿量不显含时间
+ 那么哈密顿量等于机械能$ H = T+V $。上述四个判据彼此等价。

#### 一点注意事项
+ **注意：拉格朗日量**$ L=T-V $**的计算方法会因为参考系的选择而不同，但给出相同的物理。**
    - 例如考虑在圆盘上沿着一个光滑凹槽，随着圆盘一起以恒定角速度$ \omega $转动的质点。
        * 沿着径向的平动动能为：

$ T_0 = \frac{1}{2}m\dot r^2 $

    - 在**地面系（惯性系）**，因为<font style="color:#DF2A3F;">旋转造成的能量改变解释为动能</font>。

$ T_r = \frac{1}{2}m \omega^2 r^2 $

    - 在**圆盘系（非惯性系）**，因为<font style="color:#DF2A3F;">旋转造成的能量改变解释为离心势能</font>。

$ V_r = -\frac{1}{2}m\omega^2 r^2 $

    - 最终，两张计算出来的拉格朗日量是同一个：
        * 地面系中：

$ L_D = T-V = T_0 + T_r =  \frac{1}{2}m\dot r^2+\frac{1}{2}m\omega^2 r^2 $

        * 圆盘系中：

$ L _P= T-V = T_0 -V_r  = \frac{1}{2}m\dot r^2 -(-\frac{1}{2}m\omega^2 r^2) = \frac{1}{2}m\dot r^2 +\frac{1}{2}m\omega^2 r^2 $

    - 最终也会给出相同的物理：
        * $ \frac{d}{dt}\frac{\partial L_D}{\partial \dot{r}} - \frac{\partial L_D}{\partial r} = 0 $会给出$ \ddot{r}- \omega^2 r = 0 $。
        * $ \frac{d}{dt}\frac{\partial L_P}{\partial \dot{r}} - \frac{\partial L_P}{\partial r} = 0 $会给出$ \ddot{r}- \omega^2 r = 0 $。
        * 即径向位置随着时间演化对于两个参考系是相同的。
        * 求导过程中用到了圆盘角速度恒定的条件，即$ \frac{d\omega}{dt}= 0 $。
    - 需要说明的是，此处两个系拉格朗日量形式相同，其实有一定程度上的巧合。
        * 非惯性系中，科里奥利力通常会出现，但在这个例子中，由于质点被约束在径向凹槽内，科里奥利力（切向方向）被凹槽的约束力抵消，因此不影响径向运动方程。
        * 后面的[一个例子](#yqtyn)会出现科里奥利力广义势出现在拉格朗日量中的情景。



+ **哈密顿力学也有相似情况**：
    - 在非惯性系下，转动解释为离心势能。惯性系下，转动解释为动能。
    - **不同参考系最终给出的运动方程可能不同，但是描述的是相同的物理。**
        * 结合坐标变换，反变换回去就是相同的结果。

#### 一点感悟：物理实在只有一个，数学描述可以有很多
+ 上面说明不论选择怎样的参考系，选择怎样的参数，得到的方程可能不同，但是最终描述的物理现象都一致。
+ 再考虑狭义相对论中的连接两个事件的四维矢量的来讲，矢量本身并不随观察者变化
    - “钟慢尺缩”等效应实际上是坐标洛伦兹变换导致的。
    - 同时还有基矢的洛伦兹逆变换，最终四维矢量不变。
    - 不同观察者的观察结果不同，并不意味着四维矢量发生变化，而是意味着两个观察者观察到这个过程的这两个观察事件不是同一个。不同也是自然的。
+ 综上，在描述物理实际时，选用不同的参考系和参数，最终得到的结果都应该一致，与参考系和参数无关。
    - 要区分事件本身和观察结果的区别。

#### 2-2 广义动量积分
+ 考虑广义动量对时间的导数：

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}  $

+ 如果$ H $不显含一个广义坐标$ q_\alpha $，那么其对应的动量就守恒（两边对$ dt $积分）：

$ p_\alpha = \mathrm{const.} $

    - 其中不显含的这个广义坐标称为**循环坐标**，守恒的动量值称为**广义动量积分**。

### 3 刘维尔定理
#### 3-1 相空间
+ 设系统有$ s $个自由度，那么广义坐标和广义动量张开一个$ 2s $维的空间，这就是**相空间**。相空间中每一个点都代表着系统的一个状态。
    - 在空间中选择一个运动的点，时时刻刻可以描述系统的状态，那么这个点被称为**代表点**。
    - 代表点在相空间的运动轨迹为**相轨道**。
    - 给定约束条件下系统所有可能得状态的集合称为**系综**。系综在相空间中是一个区域。
+ 考虑相空间的一个体积元$ dv $，其中有$ dN $个代表点，那么密度可以定义为：

$ \rho = \frac{dN}{dv} $

    - 其中，相空间的体积元定义为：

$ dv = d q_1 d q_2\cdots d q_sd p_1 d p_2\cdots d p_s $

#### 3-2 刘维尔定理的内容
+ 刘维尔定理表述为：系统的代表点在相空间中运动时，密度$ \rho $不变。

$ \frac{d\rho}{dt} = \frac{\partial \rho}{\partial t} +\sum_{\alpha=1}^s (\frac{\partial \rho}{\partial p_\alpha}\dot p_\alpha+\frac{\partial \rho}{\partial q_\alpha}\dot q_\alpha) = 0 $

+ 定理的意思是，跟着代表点一起去运动观察密度的时间变化率为零。

$ \frac{d\rho}{dt} = 0 $

    - $ \frac{\partial \rho}{\partial t} $代表的是相空间中静止点本地的变化率。
    - $ \sum_{\alpha=1}^s (\frac{\partial \rho}{\partial p_\alpha}\dot p_\alpha+\frac{\partial \rho}{\partial q_\alpha}\dot q_\alpha) $代表的是因为附近代表点运动造成的密度变化。

#### 3-3 刘维尔定理的证明
+ 首先推导$ \frac{\partial \rho}{\partial t} $的表达式：
    - 考虑代表点出发点附近的一个微小体元$ dv $，与$ q_1 $方向垂直的面为$ dA_1 $。那么在$ q_1 $方向上造成的代表点净增量速率为：

$ \frac{\partial N_1}{\partial t} = [\rho \dot q_1 dA_1]|_{q_1}-[\rho \dot q_1 dA_1]|_{q_1+dq_1} = -\frac{\partial}{\partial q_1}(\rho \dot q_1)dq_1dA_1 
=-\frac{\partial}{\partial q_1}(\rho \dot q_1)dv $

    - 相应的密度增量速率为：

$ \frac{\partial \rho _1}{\partial t} = \frac{1}{dv}\frac{\partial N_1}{\partial t}  = -\frac{\partial}{\partial q_1}(\rho \dot q_1) $

    - 其他方向的可以同理思考：

$ \frac{\partial \rho}{\partial t} = -\sum_{\alpha=1}^s [\frac{\partial}{\partial q_\alpha}(\rho \dot q_\alpha) + \frac{\partial}{\partial p_\alpha}(\rho \dot p_\alpha)]
 $

+ 代入表达式计算$ \frac{d\rho}{dt} $：

$ \frac{d\rho}{dt} = -\sum_{\alpha=1}^s [\frac{\partial}{\partial q_\alpha}(\rho \dot q_\alpha) + \frac{\partial}{\partial p_\alpha}(\rho \dot p_\alpha)] +\sum_{\alpha=1}^s (\frac{\partial \rho}{\partial p_\alpha}\dot p_\alpha+\frac{\partial \rho}{\partial q_\alpha}\dot q_\alpha)
\\= -\rho \sum_{\alpha=1}^s (\frac{\partial\dot p_\alpha}{\partial p_\alpha}+\frac{\partial\dot q_\alpha}{\partial q_\alpha}) $

    - 代入下述关系：

$ \dot p_\alpha = -\frac{\partial H}{\partial q_\alpha}; \;\dot q_\alpha = \frac{\partial H}{\partial p_\alpha},\alpha =1,2,...,s $ 

    - 得到：

$ \frac{d\rho}{dt} = -\rho \sum_{\alpha=1}^s (-\frac{\partial^2H}{\partial p_\alpha q_\alpha}+\frac{\partial^2H}{\partial q_\alpha p_\alpha}) = 0 $

$ \mathrm{Q.E.D.} $

#### 4-2 非惯性系拉格朗日量、哈密顿量的一般讨论
+ 为了写出非惯性系$ Oxyz $中的哈密顿量$ H $和运动方程，应该如此操作：
+ 在惯性系$ O\xi \eta \zeta $中求出动能$ T $和势能$ V $，计算出拉格朗日量$ L $。
    - 使用坐标变换方程，将拉格朗日量$ L(\xi,\eta,\zeta,\dot \xi,\dot\eta,\dot\zeta,t) $用旋转坐标系的坐标$ x,y,z $表示为$ \bar{L}(x,y,z,\dot x,\dot y,\dot z,t) $。
        * 坐标变换方程常常显含时间：

$ x = x(\xi,\eta,\zeta,t) $

$ y = y(\xi,\eta,\zeta,t) $

$ z = z(\xi,\eta,\zeta,t) $

        * 由此反解出来：

$ \xi = \xi (x,y,z,t) $

$ \eta = \eta (x,y,z,t) $

$ \zeta = \zeta (x,y,z,t) $

        * 最终用旋转坐标系坐标$ x,y,z $表示为$ \bar{L}(x,y,z,t) $：

$ \bar{L}(x,y,z,\dot x,\dot y,\dot z,t) = L(\xi,\eta,\zeta,\dot \xi,\dot\eta,\dot\zeta,t) $

+ 根据下面的式子计算出**旋转坐标系**$ Oxyz $**广义坐标**$ x,y,z $**对应的广义动量**$ p_x,p_y,p_z $：

$ p_x = \frac{\partial \bar{L}}{\partial \dot x} $$ p_y = \frac{\partial \bar{L}}{\partial \dot y} $$ p_z = \frac{\partial \bar{L}}{\partial \dot z} $

    - <font style="color:#DF2A3F;">其中广义速度</font>$ \dot x,\dot y,\dot z $<font style="color:#DF2A3F;">的表达式可以通过上述表达式用</font>$ x,y,z,p_x,p_y,p_z $<font style="color:#DF2A3F;">反解出来：</font>

$ \dot x = \dot x (x,y,z,p_x,p_y,p_z) $

$ \dot y = \dot y (x,y,z,p_x,p_y,p_z) $

$ \dot z = \dot z (x,y,z,p_x,p_y,p_z) $

+ 最后根据哈密顿量的定义式计算出**旋转坐标系**$ Oxyz $**的哈密顿量**$ \bar{H} $：

$ \bar{H} = \sum \dot q_\alpha p_\alpha - \bar L = (\dot x p_x + \dot y p_y + \dot z p_z) -\bar{L} $

    - <font style="color:#DF2A3F;">哈密顿量中不应该出现</font>$ \dot x,\dot y,\dot z $<font style="color:#DF2A3F;">，所以应该将其用</font>$ x,y,z,p_x,p_y,p_z $<font style="color:#DF2A3F;">表示。</font>
+ **旋转坐标系**$ Oxyz $**哈密顿正则方程**即为：

$ \dot x = \frac{\partial \bar H}{\partial p_x} $$ \dot p_x = -\frac{\partial \bar H}{\partial x} $

$ \dot y = \frac{\partial \bar H}{\partial p_y} $$ \dot p_y = -\frac{\partial \bar H}{\partial y} $

$ \dot z = \frac{\partial \bar H}{\partial p_z} $$ \dot p_z = -\frac{\partial \bar H}{\partial z} $

+ <font style="color:#DF2A3F;">总结：之所以在惯性系获取拉格朗日量</font>
    - <font style="color:#DF2A3F;">是因为第二类拉格朗日方程</font>$ \frac{d}{dt}\frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} =0 $<font style="color:#DF2A3F;">只在惯性系中成立。</font>
        * <font style="color:#DF2A3F;">如果想在非惯性系直接写拉格朗日方程，则需要引入惯性力相应的势能。</font>
        * <font style="color:#DF2A3F;">旋转参考系中，就是科里奥利力对应的广义势和离心势能。</font>
    - <font style="color:#DF2A3F;">就像牛顿第二定律</font>$ F = ma $<font style="color:#DF2A3F;">只在惯性系中成立一样。</font>
        * <font style="color:#DF2A3F;">如果想非惯性系用牛顿第二定律，则需要引入惯性力，或者说逆转力。</font>
        * <font style="color:#DF2A3F;">旋转参考系中，就是科里奥利力和离心力。</font>
    - 这在之前的一个[例题](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/os38ueagof3ug0u1#OXi4Y)中也进行过论述。

### 6 泊松括号
#### 6-1 泊松括号的定义
+ 力学量$ f,g $之间的泊松括号记为$ [f,g] $，其定义为：

$ [f,g] = \sum_{\alpha=1}^s (\frac{\partial f}{\partial q_\alpha}\frac{\partial g}{\partial p_\alpha} -\frac{\partial f}{\partial p_\alpha}\frac{\partial g}{\partial q_\alpha}) $

    - 或许还可以写为：

$ [f,g] = \sum_{\alpha=1}^s \det [\frac{\partial(f,g)}{\partial(q_\alpha,p_\alpha)}] $

    - 其中$ \frac{\partial(f,g)}{\partial(q_\alpha,p_\alpha)} $为雅可比行列式。

#### 6-2 泊松括号的数学性质
1. 反对称性：

$ [f,g] = - [g,f] $

2. 线性性：

$ [f,c_1g_1+c_2g_2] = c_1[f,g_1]+c_2[f,g_2] $

3. 涉及常数的泊松括号为零：

$ [f,C] = 0, \; C = \mathrm{const.} $

4. 对时间的偏微商法则：

$ \frac{\partial}{\partial t}[f,g] = [\frac{\partial f}{\partial t},g]+ [f,\frac{\partial g}{\partial t}] $

5. 乘积性质：

$ [f,gh] = g[f,h]+[f,g]h $

    - 此处没有写成$ [f,gh] = g[f,h]+h[f,g] $的形式，而是保持了$ g,h $的相对顺序
    - 是因为在量子力学中$ f,g,h $可能不是数而是算符，就不能随便交换了。
6. 泊松恒等式（轮转交换相加等于零）：

$ [f,[g,h]] + [g,[h,f]]+[h,[f,g]] = 0 $

#### 6-3 力学量的泊松括号
1. 力学量的时间偏导数：

$ \frac{df}{dt} = \frac{\partial f}{\partial t} + \sum_{i=1}^{s}(\frac{\partial f} {\partial q_\alpha}\dot q_\alpha+\frac{\partial f} {\partial p_\alpha}\dot p_\alpha)
= \frac{\partial f}{\partial t} + \sum_{i=1}^{s}(\frac{\partial f} {\partial q_\alpha} \frac{\partial H} {\partial p_\alpha}-\frac{\partial f} {\partial p_\alpha}\frac{\partial H} {\partial q_\alpha}) \\ = \frac{\partial f}{\partial t} + [f,H] $

    - 力学量$ f(q,p,t) $随着时间的演化速率由$ \frac{\partial f}{\partial t} $以及与哈密顿量的泊松括号决定。
    - 如果$ f = f(q,p) $即$ f $不显含$ t $而且$  [f,H]=0 $，那么$ f $不随时间变化。
2. 基本的泊松括号：

$ [q_\alpha,q_\beta] = 0 $

$ [p_\alpha,p_\beta] = 0 $

$ [q_\alpha,p_\beta] =\delta_{\alpha\beta} $

    - 即仅有相互对应的广义坐标和广义动量的泊松括号为1。
3. 用泊松括号将哈密顿正则方程写成更加对称的形式：

$ \dot q_\alpha = [q_\alpha,H] ; \; \dot p_\alpha = [p_\alpha,H], \; \alpha = 1,2,...,s $

#### 6-3 用泊松括号描述刘维尔定理
+ 将密度$ \rho $也看做一个力学量：

$ \frac{d\rho}{dt} = \frac{\partial \rho}{\partial t} + [\rho,H] = 0 $

#### 6-4 从经典力学到量子力学
+ 量子力学中，力学量都变成了算符。
