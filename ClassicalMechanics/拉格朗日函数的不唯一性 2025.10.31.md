#### 多个拉格朗日函数可能给出相同的物理
+ 设一个函数$ f $只是广义坐标和时间的函数：

$ \begin{equation}
f = f(q,t)
\end{equation} $

+ 拉格朗日函数的不唯一性，指的是对于拉格朗日函数$ L $和$ L' = L+\frac{df}{dt} $给出相同的物理，即下述两个方程形式相同：

$ \begin{equation}\frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} = 0
\end{equation} $

$ \begin{equation} \frac{d}{dt} \frac{\partial L'}{\partial \dot q_\alpha} - \frac{\partial L'}{\partial q_\alpha} = 0 \end{equation} $

+ 下面给出证明。

#### 拉格朗日函数的不唯一性证明
+ 如果能证明：

$ \begin{equation}
\frac{d}{dt} \frac{\partial \dot f}{\partial \dot q_\alpha} - \frac{\partial \dot f}{\partial q_\alpha} = 0
\end{equation} $

    - 那么有：

$ \frac{d}{dt} \frac{\partial L'}{\partial \dot q_\alpha} - \frac{\partial L'}{\partial q_\alpha} =\frac{d}{dt} \frac{\partial (L+\dot f)}{\partial \dot q_\alpha} - \frac{\partial (L+\dot f)}{\partial q_\alpha} \\ $

$ = \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} - (\frac{d}{dt} \frac{\partial \dot f}{\partial \dot q_\alpha} - \frac{\partial \dot f}{\partial q_\alpha}) = \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} - 0 $

$ \\=\frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} = 0 $

    - 即：

$ \frac{d}{dt} \frac{\partial L}{\partial \dot q_\alpha} - \frac{\partial L}{\partial q_\alpha} = 0 $

$ \frac{d}{dt} \frac{\partial L'}{\partial \dot q_\alpha} - \frac{\partial L'}{\partial q_\alpha} = 0 $

+ 下面对$ (4) $给出证明：
    - 首先写出$ \dot f $的表达式：

$ \begin{equation}
\dot f = \frac{df(q,t)}{dt}  = \sum_{\beta =1}^s \frac{\partial f}{\partial q
_\beta}\dot q_\beta +\frac{\partial f}{\partial t}
\end{equation} $

    - $ \dot f $对$ \dot q_\alpha $求偏导，注意角标：

$ 
\frac{\partial \dot f}{\partial \dot q_\alpha} = \frac{\partial }{\partial \dot q_\alpha } (\sum_{\beta =1}^s \frac{\partial f}{\partial q
_\beta}\dot q_\beta+\frac{\partial f}{\partial t})
= \frac{\partial f}{\partial  q_\alpha}

 $

        * 得到一个非常有意思的结果：

$ \begin{equation}
\boxed{\frac{\partial \dot f}{\partial \dot q_\alpha} = \frac{\partial  f}{\partial  q_\alpha}}
\end{equation} $

    - $ \dot f $对$ q_\alpha $求偏导，注意角标：

$ \frac{\partial \dot f}{\partial  q_\alpha} = \frac{\partial }{\partial q_\alpha } (\sum_{\beta =1}^s \frac{\partial f}{\partial q
_\beta}\dot q_\beta+\frac{\partial f}{\partial t})
=\sum_{\beta =1}^s \frac{\partial^2 f}{\partial q_\alpha\partial q
_\beta}\dot q_\beta+\frac{\partial^2 f}{\partial q_\alpha\partial t}
= \frac{d}{dt} (\frac{\partial f}{\partial q_\alpha})

 $

        * 这也是一个非常有意思的结果：

$ \begin{equation}
\boxed{\frac{\partial \dot f}{\partial  q_\alpha}
= \frac{d}{dt} (\frac{\partial f}{\partial q_\alpha})}
\end{equation}

 $

        * 将$ (6) $代入其中，得到：

$ \frac{\partial \dot f}{\partial  q_\alpha}
= \frac{d}{dt} (\frac{\partial f}{\partial q_\alpha})
= 
\frac{d}{dt}
(\frac{\partial \dot f}{\partial \dot q_\alpha})
 $



+ 移项即得：

$ \frac{d}{dt} \frac{\partial \dot f}{\partial \dot q_\alpha} - \frac{\partial \dot f}{\partial q_\alpha} = 0 $

+ 拉格朗日函数的不唯一性，指的是对于拉格朗日函数$ L $和$ L' = L+\frac{df}{dt} $给出相同的物理，其中$ f = f(q,t) $。

$ \mathrm{Q.E.D.} $

#### 结果意义讨论
+ 不同拉格朗日函数可能给出相同的物理，彼此差上一个只显含广义坐标和时间的函数对时间的全导数$ \dot f = \frac{df(q,t)}{dt} $。

#### 一个有趣的结果
+ 我们在推导中发现，对于函数$ f = f(q,t) $有：

$ \frac{\partial \dot f}{\partial \dot q_\alpha} = \frac{\partial  f}{\partial  q_\alpha} $

$ \frac{\partial \dot f}{\partial  q_\alpha}
= \frac{d}{dt} (\frac{\partial f}{\partial q_\alpha}) $

    - 这让我们想到了拉格朗日方程基础一章中我们推到过的[拉格朗日关系](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/in7z7g1ogm3bl9ma#rbWmP)：

$ \frac{\partial\dot{\vec{r_i}}}{\partial \dot{q}_\alpha} 
=\frac{\partial\vec{r_i}}{\partial q_\alpha}
 $

$ \frac{\partial\dot{\vec{r_i}}}{\partial q_\alpha} = \frac{d}{dt} (\frac{\partial \vec{r_i}}{\partial q_\alpha}) $

+ 现在看来$ \vec{r_i}(q,t) $只是$ f = f(q,t) $的一种。
