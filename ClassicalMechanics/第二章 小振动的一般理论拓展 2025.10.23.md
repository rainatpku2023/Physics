### 2-3 有限振幅振动：以单摆为例
#### 情景概述
+ 考虑一个摆长为$ l $，质量为$ m $的单摆，所有阻力都可以忽略。
+ 选定摆线与铅直方向夹角为$ \phi $，拉格朗日量为：

$ \begin{equation}
L = \frac{1}{2}ml^2\dot \phi^2 +mgl\cos\phi
\end{equation} $

+ 代入拉格朗日方程：

$ \begin{equation}
\frac{d}{dt}\frac{\partial L}{\partial \dot \phi} - \frac{\partial L}{\partial \phi} =0
\end{equation} $

+ 并对两边积分得到：

$ \begin{equation}
\frac{1}{2}ml^2\dot \phi^2 - mgl \cos \theta = E
\end{equation} $

    - 这其实就是能量守恒。

#### 周期求解
+ 假设最大摆角度为$ \phi_0 $，那么显然有：

$ \begin{equation}
E\label{a} = -mgl\cos \phi_0
\end{equation} $

+ 代入能量守恒方程：

$ \begin{equation}
\frac{1}{2}ml^2\dot \phi^2 - mgl \cos \theta = -mgl\cos \phi_0
\end{equation} $

+ 求解$ d\phi/dt $：

$ \begin{equation}
\frac{d\phi}{dt} = \dot \phi =  \pm \sqrt{\frac{2g}{l} (\cos \phi - \cos\phi_0)}
\end{equation} $

+ 考虑从最高点到最低点的$ 1/4 $周期内，符号取负。将$ dt $移动到等号右侧并积分：

$ \begin{equation}
\int_{\phi_0}^{\phi} \frac{d\phi'}{\sqrt{ (\cos \phi' - \cos\phi_0)}} =  -\int_{0}^{t} \sqrt{\frac{2g}{l} }dt' = -\sqrt{\frac{2g}{l}} t
\end{equation} $

    - 其中$ \phi ',t' $为积分过程中引入的哑变量。
+ 考虑到三角函数关系：

$ \begin{equation}
\cos\phi = 1-2\sin^2 \frac{\phi}{2}
\end{equation} $

    - 可以对上述积分进行进一步化简：

$ \begin{equation}
\int_{\phi_0}^{\phi} \frac{d\phi'}{\sqrt{ (\sin^2 \frac{\phi'}{2} - \sin^2 \frac{\phi_0}{2})}}  = -2\sqrt{\frac{g}{l}} t
\end{equation} $

#### 非小振动情形求解
+ 引入变量$ u $，令：

$ \begin{equation} \sin u = \frac{\sin{\frac{\phi}{2}}}{\sin\frac{\phi_0}{2}}\end{equation} $

    - 注意到分母$ \sin \frac{\phi_0}{2} $是常数，令其为$ k $，则：

$ \begin{equation} k = \sin \frac{\phi_0}{2} \end{equation} $

$ \begin{equation} \sqrt{ (\sin^2 \frac{\phi'}{2} - \sin^2 \frac{\phi_0}{2})} 
 = k\sqrt{1-\sin^2u}\end{equation} $

$ \begin{equation}
d\phi = \sqrt{\frac{1-\sin ^2u}{1-k^2 \sin^2 u}}2kdu
\end{equation} $

+ 两者代入积分表达式，得：

$ \begin{equation}
\int_{\phi_0}^{\phi} \frac{du}{\sqrt{ 1-k^2 \sin^2 u}}  = -\sqrt{\frac{g}{l}} t
\end{equation} $

+ 考虑到：$ t = 0, \phi = \phi_0, \sin u = 1, u =\pi/2 $以及$ t = T/4, \phi = 0, \sin u =0, u = 0 $

$ \begin{equation}
\int_{\pi/2}^{0} \frac{du}{\sqrt{ 1-k^2 \sin^2 u}}  = -\sqrt{\frac{g}{l}} \frac{T}{4}
\end{equation} $

    - 整理得到：

$ \begin{equation}
T = 4\sqrt{\frac{l}{g}}\int_{0}^{\pi/2} \frac{du}{\sqrt{ 1-k^2 \sin^2 u}} = 4\sqrt{\frac{l}{g}} K(k)
\end{equation} $

+ 其中$ K(k) $为椭圆积分，其定义为：

$ \begin{equation}
K(k) = \int_0^{\pi/2} \frac{du}{\sqrt{1-k^2\sin^2u}}
\end{equation} $

    - 对于$ k<1 $，椭圆积分可以展开成$ \mathrm{Taylor} $级数的形式：

$ \begin{equation}
K(k) = \frac{\pi}{2}[1^2+(\frac{1}{2})^2k^2 + (\frac{1\cdot 3}{2 \cdot 4})^2 k^4 +  (\frac{1\cdot 3\cdot 5}{2 \cdot 4 \cdot 6})^2 k^6+ \cdots ] =
\frac{\pi}{2}\sum_{n=0}^{\infin} [\frac{(2n-1)!!}{(2n)!!}]^2k^{2n}
\end{equation} $

+ 考虑$ k $的表达式（11）可知，当$ \phi_0 < \pi $时周期也可以展开成：

$ \begin{equation}
T = 2\pi \sqrt{\frac{l}{g}}(1+\frac{1}{4}\sin^2 \frac{\phi_0}{2} + \cdots)
\end{equation} $

#### 小振动近似的合理性
+ 进行估计，当$ \phi_0 = 15° $时，

$ T = (1+0.42\%) 2\pi \sqrt{\frac{l}{g}} $

    - 说明准确结果和小振动近似相比只有$ 0.42\% $的修正。
    - 可见小振动近似在周期方面非常合理。

### 2-5 受迫振动
#### 情景概述
+ 假设策动力$ Q $是周期性的，即：

$ Q = Q_0 \cos \omega t $

+ 代入拉格朗日方程，得到运动方程：

$ \ddot x+ \omega_0^2 x = Q_0 \cos \omega t $

    - 其中$ \omega_0 $为自由振动时频率。

#### 方程求解
+ 猜测受迫振动部分的解为：

$ x = B \cos \omega t $

    - 代入运动方程，可以算出受迫振动部分的振幅：

$ B = \frac{Q_0}{\omega_0^2-\omega^2} $

    - 可见：
        * 受迫振动振幅$ \omega $与本征频率和策动力频率$ \omega_0 $的关系有关，二者越接近受迫振动振幅越大。
        * 当策动力频率$ \omega $小于本征频率$ \omega_0 $时，振幅$ B $为正，说明振动方向与策动力方向相同。
        * 当策动力频率$ \omega $小于本征频率$ \omega_0 $时，振幅$ B $为负，说明振动方向与策动力方向相反。
+ 最终一般解应该是是无阻尼本征振动和受迫振动的叠加：

$ x(t) = A\cos(\omega_0 t+\phi_0) + \frac{Q}{\omega_0^2- \omega^2}\cos \omega t $

### 2-6 阻尼振动
#### 情景概述
+ 在速度较小的情况下，收到的阻力与速度大小成正比，方向与速度相反。引入耗散函数概念。

[第一章 拉格朗日方程拓展 2025.09.26](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/db8pwwg08cgt84qe#bUbLp)

+ 假设小振动的广义坐标已经选定为简正坐标$ \xi_1,\xi_2,\xi_3,...,\xi_s $，那么拉格朗日方程：

$ \frac{d}{dt} \frac{\partial L}{\partial \dot \xi_i}-\frac{\partial L}{\partial  \xi_i} = Q_i $

#### 方程求解
+ 代入具体表达式后，运动方程变为：

$ \ddot \xi_i + b_i \dot \xi_i+ \omega_i^2 \xi_i = 0 $

+ 取试探解为：

$ \xi_i = c_i e^{\lambda_i t} $

    - 带入后得到代数方程

$ \lambda_i^2 + b_i \lambda_i +\omega_i^2 = 0 $

    - 得到解为：

$ \lambda_i = -\frac{b_i}{2} \pm \sqrt{\frac{b_i^2}{4}-\omega_i^2} $

+ 弱阻尼条件下$ b_i < 2\omega_i $，根号下是负数，$ \lambda_i $实部和虚部都不为零
    - 虚部：不为零，说明振荡，震荡周期为：

$ \omega_i' = \sqrt{\omega_i^2 - \frac{b_i^2}{4}} $

        * 可见阻尼作用会让振子频率减小。
    - 实部：负数，说明振幅不断下降
+ 强阻尼条件下，$ b_i > 2\omega_i $，根号下是正数，$ \lambda_i $两个值都是实数。
    - 经过计算后可以发现$ \lambda_i $两个值都是负数，因此运动表现为衰减。
