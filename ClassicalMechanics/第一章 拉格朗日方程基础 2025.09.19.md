### 1-1 约束与广义坐标
#### 约束的定义
+ **定义**：约束就是对运动的限制
+ 约束的数学表达
    - 考虑由$ n $个质点组成的力学体系，它们的位置矢量分别用$ \vec{r_1},\vec{r_2},...,\vec{r_n} $表示。
    - 在笛卡尔坐标系中，可以用$ 3n $个坐标表示：$ x_1,x_2,x_3,...,x_{3n} $
    - 约束数学形式就是**约束方程**：

$ f(x_1,x_2,...,x_{3n};\dot{x}_1,\dot{x}_2,...,\dot{x}_{3n};t) \sim 0 $

        * 为了简便，可以缩写成如下形式：

$ f(x_i;\dot{x}_i;t) \sim 0 $

#### 约束的分类
##### 单侧约束与双侧约束
+ 单侧约束：一侧可以解除，另一侧不可以解除的约束，如软线单摆，用不等式表示：

$ f(x_i;\dot{x}_i;t) < 0 $

+ 双侧约束：两侧都不可以解除的约束，如刚性杆，用等式表示：

$ f(x_i;\dot{x}_i;t) = 0 $

##### 完整约束与非完整约束
+ 完整约束：又称**几何约束**，指约束方程中**不显含速度**或者可以化成不显含速度的形式的约束：

$ f(x_i;t) = 0 $

+ 非完整约束：又称**微分约束**，指约束方程中**显含速度**而且无法化成不显含速度的形式的约束，约束的是无穷小位移的关系：

$ f(x_i;\dot{x}_i;t) = 0 $

##### 稳定约束与非稳定约束
+ 稳定约束：又称**非含时约束**，指约束方程中**不显含时间**或者可以化成不显含时间的形式的约束，例如蚂蚁在固定的地球仪表面上移动：

$ f(x_i;\dot{x}_i) = 0 $

+ 非稳定约束：又称**含时约束**，指约束方程中**显含时间**而且无法化成不显含时间的形式的约束，l例如蚂蚁在不断膨胀的球形气球表面上移动：

$ f(x_i;\dot{x}_i;t) = 0 $

### 1-2 虚位移与虚功
#### 虚位移
+ 虚位移的定义：在某一时刻，将所有的约束冻结（不随时间变化），满足约束的任意无穷小位移。
    - 特点：某一时刻；满足约束；约束冻结；无穷小位移
+ 虚位移与实位移
    - 实位移：产生在时间间隔内，符合力学规律的无穷小位移；物理上实际发生的位移
    - <font style="color:#DF2A3F;">对于稳定约束</font>，无限小实位移就是虚位移中的某一个 -> 要求约束不随时间变化
+ 虚位移的方向：与约束函数的梯度垂直

:::tips
+ 证明：
    - 设一个质点的受到约束的方程为：

$ f(x,y,z;t) = 0 $

    - 对两侧求变分，得：

$ \frac{\partial f}{\partial x} \delta x+\frac{\partial f}{\partial y} \delta y+\frac{\partial f}{\partial z} \delta z
+\frac{\partial f}{\partial t} \delta t = 0 $

    - 考虑到虚位移是等式变分$ \delta t = 0 $：

$ \frac{\partial f}{\partial x} \delta x+\frac{\partial f}{\partial y} \delta y+\frac{\partial f}{\partial z} \delta z = 0 $

    - 使用拉普拉斯算符，可以写为：

$ \nabla f\cdot \delta \vec{r} = 0 $

    - 即虚位移与约束函数的梯度垂直，或者说沿着约束平面。

$ \mathrm{Q.E.D.} $

:::

#### 虚功与理想约束
+ 虚功的定义：一个力的虚功是其与虚位移的内积：

$ \boxed{\delta W \equiv \vec{F} \cdot \delta \vec{r}} $

+ 考虑所有质点的受到的主动力$ \{\vec{F}_i^{(a)}\} $和约束力$ \{\vec{N}_i\} $的虚功之和：

$ \sum_i \delta W_i = \sum_i (\vec{F}_i^{(a)}\cdot \delta \vec{r}+\vec{N}_i\cdot \delta \vec{r}) $

+ 理想约束：这样的一组约束力，对于所有可能的虚位移的做功之和都为零，即

$ \sum_i \vec{N}_i\cdot \delta \vec{r}=0 $

    - 在<font style="color:#DF2A3F;">理想约束的条件</font>下，总虚功可以表示为：

$ \delta W = \sum_{i=1}^{3n} \delta W_i= \sum_{i=1}^{n} \vec{F}_i^{(a)}\cdot \delta \vec{r}= \sum_{i=1}^{3n}F_i^{(a)}\cdot \delta x_i $

    - 理想约束的特征：非耗散

### 1-3 虚功原理，广义坐标与广义力
#### 虚功原理
+ 地位：第一性原理，无法被推出的内容，相当于牛顿力学中的牛顿第一定律
+ 内容：
    - <font style="color:#DF2A3F;">理想约束</font>下，体系的**平衡条件**为：作用在体系中的**所有主动力虚功为零**。

$ \boxed {\sum_{i=1}^{3n}F_i^{(a)}\cdot \delta x_i = 0} $

+ 非理想约束的处理方式：将约束解除，用约束力代替，约束力列入主动力中。

#### 广义坐标
+ 对于由$ n $个质点组成的系统，在笛卡尔坐标系中，系统的状态可以由$ 3n $个坐标来描述，他们是：

$ x_1,x_2,x_3,...,x_{3n} $

+ 当系统受到$ k $个约束时，系统的坐标只有$ s=3n-k $个可以独立变化，剩下的都可以被它们标出。
    - $ s $被称为系统的**自由度**。
    - 此时可以引入$ s $个相互独立的广义坐标$ q_1,q_2,....,q_s $，用来代替上述$ 3n $个坐标：

$ \boxed {x_i = x_i(q_1,q_2,....,q_s;t)} $

    - 上面以$ q_1,q_2,....,q_s $作为自变量，写出$ x_i $的表达式，这是**坐标变换方程**。
+ 注意：<font style="color:#DF2A3F;">广义坐标的选取依赖于约束方程</font>！
+ 同理可以定义：
    - 广义实位移$ d q_i $
    - 广义虚位移$ \delta q_i $



+ 下面推导**坐标与广义坐标的变分关系/坐标变换方程的等时变分**：
    - 对坐标变换方程两侧取变分：

$ \delta x_i = \sum_{\alpha=1}^{s} \frac{\partial x_i}{\partial q_\alpha} \cdot \delta q_\alpha + \frac{\partial x_i}{\partial t}\cdot \delta t $

    - 当变分为等时变分时，$ \delta q_\alpha $为虚位移，$ \delta t = 0 $：

$ \delta x_i = \sum_{\alpha=1}^{s} \frac{\partial x_i}{\partial q_\alpha} \delta q_\alpha  $

#### 广义力
+ 理想约束条件下，考虑系统受到主动力的虚功：

$ \delta W = \sum_{i=1}^{3n}F_i^{(a)}\cdot \delta x_i $

+ 将坐标变换方程的等时变分形式代入：

$ \delta W = \sum_{i=1}^{3n}F_i^{(a)}\cdot (\sum_{\alpha=1}^{s} \frac{\partial x_i}{\partial q_\alpha} \delta q_\alpha) $

+ 交换求和次序：

$ \delta W = \sum_{\alpha=1}^{s}(\sum_{i=1}^{3n}F_i^{(a)} \frac{\partial x_i}{\partial q_\alpha}) \cdot \delta q_\alpha $

+ 括号内的部分定义为广义力$ Q_\alpha $，与广义坐标$  q_\alpha $相对应：

$ \boxed{Q_\alpha = \sum_{i=1}^{3n}F_i^{(a)} \frac{\partial x_i}{\partial q_\alpha}} $

+ 此时虚功可以用更简洁的形式表出：

$ \boxed{\delta W = \sum_{\alpha=i}^{s} Q_\alpha\cdot \delta q_\alpha}  $

#### 广义坐标下的虚功原理：广义力均为零
+ 根据上文，虚功原理可以表示为：

$ \delta W = \sum_{\alpha=i}^{s} Q_\alpha\cdot \delta q_\alpha = 0  $

+ 因为选取广义坐标$ q_1,q_2,....,q_s $时保证了它们相互独立
    - 所以广义虚位移$ \delta q_1,\delta q_2,....,\delta q_s $可以独立变化，而且不一定取零值
    - 所以虚功原理进一步简化为广义力全部为零：

$ \boxed{Q_\alpha = 0,\; \mathrm{for} \: \mathrm{all} \: \alpha.} $

### 1-4 达朗贝尔原理（动力学问题，牛顿第二定律）
+ 按照牛顿运动定律，力学系统的第$ i $个质点的运动方程为：

$ \vec{F}_i^{(a)} + \vec{N}_i = m_i \ddot{\vec{r_i}} $

+ 移项得：

$ \vec{F}_i^{(a)} + \vec{N}_i -m_i \ddot{\vec{r_i}} = 0 $

    - 其中$ -m_i \ddot{\vec{r_i}} $被称为达朗贝尔力，逆转力或者倒转有效力。
+ 乘以虚位移并对质点求和得：

$ \sum_{i=1}^{n}(\vec{F}_i^{(a)} + \vec{N}_i -m_i \ddot{\vec{r_i}})\cdot \delta \vec{r}= 0 $

+ 标量化：

$ \sum_{i=1}^{3n}(F_i^{(a)} + N_i -m_i \ddot{x_i})\cdot \delta x_i= 0 $

+ 对于<font style="color:#DF2A3F;">理想约束</font>，约束力虚功总和为零，得到**达朗贝尔原理**：

$ \boxed{\sum_{i=1}^{3n}(F_i^{(a)} -m_i \ddot{x_i})\cdot \delta x_i= 0} $

    - 其作用相当于牛顿第二定律。

### 1-5 第一类拉格朗日方程（λ乘子法，约束力的求解）
#### 第一类拉格朗日方程的导出
+ 对于<font style="color:#DF2A3F;">完整的理想约束</font>，约束方程（组）为：

$ f_\alpha = f_\alpha(x_1,x_2,...,x_{3n};t) = 0, \;\alpha  = 1,2,...,k $

    - 显然，$ k $为约束的个数。
    - 对约束方程组两边取等时变分：

$ \sum_{i=1}^{3n} \frac{\partial f_\alpha}{\partial x_i}\cdot \delta x_i = 0 $

    - 引入$ k $个$ \lambda $乘子，第$ \alpha $个约束方程对应的乘子为$ \lambda_\alpha $：

$ \lambda_{\alpha}\sum_{i=1}^{3n} \frac{\partial f_\alpha}{\partial x_i}\cdot \delta x_i = 0 $

    - 对于$ \alpha $求和并交换求和次序：

$ \sum_{i=1}^{3n} (\sum_{\alpha =1}^{s}\lambda_{\alpha}\frac{\partial f_\alpha}{\partial x_i})\cdot \delta x_i = 0 $

+ 回顾推导达朗贝尔原理时的中间结果：

$ \sum_{i=1}^{3n}(F_i^{(a)} + N_i -m_i \ddot{x_i})\cdot \delta x_i= 0 $

    - 将表达式代入$ N_i $得：

$ \sum_{i=1}^{3n}(F_i^{(a)}  -m_i \ddot{x_i}+\sum_{\alpha =1}^{k}\lambda_{\alpha}\frac{\partial f_\alpha}{\partial x_i})\cdot \delta x_i= 0 $

    - 此时引入$ k $个$ \lambda $乘子后，全部约束解除，$ \delta x_1,\delta x_2,...,\delta x_{3n} $均可自由变化，求和号内每一项都为零，得到$ 3n $个方程：

$ F_i^{(a)}  -m_i \ddot{x_i}+\sum_{\alpha =1}^{k}\lambda_{\alpha}\frac{\partial f_\alpha}{\partial x_i} = 0, \; \mathrm{for} \;i = 1,2,...,3n $

    - 考虑到总共有$ x_1,x_2,...,x_{3n},\lambda_1,\lambda_2,...,\lambda_k $总共$ 3n+k $个未知量，引入全部约束方程（$ k $个）联立求解：

$ \begin{cases}
F_i^{(a)}  -m_i \ddot{x_i}+\sum_{\alpha =1}^{k}\lambda_{\alpha}\frac{\partial f_\alpha}{\partial x_i} = 0 \\
f_\alpha(x_1,x_2,...,x_{3n};t) = 0
\end{cases} $

+ 这就是**第一类拉格朗日方程**

#### 约束力表达式
+ 最后求解的约束力为：

$ \boxed{\vec{N_i} = \sum_{\alpha=1}^{k} \lambda_\alpha \nabla_i f_\alpha} $

    - 其中$ \lambda_\alpha \nabla_i f_\alpha $是第$ \alpha $个约束对第$ i $个质点的约束力。
    - 在<font style="color:#DF2A3F;">笛卡尔坐标系</font>中，可以进一步写为：

$ \vec{N_i}= \sum_{\alpha=1}^{k} \lambda_\alpha (\hat{x}_i\frac{\partial}{\partial x_i}+\hat{y}_i\frac{\partial}{\partial y_i}+\hat{z}_i\frac{\partial}{\partial z_i}) f_\alpha $

### 1-6 第二类拉格朗日方程（即拉格朗日方程，动力学问题）
#### 拉格朗日关系
+ 第一个拉格朗日关系：

$ \frac{\partial\dot{\vec{r_i}}}{\partial \dot{q}_\alpha} 
= \frac{\partial}{\partial \dot{q}_\alpha}(\frac{\partial\vec{r_i}}{\partial t}+\sum_{\beta=1}^{s}\frac{\partial\vec{r_i}}{\partial q_\beta} \dot{q}_\beta)
=\frac{\partial}{\partial \dot{q}_\alpha}(\frac{\partial\vec{r_i}}{\partial q_\alpha} \dot{q}_\alpha) 
=\frac{\partial\vec{r_i}}{\partial q_\alpha}  $

    - 推导中用了$ \frac{\partial \vec{r_i}}{\partial t} $和$ \frac{\partial \vec{r_i}}{\partial q_\beta} $与$ \dot{q}_\beta $无关的假设。
+ 第二个拉格朗日关系：

$ \frac{\partial\dot{\vec{r_i}}}{\partial q_\alpha} 
= \frac{\partial}{\partial q_\alpha}(\frac{\partial\vec{r_i}}{\partial t}+\sum_{\beta=1}^{s}\frac{\partial\vec{r_i}}{\partial q_\beta} \dot{q}_\beta)
= \frac{\partial^2 \vec{r_i}}{\partial q_\alpha \partial t}+ \sum_{\beta=1}^{s} \frac{\partial^2 \vec{r_i}}{\partial q_\alpha \partial q_\beta} \dot{q}_\beta $

$ =\frac{\partial}{\partial t}(\frac{\partial\vec{r_i}}{\partial q_\alpha}) +\sum_{\beta = 1}^{s} \frac{\partial}{\partial q_\beta}(\frac{\partial\vec{r_i}}{\partial q_\alpha})\dot{q}_\alpha
= \frac{d}{dt} (\frac{\partial \vec{r_i}}{\partial q_\alpha}) $

+ 综上，拉格朗日关系为：

$ \frac{\partial\dot{\vec{r_i}}}{\partial \dot{q}_\alpha} 
=\frac{\partial\vec{r_i}}{\partial q_\alpha}
 $

$ \frac{\partial\dot{\vec{r_i}}}{\partial q_\alpha} = \frac{d}{dt} (\frac{\partial \vec{r_i}}{\partial q_\alpha}) $

#### 第二类拉格朗日方程的导出
+ 达朗贝尔原理（要求<font style="color:#DF2A3F;">约束为理想约束</font>）：

$ \sum_{i=1}^{n}(\vec{F}_i^{(a)} -m_i \ddot{\vec{r_i}})\cdot \delta \vec{r_i}= 0 $

+ 其中第一项可以改写为：

$ \sum_{i=1}^{n}\vec{F}_i^{(a)} \cdot \delta \vec{r_i} = \sum_{\alpha=1}^s (\sum_{i=1}^n\vec{F}_i^{(a)} \cdot \frac{\partial \vec{r}_i}{\partial q_\alpha })  \delta q_\alpha=\sum_{\alpha=1}^s Q_\alpha \delta q_\alpha $

+ 第二项可以改写为：

$  -\sum_{i=1}^{n}m_i \ddot{\vec{r_i}} \cdot \delta \vec{r_i}=
 -\sum_{\alpha=1}^s (\sum_{i=1}^nm_i \ddot{\vec{r_i}}  \cdot \frac{\partial \vec{r}_i}{\partial q_\alpha })  \delta q_\alpha $

    - 其中$ \sum_{i=1}^nm_i \ddot{\vec{r_i}}  \cdot \frac{\partial \vec{r}_i}{\partial q_\alpha } $可以改写为：

$ \sum_{i=1}^nm_i \ddot{\vec{r_i}}  \cdot \frac{\partial \vec{r}_i}{\partial q_\alpha}
= \sum_{i=1}^nm_i \frac{d\dot{\vec{r_i}}}{dt} \cdot \frac{\partial \vec{r}_i}{\partial q_\alpha}
= \sum_{i=1}^nm_i \frac{d}{dt} \cdot (\dot{\vec{r_i}}\cdot \frac{\partial \vec{r}_i}{\partial q_\alpha}) - \sum_{i=1}^n m_i \dot{\vec{r_i}} \cdot \frac{d}{dt} ( \frac{\partial \vec{r}_i}{\partial q_\alpha})  $

    - 代入拉格朗日关系：

$ 
= \sum_{i=1}^nm_i \frac{d}{dt} \cdot (\dot{\vec{r_i}}\cdot \frac{\partial \dot{\vec{r}_i}}{\partial \dot{q}_\alpha}) - \sum_{i=1}^n m_i \dot{\vec{r_i}} \cdot \frac{\partial\dot{\vec{r_i}}}{\partial q_\alpha} $

$ =\frac{d}{dt}\frac{\partial}{\partial \dot{q}_\alpha}(\sum_{i=1}^{n}\frac{1}{2}m_i\vec{r_i}^2) - \frac{\partial}{\partial q_\alpha}(\sum_{i=1}^{n}\frac{1}{2}m_i\vec{r_i}^2) $

    - 引入动能函数$ T $：

$ \sum_{i=1}^nm_i \ddot{\vec{r_i}}  \cdot \frac{\partial \vec{r}_i}{\partial q_\alpha}
=\frac{d}{dt}\frac{\partial T}{\partial \dot{q}_\alpha} - \frac{\partial T}{\partial q_\alpha} $

+ 第一项与第二项合并得：

$ \sum_{i=1}^{n}(\vec{F}_i^{(a)} -m_i \ddot{\vec{r_i}})\cdot \delta \vec{r_i} =  \sum_{\alpha=1}^{s}(Q_\alpha - \frac{d}{dt}\frac{\partial T}{\partial \dot{q}_\alpha} + \frac{\partial T}{\partial q_\alpha})\delta q_\alpha = 0 $

+ 考虑到$ \delta q_\alpha $相互独立，括号内都应该为零，得到**（第二类）拉格朗日方程**：

$ \boxed{\frac{d}{dt}\frac{\partial T}{\partial \dot{q}_\alpha} - \frac{\partial T}{\partial q_\alpha}=Q_\alpha, \;\alpha  = 1,2,...,s} $

#### 主动力全都是保守力的拉格朗日方程
+ 对于<font style="color:#DF2A3F;">主动力全都是保守力</font>的情形，可以找到势能函数$ V $：

$ \vec{F}_i^{(a)} = -\nabla_i V $

+ 则广义力$ Q_\alpha $可以表示为：

$ Q_\alpha = \sum_{i=1}^{n}\vec{F}_i^{(a)} \frac{\partial \vec{r_i}}{\partial q_\alpha}
= -\sum_{i=1}^{n}\nabla_i V\frac{\partial \vec{r_i}}{\partial q_\alpha}
= -\frac{\partial V}{\partial q_\alpha}, \;\alpha  = 1,2,...,s $

+ 代入第二类拉格朗日方程中：

$ \frac{d}{dt}\frac{\partial T}{\partial \dot{q}_\alpha} - \frac{\partial (T-V)}{\partial q_\alpha}=0, \;\alpha  = 1,2,...,s $

+ 考虑到势能$ V $只是广义坐标的函数而与广义速度无关：

$ \frac{d}{dt}\frac{\partial (T-V)}{\partial \dot{q}_\alpha} - \frac{\partial (T-V)}{\partial q_\alpha}=0, \;\alpha  = 1,2,...,s $

+ 定义**拉格朗日函数**$ L $：

$ \boxed{L = T-V} $

+ 得**<font style="color:#DF2A3F;">主动力全都是保守力的拉格朗日方程</font>**：

$ \boxed{\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha}=0, \;\alpha  = 1,2,...,s} $

### 1-7 运动积分，对称性与守恒律
#### 初积分
+ 考虑保守系的拉格朗日方程：

$ \frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha}=0, \;\alpha  = 1,2,...,s $

    - 如果拉格朗日函数$ L $不显含$ q_{\alpha} $，即：

$ \frac{\partial L}{\partial q_\alpha} = 0 $

    - 代入拉格朗日方程：

$ \frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha}=0 $

    - 对时间$ t $积分得：

$ \frac{\partial L}{\partial \dot{q}_\alpha}=\mathrm{const.} $

        * 这个式子称为初积分。

#### 广义动量
+ 定义广义动量：

$ p_\alpha = \frac{\partial L}{\partial \dot{q}_\alpha} $

    - 则上述结论可以表达为：
    - 如果拉格朗日量$ L $不显含某个广义坐标$ q_\alpha $，则与之对应的广义动量$ p_\alpha $守恒。

### 1-8 广义能量积分
#### 哈密顿量的引入
+ 考虑拉格朗日量$ L $对时间$ t $的全微商：

$ \frac{dL}{dt} = \sum_{\alpha=1}^{s} \frac{\partial L}{\partial q_\alpha} \dot{q}_\alpha +\sum_{\alpha=1}^{s} \frac{\partial L}{\partial \dot{q}_\alpha} \ddot{q}_\alpha + \frac{\partial L}{\partial t} $

    - 将保守系的拉格朗日方程变形为$ \frac{\partial L}{\partial q_\alpha}=\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} $并代入上式：

$ \frac{dL}{dt} = \sum_{\alpha=1}^{s} \frac{d}{dt}(\frac{\partial L}{\partial \dot{q}_\alpha}) \dot{q}_\alpha + \sum_{\alpha=1}^{s}\frac{\partial L}{\partial \dot{q}_\alpha} \ddot{q}_\alpha + \frac{\partial L}{\partial t} $

    - 代入代入广义动量表达式$ p_\alpha = \frac{\partial L}{\partial \dot{q}_\alpha} $：

$ \frac{dL}{dt} = \sum_{\alpha=1}^{s} \frac{dp_\alpha}{dt}\dot{q}_\alpha +\sum_{\alpha=1}^{s} p_\alpha \frac{d\dot{q}_\alpha}{dt} + \frac{\partial L}{\partial t} = \sum_{\alpha=1}^{s}\frac{d}{dt}(p_\alpha \dot{q}_\alpha) + \frac{\partial L}{\partial t} $

    - 换一种表达形式：

$ \frac{d}{dt}[\sum_{\alpha=1}^{s}p_\alpha \dot{q}_\alpha-L] =- \frac{\partial L}{\partial t} $

+ 当拉格朗日量$ L $不显含时间，即：

$ L = L(q_\alpha,\dot{q}_\alpha) $

    - <font style="color:#8A8F8D;">其实就是势能、动能之差不显含时间。后文可以看到动能不显含时间要求为稳定约束。</font>
    - 有：

$ \frac{d}{dt}[\sum_{\alpha=1}^{s}p_\alpha \dot{q}_\alpha-L] =- \frac{\partial L}{\partial t} = 0 $

    - 对两侧进行积分，得：

$ \sum_{\alpha=1}^{s}p_\alpha \dot{q}_\alpha-L=\mathrm{const.} $

+ 引入哈密顿量$ H $（又称为广义能量积分），定义为：

$ H = \sum_{\alpha=1}^{s}p_\alpha \dot{q}_\alpha-L $

+ 与$ t $相关的守恒律表示为：
    - 如果保守系拉格朗日量$ L $的表达式不显含时间$ t $，系统哈密顿量守恒：

$ \frac{dH}{dt} = -\frac{\partial L}{\partial t} = 0, \; \mathrm{if}\;L = L(q_\alpha,\dot{q}_\alpha) $

$ H = \mathrm{const.}, \; \mathrm{if}\;L = L(q_\alpha,\dot{q}_\alpha) $

#### 哈密顿量与机械能关系的讨论
+ 坐标变换方程可以写为：

$ \vec{r_i} = \vec{r_i}(q_1,q_2,....,q_s;t) $

    - 对于稳定约束，坐标变换方程不含时：

$ \vec{r_i} = \vec{r_i}(q_1,q_2,....,q_s) $

+ 再讨论动能按照广义坐标的展开：

$ T = \sum_{i=1}^{n}\frac{1}{2} m_i \vec{r_i}^2 = \sum_{i=1}^{n}\frac{1}{2} m_i(\sum_{\alpha=1}^s \frac{\partial \vec{r_i}}{\partial q_\alpha}\dot{q}_\alpha + \frac{\partial \vec{r_i}}{\partial t})(\sum_{\alpha=1}^s \frac{\partial \vec{r_i}}{\partial q_\alpha}\dot{q}_\alpha + \frac{\partial \vec{r_i}}{\partial t}) $

    - 合并同类项：

$ T = \frac{1}{2}\sum_{\alpha,\beta=1}^s\sum_{i=1}^{n} m_i \frac{\partial \vec{r_i}}{\partial q_\alpha}\frac{\partial \vec{r_i}}{\partial q_\beta} \dot{q}_\alpha \dot{q}_\beta 
+\sum_{\alpha=1}^s\sum_{i=1}^{n} m_i \frac{\partial \vec{r_i}}{\partial q_\alpha}\frac{\partial \vec{r_i}}{\partial t} \dot{q}_\alpha
+ \frac{1}{2}\sum_{n=1}^{s}m_i(\frac{\partial \vec{r_i}}{\partial t})^2 $

    - 注意到三项分别是广义速度的二次、一次、零次齐次式，记为：

$ T_2 = \frac{1}{2}\sum_{\alpha,\beta=1}^s\sum_{i=1}^{n} m_i \frac{\partial \vec{r_i}}{\partial q_\alpha}\frac{\partial \vec{r_i}}{\partial q_\beta} \dot{q}_\alpha \dot{q}_\beta  $

$ T_1 = \sum_{\alpha=1}^s\sum_{i=1}^{n} m_i \frac{\partial \vec{r_i}}{\partial q_\alpha}\frac{\partial \vec{r_i}}{\partial t} \dot{q}_\alpha $

$ T_0 = \frac{1}{2}\sum_{n=1}^{s}m_i(\frac{\partial \vec{r_i}}{\partial t})^2 $

+ 考虑哈密顿量$ H $的表达式：

$ H = \sum_{\alpha=1}^{s}p_\alpha \dot{q}_\alpha-L 
= \sum_{\alpha=1}^{s}(\frac{\partial L}{\partial \dot{q}_\alpha})\dot{q}_\alpha-L
= \sum_{\alpha=1}^{s}\frac{\partial (T-V)}{\partial \dot{q}_\alpha}\dot{q}_\alpha-L $

    - 对于保守系，势能$ V $对广义速度$ \dot{q}_\alpha $的偏导数为零：

$ H = \sum_{\alpha=1}^{s}p_\alpha \dot{q}_\alpha-L 
= \sum_{\alpha=1}^{s}\frac{\partial T}{\partial \dot{q}_\alpha}\dot{q}_\alpha-L
 $

    - 将拉格朗日量$ L $和动能函数$ T $展开：

$ H= \sum_{\alpha=1}^{s}\frac{\partial T_2}{\partial \dot{q}_\alpha}\dot{q}_\alpha+\sum_{\alpha=1}^{s}\frac{\partial T_1}{\partial \dot{q}_\alpha}\dot{q}_\alpha+\sum_{\alpha=1}^{s}\frac{\partial T_0}{\partial \dot{q}_\alpha}\dot{q}_\alpha-(T_2+T_1+T_0-V) $

    - 对于$ n $次齐次式$ f = f(x_1,x_2,...,x_s) $，有欧拉定理：

$ \sum_{i=1}^s x_s\frac{\partial f}{\partial x_s} = nf $

    - 因此，哈密顿量可以表示为：

$ H= 
2T_2 + 1T_1 +0T_0-(T_2+T_1+T_0-V) = T_2-T_0 +V $

    - 综上，**一般情形的哈密顿量/广义能量积分**的表达式为：

$ \boxed{H = T_2-T_0+V} $

+ 对于稳定约束，坐标变换方程不含时，即：

$ \frac{\partial \vec{r_i}}{\partial t} = 0 $

    - 在此条件下动能函数只剩下二次项非零：

$ T_2 \neq 0 ;\; T_1 = 0 ;\; T_0 = 0 $

$ T = T_2 $

    - 哈密顿量变为：

$ H = T+V $

    - **因此，稳定约束条件下，坐标变换方程不含时，哈密顿量等于系统机械能。**

#### 欧拉定理
+ $ n $次齐次函数$ f $的定义为：
    - $ f=f(x_1,x_2,...,x_s) $每一项的幂次和均为$ n $。
    - 例如：$ f(x,y,z) = x^2 + 3y^2 + 2z^2 +6xy + 7yz $是二次齐次式。
+ 欧拉定理表述为：
    - 对于$ n $次齐次函数$ f $有：

$ \boxed{\sum_{i=1}^{s}x_i \frac{\partial f}{\partial x_i} = nf} $

### 拉格朗日方程回顾
+ 坐标变换方程并不要求理想约束！！！
+ **第一类拉格朗日方程**：
    - 形式为：

$ \begin{cases}
F_i^{(a)}  -m_i \ddot{x_i}+\sum_{\alpha =1}^{k}\lambda_{\alpha}\frac{\partial f_\alpha}{\partial x_i} = 0 \\
f_\alpha(x_1,x_2,...,x_{3n};t) = 0
\end{cases} $

    - <font style="color:#DF2A3F;">使用条件：约束为理想约束，约束力不做虚功（e.g.没有摩擦）。</font>
    - 变通：如果有耗散力，可以把它列入主动力中。
+ **第二类拉格朗日方程（最广泛）**：
    - 形式为：

$ \frac{d}{dt}\frac{\partial T}{\partial \dot{q}_\alpha} - \frac{\partial T}{\partial q_\alpha}=Q_\alpha, \;\alpha  = 1,2,...,s $

    - <font style="color:#DF2A3F;">使用条件：约束为理想约束，坐标变换方程</font>。坐标变换方程的定义要求不含广义速度。

$ x_i = x_i(q_1,q_2,....,q_s;t) $

+ **主动力全是保守力的拉格朗日方程**：
    - 形式为：

$ \frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha}=0, \;\alpha  = 1,2,...,s $

    - <font style="color:#DF2A3F;">使用条件：坐标变换方程；主动力全是保守力；约束为理想约束</font>。
    - 变通1：如果有一部分为理想约束，一部分不是理想约束，可以将非理想的约束解开视为主动力：

$ \frac{d}{dt}\frac{\partial L'}{\partial \dot{q}_\alpha} - \frac{\partial L'}{\partial q_\alpha}=Q_\alpha^{non-ideal \;constraints}, \;\alpha  = 1,2,...,s $

    - 变通2：如果有一部分为保守力，一部分不是保守力，可以将非保守力移动到右侧：

$ \frac{d}{dt}\frac{\partial L'}{\partial \dot{q}_\alpha} - \frac{\partial L'}{\partial q_\alpha}=Q_\alpha^{non-conservative \;forces}, \;\alpha  = 1,2,...,s $
