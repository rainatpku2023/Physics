### 带电粒子相对论性哈密顿函数
#### 情景概述
+ 设带电粒子的静止质量为$ m_0 $，电荷量为$ e $，速度为$ v $，处于电磁场中标势为$ \phi $，矢势为$ \vec{A} $。
+ 则其拉格朗日量$ L $为：

$ L = -m_0 c^2 \sqrt{1-\frac{v^2}{c^2}} - e(\phi - \vec{v} \cdot \vec{A}) $

#### 求解广义动量
+ 根据勒让德变换，希望求出广义动量：

$ p_i = \frac{\partial L}{\partial \dot x_i} = \frac{m_0 v_i}{\sqrt{1-\frac{v^2}{c^2}}} + eA_i = \frac{m_0 v_i}{\sqrt{1-\frac{1}{c^2}(\dot x_1^2+\dot x_2^2+\dot x_3^2)}} + eA_i $

    - 代入$ v_i = \dot x_i $，得到：

$ \boxed{p_i = \frac{m_0 \dot x_i}{\sqrt{1-v^2/c^2}} + eA_i} $

#### 用广义坐标和广义动量表示广义速度
+ 移项得：

$ p_i - eA_i =  \frac{m_0 v_i}{\sqrt{1-\frac{v^2}{c^2}}} $

    - 两侧平方并对$ x,y,z $求和：

$ \sum_i (p_i - eA_i)^2 =  \sum_i(\frac{m_0 v_i}{\sqrt{1-\frac{v^2}{c^2}}})^2 = \frac{m_0^2v^2}{1-\frac{v^2}{c^2}} = \frac{m_0^2 c^2}{1-\frac{v^2}{c^2}} - m_0^2 c^2 $

    - 整理可得：

$ \sqrt{1-\frac{v^2}{c^2}} = \frac{m_0 c}{\sqrt{m_0^2c^2 + \sum_i (p_i - eA_i)^2 }} $

    - 将这个表达式代入广义动量表达式中：

$ p_i - eA_i = \frac{m_0 v_i}{m_0 c/\sqrt{m_0^2c^2 + \sum_i (p_i - eA_i)^2 }} $

    - 从这个式子中反解出来$ \dot x_i $：

$ \boxed{\dot x_i = v_i = \frac{c(p_i - eA_i)}{\sqrt{m_0^2 c^2 + \sum_i (p_i-eA_i)^2}}} $

#### 勒让德变换求解哈密顿量
+ 根据勒让德变化，求解哈密顿量$ H $：

$ H = \sum_i \dot x_i p_i - L $

    - 代入具体表达式：

$ RHS = \sum_i \dot x_i p_i +m_0 c^2 \sqrt{1-\frac{v^2}{c^2}}+e(\phi -\vec{v}\cdot \vec{A}) $

    - 将矢势部分和前方求和合并：

$ RHS = \sum_i \dot x_i (p_i-eA_i) +m_0 c^2 \sqrt{1-\frac{v^2}{c^2}}+e\phi  $

    - 代入$ \dot x_i $表达式：

$ RHS = \sum_i \frac{c(p_i - eA_i)}{\sqrt{m_0^2 c^2 + \sum_i (p_i-eA_i)^2}}  (p_i-eA_i) +m_0 c^2 \sqrt{1-\frac{v^2}{c^2}}+e\phi  $

$ = \frac{c}{\sqrt{m_0^2 c^2 + \sum_i (p_i-eA_i)^2}}\sum_i   (p_i-eA_i)^2 +m_0 c^2 \sqrt{1-\frac{v^2}{c^2}}+e\phi  $

    - 其中$ \frac{c}{\sqrt{m_0^2 c^2 + \sum_i (p_i-eA_i)^2}} $和$ \sum_i   (p_i-eA_i)^2 $可以分别被替换：

$ RHS= \frac{1}{m_0}\sqrt{1-\frac{v^2}{c^2}} \cdot \frac{m_0^2v^2}{1-\frac{v^2}{c^2}} +m_0 c^2 \sqrt{1-\frac{v^2}{c^2}}+e\phi  $

$ =\frac{m_0 v^2}{\sqrt{1-\frac{v^2}{c^2}}}+m_0c^2\frac{1-\frac{v^2}{c^2}}{\sqrt{1-\frac{v^2}{c^2}}}+e\phi  $

$ =\frac{m_0 c^2}{\sqrt{1-\frac{v^2}{c^2}}}+e\phi  $

    - 考虑到运动质量$ m = \frac{m_0}{\sqrt{1-\frac{v^2}{c^2}}} $此时可以**不严谨地将哈密顿量写为**：

$ H = m c^2 + e\phi $

+ 考虑到$ \sqrt{1-\frac{v^2}{c^2}} = \frac{m_0 c}{\sqrt{m_0^2c^2 + \sum_i (p_i - eA_i)^2 }} $哈密顿量可以严谨地写为：

$ \boxed{H = c\sqrt{m_0^2 c^2 + \sum_i(p_i-eA_i)^2} + e\phi} $
