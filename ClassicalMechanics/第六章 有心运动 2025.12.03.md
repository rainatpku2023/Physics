### 分析力学
#### 两体问题
[第一章 拉格朗日方程拓展 2025.09.26](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/db8pwwg08cgt84qe#wMO9w)

#### 极坐标系下拉格朗日方程导出动力学方程
+ 极坐标系下，有心运动质点的拉格朗日量为：

$ L = \frac{1}{2}m(\dot \rho^2+\rho^2  \dot \phi^2)-V(\rho) $

    - 式子中不显含$ \phi $，所以相应的广义动量守恒：

$ p_\phi = \frac{\partial L}{\partial \phi}= m \rho^2 \dot \phi = \mathrm{const.} $

    - 式子中不显含$ t $，所以哈密顿量守恒：

$ H = \frac{1}{2}m(\dot \rho^2+\rho^2  \dot \phi^2)+V(\rho) = \mathrm{const.} $

### 轨道方程的微分形式
#### 轨道方程的一般求法
+ 列拉格朗日方程，求出：

$ \dot r = \dot r(r) $

$ \dot \theta = \dot \theta(r) $

    - 的函数形式。
+ 计算：

$ \boxed{
\frac{dr}{d\theta} = \frac{\dot r}{\dot \theta} = \frac{\dot r(r)}{\dot \theta(r)}=f(r)} $

+ 移项并积分：

$ \int d\theta = \int\frac{dr}{f(r)}+C $

#### 比内公式：已知能量或者力随着距离变化
+ 对于有心运动来说，能量$ E $守恒；角动量守恒，面积速度守恒，定义：

$ \rho^2 \dot{\phi} \equiv C $

    - 考虑到可以将距离$ \rho $写为角度$ \phi $的隐函数，所以有：

$ \dot \rho = \frac{d\rho}{dt} = \frac{d\rho}{d\phi} \frac{d\phi}{dt} = \frac{d\rho}{d\phi} \dot \phi = \frac{d\rho}{d\phi} \frac{C}{\rho^2} = -C\frac{d(1/\rho)}{d\phi} $

    - 定义$ u = 1/\rho $，可以写为：

$ \dot \rho = -C\frac{du}{d\phi} $

    - 类似地：

$ \ddot \rho = \frac{d\dot \rho}{dt} = \frac{d\phi}{dt} = -C \frac{d^2u}{d\phi^2} \frac{C}{\rho^2} = -C^2 u^2 \frac{d^2 u}{d\phi^2} $

    - 角动量：

$ L = m\rho^2  \dot \phi = mC $

+ 对于拉格朗日方程导出的运动方程：

$ m(\ddot \rho - \rho \dot \phi^2) = -\frac{dV}{d\rho} = F(\rho) $

    - 代入知：

$ \boxed{
-mC^2u^2(\frac{d^2u}{d\phi^2}+u) = F(1/u)} $

+ 对于能量初积分：

$ \frac{1}{2}m\dot \rho^2+\frac{L^2}{2m\rho^2}+V=E $

    - 代入知：

$ \boxed{\frac{1}{2}mC^2[(\frac{du}{d\phi})^2+u^2]+V(1/u)=E} $

    - 加框的是**比内公式的两种形式**。

#### 圆轨道稳定性
+ 径向运动方程为：

$ m(\ddot \rho - \rho \dot \phi^2) =  F(\rho) $

    - 这个方程可以改写为：

$ m\ddot \rho - \frac{mC^2}{\rho^3}=F(\rho) $

+ 对于半径为$ a $的圆轨道来说$ \ddot \rho =0 $：

$ -mC^2 = a^3 F(a) $

+ 半径发生微小的变化，即$ \rho \rightarrow a + x $，有：

$ m\ddot x - \frac{mC^2}{(a+x)^3} = F(a+x) $

    - 两侧对小量$ x $进行泰勒展开：

$ m\ddot x - \frac{mC^2}{a^3} (1-3\frac{x}{a}+\cdots) = F(a) + F'(a)x+\cdots $

    - 代入$ -mC^2 = a^3 F(a) $，保留前一项修正：

$ \ddot x - \frac{1}{m} [\frac{3}{a}F(a)-F'(a)]x \approx 0 $

+ 下面对稳定性进行讨论：
    - 当$ \frac{3}{a}F(a)-F'(a)<0 $时，解为三角函数型，轨道稳定。
    - 当$ \frac{3}{a}F(a)-F'(a)>0 $时，解为指数型，轨道不稳定。
    - 当$ \frac{3}{a}F(a)-F'(a)=0 $时，需要引入更高阶项进行判断。
+ 解题基本思路：计算平衡条件，计算稳定条件，将平衡条件代入稳定条件消元判断。

### 平方反比力：开普勒三定律
#### 开普勒轨道
+ 对于万有引力来说，势能的形式为：

$ V(\rho) = -\frac{GMm}{\rho}=-GMmu $

    - 其中$ G $是万有引力常数，$ M $是力心天体的质量。
+ 代入第二种形式的比内公式：

$ \frac{1}{2}mC^2[(\frac{du}{d\phi})^2+u^2]-GMmu=E $

    - 移项，进行变量代换：

$ (\frac{du}{d\phi})^2 = A^2 - (B-u)^2 $

    - 其中：

$ A \equiv (\frac{2E}{mC^2} + \frac{G^2m^2}{C^4})^{1/2} \geq 0 $

$ B \equiv \frac{GM}{C^2}>0 $

+ 开方，移项得：

$ \frac{du}{\sqrt{A^2 - (B-u)^2}} = \pm d\phi $

    - 两侧积分，考虑到$ \int \frac{1}{\sqrt{1-x^2}}dx = \arccos x+C $：

$ \arccos \frac{u-B}{A} = \pm \phi + \alpha $

    - 整理得：

$ u = B + A \cos (\phi \pm \alpha) $

    - 由此可以得到运动方程：

$ \rho = \frac{1/B}{1+A/B \cos (\phi-\beta)} = \frac{p}{1+e\cos(\phi-\beta)} $

+ 这就是轨道方程：

$ \boxed{
\rho =  \frac{p}{1+e\cos(\phi-\beta)}} $

    - 其中$ p $是半交弦，$ e $是偏心率：

$ \boxed{p = \frac{1}{B} = \frac{C^2}{GM}>0} $

$ \boxed{e = \frac{A}{B} = \sqrt{1+ \frac{2EC^2}{G^2M^2m}}} $

#### 轨道的能量与形状的关系
+ 根据这两个式子可以反解出二倍面积速度$ C $和能量$ E $：

$ \boxed{C = \sqrt{GM\rho}} $

$ \boxed{E = \frac{GMm}{2p} (e^2-1)} $

    - 根据轨道的形状反解出来轨道的能量。
+ 下面对于轨道的形态进行讨论：
    - 圆：偏心率$ e=1 $，能量$ E<0 $
    - 椭圆：偏心率$ 0<e<1 $，能量$ E <0 $
    - 抛物线：偏心率$ e=1 $，能量$ E = 0 $
    - 双曲线：偏心率$ e>1 $，能量$ E>0 $
+ 特别地，对于椭圆，引入半长轴$ a $：
    - 根据几何关系：$ 2a = \rho_{min}+ \rho_{max} = \frac{p}{1+e} +\frac{p}{1-e} = \frac{2p}{1-e^2} $，其中$ a $是轨道半长轴：

$ \boxed{E = \frac{GMm}{2p} (e^2-1) = -\frac{GMm}{2a} \propto -\frac{1}{2a}} $

    - 可见平方反比力下的椭圆轨道的能量与半短轴无关，与半长轴长度成反比。

### 平方反比力下的散射问题
#### 散射轨道方程的导出
+ 对于$ \alpha $粒子被金核散射，我们有：

$ V = \frac{k}{\rho} = \frac{qQ}{4\pi \epsilon_0} \frac{1}{\rho} $

    - 类比开普勒圆轨道的公式，相当于做替换：

$ GM\Rightarrow -\frac{k}{m} $

    - 代入得：

$ A \equiv (\frac{2E}{mC^2}+\frac{k^2}{m^2C^4})^{1/2}>0 $

$ B \equiv -\frac{k}{mC^2}<0 $

+ 令：

$ p = |\frac{1}{B}| = \frac{mC^2}{k} $

$ e = |\frac{A}{B}| = (1+\frac{2EC^2 m}{k^2})^{1/2} $

    - 类似地，得到轨道方程：

$ \rho = \frac{-p}{1-e\cos(\phi-\beta)} $

        * $ e $前面的负号可以吸收到$ \beta $中去，即$ \beta\rightarrow \beta +\pi $。

#### 散射角
+ 为了简单起见，考虑$ \beta=0 $的情形：

$ \rho = \frac{p}{1+e\cos\phi} $

    - 观察分母，让分母取无穷大的两个值是$ \pm \alpha $：

$ \alpha = \arccos \frac{1}{e} $

    - 设散射角为$ \theta $，$ \theta = \pi - 2\alpha $，则：

$ \cot \frac{\theta}{2} = \cot (\frac{\pi}{2}-\alpha) = \tan \alpha = \sqrt{e^2-1} $

#### 微分散射截面
+ 微分散射截面：直观上来说就是一定瞄准距离$ b\sim b+db $范围内**面积**与相应散射角$ \theta \sim \theta + d\theta $对应的**空间角**的**比值**。

$ \sigma = \frac{dA}{d\Omega} = \frac{2\pi b}{2\pi \sin \theta |d\theta|} = \frac{b}{\sin \theta} |\frac{db}{d\theta}| $

#### 卢瑟福散射公式
+ 对于平方反比斥力立场，微分散射截面的表达式被称为**卢瑟福散射公式**。
+ 联立下面三个表达式：

$ \cot \frac{\theta}{2} = \cot (\frac{\pi}{2}-\alpha) = \tan \alpha = \sqrt{e^2-1} $

$ e = |\frac{A}{B}| = (1+\frac{2EC^2 m}{k^2})^{1/2} $

$ C = \frac{L}{m} = bv_0 = b\sqrt{\frac{2E}{m}} $

    - 可以得到：

$ \cot \frac{\theta}{2} = \frac{2Eb}{k} $

+ 两边取微分：

$ d\cot \frac{\theta}{2} =  - \frac{d\theta}{2 \sin^2 (\theta/2)} = \frac{2E}{k}db $

    - 移项得：

$ |\frac{db}{d\theta}| = \frac{k}{4E} \frac{1}{\sin^2(\theta/2)} $

+ 联立下面三个表达式：

$ |\frac{db}{d\theta}| = \frac{k}{4E} \frac{1}{\sin^2(\theta/2)} $

$ b = \frac{k}{2E}\cot \frac{\theta}{2}  $

$ \sigma = \frac{b}{\sin \theta} |\frac{db}{d\theta}| $

    - 可以得到：

$ \boxed{
\sigma(\theta) = (\frac{k}{4E})^2 \frac{1}{\sin^4(\theta/2)}} $

    - 这就是**卢瑟福散射公式**。
