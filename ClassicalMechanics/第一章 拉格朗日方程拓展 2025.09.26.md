### 1-9 瞬时拉格朗日方程
#### 从牛顿力学谈起
+ 考虑台球杆撞击台球：
    - 力的大小难以测量，时间也难以测量，难以使用牛顿第二定律的基本形式：

$ \vec{F} = m\vec{a} $

    - 但是冲量是有可能描述的，这就是动量定理：

$ \vec{I} = \vec{p} - \vec{p_0} $

#### 向拉格朗日力学中引入动量定理（感觉老师这里讲的也比较模棱两可）
+ 保守系拉格朗日方程难以处理瞬时作用：

$ \frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha} = 0 $

    - 撞击力不是理想约束，因为撞了之后台球就脱离出去了，而且中间可能有耗散的成分
    - 撞击力也很难看成保守力，而且难以测量
+ 考虑使用带有非保守力的拉格朗日方程：

$ \frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha} = Q_\alpha^{non-conservative \; forces} $

    - 假设碰撞在$ t \sim t+\Delta t $时间段内发生，那么将上式两端对时间积分：

$ \int_t^{t+\Delta t}\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha}dt - \int_t^{t+\Delta t}\frac{\partial L}{\partial q_\alpha}dt = \int_t^{t+\Delta t}Q_\alpha^{ncf}dt $

    - 左侧第一项：

$ \int_t^{t+\Delta t}\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha}dt 
=\int_t^{t+\Delta t}d(\frac{\partial L}{\partial \dot{q}_\alpha})
= \frac{\partial L}{\partial \dot{q}_\alpha}|_{t+\Delta t} - \frac{\partial L}{\partial \dot{q}_\alpha}|_t $

    - 左侧第二项：
        * 因为短时间内位置不会发生突变，即$ |\frac{\partial V}{\partial q_\alpha}| \leq M $：

$ |\int_t^{t+\Delta t}\frac{\partial L}{\partial q_\alpha}dt| = |\int_t^{t+\Delta t}\frac{\partial V}{\partial q_\alpha}dt| \leq \int_t^{t+\Delta t}Mdt = M\Delta t $

    - 右侧项：

$ \int_t^{t+\Delta t}Q_\alpha^{ncf}dt = I_\alpha^{ncf} $

    - 整理得：

$ \frac{\partial L}{\partial \dot{q}_\alpha}|_{t+\Delta t} - \frac{\partial L}{\partial \dot{q}_\alpha}|_t - \int_t^{t+\Delta t}\frac{\partial L}{\partial q_\alpha}dt= \hat{I}_\alpha^{ncf} $

        * 两端求极限$ \Delta t \rightarrow 0 $：

$ \boxed{\frac{\partial L}{\partial \dot{q}_\alpha}|_{t+\Delta t} - \frac{\partial L}{\partial \dot{q}_\alpha}|_t = \hat{I}_\alpha^{ncf} , \;\Delta t \rightarrow 0} $

        * 这就是**瞬时拉格朗日方程**。

#### 广义冲量
        * 其中$ \hat{I}_\alpha^{ncf} $外界撞击力的广义冲量。模仿[广义力](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/dw1fe5/in7z7g1ogm3bl9ma#vx3Hb)的数学形式，可以给出广义冲量的表达式：

$ \hat{I}_\alpha^{ncf} = \sum_{i=1}^{n}\vec{I}^{ncf}\cdot\frac{\partial \vec{r_i}}{\partial q_\alpha} $

### 1-10 拉格朗日方程在两体问题中的应用
#### 质心坐标与相对坐标表示质点位置
+ 考虑两个处于一势场的质点，**两个质点的间距远小于质点到势场中心的距离**。
+ 两个质点的总动能$ T $为：

$ T = \frac{1}{2} m_1 \dot{\vec{r_1}}^2 + \frac{1}{2} m_2 \dot{\vec{r_2}}^2 $

    - 两个质点的质心位置为：

$ \vec{r}_c = \frac{m_1\vec{r_1}+m_2\vec{r_2}}{m_1+m_2} $

    - 质点$ 1 $相对于质点$ 2 $的相对位置为：

$ \vec{r} = \vec{r_1}-\vec{r_2} $

+ 用质心位置和相对位置反解出来$ \vec{r_1} $和$ \vec{r_2} $：

$ \vec{r_1} = \vec{r_c} + \frac{m_2}{m_1+m_2} \vec{r} $

$ \vec{r_2} = \vec{r_c} - \frac{m_1}{m_1+m_2} \vec{r} $

+ 根据柯尼希定理（质点系动能）：

$ T = \frac{1}{2}(m_1+m_2)\dot{\vec{r_c}}^2 + \frac{1}{2}\frac{m_1 m_2}{m_1+m_2}\dot{\vec{r}}^2 $

    - 质点系的拉格朗日量为：

$ L = T-V = \frac{1}{2}(m_1+m_2)\dot{\vec{r_c}}^2 + \frac{1}{2}\frac{m_1 m_2}{m_1+m_2}\dot{\vec{r}}^2 -V(\vec{r},\vec{r_c}) $

    - 引入折合质量$ \mu $：

$ \mu = \frac{m_1 m_2}{m_1+m_2} $

#### 将拉格朗日量拆成两部分和
+ 假设势能可以拆成两部分的和，即：

$ V(r_c,r) = V(r_c) + V(r) $

+ 代入拉格朗日量得表达式，可以将整体的拉格朗日量分为两部分：

$ L_c = T_c - V(r_c) = \frac{1}{2}(m_1+m_2) \dot{\vec{r_c}}^2 - V(r_c) $

$ L' = T' - V(r) = \frac{1}{2}\mu \dot{\vec{r}}^2 - V(r) $

    - 前者$ L_c $是两个质点质心的拉格朗日量。
    - 后者$ L' $是两个质点之间相互作用、相对运动对应的拉格朗日量。
    - 两者与整体拉格朗日两$ L $的关系为：

$ L = L_c +L' $

### 1-11 带有广义势的拉格朗日方程
#### 洛伦兹力
+ 洛伦兹力对应的广义势$ U $为：

$ \boxed{U  = e(\phi-\vec{v} \cdot \vec{A})} $

    - 其中$ \phi $是标势，$ \vec{A} $是矢势。与电磁场的关系为：

$ \vec{E} = -\nabla \phi - \frac{\partial \vec{A}}{\partial t}  $

$ \vec{B} = \nabla \times \vec{A} $

    - 需要证明推广的拉格朗日函数$ L' = T-U $满足拉格朗日方程：

$ \frac{d}{dt}(\frac{\partial L}{\partial \dot{x}_i}) - \frac{\partial L}{\partial x_i} = 0 $

        * 即需要证明洛伦兹力可以写成如下形式：

$ F_i = \frac{d}{dt}(\frac{\partial U}{\partial \dot{x}_i}) - \frac{\partial U}{\partial x_i} $

+ 下面给出证明：
    - 根据洛伦兹力定义：

$ \vec{F} = e(\vec{E}+\vec{v}\times\vec{B}) $

    - 代入$ \vec{E},\vec{B} $表达式：

$ \vec{F} = e[-\nabla \phi - \frac{\partial \vec{A}}{\partial t} +\vec{v}\times(\nabla \times \vec{A})] $

    - 考察括号中第二项在$ x $方向上的分量，即：

$ [\vec{v}\times(\nabla \times \vec{A})]_x = v_y (\frac{\partial A_y}{\partial x}-\frac{\partial A_x}{\partial y}) - v_z (\frac{\partial A_x}{\partial z}-\frac{\partial A_z}{\partial x}) 
\\ = v_x\frac{\partial A_x}{\partial x} +v_y\frac{\partial A_y}{\partial x}+v_z\frac{\partial A_z}{\partial x} - (v_x \frac{\partial A_x}{\partial x}+v_y \frac{\partial A_y}{\partial x}+v_z \frac{\partial A_x}{\partial z}) $

        * 考虑到$ v_x $与$ x,y,z $相互独立，偏导数为零，上面可以进一步化简为：

$ [\vec{v}\times(\nabla \times \vec{A})]_x = \frac{\partial}{\partial x}(\vec{v}\times\vec{A}) - (\vec{v}\cdot\nabla)A_x $

        * 观察右侧第二项，有以下关系：

$ (\vec{v}\cdot\nabla)A_x = \dot{x}\cdot\frac{\partial A_x}{\partial x} + \dot{y}\cdot\frac{\partial A_x}{\partial y} +\dot{z}\cdot\frac{\partial A_x}{\partial z} = \frac{dA_x}{dt} - \frac{\partial A_x}{\partial t} $

        * 代入得：

$ [\vec{v}\times(\nabla \times \vec{A})]_x = \frac{\partial}{\partial x}(\vec{v}\cdot\vec{A}) - \frac{dA_x}{dt} + \frac{\partial A_x}{\partial t} $

    - 代入力的表达式得：

$ \vec{F} = e[-\nabla \phi - \frac{\partial \vec{A}}{\partial t} +\nabla(\vec{v}\cdot\vec{A}) - \frac{d\vec{A}}{dt} + \frac{\partial \vec{A}}{\partial t}]
= e[-\nabla (\phi-\vec{v}\cdot\vec{A}) - \frac{d\vec{A}}{dt}  ] $

        * 其中$ x $方向上的分量为：

$ F_x
= e[-\frac{\partial}{\partial x} (\phi-\vec{v}\cdot\vec{A}) - \frac{dA_x}{dt}]  $

    - 考虑下面的式子，得到$ A_x $的新表达式：

$ \frac{\partial}{\partial v_x}(\vec{v}\cdot\vec{A}) = \frac{\partial}{\partial v_x} （v_x A_x + v_y A_y + v_z A_z) = A_x $

$ \frac{\partial \phi}{\partial v_x} = 0 $

        * 两个式子用到了$ \vec{A},\phi $与速度无关的性质，第一个式子还用到了各个方向速度无关的性质。
        * 因此，$ A_x $有了一个新的表达式：

$ A_x = -\frac{\partial}{\partial v_x}(\phi - \vec{v}\cdot\vec{A}) $

    - 代入$ F_x $的表达式中：

$ F_x
= e[-\frac{\partial}{\partial x} (\phi-\vec{v}\cdot\vec{A}) + \frac{d}{dt}\frac{\partial(\phi - \vec{v}\cdot\vec{A})}{\partial v_x}]  $

        * 考虑到$ v_x = \dot{x} $并代入$ U = e(\phi-\vec{v}\cdot\vec{A}) $，得：

$ F_x
= -\frac{\partial U}{\partial x}+ \frac{d}{dt}\frac{\partial U}{\partial \dot{x}} $

$ \mathrm{Q.E.D.} $

+ 类似地，科里奥利力：

$ F_{cor} = 2m \vec{v} \times \vec{\omega} $

    - 也可以定义相应的广义势：

$ U_{cor} = -m\vec{v} \cdot A_{cor} $



#### 摩擦力与耗散函数
+ 考虑流体中，流速$ \vec{v} $较小时，有层流$ \vec{f} = -b\vec{v} $，希望找到拉格朗日量$ L' $满足拉格朗日方程形式。
+ 定义耗散函数$ \mathcal{J} $：

$ \boxed{\mathcal{J} = \sum_{i=1}^{3n}\frac{1}{2} b_i \dot{x}_i^2} $

    - 结合层流情形速度表达式，可知：

$ \boxed{f = -\frac{\partial \mathcal{J}}{\partial \dot{x}_i}} $

+ 广义力的表达式为：

$ Q_\alpha = \sum_{i=1}^{3n} f_i \frac{\partial x_i}{\partial q_\alpha} = -\sum_{i=1}^{3n}\frac{\partial \mathcal{J}}{\partial \dot{x}_i}\frac{\partial x_i}{\partial q_\alpha} $

    - 结合拉格朗日关系$ \frac{\partial x_i}{\partial q_\alpha} = \frac{\partial \dot{x}_i}{\partial \dot{q}_\alpha} $，代入知：

$ Q_\alpha= -\sum_{i=1}^{3n}\frac{\partial \mathcal{J}}{\partial \dot{x}_i}\frac{\partial \dot{x}_i}{\partial \dot{q}_\alpha}
= - \frac{\partial \mathcal{J}}{\partial \dot{q}_\alpha} $

+ 第二类拉格朗日方程的原始形式为：

$ \frac{d}{dt}(\frac{\partial L}{\partial \dot{q}_\alpha}) - \frac{\partial L}{\partial q_\alpha} = Q_\alpha $

    - 代入得：

$ \frac{d}{dt}(\frac{\partial L}{\partial \dot{q}_\alpha}) - \frac{\partial L}{\partial q_\alpha} =- \frac{\partial \mathcal{J}}{\partial \dot{q}_\alpha} $

    - 两边同时乘以$ \dot{q}_\alpha $并对$ \alpha $求和：

$ \sum_{\alpha=1}^{s}[\frac{d}{dt}(\frac{\partial L}{\partial \dot{q}_\alpha}) - \frac{\partial L}{\partial q_\alpha}]\dot{q}_\alpha =
- \sum_{\alpha=1}^{s}\frac{\partial \mathcal{J}}{\partial \dot{q}_\alpha} \dot{q}_\alpha $

+ 等式左边可以化为：

$ LHS = \sum_{\alpha=1}^{s}[\frac{d}{dt}(\dot{q}_\alpha \frac{\partial L}{\partial \dot{q}_\alpha}) - \ddot{q}_\alpha \frac{\partial L}{\partial \dot{q}_\alpha} - \dot{q}_\alpha\frac{\partial L}{\partial q_\alpha}] $

    - 考虑到$ \frac{dL}{dt} = \sum_{\alpha=1}^{s} \frac{\partial L}{\partial q_\alpha}\dot{q}_\alpha
+\sum_{\alpha=1}^{s} \frac{\partial L}{\partial \dot{q}_\alpha} \ddot{q}_\alpha + \frac{\partial L}{\partial t} $，左边可以继续化简：

$ LHS = \sum_{\alpha=1}^{s}[\frac{d}{dt}(\dot{q}_\alpha \frac{\partial L}{\partial \dot{q}_\alpha})] - (\frac{dL}{dt} - \frac{\partial L}{\partial t}) $

    - 在<font style="color:#DF2A3F;">稳定约束</font>条件下，$ \frac{\partial x_i}{\partial t} = 0 $，广义动能仅含有二阶项，不含有零阶和一阶项。

$ LHS = 2\frac{dT}{dt}- \frac{d(T-V)}{dt} - \frac{\partial L}{\partial t}
= \frac{d(T+V)}{dt} - \frac{\partial L}{\partial t} $

    - 如果<font style="color:#DF2A3F;">拉格朗日量不显含时间</font>，那么：

$ LHS = \frac{d(T+V)}{dt} $

+ 考虑到$ \mathcal{J} $是$ \dot{q}_\alpha $的二次齐次函数，右侧可以化为：

$ RHS = -2\mathcal{J} $

+ 两边相等，联立得：

$  \frac{d(T+V)}{dt} = -2\mathcal{J} $

    - 整理得：

$ \boxed{\mathcal{J} =  -\frac{1}{2}\frac{d}{dt} (T+V)} $

### 1-12 相对论的拉格朗日方程
#### 相对论中的拉格朗日量
+ 如果引入下述形式的拉格朗日量：

$ \boxed{L = -m_0 c^2 \sqrt{1-\frac{v^2}{c^2}} - V} $

+ 那么代入第二类拉格朗日方程可以得到运动方程：

$ \frac{d}{dt} (\frac{\partial L}{\partial \dot{q}_\alpha}) - \frac{\partial L}{\partial q_\alpha} = 0 $

#### 证明过程
+ 证明要求满足：
    - 满足动力学方程。
    - 当$ \frac{v}{c} << 1 $时给出非相对论性的近似，即：

$ L = -m_0 c^2 + \frac{1}{2} m_0 v^2 -V $

        * 这和经典的拉格朗日量定义有所出入，但是拉格朗日量本身就不是可观测量，可以有多个选择。
+ 先对第一点给出证明：
    - 首先考虑拉格朗日量$ L $对速度$ \dot{x} $的偏导数：

$ \frac{\partial L}{\partial v_x} = \frac{\partial L}{\partial \dot{x}} =
\frac{\partial}{\partial v_x} [-m_0c^2(1-\frac{v_x^2+v_y^2+v_z^2}{c^2})^{1/2}-V(x,y,z,t)] \\
=-\frac{1}{2} m_0 c^2 (1-\frac{v^2}{c^2})^{-\frac{1}{2}}(-\frac{2v_x}{c^2})
 = \frac{m_0v_x}{\sqrt{1-v^2/c^2}} = mv_x $

        * 考虑动量表达式：

$ p_x = \frac{\partial L}{\partial \dot{x}} $

    - 再考虑拉格朗日量$ L $对坐标$ x $的偏导数：

$ \frac{\partial L}{\partial x} = -\frac{\partial V}{\partial x} = F_x $

    - 代入第二类拉格朗日方程：

$ \frac{d p_x}{dt} - F_x = 0 $

        * 整理得：

$ F_x = \frac{d p_x}{dt}  $

        * 这符合相对论性的动力学方程。
+ 再对第二点给出证明：
    - 首先在$ \frac{v}{c} << 1 $（非相对论性近似）时给出泰勒展开：

$ \sqrt{1-\frac{v^2}{c^2}} = 1-\frac{1}{2}\frac{v^2}{c^2}+-\cdots $

    - 代入拉格朗日量：

$ L = -m_0 c^2 (1+\frac{1}{2}\frac{v^2}{c^2}+\cdots) - V
= -m_0 c^2 + \frac{1}{2} m_0 v^2 -V + \cdots $

    - 可见，非相对论性近似下，有：

$ L = -m_0 c^2 + \frac{1}{2} m_0 v^2 -V $

$ \mathrm{Q.E.D.} $
