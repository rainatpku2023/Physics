#### 波函数的内积
+ 波函数的内积定义为：

$ (\psi,\phi) = \int \psi^* \phi d\tau $

    - 其中$ d\tau $为空间体积元。
+ 显然内积保证：

$ (\psi,\psi) = 0 $

$ (\psi,\phi)^* = (\phi,\psi) $

$ (\psi,c_1 \phi_1+c_2 \phi_2) = c_1(\psi,\phi_1)+ c_2(\psi,\phi_2) $

$ (c_1\psi_1+c_2\psi_2,\phi) = c_1^*(\psi_1,\phi)+c_2^*(\psi_2,\phi) $

    - 其中$ c_1,c_2 $是任意两个复常数

### 算符的运算规则
#### 算符
+ 算符作用一个函数，输出一个函数，可以看成一个从函数空间到函数空间的映射。

#### 线性算符（算符类型）
+ 一个算符$ \hat{A} $被称为是线性，如果有：

$ \hat{A}(c_1 \psi_1+c_1 \psi_1) = c_1 \hat{A}\psi_1 + c_2 \hat{A}\psi_2 $

    - 其中$ c_1,c_2 $是两个**任意的常复数**。$ \psi_1,\psi_2 $是**任意两个波函数**。

#### 单位算符（算符类型）
+ 单位算符$ \hat{I} $作用于任意一个波函数都返回其本身，即：

$ \hat{I}\psi = \psi $

    - 其中$ \psi $是**任意一个**波函数。

#### 算符相等
+ 两个算符$ \hat{A},\hat{B} $被称为是相等的，即$ \hat{A} = \hat{B} $。如果对于对于**任何一个**波函数$ \psi $都有：

$ \hat{A}\psi = \hat{B}\psi $

#### 算符之和
+ 算符$ \hat{A},\hat{B} $之和$ (\hat{A}+\hat{B}) $定义为：

$ (\hat{A} + \hat{B})\psi = \hat{A}\psi +\hat{B}\psi $

    - 其中$ \psi $是**任何一个**波函数。
+ 算符之和满足交换律和结合律。

#### 算符之积
+ 算符$ \hat{A},\hat{B} $之积$ \hat{A}\hat{B} $定义为：

$ (\hat{A}\hat{B})\psi = \hat{A}(\hat{B}\psi) $

    - 即先让算符$ \hat{B} $作用于波函数$ \psi $，然后将得到结果$ (\hat{B}\psi) $整个地被$ \hat{A} $作用。

+ **<font style="color:#DF2A3F;">需要注意，算符之积不满足交换律，因为</font>**$ \hat{A},\hat{B} $**<font style="color:#DF2A3F;">不一定对易。</font>**

#### 对易式
+ 为了考察算符$ \hat{A},\hat{B} $的不对易的程度，定义对易式：

$ [\hat{A},\hat{B}]:=\hat{A}\hat{B} -\hat{B}\hat{A} $

+ 下面是对易式的一些性质：
    - 反对称性：

$ [\hat{A},\hat{B}] = -[\hat{B},\hat{A}] $

    - 某种“分配律”：

$ [\hat{A},\hat{B}+\hat{C}]=[\hat{A},\hat{B}]+[\hat{A},\hat{C}] $

    - 乘法规则：

$ [\hat{A}\hat{B},\hat{C}] = \hat{A}[\hat{B},\hat{C}] +[\hat{A},\hat{C}]\hat{B} $

$ [\hat{A},\hat{B}\hat{C}] = \hat{B}[\hat{A},\hat{C}]+[\hat{A},\hat{B}]\hat{C} $

        * **<font style="color:#DF2A3F;">注意，展开的算符需要保持顺序！</font>**
    - $ \mathrm{Jacobi} $恒等式：

$ [\hat{A},[\hat{B},\hat{C}]]+[\hat{B},[\hat{C},\hat{A}]]+[\hat{C},[\hat{A},\hat{B}]] = 0 $

        * 记忆方法就是轮转相加为$ 0 $。

#### 逆算符（运算关系）
+ 设 $ \hat{A} $ 是定义在希尔伯特空间 $ \mathcal{H} $ 的子集 $ D(\hat{A}) $ 上，且值域为 $ R(\hat{A}) $ 的线性算符。若对于每一个 $ y \in R(\hat{A}) $，方程 $ \hat{A}x = y $ 有且仅有一个解 $ x \in D(\hat{A}) $，则称算符 $ \hat{A} $ 具有**逆算符**，记作 $ \hat{A}^{-1} $。
+ 其数学关系满足：
1. **定义域与值域的互换**：$ D(\hat{A}^{-1}) = R(\hat{A}) $ 且 $ R(\hat{A}^{-1}) = D(\hat{A}) $。
2. **算符恒等关系**：
    - 在 $ D(\hat{A}) $ 上：$ \hat{A}^{-1}\hat{A} = \hat{I} $ （左逆）
    - 在 $ R(\hat{A}) $ 上：$ \hat{A}\hat{A}^{-1} = \hat{I} $ （右逆）

其中 $ \hat{I} $ 为恒等算符。

#### 算符的函数
+ 给定一个函数$ F(x) $，其各阶导数均存在，幂级数展开收敛，即：

$ F(x) = \sum_{i=1}^{\infin} \frac{F^{(n)}(0)}{n!} x^n $

    - 那么可以定义算符的函数$ F(\hat{A}) $，定义为：

$ F(\hat{A}) = \sum_{i=1}^{\infin} \frac{F^{(n)}(0)}{n!} \hat{A}^n $

#### 转置算符（运算关系）
+ 一个算符$ \hat{A} $的转置算符$ \hat{A}^T $定义为：

$ \int \psi^* \hat{A}^{T} \phi d\tau = \int \phi\hat{A} \psi^*d\tau $

    - 或：

$ (\psi,\hat{A}\phi) = (\phi,\hat{A}\psi^*) $

    - 其中$ \psi,\phi $是**任意两个波函数**。
+ 复共轭算符的运算规则为：

$ (\hat{A}\hat{B})^T = \hat{B}^T\hat{A}^T $

#### 复共轭算符（运算关系）
+ 算符$ \hat{A} $的复共轭算符$ \hat{A}^* $的定义为：

$ \hat{A}^*(\psi) = (\hat{A}\psi^*)^* $

+ 复共轭算符的运算规则为：

$ (\hat{A}\hat{B})^* = \hat{A}^*\hat{B}^* $

+ **<font style="color:#DF2A3F;">注意，不同表象下的</font>**$ \hat{A}^* $**<font style="color:#DF2A3F;">的表达式可能不同</font>**，例如：
    - $ \hat{p} $在坐标表象下的复共轭算符为：

$ \hat{p}^* = (-i\hbar\nabla)^* = i\hbar\nabla = -\hat{p} $

    - $ \hat{p} $在动量表象下的复共轭算符为：

$ \hat{p}^* = \vec{p}^* = \vec{p} = \hat{p} $

#### 厄米共轭算符（运算关系）
+ 算符$ \hat{A} $的厄米共轭算符$ \hat{A}^{\dagger} $的定义为：

$ (\psi,\hat{A}^{\dagger}\phi) = (\hat{A}\psi,\phi) $

+ 厄米共轭有如下性质：
    - 厄米共轭等价于转置复共轭：

$ \hat{A}^\dagger = (\hat{A}^T)^* $

    - 算符之积的厄米共轭等于每一个分别取厄米共轭并反着排列：

$ (\hat{A}\hat{B})^{\dagger} = \hat{B}^{\dagger}\hat{A}^{\dagger} $

#### 厄米算符（算符类型）
+ 一个算符被称为厄米算符，或者自共轭算符，如果：

$ (\psi,\hat{A}\phi) = (\hat{A}\psi,\phi) $

    - 或者更简单地记为：

$ \hat{A}^\dagger = \hat{A} $

+ 厄米算符的性质
    - 定理1：厄米算符的本征值是实数。
    - 定理2：体系任何状态下，厄米算符的平均值必为实数。
    - 逆定理2：任何状态下平均值均为实数的算符必为厄米算符。
+ **<font style="color:#DF2A3F;">注意：两个厄米算符的乘积一般不是厄米算符。</font>**

$ (\hat{A}\hat{B})^{\dagger} = \hat{B}^\dagger \hat{A}^\dagger =\hat{B}\hat{A} \neq \hat{A}\hat{B} $

### 厄米算符的本征值与本征函数
#### 定理1：厄米算符的本征值必为实数
+ 证明：
    - 考虑厄米算符$ \hat{A} $与其对应本征值为$ A $的本征态$ \psi $，那么有：

$ (\psi,\hat{A}\psi)  = (\psi,A\psi) = A(\psi,\psi) $

        * 这里只用到了本征值与本征函数的定义，以及内积的线性性。
    - 以及：

$ (\psi,\hat{A}\psi) = (\hat{A}^{\dagger}\psi,\psi) = (\hat{A}\psi,\psi) = (A\psi,\psi) = A^* (\psi,\psi)  $

        * 第一个等号用到了算符与伴算符的性质。
        * 第二个等号用到了厄米算符的性质，即$ \hat{A} = \hat{A}^\dagger $。
        * 第三个等号用到了本征值与本征函数的定义$ \hat{A} \psi = A\psi $。
        * 第四个等号用到了内积的线性性，并考虑到左矢内积时要取复共轭。
    - 对比这两个式子，不难发现：

$ A(\psi,\psi) = A^*(\psi,\psi) $

    - 考虑到$ (\psi,\psi)\neq 0 $，那么一定有：

$ A = A^* $

    - 即厄米算符的本征值一定是实数。

$ \mathrm{Q.E.D.} $

#### 定理2：厄米算符属于不同本征值的本征函数彼此正交
+ 证明：
    - 考虑厄米算符$ \hat{A} $，本征函数$ \psi_n $对应的本征值是$ A_n $，本征函数$ \psi_m $对应的本征值是$ A_m $，而且$ A_n\neq A_m $。根据定义：

$ \hat{A}\psi_n = A_n\psi_n $

$ \hat{A}\psi_m = A_m\psi_m $

        * 对第一个式子左乘$ \psi_m $并进行积分：

$ (\psi_m,\hat{A}\psi_n) = (\psi_m,A_n\psi_n) = A_n(\psi_m,\psi_n) $

        * 根据厄米算符性质：

$ (\psi_m,\hat{A}\psi_n) = (\hat{A}^{\dagger}\psi_m,\psi_n) = (\hat{A}\psi_m,\psi_n) = (A_m\psi_m,\psi_n) = A_m(\psi_m,\psi_n) $

        * 根据这两个式子可以看出来：

$ (A_n-A_m)(\psi_m,\psi_n) = 0 $

    - 如果对应的不是同一个本征值，$ A_n\neq A_m $即$ A_n- A_m\neq 0 $，那么：

$ (\psi_m,\psi_n) = 0 $

    - 厄米算符属于不同本征值的本征函数彼此正交。

$ \mathrm{Q.E.D.} $

+ 评述：对于相同的本征值，可能存在多个本征函数，这种现象称为简并。
    - 可以通过施密特正交化方案得到正交归一的本征函数组。
    - 对于狄利克雷型边界条件的斯图姆刘维尔本征值问题，本征值不存在简并。

#### 涨落与本征态
+ 物理量$ A $在态$ \psi $下的**涨落**定义为：

$ \overline{\Delta A^2} = \overline{(\hat{A}-\bar{A})^2} = \int\psi^*(\hat{A}-\bar{A})^2\psi d\tau $

    - 因为$ \hat{A} $是厄米算符，而$ \bar{A} $是实数也可以看成厄米算符，所以$ \Delta \hat{A} = (\hat{A}-\bar{A}) $也是厄米算符，由此，可以推知：

$ (\psi,\Delta \hat{A}^2\psi) = (\Delta \hat{A}^\dagger \psi,\Delta \hat{A}\psi) = (\Delta \hat{A}\psi,\Delta \hat{A}\psi) \geq 0 $

        * 第一个等号用到了算符与厄米共轭算符的定义。
        * 第二个等号用到了厄米算符的性质$ \Delta \hat{A}^{\dagger} = \Delta \hat{A} $。
        * 第三个等号用到了内积的定义（模方大于等于零，当且仅当函数为零时模方为零）。
    - 可见：
        * 涨落一定是非负的。
        * 当且仅当$ \psi $是$ \Delta \hat{A} $退化核中的元素时涨落为零，即：

$ \Delta \hat{A}\psi = 0 $

$ \Rightarrow \hat{A}\psi = \bar{A}\psi $

            + 即$ \psi $是$ \hat{A} $的本征态。
        * 当且仅当波函数处于力学量$ A $对应的算符$ \hat{A} $的本征态时，力学量$ A $的涨落$ \overline{\Delta A^2} $为零。

#### 共同本征态
+ 这个定理在曾谨言《量子力学教程》中并非在此处给出，但是还是提前一下：

> + 对于两个非退化线性算符$ \hat{A},\hat{B} $相互对易$ [\hat{A},\hat{B}] = 0 $。
>     - 如果$ \hat{A} $的某一个本征值$ A_n $不存在简并，即存在$ \psi $满足：

>
>     - 那么$ \psi $一定是$ \hat{B} $的一个本征函数，即：
>
> $ \hat{B}\psi = B_m \psi $
>
>     - 对于其他满足上述关系的$ \phi $，一定和$ \psi $只差了一个非零常数$ c $：
>
> $ \phi = c\psi, \text{  c = const.} $
>
>         * 即二者是同一个本征态。
> + **上述定理可以重新表述为：**
>     - **对于两个非退化线性算符**$ \hat{A},\hat{B} $**相互对易**$ [\hat{A},\hat{B}] = 0 $**，如果**$ \psi $**是算符**$ \hat{A} $**不简并的本征值对应的本征态，那么**$ \psi $**一定是算符**$ \hat{B} $**的一个本征态。**

### 共同本征函数
#### 不确定度关系的严格证明
+ 对于两个任意的力学量$ A,B $，考虑下列显然成立的积分不等式：

$ I(\xi) = \int|\xi \hat{A}\psi+i\hat{B}\psi|^2 d\tau \geq 0 $

    - 其中$ \xi $为**任意实数参数**，$ \psi $为**任意**的一个量子态。
    - 力学量算符$ \hat{A},\hat{B} $都是厄米算符，我们可以进行转化：

$ I(\xi) = (\xi\hat{A}\psi+i\hat{B}\psi,\xi\hat{A}\psi+i\hat{B}\psi)  $

$ =\xi^2(\hat{A}\psi,\hat{A}\psi)+i\xi(\hat{A}\psi,\hat{B}\psi)-i\xi(\hat{B}\psi,\hat{A}\psi) + (\hat{B}\psi,\hat{B}\psi) $

$ =\xi^2(\psi,\hat{A}^2\psi)+ i\xi(\psi,[\hat{A},\hat{B}]\psi)+(\psi,\hat{B}^2\psi) $

        * 第二个等号使用了内积的双线性。
        * 第三个等号使用了厄米算符的定义以及对易子的定义。
    - 为了进行简写，引入厄米算符$ \hat{C} = [\hat{A},\hat{B}]/i = \hat{C}^\dagger $，式子可以进行简化：

$ I(\xi) = \xi^2 \overline{A^2}-\xi\overline{C} + \overline{B^2} $

$ = \overline{A^2}(\xi-\overline{C}/2\overline{A^2})^2+(\overline{B^2}-\overline{C}^2/4\overline{A^2})\geq 0 $

        * 首先证明$ \hat{C} $是厄米算符：

$ \hat{C}^{\dagger} = ([\hat{A},\hat{B}]/i)^{\dagger} = i(\hat{A}\hat{B}-\hat{B}\hat{A})^\dagger = i(\hat{B}^{\dagger} \hat{A}^{\dagger} - \hat{A}^{\dagger}\hat{B}^{\dagger}) = i(\hat{B} \hat{A} - \hat{A}\hat{B}) \\= (\hat{A}\hat{B}-\hat{B}\hat{A})/i = ([\hat{A},\hat{B}]/i) = \hat{C} $

    - 不妨取$ \xi-\overline{C}/2\overline{A^2} = 0 $，那么有：

$ \overline{B^2}-\overline{C}^2/4\overline{A^2}\geq 0 $

    - 这个式子可以改写为：

$ \sqrt{\overline{A^2}\cdot\overline{B^2}}\geq \frac{1}{2} |\overline{C}| = \frac{1}{2}|\overline{[\hat{A},\hat{B}]/i} | = \frac{1}{2}|\overline{[\hat{A},\hat{B}]}| $

    - 上列不等式对于任意两个厄米算符$ \hat{A},\hat{B} $都成立，我们注意到$ \bar{A},\bar{B} $都是实数，所以$ \Delta \hat{A} = \hat{A}-\overline{A} $和$ \Delta \hat{B} = \hat{B}-\overline{B} $也是厄米算符，不难证明：

$ [\Delta \hat{A},\Delta \hat{B}] = [\hat{A},\hat{B}] $

        * 其中用到了常数与任何线性算符都对易，而且常数之间也相互对易。
    - 所以我们就得出：

$ \sqrt{\overline{\Delta A^2}\cdot\overline{\Delta B^2}} \geq \frac{1}{2}|\overline{[\hat{A},\hat{B}]}| $

    - 常常简记为：

$ \Delta A\cdot \Delta B \geq \frac{1}{2}|\overline{[\hat{A},\hat{B}]}| $

    - 这就是不确定度关系。
    - 它说明，如果有两个力学量$ \hat{A},\hat{B} $，那么它们的测量值的不确定度（涨落）的乘积不低于$ \frac{1}{2}|\overline{[\hat{A},\hat{B}]}| $。

#### 对易力学量完全集（CSCO）
+ 设有一组彼此独立而且相互对易的厄米算符$ \{\hat{A}_1,\hat{A}_2,\hat{A}_3,...\} $，它们的共同本征态记为$ \psi_{\{\alpha\}} $，$ \{\alpha\} $表示一组完备的量子数之后，就能确定体系的一个唯一一个可能的状态，我们就称构成一组对易可观测量完全集。
    - 首先强调要相互对易：$ [\hat{A}_i,\hat{A}_j] = 0, \forall i,j $
    - 其次存在要彼此独立，就是说抽出任何一个可观测量都要不构成CSCO。
    - 最后也要求完备，一般来说可观测量数目等于体系的自由度数目。但是也有时存在大于体系自由度数目的情形。

#### 对易守恒量完全集（CSCCO）
+ 一组对易力学量完全集，如果其中**<font style="color:#DF2A3F;">包含哈密顿量</font>**$ \hat{H} $**<font style="color:#DF2A3F;">，而且哈密顿量不显含时间，而且任意的其他的力学量算符都不显含时间且都与哈密顿量算符相互对易，那么这样的一组对易力学量完全集又称为</font>****对易守恒量完全集**。包括$ \hat{H} $的守恒量的完全集的共同本征态是**定态**，相应的量子数都是**好量子数**。还可以发现，对于任意一个波函数的展开的系数模方$ |a_{\{\alpha\}}|^2 $是不随时间改变的。
    - 后面可以证明，如果哈密顿量$ \hat{H} $不显含时间，那么$ \hat{H} $是守恒量。
    - 后面还可以证明，如果力学量$ \hat{A} $不显含时间$ \frac{\partial \hat{A}}{\partial t} = 0 $而且与哈密顿量对易$ [\hat{A},\hat{H}] = 0 $，那么$ \hat{A} $是守恒量。

### 箱归一化
+ 无穷大平面波一般来说是无法进行归一化的。
+ 箱归一化是先将粒子局限在$ [-L/2,L/2] $的有限空间内运动，最后才让$ L\rightarrow \infin $。
    - 为了保证动量算符$ \hat{p}_x $是厄米算符，波函数算符需要满足周期性边界条件，不难求出正交归一完备本征函数组：

$ \psi_{p_n}(x) = \frac{1}{\sqrt{L}}e^{ip_n x/\hbar} = \frac{1}{\sqrt{L}}e^{i2\pi nx/L} , n =0,\pm 1,\pm2,...  $

    - 它们的正交归一关系可以写为：

$ \int_{-L/2}^{L/2}\psi_{p_n}^*(x)\psi_{p_m}(x)dx = \delta_{nm} $

+ 根据$ \delta $函数的性质：

$ \delta(x-x') = \sum_{n}\psi_n^*(x')\psi_n(x) $

    - 其中$ \psi_n $为正交归一完备本征函数组。
    - 可以写出：

$ \delta (x-x') = \frac{1}{L}\sum_{n = -\infin}^{+\infin} e^{ip_n(x-x')/\hbar} = \frac{1}{L}\sum_{n = -\infin}^{+\infin} e^{i2\pi n(x-x')/L} $

+ 现在让$ L\rightarrow \infin,\Delta p_n = p_{n+1}-p_n = (h/L)\rightarrow 0 $，动量的可能取值趋于连续变化。
    - 此时，可以把$ h/L\rightarrow dp $，而：

$ \sum_{n = -\infin}^{+\infin}\Delta p_n = \frac{h}{L}\sum_{n = -\infin}^{+\infin} \rightarrow \int_{-\infin}^{+\infin}dp $

$ \sum_{n = -\infin}^{+\infin} \rightarrow \frac{L}{h}\int_{-\infin}^{+\infin}dp $

    - $ \delta $函数的表达式趋近于：

$ \delta(x-x') = \frac{1}{2\pi \hbar} \int_{-\infin}^{+\infin}dpe^{ip(x-x')/\hbar} =  \frac{1}{2\pi} \int_{-\infin}^{+\infin}dke^{ik(x-x')} $

+ 在三维情形中：

$ \sum_{n,l,m= -\infin}^{+\infin} \rightarrow \frac{L^3}{h^3}\int_{-\infin}^{+\infin}dp_xdp_ydp_z $

$ \delta(x-x') = \frac{1}{h^3} \int_{-\infin}^{+\infin}dpe^{i\vec{p}\cdot (\vec{r}-\vec{r}')/\hbar}  $

    - 这个式子表明相空间一个体积元$ h^3 $相当于有一个量子态，这个结论在量子统计物理中经常用到。

### 附录
#### 量子力学基本对易式
+ 考虑到：

$ (\hat{x}\hat{p}_x-\hat{p}_x\hat{x})\psi = x(-i\hbar\frac{d\psi}{dx})-(-i\hbar\frac{d}{dx}(x\psi)) = i\hbar\psi $

    - 最后一步用到了分部积分。
+ 此外还有：

$ (\hat{y}\hat{p}_x-\hat{p}_x\hat{y})\psi = y(-i\hbar\frac{\partial\psi}{\partial x})-(-i\hbar\frac{\partial}{\partial x}(y\psi)) = 0 $

    - 用到了偏导数$ \frac{\partial}{\partial x} y = 0 $所以直接把$ y $提出去。
+ 类似地，总结一下就是：

$ \hat{x}_\alpha\hat{p}_\beta-\hat{p}_\beta\hat{x}_\alpha = i\hbar\delta_{\alpha\beta} $

+ 用对易式写为：

$ [\hat{x}_\alpha,\hat{p}_\beta] =i\hbar\delta_{\alpha\beta} $

#### 一些对易子
+ **角动量算符**的定义为：

$ \hat{l} = \hat{r} \times \hat{p} $

    - 分量式可以写为：

$ \hat{l}_\gamma = \epsilon_{\alpha\beta\gamma} \hat{x}_{\alpha}\hat{p}_{\beta} $

    - 可以证明：

$ [\hat{l}_\alpha,\hat{x}_\beta] = \epsilon_{\alpha\beta\gamma} \hat{x}_\gamma $

$ [\hat{l}_\alpha,\hat{p}_\beta] = \epsilon_{\alpha\beta\gamma} \hat{p}_\gamma $

$ [\hat{l}_\alpha,\hat{l}_\beta] = \epsilon_{\alpha\beta\gamma} \hat{l}_\gamma $

$ \hat{l}\times\hat{l} = i\hbar \hat{l} $

+ **角动量平方算符**的定义为：

$ \hat{l}^2 = \hat{l}_x^{2}+\hat{l}_y^{2}+\hat{l}_z^{2} $

    - 容易证明：

$ [\hat{l}^2,\hat{l}_\alpha] = 0 $

+ **角动量的升降算符（z方向）**的定义为：

$ \hat{l}_{\pm} = \hat{l}_x \pm i\hat{l}_y $

    - 容易证明：

$ [\hat{l}_z,\hat{l}_{\pm}] = \pm \hbar \hat{l}_{\pm} $

$ [\hat{l}_{+},\hat{l}_{-}] = 2\hbar \hat{l}_z $

#### 常见的转置算符
+ $ (\partial_x)^T = -\partial_x $
+ 证明：

$ \int_{-\infin}^{+\infin}\phi \frac{\partial}{\partial x}\psi^{*}dx = \phi \psi^*|^{+\infin}_{-\infin} - \int_{-\infin}^{+\infin}\psi^* \frac{\partial}{\partial x}\phi dx =  \int_{-\infin}^{+\infin}\psi^* (-\frac{\partial}{\partial x})\phi dx  $

    - 其中第二个等号是因为波函数在无穷远处趋近于零。
