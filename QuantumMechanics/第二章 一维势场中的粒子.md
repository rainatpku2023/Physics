### 一维势场中粒子能量本征态的一般性质
#### 定态薛定谔方程
+ 如果势能不变，那么一维的$ \mathrm{Schr\ddot{o}dinger} $方程可以写为：

$ i\hbar \frac{\partial}{\partial t}\psi(x,t) = \left[-\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2}+V(x)\right]\psi(x,t) $

+ 可以进行变量分离：

$ \psi(x,t) = \psi(x)f(t) $

    - 不难发现：

$ f(t) = e^{-iEt/\hbar} $

$ \left[-\frac{\hbar^2}{2m}\frac{d^2}{d x^2}+V(x)\right]\psi(x) = E\psi(x) $

    - 其中第二个式子被称为**定态**$ \mathrm{Schr\ddot{o}dinger} $**方程**。

#### 定理1
+ 定理内容：如果$ \psi(x) $是定态$ \mathrm{Schr\ddot{o}dinger} $方程的一个解，对应的本征值为$ E $，那么$ \psi^*(x) $也是定态$ \mathrm{Schr\ddot{o}dinger} $方程的一个解，对应的本征值也是$ E $。
+ 证明：
    - 对定态$ \mathrm{Schr\ddot{o}dinger} $方程取复共轭：

$ \left[-\frac{\hbar^2}{2m}\frac{d^2}{d x^2}+V(x)\right]\psi^*(x) = E\psi^*(x) $

+ 可见$ \psi^*(x) $也是定态$ \mathrm{Schr\ddot{o}dinger} $方程的一个解，对应的本征值也是$ E $。

$ \mathrm{Q.E.D.} $

+ **<font style="color:#DF2A3F;">需要说明的是，</font>**$ \psi(x) $**<font style="color:#DF2A3F;">和</font>**$ \psi^*(x) $**<font style="color:#DF2A3F;">不一定描述同一个量子态</font>**
    - **<font style="color:#DF2A3F;">例如自由粒子的</font>**$ e^{ikx} $**<font style="color:#DF2A3F;">和</font>**$ e^{-ikx} $**<font style="color:#DF2A3F;">波函数分别代表向左传播和向右传播的波函数。</font>**
    - $ \psi_1,\psi_2 $**<font style="color:#DF2A3F;">是同一个量子态的条件是存在一个复常数</font>**$ c $**<font style="color:#DF2A3F;">使得</font>**$ \psi_1 = c\psi_2 $**<font style="color:#DF2A3F;">，而非概率分布相同</font>**$ |\psi_1|^2 = c^2|\psi_2|^2 $**<font style="color:#DF2A3F;">。</font>**

#### 定理2
+ 定理内容：对应于能量的某个本征值$ E $，总可以找到原来定态$ \mathrm{Schr\ddot{o}dinger} $方程的一组实解，凡是属于$ E $的任何一个解，都可以表示为这一组实解的线性叠加。
+ 证明：
    - 对于刚找到的一个解$ \psi(x) $，可以做如下操作：
    - 前文提到，如果$ \psi(x) $是能量本征值$ E $的解，那么$ \psi^*(x) $也是本征值$ E $的解。根据线性微分方程解的叠加性，可以得到另外两个本征值为$ E $的实数解：

$ \phi(x) = \psi(x)+\psi^{*}(x),\; \chi(x) = -i(\psi(x)+\psi^{*}(x)) $

    - 反过来，可知：

$ \psi = \frac{1}{2}(\phi+i\chi),\;\psi^* = \frac{1}{2}(\phi-i\chi) $

$ \mathrm{Q.E.D.} $

#### 定理3
+ 定理内容：如果势能函数$ V(x) $具有空间反射不变性，即$ V(x) = V(-x) $，如果$ \psi(x) $是定态$ \mathrm{Schr\ddot{o}dinger} $方程的一个解，对应的本征值为$ E $，那么$ \psi(-x) $也是定态$ \mathrm{Schr\ddot{o}dinger} $方程的一个解，对应的本征值也是$ E $。
+ 证明：
    - 首先对薛定谔方程做代换$ x\rightarrow -x $：

$ \left[-\frac{\hbar^2}{2m}\frac{d^2}{d (-x)^2}+V(-x)\right]\psi(-x) = E\psi(-x) $

    - 考虑到$ V(x) = V(-x) $，$ \frac{d^2}{dx^2} = \frac{d^2}{d(-x)^2} $，有：

$ \left[-\frac{\hbar^2}{2m}\frac{d^2}{d x^2}+V(x)\right]\psi(-x) = E\psi(-x) $

    - 可见$ \psi(-x) $也是本征值为$ E $的一个解。

$ \mathrm{Q.E.D.} $

#### 空间反射算符与宇称
+ 空间反射算符$ P $的定义为：

$ P\psi(\vec{r}) = \psi(-\vec{r}) $

    - 在一维中的结果，就是：

$ P\psi(x) = \psi(-x) $

#### 推论：如果对应于某个能量E，V(x)=V(-x)的一维薛定谔方程方程无简并，则解必然有确定的宇称
+ 推论内容：如果对应于某个能量$ E $，$ V(x)=V(-x) $的一维薛定谔方程方程无简并，则解必然有确定的宇称
+ 证明：
    - 因为原来的方程对于本征值$ E $没有简并，所以$ \psi(x),\psi(-x) $必然对应同一个量子态，即：

$ P \psi(x) = \psi(-x)= C\psi(x) $

    - 再用宇称算符$ P $左右一次，得到：

$ \psi(x) =P^2\psi(x) =C P\psi(x ) =  C^2\psi(x)  $

    - 因此：$ C = \pm 1 $。
+ 对于$ C=+ 1 $的解$ \psi(x) $称为偶宇称解，它满足$ P\psi(x) = \psi(-x) = \psi(x) $。
+ 对于$ C = -1 $的解$ \psi(x) $称为奇宇称解，它满足$ P\psi(x) = \psi(-x) = -\psi(x) $。
+ **具有偶宇称或者奇宇称性质的波函数，被称为具有确定的宇称。其他情况都属于不具有确定的宇称。**

$ \mathrm{Q.E.D.} $

+ **<font style="color:#DF2A3F;">注意：对于能级有简并的情况，</font>**$ \psi(x),\psi(-x) $**<font style="color:#DF2A3F;">可能不是同一个量子态，</font>**$ \psi(x),\psi(-x) $**<font style="color:#DF2A3F;">二者也就不具有确定的宇称。</font>**

#### 定理4
+ 定理内容：设$ V(-x)=V(x) $，则对应于任何一个能量本征值$ E $，总可以找到方程的一组解，且每一个解都有确定的宇称。而属于能量本征值$ E $的任何解，都可以用它们来展开。
+ 证明：
    - 对于能量本征值为$ E $的解$ \psi(x) $，如果它没有确定的宇称，考虑到$ \psi(-x) $也是能量本征值为$ E $的解$ \psi(x) $，则可以构造：

$ f(x) = \psi(x)+\psi(-x),\; g(x) = \psi(x)-\psi(-x) $

    - 显然前者具有偶宇称，后者具有奇宇称，二者都有确定的宇称。
    - 根据线性微分方程解的叠加性质，可以判断二者都属于能量本征值为$ E $的解。
    - 也可以反过来由它们解出来：

$ \psi(x) = \frac{1}{2}(f(x)+g(x)),\;\psi(-x) = \frac{1}{2}(f(x)-g(x)) $

$ \mathrm{Q.E.D.} $

#### 定理5
+ 定理内容：对于阶梯型的方位势：

$ V(x) = \begin{cases}V_1,x<a\\V_2,x>a\end{cases} $

    - 如果$ (V_2-V_1) $有限，那么能量本征函数$ \psi(x) $及其导数$ \psi'(x) $必定是连续的。
    - 但如果$ |V_2-V_1|\rightarrow \infin $，则定理不成立
+ 证明：
    - 定态$ \mathrm{Schr\ddot{o}dinger} $方程可以改写为：

$ \frac{d^2}{dx^2}\psi =-\frac{2m}{\hbar^2}[E-V(x)]\psi $

    - 在$ (a-\epsilon,a+\epsilon) $范围内进行积分：

$ \psi'(a+\epsilon)-\psi'(a-\epsilon) =\int _{a+\epsilon}^{a-\epsilon}-\frac{2m}{\hbar^2}[E-V(x)]\psi dx $

        * 考虑到被积函数都有界，那么当$ \epsilon \rightarrow 0^+ $时：

$ \psi'(a+0^+)-\psi'(a-0^+) =0 $

        * 即$ \psi'(x) $连续。
    - 既然$ \psi'(x) $连续，那么$ \psi'(x) $有界，因而$ \psi(x) $也是连续的。

$ \mathrm{Q.E.D.} $

#### 定理6
+ 定理内容：对于一维粒子，设$ \psi_1(x),\psi_2(x) $都是对应于方程同一能量本征值$ E $的解，那么：

$ \psi_1\psi_2'-\psi_2\psi_1' = \mathrm{const.}\;(这个常数与x无关) $

+ 证明：
    - 定态$ \mathrm{Schr\ddot{o}dinger} $方程可以改写为：

$ \psi_1'' +\frac{2m}{\hbar^2}[E-V(x)]\psi_1=0 $

$ \psi_2'' +\frac{2m}{\hbar^2}[E-V(x)]\psi_2=0 $

    - 用$ \psi_2,\psi_1 $分别乘以两个式子并相减，得：

$ \psi_2 \left[\psi_1'' +\frac{2m}{\hbar^2}[E-V(x)]\psi_1\right] - \psi_1 \left[\psi_2'' +\frac{2m}{\hbar^2}[E-V(x)]\psi_2\right] = 0 $

$ \Rightarrow \psi_1\psi_2'' - \psi_2\psi_1''=0 $

$ \Rightarrow (\psi_1\psi_2' - \psi_2\psi_1')'=0 $

    - 两侧积分得到：

$ \psi_1\psi_2'-\psi_2\psi_1' = \mathrm{const.}\;(这个常数与x无关) $

$ \mathrm{Q.E.D.} $

#### 推论
+ 推论内容：对于**束缚态**一维粒子，设$ \psi_1(x),\psi_2(x) $都是对应于方程同一能量本征值$ E $的解，那么上述定理可以强化为：

$ \psi_1\psi_2'-\psi_2\psi_1' = 0 $

+ 证明：
    - 因为是束缚态，所以在$ |x|\rightarrow \infin $时，$ \psi_1,\psi_2\rightarrow 0 $，即：

$ \lim_{|x|\rightarrow \infin}(\psi_1\psi_2'-\psi_2\psi_1') = 0 $

    - 根据前文证明，$ \psi_1\psi_2'-\psi_2\psi_1' $的值是一个与$ x $无关的常数，所以可以由无穷远处为零推出处处为零，即：

$ \psi_1\psi_2'-\psi_2\psi_1' = 0 $

$ \mathrm{Q.E.D.} $

#### 定理7
+ 定理内容：假设粒子在规则（没有奇点）的势场$ V(x) $中运动，如果存在束缚态，那么这些束缚态一定是不简并的。
+ 证明：
    - 根据前文的推论可知：

$ \psi_1\psi_2'-\psi_2\psi_1' = 0 $

    - 稍加整理可以得到：

$ \psi_1'/\psi_1 = \psi_2'/\psi_2 $

        * $ V(x) $不具有奇异性实际上是保证了这二者都存在。
    - 注意两侧分别是$ \ln \psi_1,\ln\psi_2 $的导数：

$ (\ln \psi_1)'=(\ln\psi_2)' $

$ \Rightarrow \ln \psi_1 = \ln \psi_2 + C_1 $

$ \Rightarrow \psi_1 = C_2 \psi_2 $

    - 可见$ \psi_1(x),\psi_2(x) $之间只差一个常数因子，因此二者描述相同的量子态，而非两个不同的能量简并态。
    - 因此，如果粒子在规则（没有奇点）的势场$ V(x) $中运动，如果存在束缚态，那么这些束缚态一定是不简并的。

$ \mathrm{Q.E.D.} $



### 方势
#### 束缚态与离散谱
+ 下面对于**束缚态对应离散谱**，进行一定的解释：
    - 定态薛定谔方程：

$ \psi'' =-\frac{2m}{\hbar^2}[E-V(x)]\psi $

    - 对于束缚态，在坑外$ E-V(x)<0 $，根据定理2总可以取实数解，有$ \psi'',\psi $二者同号：
        * 若$ \psi>0 $则$ \psi''>0 $：即波函数$ \psi $在$ x $轴上方时为下凸函数。
        * 若$ \psi<0 $则$ \psi''<0 $：即波函数$ \psi $在$ x $轴下方时为上凸函数。
    - 这两种情况都表明，波函数的凸起方向总是远离$ x $轴的。
    - 
    - 根据波函数的连续性，在$ \psi $在有限高势能边缘一定是连续的，此处不妨设为正数。
        * 如果能找到一个本征值$ E $，那么当$ x\rightarrow +\infin $时，一定是因为$ \psi,\psi'' $刚好能同时趋于零。
        * 稍微对不上，使$ \psi'' $大一点点$ \psi $就会加速远离$ x $轴使整个波函数无法归一化，使$ \psi'' $小一点点$ \psi $就会穿越$ x $轴并从反方向加速远离$ x $轴使整个波函数无法归一化。
        * 这是一个非常精细的过程，所以是离散谱。

#### 束缚态波函数的节点数
+ 束缚态，能量越高，波函数节点数越多，这是因为根据：

$ \psi'' =-\frac{2m}{\hbar^2}[E-V(x)]\psi $

    - 在坑里面$ E-V(x)>0 $，比值系数的绝对值是负值，因此$ \psi(x) $为振荡形式解。
    - 随着能量上升比值系数的绝对值越来越大，振荡也越来越厉害，因此波函数的零点（节点）越来越多。

+ 一维谐振子的波函数求解有可圈可点之处，下面进行梳理：
    - 首先进行无量纲化，将式子变得简单，其中能量的无量纲化$ \lambda = E/(\frac{1}{2}\hbar \omega) $有一定的后见之明。
    - 分析波函数的渐进行为，此处主要就是$ \xi \rightarrow \pm \infin $时$ \psi \sim e^{\pm \xi^2/2} $，其中正号不满足束缚态边界条件。然后把这部分分离出去，具体来说就是设$ \psi(\xi) = e^{- \xi^2/2}u(\xi) $。根据$ \psi $满足的微分方程求出$ u $满足的微分方程，然后进行求解。
    - 后面就是常规的二阶线性常微分方程级数法求解，为了保证$ u(\xi) $收敛，必然截断为多项式，这就是厄米多项式$ H_n(\xi) $。
    - 因为我们可以认为本征波函数本身是正交归一的我们已经分离出去了$ e^{- \xi^2/2} $，所以猜测厄米多项式的权函数是$ e^{-\xi^2} $。
