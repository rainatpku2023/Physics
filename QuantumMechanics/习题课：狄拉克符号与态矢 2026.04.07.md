### Dirac符号的引入
#### 从Delta函数谈起
+ 对于坐标表象下的波函数$ \psi(x) $，可以写成以下形式：

$ \psi(x) = \int \psi(x_0)\delta(x-x_0)dx_0 $

    - 这课式子可以如此解读：$ \delta(x-x_0) $是坐标的本征函数，$ \psi(x_0) $是叠加系数。
+ 再考虑对于同一个波函数，坐标表象下形式$ \psi(x) $与动量表象下形式$ \phi(p) $的关系为：

$ \phi(p) = \frac{1}{\sqrt{2\pi\hbar}}\int e^{-ipx/\hbar}\psi(x)dx $

$ \psi(x) = \frac{1}{\sqrt{2\pi\hbar}}\int e^{ipx/\hbar}\phi(p)dp $

+ 希望找到一种不依赖于表象的波函数表示方法，即引入态空间。

#### Dirac符号与态空间
+ 将波函数替换为不依赖于表象的**态矢**：

$ \psi(\vec{r})\in \mathscr{L}^2 \rightarrow \ket{\psi} \in \mathscr{H} $

+ 考虑线性泛函$ \epsilon: \mathscr{H} \rightarrow \mathbb{C} $，$ \epsilon(\ket{\psi}) = \text{some number in }\mathbb{C} $。
    - 线性泛函$ \epsilon $张成一个线性的对偶空间。
+ 此时重新定义**内积**：

$ \braket{\phi|\psi} = \int \phi^* \psi d\tau $

#### 投影算符
+ 考虑恒等式：

$ \ket{\phi}\bra{\phi}(\ket{\psi}) = \braket{\phi|\psi}\ket{\phi} $

    - 可以认为是$ \ket{\phi}\bra{\phi} $将$ \ket{\psi} $投影到了$ \ket{\phi} $上。$ \braket{\phi|\psi} $是投影结果的坐标。
+ 如果$ \ket{\phi_i} $是正交归一完备基矢，那么有：

$ \sum_i\ket{\phi_i}\bra{\phi_i}(\ket{\psi}) = \sum_i\braket{\phi_i|\psi}\ket{\phi_i} =\ket{\psi} $

    - 即：

$ \sum_i\ket{\phi_i}\bra{\phi_i} =\mathbb{I}（恒等算符） $

#### 一些有意思的结果
+ 在坐标表象下，一些态矢对应的波函数：

$ \ket{x_0} \rightarrow \braket{x|x_0} = \delta (x-x_0) $

$ \ket{p_0} \rightarrow \braket{x|p_0} = \frac{1}{\sqrt{2\pi \hbar}}e^{ixp_0/\hbar} $

$ \ket{\psi} \rightarrow \braket{x|\psi} = \psi(x) $

    - 实际上，上面的过程是把$ \braket{x|\cdot} $这一内积运算看成了一种线性泛函：

$ \braket{x|\cdot} :\ket{x_0} \rightarrow \braket{x|x_0} = \delta (x-x_0) $

$ \braket{x|\cdot} :\ket{p_0} \rightarrow \braket{x|p_0} = \frac{1}{\sqrt{2\pi \hbar}}e^{ixp_0/\hbar} $

$ \braket{x|\cdot} :\ket{\psi} \rightarrow \braket{x|\psi} = \psi(x) $

+ 这个式子也有了新的解释：

$ \psi(x) = \frac{1}{\sqrt{2\pi\hbar}}\int e^{ipx/\hbar}\phi(p)dp $

    - 其实就是：

$ \braket{x|\psi}= \int\braket{x|p}\braket{p|\psi}dp $

    - 其中：

$ \braket{x|\psi} = \psi(x) $

$ \braket{x|p}  = \frac{1}{\sqrt{2\pi \hbar}}e^{ixp/\hbar} $

$ \braket{p|\psi} = \phi(p) $

#### 厄米共轭算符的定义
+ 算符$ \hat{A} $：

$ \hat{A}: \ket{\psi}\rightarrow \hat{A}\ket{\psi} = \ket{\psi'} $

+ 其厄米共轭算符$ \hat{A}^\dagger $定义为：

$ \hat{A}^{\dagger}: \bra{\psi}\rightarrow \hat{A}^{\dagger}(\bra{\psi}) = \bra{\psi}\hat{A}^{\dagger} = \bra{\psi'} $

#### 厄米算符的定义
+ 一个算符$ \hat{A} $称为厄米的，如果：

$ \hat{A} = \hat{A}^{\dagger} $

    - 即厄米共轭算符等于本身。

#### 矩阵元与算符的厄米性
+ 在**正交基**下的矩阵元组成的矩阵，转置复共轭等于本身，这是厄米算符的性质。
+ 算符的厄米性是内禀几何属性，不依赖于表象：
    - 以$ \hat{p} $为厄米算符进行证明：

$ (\braket{\psi|\hat{p}|\phi})^* = \left[ \int\braket{\psi|x}\hat{p}\braket{x|\phi}dx\right] ^* = \left[\int\psi^*(x) (-i\hbar \frac{d}{dx})\phi(x)dx\right]^* $

$ =\left[-i\hbar \int\psi^*(x) \frac{d}{dx}\phi(x)dx \right]^* $

$ = \left[-i\hbar\psi^*(x)\phi(x)|_{-\infin}^{+\infin} + i\hbar  \int\phi(x) \frac{d}{dx}\psi^*(x)dx\right]^* $

$ =\left[i\hbar  \int\phi(x) \frac{d}{dx}\psi^*(x)dx\right]^* $

$ = \int \phi^*(x)(-i\hbar \frac{d}{dx})\psi(x)dx $

$ =\braket{\phi|\hat{p}|\psi} $

    - 我们看到：

$ (\braket{\psi|\hat{p}|\phi})^* = \braket{\phi|\hat{p}|\psi} $

    - 即在**正交基**下的矩阵元组成的矩阵，转置复共轭等于本身，这是厄米算符的性质。
+ <font style="color:#DF2A3F;">下面这种推导方式是错误的：</font>

$ \hat{p} = -i\hbar \frac{d}{dx} \Rightarrow \hat{p}^{\dagger} \underbrace{=}_{此处有错} (-i\hbar\frac{d}{dx})^* = i\hbar \frac{d}{dx} = -\hat{p} $

    - <font style="color:#DF2A3F;">这是因为转置时</font>$ d/dx $<font style="color:#DF2A3F;">不再作用在右矢上，而是转而作用在左矢上，需要进行分部积分。</font>

### 量子力学的基本假设
1. $ t_0 $时刻状态使用$ \mathrm{Hilbert} $空间中的态矢$ \ket{\psi(t_0)} $表示。
2. 物理量$ A $在$ \mathrm{Hilbert} $空间中用$ \hat{A} $表示，而且有$ \hat{A} = \hat{A}^{\dagger} $。
3. 测量结果只可能是$ \hat{A} $的本征矢。
4. $ \ket{\psi} $态下测量$ A $的结果为$ a_n $的概率为：

$ P_n = |\braket{\phi_n|\psi}|^2 $

    - 其中$ \hat{A}\ket{\phi_n} = a_n\ket{\phi_n} $，即$ \ket{\phi_n} $为$ \hat{A} $本征值$ a_n $对应的一个本征矢。
5. 测量后，态矢坍缩到所测力学量本征态。
6. 薛定谔方程成立。
7. 全同粒子假设，费米子自旋为半整数，玻色子自旋为整数。

### 升降算符
#### 中心力场中轨道角动量的升降算符
+ 中心力场中，选用对易守恒量完全集$ \{H,L^2,L_z\} $，它们的作用是：

$ \hat{L}^2 \ket{lm} = l(l+1)\hbar^2\ket{lm} $

$ \hat{L}_z \ket{lm} = m\hbar\ket{lm} $

+ 此时定义升降算符：

$ \hat{L}_+ = \hat{L}_x+i\hat{L}_y $

$ \hat{L}_- = \hat{L}_x-i\hat{L}_y $
