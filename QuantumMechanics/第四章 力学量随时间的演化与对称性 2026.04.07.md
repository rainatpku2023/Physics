### 力学量随时间的演化
#### 力学量随着时间的演变
+ 首先考虑一个力学量$ A $的平均值：

$ \bar{A}(t) = (\psi(t),\hat{A}\psi(t)) $

    - 考察其时间变化率：

$ \frac{d}{dt}\bar{A}(t) = (\frac{\partial \psi}{\partial t},\hat{A}\psi)+(\psi,\hat{A}\frac{\partial \psi}{\partial t}) +(\psi,\frac{\partial \hat{A}}{\partial t}\psi) $

$ =(\frac{\hat{H}\psi}{i\hbar},\hat{A}\psi)+(\psi,\hat{A}\frac{\hat{H}\psi}{i\hbar}) + (\psi,\frac{\partial \hat{A}}{\partial t}\psi) $

$ = -\frac{1}{i\hbar}(\psi,\hat{H}\hat{A}\psi)+\frac{1}{i\hbar}(\psi,\hat{A}\hat{H}\psi)+ (\psi,\frac{\partial \hat{A}}{\partial t}\psi) $

$ =\frac{1}{i\hbar}(\psi,[\hat{A},\hat{H}]\psi)+ (\psi,\frac{\partial \hat{A}}{\partial t}\psi) $

$ =\frac{1}{i\hbar} \overline{[\hat{A},\hat{H}]} + \overline{\frac{\partial A}{\partial t}} $

    - 这是一个对于哈密顿量不显含时体系的一个公式：

$ \frac{d}{dt}\bar{A}(t) =\frac{1}{i\hbar} \overline{[\hat{A},\hat{H}]} + \overline{\frac{\partial A}{\partial t}} $

#### 守恒量
+ 守恒量的定义：
    - 对于哈密顿量$ H $不含时的体系，如果$ \hat{A} $也不显含时间，即$ \frac{\partial \hat{A}}{\partial t} = 0 $，那么$ A $称为守恒量，如果：

$ [\hat{A},\hat{H}] = 0 $

+ 守恒量的性质：
    - 如果$ A $是守恒量，那么可以$ \hat{A} $找到一组与$ \hat{H} $的共同本征函数。
    - 如果$ A $是守恒量，那么$ A $测量的平均值和概率分布都不随时间改变。
        * **<font style="color:#DF2A3F;">但是如果想要取确定值还需要更强的条件，即处于</font>**$ \hat{A} $**<font style="color:#DF2A3F;">某一个本征值的某一个本征态。</font>**
        * **<font style="color:#DF2A3F;">可以认为确定值就是一种</font>**$ \mathrm{Dirac \;Delta} $**<font style="color:#DF2A3F;">函数式的分布。</font>**

    - 量子体系的各个守恒量不一定能同时取确定值，例如中心力场中的三个角动量分量均为守恒量，但是因为不对易，所以一般不能同时取确定值。但是，对于$ l =0 $的$ s $态，它们两两之间的对易子恰好为$ 0 $（而非恒为零），这个态内可以同时取确定值。

#### 定态
+ 定态的定义：
    - 定态就是能量本征态。
+ 定态的性质：
    - 一切力学量测量的平均值和概率分布都不随时间变化。

#### 能级简并与守恒量的关系
+ 定理：设体系有两个彼此不对易的守恒量$ F,G $，即：

$ [\hat{F},\hat{G}]\neq 0 $

$ [\hat{F},\hat{H}] =  0 $

$ [\hat{G},\hat{H}] =  0 $

    - 那么体系能级一般是（对$ \hat{H} $的本征值来说）是简并的。



+ 推论：如果一个体系有一个守恒量$ F $，而体系的某条能级不简并，即存在$ \psi_E $唯一地对应$ \hat{H} $的本征值$ E $，那么$ \psi_E $一定是$ \hat{F} $的一个本征态。

#### 位力定理
+ 当体系处于定态下，关于平均值随着时间的变化：

$ H = \frac{p^2}{2m}+V(\vec{r}) $

+ 考虑$ \vec{r}\cdot\vec{p} $的平均值随着时间的变化，有：

$ i\hbar \frac{d}{dt} \overline{\vec{r}\cdot\vec{p}} =\overline{[\vec{r}\cdot \vec{p},H]} $

$ =\frac{1}{2m} \overline{[\vec{r}\cdot \vec{p},p^2]}+ \overline{[\vec{r}\cdot \vec{p},V(\vec{r})]} $

$ = i\hbar\left(\frac{1}{m}\overline{p^2}-\overline{\vec{r}\cdot \nabla V}\right) $

+ 定态下所有力学量的测量值的平均值与分布都不变：

$ \frac{d}{dt} \overline{\vec{r}\cdot\vec{p}} = 0 $

    - 因此：

$ \frac{1}{m}\overline{p^2}-\overline{\vec{r}\cdot \nabla V} = 0 $

    - 即：

$ 2\overline{T} = \overline{\vec{r}\cdot \nabla V} $

### 波包的运动，Ehrenfest定理
#### Ehrenfest定理
+ 与经典粒子运动对应的量子态必然是非定态，设粒子的哈密顿量为：

$ H = \frac{p^2}{2m}+V(\vec{r}) $

+ 考察平均位置随时间的演化：

$ \frac{d}{dt}\overline{\vec{r}} = \frac{1}{i\hbar}\overline{[\vec{r},H]} = \frac{\overline{\vec{p}}}{m} $

    - 然后再计算平均时间随着动量的演化：

$ \frac{d}{dt}\overline{\vec{p}} = \frac{1}{i\hbar}\overline{[\vec{r},H]} = -\overline{\nabla V(\vec{r})}:= \overline{\vec{F}(\vec{r})} $

    - 代入后得到：

$ m\frac{d^2}{dt^2}\overline{\vec{r}} = \overline{\vec{F}(\vec{r})} $

    - 这个就是$ \mathrm{Ehrenfest} $定理，与经典$ \mathrm{Newton} $方程相似。
+ **但当只有**$ \overline{\vec{F}(\vec{r})} $**可以用**$ \vec{F}(\overline{\vec{r}}) $**近似代替时，波包中心**$ \overline{\vec{r}} $**的运动规律才与经典粒子相同。**
+ **从物理上讲，想用波包来描述粒子运动需要：**
    - **波包很狭窄，波包大小与粒子大小相当。**
    - **势场的变化很缓慢，使波包中心感受到的势**$ V(\overline{\vec{r}}) $**和粒子感受到的势**$ V(\vec{r}) $**很接近。**
    - **在研究的运动过程中，波包扩散的不能太快。**

### Schrodinger图像与Heisenberg图像
#### Schrodinger 图像
+ 薛定谔图像认为算符本身是是不随时间演化的，即：

$ \bar{A}(t) = (\psi(t),A\psi(t)) $

+ 描述物理变化的是薛定谔方程：

$ i\hbar\frac{\partial}{\partial t}\psi(t) = H \psi(t) $

+ 我们可以得到：

$ \frac{d}{dt}\bar{A}(t) = \frac{1}{i\hbar} \overline{[\hat{A},\hat{H}]} $

    - 值得小心注意的是前面的系数为$ 1/i\hbar $。

#### Heisenberg 图像
+ 海森堡图像认为算符本身是随时间变化的，即：

$ \bar{A}(t)= (U(t,0)\psi(0),AU(t,0)\psi(0))  $

$ = (\psi(0),U^\dagger(t,0) A U(t,0)\psi(0)) $

$ =(\psi(0),A(t)\psi(0)) $

    - 其中：

$ A(t) = U^\dagger(t,0) A U(t,0) = e^{iHt/\hbar} A e^{-iHt/\hbar} $

+ 不难证明海森堡方程：

$ \frac{d}{dt}A(t) = \frac{1}{i\hbar} [A(t),H] $

#### 空间平移不变性与动量守恒
+ 定义空间平移算符$ \hat{D}(a) $：

$ \hat{D}(a)\psi(x) = \psi(x-a) $

    - $ \hat{D}(a) $可以将作用于的函数向右平移$ a $。
+ 显然，根据泰勒展开，有：

$ \psi(x-a) = \sum_{n=0}^{\infin} \frac{(-a)^n}{n!} \frac{\partial^n}{\partial x^n}\psi(x) = \exp (-a \frac{\partial}{\partial x})\psi(x) = \exp (-ia\hat{p}_x/\hbar )\psi(x) $

    - 可见：

$ \hat{D}(a) = \exp (-a \frac{\partial}{\partial x})= \exp (-ia\hat{p}_x/\hbar ) $

+ 如果空间具有分立的平移不变性，那么$ [\hat{D},\hat{H}] = 0 $。
+ 如果空间具有连续的平移不变形，$ a $可以取任意小的值，那么$ [\hat{p}_x,\hat{H}] = 0 $，即动量守恒。
+ 上述讨论可以很方便地推广到三维：

$ \psi(\vec{r}-\vec{a}) = \sum_{n=0}^{\infin} \frac{(-)^n}{n!}\vec{a}\vec{a}...\vec{a}\underbrace{:}_{n点积}\nabla \nabla...\nabla  \psi(x)  $

$ \hat{D}(a) = \exp (-a \frac{\partial}{\partial \vec{r}})= \exp (-i\vec{a}\cdot \vec{p}/\hbar ) $

#### 空间旋转不变性与角动量守恒
+ 定义空间旋转算符$ \hat{R}(\alpha) $：

$ \hat{R}(\alpha)\psi(\phi) = \psi(\phi-\alpha) $

    - $ \hat{R}(\alpha) $可以将作用于的函数绕着$ z $轴逆时针旋转$ \alpha $。
+ 显然，根据泰勒展开，有：

$ \psi(\phi-\alpha) = \sum_{n=0}^{\infin} \frac{(-\alpha )^n}{n!} \frac{\partial^n}{\partial \phi ^n}\psi(\phi) = \exp (-a \frac{\partial}{\partial \phi})\psi(\phi) = \exp (-i\alpha \hat{l}_z/\hbar )\psi(\phi) $

    - 可见：

$ \hat{R}(\alpha) = \exp (-a \frac{\partial}{\partial \phi}) = \exp (-i\alpha \hat{l}_z/\hbar ) $

+ 如果空间具有分立的旋转不变性，那么$ [\hat{R},\hat{H}] = 0 $。
+ 如果空间具有连续的平移不变形，$ \alpha $可以取任意小的值，那么$ [\hat{l}_z,\hat{H}] = 0 $，即角动量守恒。

### 全同粒子体系与波函数的交换对称性
#### 全同粒子体系的交换对称性
+ 全同粒子：所有内禀属性都相同的粒子。
    - 全同性作用的条件：全同粒子，波包有所重叠。
+ 交换对称性：全同粒子多体系，任何两个粒子交换一下，其量子态不变。
    - 玻色子：波函数满足交换任意两个粒子波函数不变，显然满足交换对称性。
    - 费米子：波函数满足交换任意两个粒子波函数反号，但是概率密度分布仍然不变，所以也满足交换对称性。

#### 交换粒子的算符
+ 考虑交换粒子的算符$ \hat{P}_{ij} $，那么对于全同粒子体系：

$ \hat{P}_{ij}\psi =C\psi $

    - 其中$ C $只是一个常数，这是因为全同粒子多体系，任何两个粒子交换一下，其量子态不变。
    - 再进行一步运算：

$ \hat{P}_{ij}\hat{P}_{ij}\psi =C^2\psi = \psi $

    - 这只有两个解$ C = \pm 1 $。
        * 对于玻色子$ C=1 $，即$ \hat{P}_{ij}\psi =\psi $。
        * 对于费米子$ C=-1 $，即$ \hat{P}_{ij}\psi =-\psi $。
            + 显然对于全同费米子体系有$ \psi = \hat{P}_{ii}\psi =-\psi $，即$ \psi = 0 $，即$ \mathrm{Pauli} $不相容原理。
+ 最终实际的波函数应该是所有$ \hat{P}_{ij},i,j = 0,1,...,n $的共同本征态。

#### N个全同无相互作用费米子组成的体系
+ 假设$ n $个全同无相互作用费米子$ q_1,q_2,...,q_n $分别处于$ \psi_{k_1},\psi_{k_2},...,\psi_{k_n} $单粒子态，那么总的波函数是：

$ \Psi^A_{k_1...k_N}(q_1,...,q_N) = \mathcal{A} \psi_{k_1}(q_1)\psi_{k_2}(q_2)\cdots \psi_{k_n}(q_n) $

    - 其中$ \mathcal{A} $是反对称化算符：

$ \mathcal{A} = \frac{1}{\sqrt{N!}} \sum_{P}\delta_PP $

        * 其中$ 1/\sqrt{N!} $是归一化系数，$ P $是某一个置换算符。
        * $ \delta_P $是与$ P $相对的符号，$ P $的置换数是偶数时$ \delta_P=1 $，$ P $的置换数是奇数时$ \delta_P=-1 $。

#### N个全同无相互作用玻色子组成的体系
+ 假设有$ n_i $个全同无相互作用玻色子$ q_{n_{i1}},q_{n_{i1}},...,q_{n_{ii}} $位于$ \psi_{k_i} $态上，总共有$ m $个态。

$ N = \sum_{i=1}^{m}\sum_{j=1}^{i}1 $

    - 为总粒子数。
    - 那么总的波函数是：

$ \Psi^S_{n_1...n_m} (q_1,q_2,...,q_N) = \sqrt{\frac{\prod_i n_i !}{N!}} \sum P[\psi_{k_1}^{n_1}\psi_{k_2}^{n_2}\cdots \psi_{k_m}^{n_m}] $

    - 需要需要注意$ \Psi^S $的下标是$ n_1...n_m $，代表对应的粒子数排布。
