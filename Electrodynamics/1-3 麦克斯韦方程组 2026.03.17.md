### 复习：真空中的静电场和静磁场的方程
+ 静电场和静磁场满足的方程为：

$ \nabla \cdot \vec{E} = \frac{\rho}{\epsilon_0} $

$ \nabla\times \vec{E} = \vec{0} $

$ \nabla\cdot \vec{B} = 0 $

$ \nabla\times \vec{B} = \mu_0 \vec{j} $

### 电磁感应定律
+ 闭合曲线上的感生电动势和通过闭合曲线围绕着的磁通量的变化率，而且方向与右手相反即：

$ \Epsilon = -\frac{d\phi_B}{dt} = -\frac{d}{dt} \int_S \vec{B}\cdot d\vec{S} $

    - 这就是$ \mathrm{Faraday} $**电磁感应定律**。
+ 另一方面，我们可以将感生电动势写成积分形式：

$ \Epsilon = \int_{\partial S} \vec{E}\cdot d\vec{l} = \int_S (\nabla\times\vec{E})\cdot d\vec{S} $

    - 第二步用到了数学上的斯托克斯定理。
+ 如果认为闭合曲线是不随时间变化的，偏微分号可以移入积分号，最终得到：

$ \boxed{\nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t}} $

### 麦克斯韦前的方程组
#### 方程的补充
+ 结合前面的电磁感应定律，我们可以进一步将方程组补充为：

$ \nabla \cdot \vec{E} = \frac{\rho}{\epsilon_0} $

$ \nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $

$ \nabla\cdot \vec{B} = 0 $

$ \nabla\times \vec{B} = \mu_0 \vec{j} $

#### 非恒定电流下的不自洽性
+ 非恒定电流情形，有：

$ \nabla \cdot \vec{j} + \frac{\partial \rho}{\partial t} = 0 \Rightarrow \nabla \cdot \vec{j} =- \frac{\partial \rho}{\partial t} $

    - 即一般来说，$ \nabla\cdot\vec{j}\neq 0 $。
    - 但是将$ \nabla\times \vec{B} = \mu_0 \vec{j} $两端求散度：

$ LHS = \nabla\cdot(\nabla \times \vec{j}) = 0（旋度无散） $

    - 可见二者是矛盾的，缺少了一些项。

### 位移电流
+ 为了解决电流连续性方程与安培环路定理的矛盾，此时引入位移电流的概念$ \vec{j}_D $：

$ \nabla\cdot \vec{j}_D = \frac{\partial\rho}{\partial t} = -\nabla \cdot \vec{j} $

    - 结合电场散度方程可以得到：

$ \nabla\cdot \vec{j}_D = \frac{\partial \rho}{\partial t} = \frac{\partial}{\partial t} (\epsilon_0 \nabla \cdot \vec{E}) = \epsilon_0 \nabla \cdot \frac{\partial \vec{E}}{\partial t} $

    - 这个时候就有：

$ \nabla\cdot (\vec{j}+\vec{j}_D) = 0 $

    - 方程可以改写为：

$ \nabla\times \vec{B} = \mu_0 (\vec{j}+\vec{j}_D) $

    - 代入$ \vec{j} $的表达时候就得到：

$ \nabla\times \vec{B}= \mu_0 \vec{j}+\mu_0 \epsilon_0 \frac{\partial\vec{E}}{\partial t} $

+ <font style="color:#8A8F8D;">一点注记：如果存在磁单极子，电场旋度式</font>$ \nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $<font style="color:#8A8F8D;">可以看成位移磁流。此外，这个式子应该也会补充上一个磁流对应的环路定理。</font>

### 真空的麦克斯韦方程组
+ 加上位移电流后，我们得到：

$ \nabla \cdot \vec{E} = \frac{\rho}{\epsilon_0} $

$ \nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $

$ \nabla\cdot \vec{B} = 0 $

$ \nabla\times \vec{B}= \mu_0 \vec{j}+\mu_0 \epsilon_0 \frac{\partial\vec{E}}{\partial t} $

+ 这就是**真空中的麦克斯韦方程组**。
+ **<font style="color:#DF2A3F;">需要说明的是，这个方程组只在真空中成立。</font>**

### 洛伦兹力公式
+ 带电粒子$ q $运动速度为$ \vec{v} $，那么它在电磁场中的受力：

$ \vec{F} = q\vec{E}+q\vec{v}\times \vec{B} $

+ 如果电荷连续分布，那么单位体积受力（**洛伦兹力密度**）为：

$ \boxed{\vec{f} = \rho \vec{E} + \vec{j}\times\vec{B}} $

    - 其中用到了：

$ \rho = \lim_{\Delta V\rightarrow 0} \frac{\Delta Q}{\Delta V} $

$ \vec{j} = \rho \vec{v} $
