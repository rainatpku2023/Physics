### 复习：真空中的麦克斯韦方程组
+ 真空中的麦克斯韦方程组为：

$ \nabla\cdot\vec{E} = \frac{\rho}{\epsilon_0} $

$ \nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $

$ \nabla \cdot \vec{B} = 0 $

$ \nabla\times\vec{B} = \mu_0 \vec{j} + \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t} $



### 线性
+ 麦克斯韦方程组是线性的，经典电磁场满足线性叠加原理：
    - 对于$ \rho_1,\vec{j}_1 $产生的电磁场$ \vec{E}_1,\vec{B}_1 $，$ \rho_2,\vec{j}_2 $产生的电磁场$ \vec{E}_2,\vec{B}_2 $。
    - 那么$ \rho_1+\rho_2,\vec{j}_1+\vec{j}_2 $产生的电磁场为$ \vec{E}_1+\vec{E}_2,\vec{B}_1+\vec{B}_2 $。

### 连续对称性
#### 洛伦兹不变性
+ 麦克斯韦方程组与爱因斯坦狭义相对论完全兼容。
+ 狭义相对论认为，在不同**惯性参考系**中的物理规律**形式**相同——相对性原理。
    - **<font style="color:#DF2A3F;">也就是说，在狭义相对论的理论体系下，非惯性参考系的物理规律形式仍然是不同的。</font>**
+ 不同惯性参考系之间的变换由洛伦兹变换来描述。
+ 经典的电磁场方程满足相对性原理。

#### 规范对称性
+ 在规范变换下，麦克斯韦方程形式不变。

### 分立对称性
+ 这块有两种看法：
    - 第一种看法：给定所有物理量的空间反射/时间反演规律，推得麦克斯韦方程组具有空间反射/时间反演对称性。
    - 第二种看法：假定麦克斯韦方程组具有空间反射/时间反演对称性，反推出各个物理量具有的空间反射/时间反演规律。

#### 空间反射对称性
+ 定义空间反射算符$ \hat{P} $，以及宇称变换：

$ \vec{r} \rightarrow \hat{P} \vec{r} = -\vec{r} $

    - 带入后可以得到：

$ \hat{P} \nabla = \hat{P}\frac{\partial}{\partial \vec{r}} = \frac{\partial}{\partial (-\vec{r})} = -\frac{\partial}{\partial \vec{r} }=-\nabla $

    - 宇称变换应该是和时间是无关的：

$ \hat{P} t = t $

$ \hat{P} \frac{\partial}{\partial t} = \frac{\partial}{\partial t} $

+ 通过直觉来看：

$ \hat{P} \rho = \rho $

    - 得到电流密度矢量的宇称变换：

$ \hat{P}\vec{J} = \hat{P}(\rho \frac{\partial \vec{r}}{\partial t}) = \rho \frac{\partial (-\vec{r})}{\partial t} = -\rho \frac{\partial \vec{r}}{\partial t} =  -\vec{J} $

+ 观察麦克斯韦方程组，如果：

$ \hat{P} \vec{E} = -\vec{E} $

$ \hat{P} \vec{B} = \vec{B} $

    - 那么麦克斯韦具有空间反射对称性。

#### 时间反演对称性
+ 定义时间反演算符$ \hat{T} $，以及时间反演变换：

$ t \rightarrow \hat{T} t = -t $

    - 代入后可以得到：

$ \hat{T} \frac{\partial}{\partial t} = \frac{\partial}{\partial (-t)} = -\frac{\partial}{\partial t} $

    - 时间反演变换应该是和空间无关的：

$ \hat{T} \vec{r}= \vec{r} $

$ \hat{T} \nabla = \nabla $

+ 通过直觉来看：

$ \hat{T} \rho = \rho $

    - 得到电流密度矢量的时间反演变换：

$ \hat{T}\vec{J} = \hat{T}(\rho \frac{\partial \vec{r}}{\partial t}) = \rho \frac{\partial \vec{r}}{\partial (-t)} = -\rho \frac{\partial \vec{r}}{\partial t} =  -\vec{J} $

+ 观察麦克斯韦方程组，如果：

$ \hat{T} \vec{E} = \vec{E} $

$ \hat{T} \vec{B} = -\vec{B} $

    - 那么麦克斯韦具有时间反演对称性。

#### 合理性分析
+ 这样的规定应该有一些合理性，以静电场和静磁场为例分析：
    - 假定：电荷在空间反射以及时间反演下不变。
    - 静电场：

$ \vec{E}(\vec{r}) = \frac{1}{4\pi\epsilon_0} \int \frac{\rho(\vec{r}')\times (\vec{r}-\vec{r}')}{|\vec{r}-\vec{r}'|^3}dV' $

        * 空间反射下$ \rho $不变，但是$ \vec{r}-\vec{r}' $变为原来相反数，所以$ \vec{E} $空间反射下变号。
        * 时间反演下$ \rho $不变，$ \vec{r}-\vec{r}' $不变，所以$ \vec{E} $时间反演下不变。
    - 静磁场：

$ \vec{B}(\vec{r}) = \frac{\mu_0}{4\pi} \int \frac{\vec{j}(\vec{r}')\times (\vec{r}-\vec{r}')}{|\vec{r}-\vec{r}'|^3}dV' $

        * 空间反射下$ \vec{j} $变为原来相反数，$ \vec{r}-\vec{r}' $也变为原来相反数，所以$ \vec{B} $空间反射下不变。
        * 时间反演下$ \vec{j} $变为原来相反数，$ \vec{r}-\vec{r}' $不变，所以$ \vec{B} $时间反演下变号。
    - 可见这样的规定和假定麦克斯韦方程组形式具有空间反射不变性和时间反演不变性得到的结果相同。

### 电磁势与规范对称性
#### 电磁势的引入
+ 根据麦克斯韦方程组第三式$ \nabla\cdot\vec{B} =0 $，可以引入矢势$ \vec{A} $满足：

$ \boxed{\vec{B} = \nabla\times \vec{A}} $

    - 显然这个表达式与矢量恒等式$ \nabla \cdot (\nabla \times \vec{A}) = 0 $自洽。
    - $ \vec{A} $到$ \vec{B} $的映射是到上的。
+ 根据麦克斯韦方程组第二式$ \nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $，带入后得到：

$ \nabla \times \vec{E} = -\nabla\times \frac{\partial \vec{A}}{\partial t} $

$ \Rightarrow \nabla \times (\vec{E} +\frac{\partial\vec{A}}{\partial t}) = \vec{0} $

    - 这个时候可以引入标量势$ \phi $：

$ \vec{E}+\frac{\partial \vec{A}}{\partial t} = -\nabla\phi $

$ \Rightarrow \boxed{\vec{E} = -\nabla \phi - \frac{\partial \vec{A}}{\partial t}} $

    - $ (\vec{A},\phi) $到$ \vec{E} $的映射是到上的。

#### 规范变换与规范对称性
+ 前面提到，$ (\vec{A},\phi) $到$ (\vec{E},\vec{B}) $的映射是到上的，但并不是一一到上的，也就是说它是多对一的。
    - 对于不同的$ (\vec{A},\phi) $$ (\vec{A}',\phi') $可能对应着同一组$ (\vec{E},\vec{B}) $。
+ 例如**规范变换**：对于任意的标量场$ \Lambda $：

$ \vec{A}' = \vec{A} +\nabla \Lambda $

$ \phi' = \phi -\frac{\partial \Lambda}{\partial t} $

    - 计算电场强度和磁感应强度：

$ \nabla\times\vec{A}' =\nabla\times(\vec{A}+\nabla\Lambda) = \nabla\times\vec{A}+\nabla\times\nabla\Lambda = \nabla\times\vec{A} =\vec{B} $

        * 其中用到了旋度无散的矢量恒等式。

$ -\nabla \phi' - \frac{\partial \vec{A}'}{\partial t} = -\nabla (\phi -\frac{\partial \Lambda}{\partial t}) - \frac{\partial}{\partial t}(\vec{A} +\nabla \Lambda) =-\nabla \phi - \frac{\partial \vec{A}}{\partial t} = \vec{E} $

        * 其中用到了$ \nabla,\frac{\partial}{\partial t} $的对易性。

#### 规范条件
+ 对于一个电磁场$ (\vec{E},\vec{B}) $，对应着多个电磁势$ (\vec{A},\phi) $。
    - 给电磁势$ (\vec{A},\phi) $附加条件，使之完全确定，这个附加条件就是**规范条件**。
+ 洛伦兹规范（又称协变规范）：

$ \nabla\cdot \vec{A}+ \frac{1}{c^2}\frac{\partial\phi}{\partial t} = 0 $

+ 库伦规范（又称辐射规范、横场规范）：

$ \nabla\cdot \vec{A}= 0 $

#### 电磁势的波动方程
+ 将$ \vec{E} = -\nabla \phi - \frac{\partial \vec{A}}{\partial t} $代入$ \nabla \cdot \vec{E} = \frac{\rho}{\epsilon_0} $：

$ \boxed{
\nabla^2 \phi + \frac{\partial}{\partial t}(\nabla \cdot \vec{A}) = -\frac{\rho}{\epsilon_0}} $

+ 将$ \vec{B} = \nabla \times \vec{A} $代入$ \nabla\times\vec{B} = \mu_0 \vec{j} + \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t} $：

$ \boxed{\nabla^2 \vec{A} - \frac{1}{c^2} \frac{\partial^2 \vec{A}}{\partial t^2} - \nabla (\nabla\cdot\vec{A} + \frac{1}{c^2} \frac{\partial \phi}{\partial t})=-\mu_0 \vec{j}} $

    - 其中用到了矢量恒等式：

$ \nabla\times(\nabla\times\vec{A}) = \nabla(\nabla \cdot \vec{A}) - \nabla^2\vec{A} $

    - 以及光速的表达式：

$ c^2 = \frac{1}{\epsilon_0 \mu_0} $

+ 采用**洛伦兹规范**$ \nabla\cdot \vec{A}+ \frac{1}{c^2}\frac{\partial\phi}{\partial t} = 0 $后：

$ \boxed{
\nabla^2 \phi -\frac{1}{c^2}\frac{\partial^2 \phi}{\partial t^2} = -\frac{\rho}{\epsilon_0}} $

$ \boxed{\nabla^2 \vec{A} - \frac{1}{c^2} \frac{\partial^2 \vec{A}}{\partial t^2}=-\mu_0 \vec{j}} $

    - 在洛伦兹规范下，真空中的麦克斯韦方程组化为相互独立的，关于**电磁势的波动方程**。
