### 例1：无穷大导体平板
#### 问题
+ 真空中与无穷大接地导体平面距离为$ a $处有一个点电荷$ +q $，求全空间电势分布。
+ 求解边值问题：

$ \nabla^2 \phi = -\frac{q}{\epsilon_0}\delta (x,y,z-a) $

$ \phi|_{z=0} = 0 $

$ \phi|_{r\rightarrow \infin} = 0 $

#### 解答
+ 拿走导体板，在$ (0,0,.-a) $处放一个点电荷作为像电荷。像电荷的电荷量为$ -a $。
+ 对于$ z>0 $区域给出尝试解：

$ \phi_{z>0} = \frac{1}{4\pi \epsilon_0}\left(\frac{q}{r_+} + \frac{-q}{r_-}\right) = \frac{1}{4\pi \epsilon_0} \left( \frac{q}{\sqrt{x^2+y^2+(z-a)^2}} - \frac{q}{\sqrt{x^2+y^2+(z+a)^2}} \right) $

    - 这个解满足原来泊松方程和其边界条件。
+ 根据唯一性定理，这个解就是$ z>0 $区域正确且唯一的解。
+ 再根据导体的性质，$ z<0 $区域应该有：

$ \phi_{z<0} =0 \; (静电屏蔽) $

### 例2：导体球
#### 问题
+ 真空中有一个半径为$ R_0 $的接地导体球，在距离球心$ a>R_0 $的位置处放一个点电荷，求空间中各处的电势。
+ 坐标原点在球心，点电荷坐标$ Q(0,0,a) $
    - 球内的电势分布满足的方程和边界条件为：

$ \nabla^2 \phi_1 = 0 $

$ \phi_1 |_{r = R_0} = 0 $

    - 球外：

$ \nabla^2 \phi_2 = -\frac{Q}{\epsilon_0} = -\frac{Q}{\epsilon_0}\delta (x,y,z-a) $

$ \phi_2|_{r=R_0} = 0 $

$ \phi_2|_{r\rightarrow \infin} = 0 $

$ \epsilon_0 \frac{\partial \phi_2}{\partial n}|_{r = R_0} = -\sigma $

#### 解答
+ 对于球外区域的电势讨论：线在$ z $轴上，引入一个负点电荷$ -Q' $，到圆心的距离为$ b $。
+ 尝试解：

$ \phi_2 = \frac{1}{4\pi \epsilon_0}\left( \frac{Q}{\sqrt{x^2+y^2+(z-a)^2}} - \frac{Q'}{\sqrt{x^2+y^2+(z-b)^2}}\right) $

+ 根据球面上任何一点$ \phi_2|_{r= R_0}= 0 $，可以推知：

$ \frac{r'}{r} = \frac{Q'}{Q} = \mathrm{const.} $

    - 这要求$ \Delta OQ'P $与$ \Delta OPQ $相似，可以推出：

$ \frac{r'}{r} = \frac{R_0}{a} = \frac{b}{R_0} $

$ \Rightarrow b = \frac{R_0^2}{a}\; Q' = \frac{R_0 Q}{a} $

    - 代入$ \phi_2 $的表达式，得到：

$ \phi_2(x,y,z) = \frac{1}{4\pi\epsilon_0} \left[\frac{Q}{\sqrt{x^2+y^2+(z-a)^2}}-\frac{R_0Q/a}{\sqrt{x^2+y^2+(z-R_0^2/a)^2}}\right] $

$ \phi_2(r,\theta) = \frac{Q}{4\pi \epsilon_0} \left(\frac{1}{\sqrt{r^2+a^2-2ra\cos\theta}}-\frac{1}{\sqrt{r^2+(ra/R_0)^2-2ra\cos\theta}}\right) $

+ 这个解满足球外区域的方程以及边界条件，根据唯一性定理可知是正确且唯一的解。
