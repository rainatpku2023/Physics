### 导体内自由电荷的分布随时间变化
+ 电荷守恒定律：

$ \nabla\cdot \vec{j}_f + \frac{\partial \rho_f}{\partial t} = 0 $

+ 欧姆定律的微分形式：

$ \vec{j}_f = \sigma \vec{E} $

+ 高斯定理：

$ \nabla \cdot \vec{D} = \rho_f $

+ 导体为线性各向同性均匀介质：

$ \vec{D} = \epsilon \vec{E} $

+ 上述式子联立得：

$  \frac{\partial \rho_f}{\partial t} = -\nabla\cdot \vec{j}_f  = -\sigma\nabla \cdot \vec{E} = -\frac{\sigma}{\epsilon }\nabla \cdot \vec{D} = -\frac{\sigma}{\epsilon}\rho_f(\vec{r},t) $

    - 解上述方程，得：

$ \rho_f(\vec{r},t) = \rho(\vec{r},0) e^{-\sigma t/\epsilon} $

    - 导体内自由电荷密度按照指数衰减。
+ 可见电荷聚集的特征寿命为：

$ \tau = \frac{\epsilon}{\sigma} $

    - 对于良导体，$ \tau\sim10^{-17}\mathrm{s} $，因此良导体内部不堆积电荷，$ \rho = 0 $。
    - 电荷只存在导体表面。
+ 对于低频电磁波，导体响应速度足够快，所以屏蔽效应较好。
+ 如果电磁波频率过高，比电荷特征寿命还要短，屏蔽就会失效。

### 导体内的麦克斯韦方程组与单色电磁波
#### 无自由电荷介质中的麦克斯韦方程
+ 前面提到，对于低频变化的电磁场，导体响应足够快，导体内部不会堆积电荷$ \rho_f = 0 $。
    - 同时假定不存在极化$ \vec{j} = \vec{j}_f $：

$ \nabla\times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $

$ \nabla\times\vec{H} = \vec{j}+\frac{\partial \vec{D}}{\partial t} $

$ \nabla\cdot \vec{D} = \rho_f = 0 $

$ \nabla \cdot \vec{B} = 0 $

#### 导体内的单色波电磁场
[【未完】4-1 平面电磁波 2026.04.25](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/lzpma1/oic0lh1eklbi8ico#rIM8G)

+ 类比真空中分离变量过程，可以得到：

$ \nabla \times \vec{E}(\vec{r}) = i\omega \mu \vec{H}(\vec{r}) $

$ \nabla\times\vec{H}  = -i\omega\epsilon \vec{E}(\vec{r}) + \sigma \vec{E}(\vec{r}) = -i\omega\epsilon'\vec{E} $

    - 其中定义了**复介电常数**：

$ \boxed{\epsilon' = \epsilon + i\frac{\sigma}{\omega}} $

+ 之前导出：

$ \vec{E}(\vec{r},t) = \vec{E}(\vec{r})e^{-i\omega t} =\vec{E}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)} $

+ 在这个情景中，**波数**也是复数：

$ k = \omega \sqrt{\mu\epsilon'} \in \mathbb{C} $

    - 引入**复波矢**：

$ \vec{k} = \vec{\beta}+i\vec{\alpha} $

    - $ \vec{\alpha},\vec{\beta} $<font style="color:#DF2A3F;">都是实矢量，而</font>$ \vec{k} $<font style="color:#DF2A3F;">就是复矢量。</font>
    - 代入后得到：

$ \boxed{\vec{E}(\vec{r},t) = \vec{E}_{0k}e^{-\vec{\alpha}\cdot \vec{r}}e^{i(\vec{\beta}\cdot\vec{r}-\omega t)}} $

+ $ \vec{\alpha} $代表振幅随着透射深度衰减的速率，$ \alpha $越大衰减越快。

#### 复波矢的求解
+ 为了求解复波矢，我们先整理一下公式：
    - 复电容率表达式：$ \epsilon' = \epsilon + i\frac{\sigma}{\omega} $
    - 复数波数的平方表达式：$ k^2 = \omega^2 \mu\epsilon'\in \mathbb{C} $
    - 复数波矢表达式：$ \vec{k} = \vec{\beta}+i\vec{\alpha} $
+ 考虑下列等式：

$ k^2 = \vec{k}\cdot\vec{k} $

    - 左侧：

$ LHS = k^2 = \omega^2 \mu\epsilon' = \omega^2 \mu \left(\epsilon + i\frac{\sigma}{\omega}\right) $

    - 右侧：

$ RHS = \vec{k}\cdot \vec{k} = (\vec{\beta}+i\vec{\alpha})\cdot(\vec{\beta}+i\vec{\alpha}) = \beta^2-\alpha^2+2i\vec{\alpha}\cdot \vec{\beta} $

    - 因此：

$ \omega^2  \mu \left(\epsilon + i\frac{\sigma}{\omega}\right)= \beta^2-\alpha^2+2i\vec{\alpha}\cdot \vec{\beta} $

+ 对比这个式子的实部和虚部，分别得到：

$ \omega^2  \mu \epsilon = \beta^2-\alpha^2 $

$ \omega  \mu\sigma=2\vec{\alpha}\cdot \vec{\beta} $

    - 结合边界条件，就可以求出$ \vec{\alpha} $和$ \vec{\beta} $。

### 趋肤效应和穿透深度（垂直入射）
#### 复波矢的求解：垂直入射的情形
+ 假设导体界面为$ Oxy $平面，$ z>0 $是道题内部。
+ 假设垂直入射，那么$ \alpha_x = \beta_x = 0 $，$ \vec{\alpha},\vec{\beta} $都沿着$ z $方向，那么：
    - 电场强度表达式：

$ \vec{E}(\vec{r},t) = \vec{E}_{0k}e^{-\alpha z}e^{i(\beta z-\omega t)} $

    - 引入复波矢后：

$ \begin{equation}\omega^2  \mu \epsilon = \beta^2-\alpha^2\end{equation} $

$ \begin{equation}\omega  \mu\sigma=2\alpha\beta\end{equation} $

+ <font style="color:#8A8F8D;">这个方程求解有一定的技巧。</font>
    - <font style="color:#8A8F8D;">首先处理</font>$ (2) $<font style="color:#8A8F8D;">式，可以得到：</font>

$ \begin{equation}\alpha^2\beta^2 = \frac{\omega^2\mu^2\sigma^2}{4}\end{equation} $

    - $ (1) $<font style="color:#8A8F8D;">式平方得：</font>

$ \begin{equation}(\beta^2-\alpha^2)^2 = (\beta^2)^2 - 2\alpha^2\beta^2+(\alpha^2)^2 = \omega^4 \mu^2 \epsilon^2\end{equation} $

    - $ (3)+4\times(2) $<font style="color:#8A8F8D;">得：</font>

$ (\beta^2+\alpha^2)^2 = (\beta^2)^2 + 2\alpha^2\beta^2+(\alpha^2)^2 = \omega^4 \mu^2 \epsilon^2+\omega^2\mu^2\sigma^2 $

    - <font style="color:#8A8F8D;">开平方：</font>

$ \begin{equation}\beta^2+\alpha^2 = \sqrt {\omega^4 \mu^2 \epsilon^2+\omega^2\mu^2\sigma^2} = \omega^2\mu\sigma \sqrt{1+\left(\frac{\sigma}{\epsilon\omega}\right)^2}\end{equation} $

+ $ (1)+(5) $除以$ 2 $并开平方：

$ \boxed{\beta = \omega \sqrt{\frac{\mu\epsilon}{2}}\left(\sqrt{1+\left(\frac{\sigma}{\epsilon\omega}\right)^2}+1\right)^{1/2}} $

+ $ (5)-(1) $除以$ 2 $并开平方：

$ \boxed{\alpha = \omega \sqrt{\frac{\mu\epsilon}{2}}\left(\sqrt{1+\left(\frac{\sigma}{\epsilon\omega}\right)^2}-1\right)^{1/2}} $

#### 良导体极限
+ 对于良导体，电导率很大$ \sigma >> \epsilon\omega $，那么：

$ \alpha \approx \beta \approx \sqrt{\frac{\omega\mu\sigma}{2}} $

+ 穿透深度：

$ \delta = \alpha \approx \sqrt{\frac{2}{\omega\mu\sigma}} $

+ 对于垂直入射的**透射波**：

$ \vec{H} = \frac{1}{i\omega\mu} \nabla \times \vec{E} = \frac{1}{i\omega\mu}i\vec{k}\times\vec{E} = \frac{1}{\omega\mu}(\beta+i\alpha)\hat{e}_z \times\vec{E} $

$ = \frac{1}{\omega\mu}(\sqrt{\frac{\omega\mu\sigma}{2}}+i\sqrt{\frac{\omega\mu\sigma}{2}})\hat{e}_z \times\vec{E} =\sqrt{\frac{\sigma}{\omega\mu}}\frac{1}{\sqrt{2}} (1+i)\hat{e}_z\times\vec{E}= \sqrt{\frac{\sigma}{\omega\mu}}e^{i\frac{\pi}{4}}\hat{e}_z\times\vec{E}  $

+ 可见**良导体磁场相位滞后电场相位**$ \frac{\pi}{4} $。
+ **良导体磁场能量与电场能量密度比**：

$ \frac{\mu H^2/2}{\epsilon E^2/2} = \frac{\mu}{\epsilon} \frac{\sigma}{\omega\mu} = \frac{\sigma}{\epsilon\omega}>>1 $

    - 可见**良导体内电磁波能量中磁场占主导**。
