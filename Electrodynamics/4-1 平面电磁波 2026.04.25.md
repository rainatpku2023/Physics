### 复习
#### 真空中的麦克斯韦方程组
+ 真空中的麦克斯韦方程组为：

$ \nabla \cdot \vec{E} = \frac{\rho}{\epsilon_0} $

$ \nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t} $

$ \nabla \cdot \vec{B} = 0 $

$ \nabla\times\vec{B} = \mu_0 \vec{j} + \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t} $

+ 洛伦兹力公式为：

$ \vec{F} = q(\vec{E}+\vec{v}\times\vec{B}) $

#### 电磁势的引入
+ 根据$ \vec{B} $无散$ (3) $式，可以引入矢量势$ \vec{A} $与：

$ \vec{B} = \nabla\times\vec{A} $

    - 再根据$ \left(\vec{E}+\frac{\partial \vec{A}}{\partial t}\right) $无旋$ (2) $式引入标量势$ \phi $：

$ \vec{E} = -\nabla\phi-\frac{\partial\vec{A}}{\partial t} $

+ 在洛伦兹规范$ \nabla \cdot \vec{A}+ \frac{1}{c^2} \frac{\partial\phi}{\partial t} = 0 $前提下，麦克斯韦方程组得到两个独立的波动方程：

$ \nabla^2\phi - \frac{1}{c^2}\frac{\partial\phi}{\partial t^2} = -\frac{\rho}{\epsilon_0} $

$ \nabla^2\vec{A} - \frac{1}{c^2}\frac{\partial\vec{A}}{\partial t^2} = -\mu_0\vec{j} $

    - <font style="color:#DF2A3F;">注意等式有右边的负号，这是因为力强度总是指向势的负梯度。</font>

#### 一维波动方程
+ 一维的波动方程形如：

$ \frac{\partial^2}{\partial z^2} - \frac{1}{v^2}\frac{\partial^2 f}{\partial t^2} = 0 $

+ 上述方程的**通解**可以写成行波解的形式，行波解又被称为达朗贝尔解：

$ f(z,t) = g(z-vt)+ h(z+vt) $

    - 右侧第一项是向右传播的行波，因为随着$ t $增大，想要得到相同的函数值需要$ z $更大。
    - 右侧第二项是向左传播的行波，因为随着$ t $增大，想要得到相同的函数值需要$ z $更小。
+ 还可以写出**特解**：

$ f(z,t) = A\cos[k(z-vt)+\delta] $

$ =A \cos (kz-\omega t+\delta) $

$ =\mathrm{Re} \{Ae^{i(kz-\omega t+\delta)}\} $

    - 其中：
        * $ A $为振幅，$ kz-\omega t+\delta $为相位，$ \delta $为初相位。
        * $ \omega $为角频率，数值是频率的$ 2\pi $倍；$ k $为角波矢，数值是波矢的$ 2\pi $倍。
+ 通解可以用特解线性叠加表示：

$ F(z,t) = \frac{1}{\sqrt{2\pi}} \int_{-\infin}^{+\infin} A(k)e^{i(kz-\omega(k)t)}dk $



### 真空中的电磁场波动方程
#### 真空中无源麦克斯韦方程组
+ 在真空无源条件下，$ \rho = 0,\vec{j} = \vec{0} $，因此：

$ \nabla \cdot \vec{E} = 0 $

$ \nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t} $

$ \nabla \cdot \vec{B} = 0 $

$ \nabla\times\vec{B} = \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t} $

#### 电磁场波动方程推导
+ 对$ (2) $式求旋度：

$ \nabla \times (\nabla\times\vec{E}) = -\nabla \times \frac{\partial\vec{B}}{\partial t} $

    - 处理左边：

$ LHS = \nabla \times (\nabla\times\vec{E}) = \nabla(\nabla \cdot \vec{E})-\nabla^2\vec{E} = -\nabla^2\vec{E} $

        * 最后一个等号用到了$ \nabla \cdot \vec{E} = 0 $。
    - 处理右边：

$ RHS =  -\nabla \times \frac{\partial\vec{B}}{\partial t} = -\frac{\partial}{\partial t}(\nabla\times\vec{B}) = -\frac{\partial}{\partial t}( \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t}) = -\mu_0\epsilon_0 \frac{\partial^2\vec{E}}{\partial t^2} $

        * 第二个等号用到了对时间导数和对空间导数可以交换顺序。
        * 第三个等号带入了$ (4) $式$ \nabla\times\vec{B} = \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t} $。
    - 两边联立即得：

$ \nabla^2 \vec{E} - \mu_0\epsilon_0 \frac{\partial \vec{E}^2}{\partial t^2} = \vec{0} $

+ 类似可得：

$ \nabla^2 \vec{B} - \mu_0\epsilon_0 \frac{\partial \vec{B}^2}{\partial t^2} = \vec{0} $

+ 令$ c = 1/\sqrt{\mu_0\epsilon_0} $，可得：

$ \nabla^2 \vec{E} - \frac{1}{c^2} \frac{\partial \vec{E}^2}{\partial t^2} = \vec{0} $

$ \nabla^2 \vec{B} - \frac{1}{c^2} \frac{\partial \vec{B}^2}{\partial t^2} = \vec{0} $

### 电磁波在介质中的色散
#### 介质中无源麦克斯韦方程组
+ 在介质无源条件下，$ \rho_f = 0,\vec{j}_f = \vec{0} $，因此：

$ \nabla \cdot \vec{D} = 0 $

$ \nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t} $

$ \nabla \cdot \vec{B} = 0 $

$ \nabla\times\vec{H} =  \frac{\partial \vec{D}}{\partial t} $

+ 介质方程组为：

$ \vec{D}(\omega) = \epsilon(\omega)\vec{E}(\omega) $

$ \vec{B}(\omega) = \mu(\omega)\vec{H}(\omega) $

+ 由于存在⾊散，$ \epsilon,\mu $与电磁波角频率$ \omega $相 关，介质中⼀般不存在$ \vec{D}(t) = \epsilon \vec{E}(t) $的线性关系。
+ 但是如果角频率$ \omega $固定，就不需要考虑到$ \epsilon,\mu $变化的问题。
    - 讨论固定$ \omega $的电磁波在介质中的传播。

### 时谐电磁波
#### 波包与单色波的关系
+ 通常电磁波为不同频率单色波的叠加，形成波包。
    - 根据前面推出的波动方程句具有线性性，波包内每一个频率的成分可以认为是单独传播的。
    - 在真空中，每一种频率的单色波$ \mu = \mu_0,\epsilon = \epsilon_0 $，因此传播速度相同，不会出现色散现象。
    - 但是如果在介质中，$ \mu,\epsilon $都与角频率$ \omega $相关，传播速度不同，因此波包会发生扩散。
+ 考虑波包的的传播：
    - 可以先将初始状态的波包进行频率分解，即做$ \mathrm{Fouier} $正变换从坐标空间转化到频率空间获得单色波振幅：

$ A(k)= \frac{1}{\sqrt{2\pi}} \int_{-\infin}^{+\infin} U(x,0)e^{-ikx}dx $

    - 然后再让各个单色波独立传播，再叠加在一起：

$ U(x,t) = \frac{1}{\sqrt{2\pi}} \int_{-\infin}^{+\infin} A(k)e^{ikx-i\omega(k)t}dk $

    - 注意时间演化是乘以$ e^{-i \omega t} $的因子，注意上面的负号。
+ 这与量子力学中的波包传播原理类似。

#### 单色波前提下麦克斯韦方程组的改写
+ **真空时谐单色波**可以拆成空间部分和时间部分，时间部分采用振荡形式：

$ \vec{E}(\vec{r},t) = \vec{E}(\vec{r})e^{-i\omega t} $$ \vec{B}(\vec{r},t) = \vec{B}(\vec{r})e^{-i\omega t} $

    - 结合介质方程组：

$ \vec{D}(\omega) = \epsilon(\omega)\vec{E}(\omega) $$ \vec{B}(\omega) = \mu(\omega)\vec{H}(\omega) $

    - 代入波动方程：

$ \nabla^2 \vec{E} - \frac{1}{c^2} \frac{\partial \vec{E}^2}{\partial t^2} = \vec{0} $$ \nabla^2 \vec{B} - \frac{1}{c^2} \frac{\partial \vec{B}^2}{\partial t^2} = \vec{0} $

    - 可以得到**电磁场空间部分**：

$ \nabla^2 \vec{E}(\vec{r}) + k^2 \vec{E}(\vec{r}) = \vec{0} $

$ \nabla^2 \vec{H}(\vec{r}) + k^2 \vec{H}(\vec{r}) = \vec{0} $

    - 其中$ k^2 = \omega^2 \mu\epsilon $。**这就是偏微分方程中被亥姆霍兹方程的方程形式。**
+ 根据麦克斯韦方程组电磁场旋度式：

$ \nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t} $

$ \nabla\times\vec{H} =  \frac{\partial \vec{D}}{\partial t} $

    - 可以得到**电磁空间部分的关系**：

$ \nabla\times\vec{E}(\vec{r}) = i\omega\mu \vec{H}(\vec{r}) $

$ \nabla\times\vec{H}(\vec{r}) = -i\omega\epsilon \vec{E}(\vec{r}) $

+ 注意应该还有**横波条件**：

$ \nabla \cdot \vec{E} = 0 $$ \nabla \cdot \vec{H} = 0 $

#### 亥姆霍兹方程的平面波解
+ 前面我们在**真空时谐单色波**前提下，改写了麦克斯韦方程组，下面摘录只与电场有关的部分：

$ \nabla^2 \vec{E}(\vec{r}) + k^2 \vec{E}(\vec{r}) = \vec{0} $

$ \nabla \cdot \vec{E} = 0 $

+ 在三维条件下，可以写出：

$ \vec{E}(\vec{r},t) = \vec{E}(\vec{r})e^{-i\omega t} =\vec{E}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)} $

    - 根据横波条件$ \nabla \cdot \vec{E} = 0 $：

$ 0 = \nabla \cdot \vec{E} = \frac{\partial}{\partial \vec{r}} \left[\vec{E}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)}\right] =  i\vec{k}\cdot \vec{E}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)}  $

$ \Rightarrow \vec{k}\cdot \vec{E}_{0k} = 0 $

    - 可见**电场强度振荡方与传播方向相互垂直**。
+ 同理可以得到磁场部分：

$ \vec{H}(\vec{r},t)  = \vec{H}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)} $

$  \vec{k}\cdot \vec{H}_{0k} = 0 $

    - **磁场强度振荡方与传播方向相互垂直**。



+ 下面我们推导$ \vec{E},\vec{H} $的方向关系，空间部分有：

$ \nabla\times\vec{H}(\vec{r}) = -i\omega\epsilon \vec{E}(\vec{r}) $

    - 左侧：

$ LHS = \nabla\times\left[\vec{H}_{0k}e^{i\vec{k}\cdot\vec{r}}\right] = \nabla\times\vec{H}_{0k}\left[e^{i\vec{k}\cdot\vec{r}}\right]+\nabla\left[e^{i\vec{k}\cdot\vec{r}}\right]\times \vec{H}_{0k} $

$ = 0+e^{i\vec{k}\cdot\vec{r}} i\vec{k}\times \vec{H}_{0k}=e^{i\vec{k}\cdot\vec{r}}i\vec{k}\times \vec{H}_{0k} $

        * 第二个等号用到了$ \nabla \times(c\vec{f}) = c\nabla\times\vec{f}+\nabla c \times \vec{f} $公式。
        * 第三个等号用到了$ \vec{H}_{0k} $是常矢量的条件。
    - 右侧：

$ RHS = -i\omega\epsilon \vec{E}(\vec{r}) = -i\omega \epsilon \vec{E}_{0k}e^{i\vec{k}\cdot \vec{r}} $

    - 联立得：

$ e^{i\vec{k}\cdot\vec{r}}i\vec{k}\times \vec{H}_{0k} = -i\omega \epsilon \vec{E}_{0k}e^{i\vec{k}\cdot \vec{r}} $

$ \Rightarrow \vec{k}\times \vec{H}_{0k} = -\omega \epsilon \vec{E}_{0k} $

$ \Rightarrow\hat{e}_k \times \vec{H}_{0k} = -\sqrt{\frac{\epsilon}{\mu}}\vec{E}_{0k} $

        * $ \vec{k} $方向上单位矢量：$ \hat{e}_k = \frac{\vec{k}}{k} = \frac{\vec{k}}{\sqrt{\omega^2 \mu\epsilon}} =  \frac{\vec{k}}{\omega\sqrt{\mu\epsilon}} $
    - **可见**$ \vec{k},\vec{E},\vec{H} $**呈现右手关系，相互正交。而且相同位置电场强度与磁场强度大小比值固定。**

+ <font style="color:#DF2A3F;">需要说明的是，实际中电磁波只能取实数，即：</font>

$ \mathrm{Re}\vec{E}(\vec{r},t) =\mathrm{Re}\vec{E}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)}=\vec{E}_{0k}\cos(\vec{k}\cdot\vec{r}-\omega t) $

$ \mathrm{Re}\vec{H}(\vec{r},t) =\mathrm{Re}\vec{H}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)} = \vec{H}_{0k}\cos(\vec{k}\cdot\vec{r}-\omega t) $

    - <font style="color:#DF2A3F;">使用复数只是代数上比较简洁。</font>

### 平面电磁波
#### 平面电磁波的性质
+ 前面推出，单色平面电磁波的解为：

$ \mathrm{Re}\vec{E}(\vec{r},t) =\mathrm{Re}\vec{E}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)}=\vec{E}_{0k}\cos(\vec{k}\cdot\vec{r}-\omega t) $

$ \mathrm{Re}\vec{H}(\vec{r},t) =\mathrm{Re}\vec{H}_{0k}e^{i(\vec{k}\cdot\vec{r}-\omega t)} = \vec{H}_{0k}\cos(\vec{k}\cdot\vec{r}-\omega t) $

+ 在方向和振幅上有：

$ \hat{e}_k \times \vec{E}_{0k} = \sqrt{\frac{\mu}{\epsilon}}\vec{H}_{0k} $

$ \hat{e}_k \times \vec{H}_{0k} = -\sqrt{\frac{\epsilon}{\mu}}\vec{E}_{0k} $

    - $ \vec{k},\vec{E},\vec{H} $呈现右手关系，相互正交。而且相同位置电场强度与磁场强度大小比值固定。
+ 传播速度：

$ v= 1/\sqrt{\epsilon\mu} $

### 能量密度、能流密度矢量、动量密度矢量、动量流密度张量
#### 能量密度
+ 一般而言，能量密度表达式：

$ \omega =\frac{1}{2}(\epsilon_0 E^2 + \frac{1}{\mu_0}B^2) $

+ 对于平面电磁波来说：

$ \omega =\frac{1}{2}\left(\epsilon_0 |\mathrm{Re}\vec{E}|^2 + \frac{1}{\mu_0}|\mathrm{Re}\vec{B}|^2\right) $

    - 结合$ \vec{B} = \frac{1}{\omega} \vec{k}\times\vec{E} = \sqrt{\mu_0\epsilon_0}\hat{e}_k\times\vec{E} $，可知：

$ \omega =\epsilon_0 |\mathrm{Re}\vec{E}|^2 = \frac{1}{\mu_0}|\mathrm{Re}\vec{B}|^2 $

#### 能流密度矢量
+ 一般而言，能流密度表达式：

$ \vec{S} = \frac{1}{\mu_0} \vec{E}\times\vec{B} $

+ 对于平面电磁波来说：

$ \vec{S} = \frac{1}{\mu_0} \mathrm{Re}\vec{E}\times (\sqrt{\mu_0\epsilon_0}\hat{e}_k\times\mathrm{Re}\vec{E}) $

$ = \frac{1}{\sqrt{\epsilon_0 \mu_0}}\epsilon_0 |\mathrm{Re}\vec{E}|^2\hat{e}_k = c\omega \hat{e}_k $

#### 动量密度矢量
+ 一般而言，动量密度矢量表达式：

$ \vec{g} = \epsilon_0 \vec{E}\times \vec{B} $

+ 对于平面电磁波来说：

$ \vec{g} = \epsilon_0 \mathrm{Re}\vec{E}\times  \mathrm{Re}\vec{B} = \frac{\vec{S}}{c^2} = \frac{\omega}{c}\hat{e}_k $

#### 动量流密度张量
+ 一般而言，动量流密度张量表达式：

$ \overleftrightarrow T = \frac{1}{2}\left( \epsilon_0 E^2 + \frac{B^2}{\mu_0}\right)\overleftrightarrow{I} -\epsilon_0 \vec{E}\vec{E}-\frac{\vec{B}\vec{B}}{\mu_0} $

+ 对于平面电磁波来说：

$ \overleftrightarrow T = \frac{1}{2}\left( \epsilon_0 |\mathrm{Re}E|^2 + \frac{|\mathrm{Re}B|^2}{\mu_0}\right)\overleftrightarrow{I} -\epsilon_0 \mathrm{Re}\vec{E}\mathrm{Re}\vec{E}-\frac{\mathrm{Re}\vec{B}\mathrm{Re}\vec{B}}{\mu_0} $

+ 设$ \hat{e}_k = \hat{e}_z,\vec{E} = E\hat{e}_x,\vec{B} = B\hat{e}_y $，那么：

$ \overleftrightarrow T = \omega \left[-\hat{e}_x\hat{e}_x-\hat{e}_y\hat{e}_y+(\hat{e}_x\hat{e}_x+-\hat{e}_y\hat{e}_y+-\hat{e}_z\hat{e}_z)\right] $

$ =\omega\hat{e}_z\hat{e}_z = \omega\hat{e}_k\hat{e}_k $

    - 其中化简时使用了：

$ \omega =\epsilon_0 |\mathrm{Re}\vec{E}|^2 = \frac{1}{\mu_0}|\mathrm{Re}\vec{B}|^2 $

+ **<font style="color:#DF2A3F;">其实这些推导过程使用</font>**$ \vec{B} = \mu\vec{H} $**<font style="color:#DF2A3F;">代入后会得到更加对称的结构。</font>**
