### 介质中的麦克斯韦方程组
#### 微分形式
$ \nabla \cdot \vec{D} = \rho_{自由} $

$ \nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t} $

$ \nabla\cdot \vec{B} = 0 $

$ \nabla\times \vec{H} = \vec{f}_{自由}+\frac{\partial \vec{D}}{\partial t} $

#### 积分形式
$ \oint_S \vec{D}\cdot d\vec{S} = \int_V \rho_{自由}dV = Q_{自由} $

$ \oint_{L}\vec{E}\cdot d\vec{l} = -\frac{d}{dt}\int_S \vec{B}\cdot d\vec{S} $

$ \oint_S \vec{B}\cdot d\vec{S} = 0 $

$ \oint_L \vec{H}\cdot d\vec{l}  = \int_S (\vec{f}_{自由}+\frac{\partial \vec{D}}{\partial t})\cdot d\vec{S} = I_{自由} +\frac{d}{dt} \int_S\vec{D}\cdot d\vec{S} $

### 介质交界处的边界条件
#### 电场强度
+ 在边界处取一个无穷小的狭长矩形回路（高度远小于宽度），同时固定回路本身不随时间变化，那么：

$ \oint_{L}\vec{E}\cdot d\vec{l} = -\int_S \frac{\partial \vec{B}}{\partial t}\cdot d\vec{S} $

    - 考虑无穷小回路的高度趋于零，因为$ \frac{\partial \vec{B}}{\partial t} $总是有限值，所以：

$ \oint_{L}\vec{E}\cdot d\vec{l}  = 0 $

    - 上式可以进一步变为：

$ \hat{n} \times (\vec{E}_2 - \vec{E}_1) = \vec{0} $

#### 磁场强度
+ 在边界处取一个无穷小的矩形回路（高度远小于宽度），同时固定回路本身不随时间变化，那么：

$ \oint_L \vec{H}\cdot d\vec{l}   = I_{自由} +\int_S \frac{\partial \vec{D}}{\partial t}\cdot d\vec{S} $

    - 考虑无穷小回路的高度趋于零，因为$ \frac{\partial \vec{D}}{\partial t} $总是有限值，所以：

$ \oint_L \vec{H}\cdot d\vec{l}   = \alpha_{自由}\Delta L+0 $

    - 其中$ \alpha_{自由} $为自由面电流密度。
    - 此时再让宽度趋于零，得到：

$ \hat{n}\times (\vec{H}_2-\vec{H}_1) = \vec{\alpha}_{自由} $

#### 电位移矢量
+ 在边界处，取一个无穷小的柱体（高度远小于底面的线度），那么：

$ \oint_S \vec{D}\cdot d\vec{S}= Q_{自由} = \sigma_{自由}\Delta S $

    - 其中$ \sigma_{自由} $为自由面电荷密度。
    - 由此可以推出：

$ \hat{n}\cdot (\vec{D}_2-\vec{D}_1) = \sigma_{自由} $

#### 磁感应强度
+ 在边界处，取一个无穷小的柱体（高度远小于底面的线度），那么：

$ \oint_S \vec{B}\cdot d\vec{S}=0 $

    - 其中$ \sigma_{自由} $为自由面电荷密度。
    - 由此可以推出：

$ \hat{n}\cdot (\vec{B}_2-\vec{B}_1) = 0 $

### 总结
+ 综上，我们得到了电磁场的边值关系：

$ \hat{n} \times (\vec{E}_2 - \vec{E}_1) = \vec{0} $

$ \hat{n}\times (\vec{H}_2-\vec{H}_1) = \vec{\alpha}_{自由} $

$ \hat{n}\cdot (\vec{D}_2-\vec{D}_1) = \sigma_{自由} $

$ \hat{n}\cdot (\vec{B}_2-\vec{B}_1) = 0 $
