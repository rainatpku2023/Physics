### 多元函数的泰勒展开
#### 分量式
+ 多元函数的泰勒展开用分量表示为：

$ f(x_1,x_2,...,x_n) = f(x_1^0,x_2^0,...,x_n^0) + \sum_{i=1}^n(x_i - x_i^0)f_i'(x_1^0,x_2^0,...,x_n^0)+\frac{1}{2!}\sum_{i,j=1}^n(x_i - x_i^0)(x_j - x_j^0)f_i'(x_1^0,x_2^0,...,x_n^0)+\cdots $

#### 用nabla表示
+ 还可以表示为：

$ f(\vec{r}_0+\vec{r}_1) = f(\vec{r}_0) + (\vec{r}_1 \cdot \nabla) f(\vec{r}_0) + \frac{1}{2!} (\vec{r}_1 \cdot \nabla)^2 f(\vec{r}_0) +\cdots $

$ = \sum_{k=0}^n \frac{1}{k!} (\vec{r}_1 \cdot \nabla)^k f(\vec{r}_0) = \sum_{k=0}^n \frac{1}{k!} \underbrace{\vec{r}_1\vec{r}_1\cdots \vec{r}_1}_{k个\vec{r}_1形成的并矢} \underbrace{\cdots}_{n点积} \underbrace{\nabla\nabla\cdots \nabla}_{k个\nabla形成的并矢} f(\vec{r}_0) $



+ 在电动力学中，常常考察源点$ \vec{r}' $对于场点$ \vec{r} $的电势影响，一般来说有$ |\vec{r}'|<<|\vec{r}| $，所以改写为：

$ f(\vec{r}-\vec{r}') = f(\vec{r}_0) - (\vec{r}' \cdot \nabla) f(\vec{r}) + \frac{1}{2!} (\vec{r}' \cdot \nabla)^2 f(\vec{r}) -+\cdots $

$ = \sum_{k=0}^n \frac{(-)^k}{k!} (\vec{r} \cdot \nabla)^k f(\vec{r})  $

#### 反比例函数的情形
+ 对于反比例函数，有：

$ \frac{1}{|\vec{r}-\vec{r}'|} = \frac{1}{|\vec{r}|} - \vec{r}'\cdot \nabla\frac{1}{r} + \frac{1}{2!} \vec{r}'\vec{r}' : \nabla\nabla\frac{1}{r}+\cdots  $

    - 结合：

$ \nabla \frac{1}{r} = -\frac{\vec{r}}{r^3} = -\frac{\hat{e}_r}{r^2} $

$ \nabla\nabla \frac{1}{r} = \frac{3\hat{e}_r\hat{e}_r - \overleftrightarrow{I}}{r^3} $

+ 可以得到：

$ \frac{1}{|\vec{r}-\vec{r}'|} = \frac{1}{|\vec{r}|} - \vec{r}'\cdot \left(-\frac{\hat{e}_r}{r^2}\right) + \frac{1}{2!} \vec{r}'\vec{r}' :\left(\frac{3\hat{e}_r\hat{e}_r - \overleftrightarrow{I}}{r^3}\right)+\cdots $

$ =  \frac{1}{r} +  \frac{\vec{r}'\cdot \hat{e}_r}{r^2}\ + \frac{1}{2!} \frac{\vec{r}'\vec{r}' :(3\hat{e}_r\hat{e}_r - \overleftrightarrow{I})}{r^3}+\cdots $

    - 结合：

$ \vec{r}'\vec{r}':\overleftrightarrow{I} = \vec{r}'\cdot\vec{r}' = r'^2 = \vec{r}'^2 \overleftrightarrow{I}:\hat{e}_r\hat{e}_r $

+ 可以得到：

$ \frac{1}{|\vec{r}-\vec{r}'|} =  \frac{1}{r} +  \frac{\vec{r}'\cdot \hat{e}_r}{r^2}\ + \frac{1}{2!} \frac{(3\vec{r}'\vec{r}' -r'^2\overleftrightarrow{I}):\hat{e}_r\hat{e}_r}{r^3}+\cdots $

### 两点电荷多极展开、电偶极矩
#### 电偶极矩的定义
+ 考虑坐标轴上相距为$ l $的两个点电荷$ q $和$ -q $，电势可以写为：

$ \phi = \frac{1}{4\pi\epsilon_0}\left( \frac{q}{r_+} - \frac{q}{r_-}\right) = \frac{1}{4\pi \epsilon_0}\frac{\vec{p}\cdot\vec{r}}{r^3} $

    - 上面考察的是远处的电势，应该有：

$ r_+,r_->> |r_+-r_-| = l $

    - 还有磁偶极矩表达式：

$ \vec{P} = q(\vec{r}_+-\vec{r}_-) = q\vec{l} $

#### 电偶极子产生的电场推导
+ 根据$ \vec{E} = -\nabla\phi $可知：

$ \vec{E} = -\frac{1}{4\pi\epsilon_0}\nabla(\frac{\vec{p}\cdot\vec{r}}{r^3}) = \frac{1}{4\pi \epsilon_0} \frac{3(\vec{P}\cdot\hat{e}_r)\hat{e}_r-\vec{P}}{r^3} $

+ 证明：
    - 想要证明上面的式子，需要先明确$ \nabla $作用在$ r $上，而与$ p $无关。
        * 这是因为$ \nabla $是对场空间的微分，而不是对源空间的微分，因此：

$ \nabla(\frac{\vec{p}\cdot\vec{r}}{r^3}) = \hat{e}_i\partial_i\left[\frac{p_jr_j}{(r_kr_k)^{3/2}}\right] = \frac{\hat{e}_i\partial_i(p_jr_j)(r_kr_k)^{3/2} - p_jr_j\hat{e}_i\partial_i(r_kr_k)^{3/2}}{(r_kr_k)^3} $

$ = \frac{\hat{e}_ip_j\partial_i(r_j)(r_kr_k)^{3/2} - p_jr_j\hat{e}_i(r_kr_k)^{1/2}\frac{3}{2}\cdot 2r_k\partial_i(r_k)}{(r_kr_k)^3} $

$ = \frac{\hat{e}_ip_j\delta_{ij}(r_kr_k)^{3/2} - 3p_jr_j\hat{e}_i(r_kr_k)^{1/2}r_k\delta_{ik}}{(r_kr_k)^3} $

$ =\frac{\hat{e}_ip_i(r_kr_k)^{3/2} - 3p_jr_jr_i\hat{e}_i(r_kr_k)^{1/2}}{(r_kr_k)^3} $

$ = \frac{\vec{P} - 3r(\vec{P}\cdot\vec{r})\vec{r}}{(r_kr_k)^{3/2}}  $

$ = \frac{\vec{P}-3(\vec{P}\cdot\hat{e}_r)\hat{e}_r}{r^3} $

    - 代入得：

$ \vec{E} = -\frac{1}{4\pi\epsilon_0}\nabla(\frac{\vec{p}\cdot\vec{r}}{r^3}) = \frac{1}{4\pi \epsilon_0} \frac{\vec{P}-3(\vec{P}\cdot\hat{e}_r)\cdot\hat{e}_r}{r^3} =  \frac{1}{4\pi \epsilon_0} \frac{3(\vec{P}\cdot\hat{e}_r)\hat{e}_r-\vec{P}}{r^3} $

$ \mathrm{Q.E.D.} $

### 连续电荷分布的多极展开
#### 多极展开：多元函数泰勒展开在电动力学中的应用
+ 考虑**连续的电荷分布产生的静电场**：

$ \phi(\vec{r}) = \frac{1}{4\pi \epsilon_0} \int\frac{\rho(\vec{r}')}{|\vec{r}-\vec{r}'|}dV' $

    - 使用多元函数的泰勒展开：

$ = \frac{1}{4\pi \epsilon_0} \int dV' \rho(\vec{r}')\left[\frac{1}{r} - \vec{r}'\cdot \nabla \frac{1}{r} +\frac{1}{2!}\vec{r'}\vec{r}':\nabla\nabla \frac{1}{r}+\cdots\right] $

    - 将积分部分合并到里面：

$ =\frac{1}{4\pi\epsilon_0} \left[ \frac{Q}{r}-\vec{P}\cdot\nabla\frac{1}{r}+ \overleftrightarrow{D}:\nabla\nabla\frac{1}{r}+\cdots\right] = \frac{1}{4\pi\epsilon_0} \left[ \frac{Q}{r}-\vec{P}\cdot\nabla\frac{1}{r}+ \frac{1}{2}\sum_{i,j}D_{ij}\frac{\partial^2}{\partial x_i \partial x_j}\frac{1}{r}+\cdots\right] $

+ 其中$ Q $表示总电荷：

$ Q = \int_{V'}\rho(\vec{r}')dV' $

    - $ \vec{P} $代表电偶极矩：

$ \vec{P} = \int_{V'}\rho(\vec{r}')\vec{r}'dV' $

    - $ \overleftrightarrow{D} $代表电四极矩：

$ \overleftrightarrow{D} = \int_{V'}\rho(\vec{r}')\vec{r}'\vec{r}'dV' $

    - $ D_{ij} $代表电四极矩的分量：

$ D_{ij} = \int_{V'}\rho(\vec{r}')x_i'x_j'dV' $

+ 可以预见，电$ 2^{n} $极矩的表达式为：

$ \bold{D} = \int_{V'}\rho(\vec{r}')\underbrace{\vec{r}'\vec{r}'...\vec{r}'}_{n个\vec{r}'缩并}dV' $

#### 电四极矩的性质
+ 电四极矩是三维空间中的二阶张量，总共有$ 3^2 = 9 $个分量。
+ 根据分量表达式：

$ D_{ij} = \int_{V'}\rho(\vec{r}')x_i'x_j'dV' $

    - 可知应该是对称张量，因此独立分量只有$ 9-3=6 $个。
+ 在电四极矩张量上增加一个正比于单位张量的项不会对物理造成影响（可以把它踢出去）。

#### 从电四极矩到约化电四极矩
+ 可以证明，用总电荷、电偶极矩、电四极矩表达电势为：

$ \phi(\vec{r}) = \frac{1}{4\pi \epsilon_0} \left[ \frac{Q}{r} + \frac{\vec{P}\cdot\hat{e}_r}{r^2}+ \frac{\overleftrightarrow{D}:(3\hat{e}_r\hat{e}_r-\overleftrightarrow{I})}{2r^3} + \cdots\right] $

    - 但是这个定义下的电四极矩会受到总电荷的严重干扰，设想原点发生平移，那么电四极矩也会发生改变。

$ \overleftrightarrow{D} = \int_{V'}\rho(\vec{r}')\vec{r}'\vec{r}'dV' $

+ 为了解决这个问题，我们引入约化电四极矩：

$ Q_{ij} = \int \rho(\vec{r}') (3x'_i x'_j - r'^2 \delta_{ij}) \, dV' $ 

    - 约化电四极矩就是电四极矩去掉迹的部分：

$ \mathrm{tr}\overleftrightarrow{Q} = Q_{xx}+Q_{yy}+Q_{zz} = 0 $

    - 此时电势表达式可以改写为：

$ \phi(\vec{r}) = \frac{1}{4\pi \epsilon_0} \left[ \frac{Q}{r} + \frac{\vec{P}\cdot\hat{e}_r}{r^2}+ \frac{\overleftrightarrow{Q}:\hat{e}_r\hat{e}_r}{2r^3} + \cdots\right] $

+ 这个推导可以参考[反比例函数的情形](#InxLC)。

### 电荷体系在外电场中的能量
+ 设外电场的电势为$ \phi_e $，具有电荷分布$ \rho(\vec{r}) $的体系在外电场中能量为：

$ W = \int \rho\phi_edV $

+ 假设电荷分布于小区域内，此时对于外电势在原点附近进行展开：

$ \phi_e(\vec{r})=\phi_e(\vec{0}) +\vec{r}\cdot \nabla \phi_{e}|_{\vec{0}}+\frac{1}{2!}\vec{r}\vec{r} :\nabla \nabla \phi_{e}|_{\vec{0}} + \cdots  $

+ 代入：

$ W = \int \rho\left[\phi_e(\vec{0}) +\vec{r}\cdot \nabla \phi_{e}|_{\vec{0}}+\frac{1}{2!}\vec{r}\vec{r} :\nabla \nabla \phi_{e}|_{\vec{0}} + \cdots \right]dV $

$ =\int \rho dV\phi_e(\vec{0})+ \int\rho \vec{r}dV \cdot \nabla\phi_e|_{\vec{0}}+ \int\rho \vec{r}\vec{r}dV \cdot \nabla\nabla\phi_e|_{\vec{0}} $

$ =Q\phi(\vec{0}) + \vec{p}\cdot \nabla \phi_e(\vec{0}) + \frac{1}{6} \overleftrightarrow{Q}:\nabla\nabla\phi_e(\vec{0})+\cdots  $
