### 复习：静电势的偏微分方程与边界条件
#### 偏微分方程
+ 在[2-1 静电势方程及边界条件 2026.03.31](https://dengbaiyu2023.yuque.com/org-wiki-dengbaiyu2023-vsgo6q/lzpma1/rculygoci2h6b1i9)中，我们根据：

$ \nabla\cdot\vec{E} = \frac{\rho_{自由}}{\epsilon} $

    - 和：

$ \vec{E} = -\nabla \phi $

    - 推出静电势满足的方程：

$ \nabla^2\phi = -\frac{\rho_{自由}}{\epsilon} $

+ 特别地，如果$ \rho_{自由} = 0 $，那么退化为拉普拉斯方程：

$ \nabla^2\phi = 0 $

#### 边界条件
$ \phi_2 - \phi_1 = 0 $

$ \epsilon_2 \frac{\partial \phi_2}{\partial n} - \epsilon_1 \frac{\partial \phi_1}{\partial n} = -\sigma_{自由} $

### 静电问题的唯一性定理  
#### 定理
+ 区域$ V $可以分成若干区域$ V_i $，每一均匀区域具有介电常数$ \epsilon_i $。
+ 如果给定$ V $内自由电荷分布为$ \rho(\vec{r}) $，并在$ V $的边界上给定电势$ \phi|_S $或者电势的法向导数$ \frac{\partial \phi}{\partial n}|_S $，那么$ V $内电势分布唯一地确定。

#### 证明
+ 假设存在两组解$ \phi',\phi'' $都满足唯一性条件，令：

$ \phi = \phi' - \phi'' $

+ 首先讨论**内部边界**：根据前面的论述，我们有：

$ \phi_i' - \phi_j' = 0 $$ \phi_i'' - \phi_j'' = 0 $

$ \epsilon_i\left( \frac{\partial \phi'}{\partial n}\right)_i - \epsilon_j\left( \frac{\partial \phi'}{\partial n}\right)_j = -\sigma_{自由} $$ \epsilon_i\left( \frac{\partial \phi''}{\partial n}\right)_i - \epsilon_j\left( \frac{\partial \phi''}{\partial n}\right)_j = -\sigma_{自由} $

    - 相减，得到：

$ \phi_i-\phi_j = 0 $

$ \epsilon_i\left( \frac{\partial \phi}{\partial n}\right)_i - \epsilon_j\left( \frac{\partial \phi}{\partial n}\right)_j = 0 $

+ 再讨论整体边界：

$ \phi|_S = \phi'|_S- \phi''|_S = 0 $

$ \frac{\partial\phi}{\partial n}|_S = \frac{\partial\phi'}{\partial n}|_S- \frac{\partial\phi''}{\partial n}|_S = 0 $

    - 两行最后一个等号都是因为这些值是给定的，相等，所以相减为零。
+ 考虑第$ i $个子区域$ V_i $以及其边界$ S_i $上的积分：

$ \int_{V_i} \epsilon_i (\nabla\phi)^2 dV = \int_{V_i}\nabla\cdot(\epsilon_i \phi\nabla\phi)dV - \int_{V_i}\epsilon_i \phi\nabla^2\phi dV $

$ =\oint_{S_i} \epsilon_i\phi \nabla\phi \cdot d\vec{S} -0 = \oint_{S_i} \epsilon_i\phi \nabla\phi \cdot d\vec{S} $

+ 对于子区域求和：

$ \sum_i\int_{V_i} \epsilon_i (\nabla\phi)^2 dV = \sum \oint_{S_i} \epsilon_i\phi \nabla\phi \cdot d\vec{S} $

    - 因为在表面上有：

$ \phi_i-\phi_j = 0 $

$ \epsilon_i\left( \frac{\partial \phi}{\partial n}\right)_i - \epsilon_j\left( \frac{\partial \phi}{\partial n}\right)_j = 0 $

    - 内部边界面积分相互抵消，只剩下表面$ S $的积分，而在表面上有：

$ \phi|_S  = 0 $

$ \frac{\partial\phi}{\partial n}|_S = 0 $

    - 所以：

$ \sum_i\int_{V_i} \epsilon_i (\nabla\phi)^2 dV = 0 $

+ 结合$ \epsilon_i>0,(\nabla\phi)^2\geq 0 $可以得到：

$ \nabla\phi = 0 $

    - 即在$ V $内$ \phi = \mathrm{const.} $，即：

$ \phi'-\phi'' = \mathrm{const.} $

    - $ \phi',\phi'' $最多只差一个常量，不影响电场强度，得证。

$ \mathrm{Q.E.D.} $

### 有导体存在时的唯一性定理
#### 定理
+ 为了简单起见，我们只讨论区域中含有一种均匀介质的情形。
+ 设区域$ V $内有一些道题，给定导体之外的电荷密度$ \rho $，给定各导体上的总电荷$ Q_i $以及$ V $的（外）边界$ S $上的$ \phi $或者$ \frac{\partial \phi}{\partial n} $值，则$ V $内的电场唯一地确定。也就是说，存在电势解，他们相互之间只差一个常量。
    - 我们之后会看到，给定各导体上的总电荷$ Q_i $是为了确定内边界上的下列积分值：

$ -\oint_{S_i} \frac{\partial \phi}{\partial n}dS = \frac{Q_i}{\epsilon} $

+ 在导体之外的区域满足泊松方程：

$ \nabla^2 \phi= -\frac{\rho_{自由}}{\epsilon} $

    - 导体表面上满足等势面条件：

$ \phi|_{S_i} = \phi_i = \mathrm{const.} $

#### 证明
+ 我们假设有两个电势解$ \phi',\phi'' $满足上述条件，令：

$ \phi = \phi' - \phi'' $

+ 显然我们有：

$ \phi|_S = \phi'|_S - \phi''|_S = 0 $

或者

$ \frac{\partial \phi}{\partial n}|_S = \frac{\partial \phi'}{\partial n}|_S - \frac{\partial \phi''}{\partial n}|_S = 0 $



$ \nabla^2 \phi'= -\frac{\rho_{自由}}{\epsilon} $$ \nabla^2 \phi''= -\frac{\rho_{自由}}{\epsilon} $



$ \phi'|_{S_i} = \mathrm{const.} $$ \phi''|_{S_i} = \mathrm{const.} $

或者

$ -\oint_{S_i} \frac{\partial \phi'}{\partial n}dS = \frac{Q_i}{\epsilon} $$ -\oint_{S_i} \frac{\partial \phi''}{\partial n}dS = \frac{Q_i}{\epsilon} $

    - 两两相减得到：

$ \nabla^2 \phi=0 $

$ \phi|_{S_i} = \mathrm{const.} $

或者

$ -\oint_{S_i} \frac{\partial \phi}{\partial n}dS = 0 $

+ 在区域$ V' $（扣除掉道题部分的$ V $）中，有：

$ \oint_{全部} \phi \nabla \phi \cdot d\vec{S} = \int_{V'}\nabla\cdot (\phi \nabla \phi)dV= \int_{V'} (\nabla \phi)^2 dV + \int_{V'}\phi\nabla^2 \phi dV=\int_{V'} (\nabla \phi)^2 dV $

    - 第一个等号用到了数学上的高斯公式。
    - 第二个等号用到了偏导数的求导法则。
    - 第三个等号用到了$ \nabla^2 \phi=0 $说明第二个积分项是$ 0 $。
+ 我们考虑所有的内部边界：

$ \oint_{内部} \phi \nabla \phi \cdot d\vec{S} =\sum_i -\phi_i \oint_{S_i} \frac{\partial \phi}{\partial n}dS = 0 $

    - 第一个等号后面的符号代表对于$ V' $外法向实际上是指向导体内部。
    - 第二个等号是带入了前面的表达式$ -\oint_{S_i} \frac{\partial \phi}{\partial n}dS = 0 $。
+ 再根据：

$ \phi|_S = \phi'|_S - \phi''|_S = 0 $

或者

$ \frac{\partial \phi}{\partial n}|_S = \frac{\partial \phi'}{\partial n}|_S - \frac{\partial \phi''}{\partial n}|_S = 0 $

    - 可知：

$ \oint_{外部} \phi \nabla \phi \cdot d\vec{S}=\oint_{S} \phi \nabla \phi \cdot d\vec{S} = 0 $

$ \Rightarrow \oint_{全部} \phi \nabla \phi \cdot d\vec{S} = \oint_{外部} \phi \nabla \phi \cdot d\vec{S} +\oint_{内部} \phi \nabla \phi \cdot d\vec{S} =0 $

$ \Rightarrow  \int_{V'} (\nabla \phi)^2 dV= 0 $

$ \Rightarrow  (\nabla \phi)^2 = 0 $

$ \Rightarrow  \nabla \phi = 0 $

    - 即在$ V $内$ \phi = \mathrm{const.} $，即：

$ \phi'-\phi'' = \mathrm{const.} $

+ 此外，导体外的电势确定后，根据边值关系：

$ -\epsilon \frac{\partial \phi}{\partial n}|_{S_i} = \sigma $

    - 导体面电荷密度也同时确定。

$ \mathrm{Q.E.D.} $
