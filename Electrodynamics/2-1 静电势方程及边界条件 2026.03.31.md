### 静电场的方程
+ 介质中的麦克斯韦方程组第一式为：

$ \nabla \cdot \vec{D} = \rho_{自由} $

    - 在线性介质中，有：

$ \vec{D} = \epsilon_0\vec{E} + \vec{P} = \epsilon_0\epsilon_r\vec{E} = \epsilon\vec{E} $

    - 可以推知：

$ \nabla\cdot\vec{E} = \frac{\rho_{自由}}{\epsilon} $

+ 结合电场与电势的关系：

$ \vec{E} = -\nabla \phi $

    - 可以推出：

$ -\nabla \cdot \nabla \phi = \frac{\rho_{自由}}{\epsilon} $

$ \Rightarrow \boxed{\nabla^2 \phi = -\frac{\rho_{自由}}{\epsilon}} $

    - 可见静电场满足的偏微分方程是泊松方程。
+ 如果所讨论区域的电荷密度处处为零，则退化成拉普拉斯方程：

$ \nabla^2 \phi = 0 $

### 静电势的边界条件
+ 我们首先写出电场强度、电位移矢量在边界上的行为：

$ \vec{n} \times (\vec{E}_2-\vec{E}_1) = \vec{0} $

$ \vec{n}\cdot (\vec{D}_2-\vec{D}_1) = \sigma_{自由} $

$ \vec{D} = \epsilon \vec{E} $

+ 从介质1到介质2的电势变化量：

$ \phi_2-\phi_1 = \nabla \phi \cdot d\vec{l} = -\vec{E}\cdot d\vec{l} $

    - 因为$ \vec{E} $是有限值，而且总可以让$ d\vec{l} $取得无穷小，那么等式右边可以趋于$ 0 $。由此：

$ \boxed{\phi_2 - \phi_1 = 0} $

+ 电位移矢量式可以改写为：

$ D_{2t} - D_{1t} = \sigma_{自由} $

    - 结合：

$ \vec{D} = \epsilon \vec{E}\Rightarrow D_t = \epsilon E_t $

$ \vec{E} = -\nabla\phi \Rightarrow E_t = -\frac{\partial \phi}{\partial n} $

    - 可以得到：

$ \boxed{
\epsilon_2 \frac{\partial \phi_2}{\partial n} - \epsilon_1 \frac{\partial \phi_1}{\partial n} = -\sigma_{自由}} $

    - 前面的负号来自于电场强度等于电势的负梯度。

#### 导体
+ 导体内部：$ \vec{E} = \vec{0} $，不带电，电荷分布在导体表面。
+ 导体表面：$ \vec{E} = \hat{e}_n E_n,\; \vec{E}_t = \vec{0} $，表面为等势面。
+ 导体和介质面的连接条件：

$ \phi = \mathrm{const.} $

$ \epsilon \frac{\partial\phi}{\partial n} = -\sigma_{自由} $

+ 导体内部等势：

$ \nabla\phi = \vec{0} $
