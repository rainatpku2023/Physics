#### 方程离散化
+ 希望对于下列二维拉普拉斯方程进行数值求解：

$ \frac{\partial^2 \phi}{\partial x^2}+\frac{\partial^2 \phi}{\partial y^2} =0 $

+ 在矩形网格中进行数值求解，进行离散化：

$ \phi(x,y)\rightarrow \phi_{ij} $

$ \frac{\partial \phi}{\partial x}\rightarrow \frac{\phi_{i+1,j}-\phi_{ij}}{h} $

$ \frac{\partial^2 \phi}{\partial x^2}\rightarrow \frac{\phi_{i+1,j}-2\phi_{ij}+\phi_{i-1,j}}{h} $

$ \frac{\partial^2 \phi}{\partial y^2}\rightarrow \frac{\phi_{i,j+1}-2\phi_{ij}+\phi_{i,j-1}}{h} $

+ 代入发现，拉普拉斯方程改写为：

$ \phi_{ij} = \frac{1}{4}(\phi_{i-1,j}+\phi_{i+1,j} + \phi_{i,j-1}+\phi_{i,j+1}) $

#### 例题：使用扩散方程解决第一类边值问题
+ 边界$ S $上$ \phi $已知（第一类边值问题），拉普拉斯方程$ \nabla^2\phi = 0 $为扩散方程$ \nabla^2\phi = \frac{\partial \phi}{\partial t} $的稳态解。

$ \frac{\phi_{ij}(t+\Delta t)-\phi_{ij}(t)}{\Delta t} = (\phi_{i-1,j}+\phi_{i+1,j} + \phi_{i,j-1}+\phi_{i,j+1} - 4\phi_{ij})/h^2 $

$ \phi_{ij}(t) \rightarrow \phi_{ij}^n $

$ \phi_{ij}(t+\Delta t) \rightarrow \phi_{ij}^{n+1} $

$ \phi_{ij}^{n+1} = \phi_{ij}^{n+1} + \frac{\Delta t}{h^2}(\phi_{i-1,j}+\phi_{i+1,j} + \phi_{i,j-1}+\phi_{i,j+1} - 4\phi_{ij}) $

    - 迭代稳定性要求$ \Delta t \leq \frac{1}{4}h^2 $，如果取最大时间间隔：

$ \phi_{ij}^{n+1} = \phi_{ij}^{n+1} + \frac{1}{4}(\phi_{i-1,j}+\phi_{i+1,j} + \phi_{i,j-1}+\phi_{i,j+1} - 4\phi_{ij}) $

+ 这种方法被称为雅可比迭代法。

```python
import numpy as np
import matplotlib.pyplot as plt

def solve_laplace(grid_size=50, max_iter=5000, tolerance=1e-5):
    # 1. 初始化网格 (N+1 x N+1)
    u = np.zeros((grid_size + 1, grid_size + 1))

    # 2. 设置边界条件
    # 根据图片：u(x, 1) = 1, 其余三边为 0
    u[-1, :] = 1.0  # 顶部 (y=1)
    # 其余边 u[0, :], u[:, 0], u[:, -1] 默认已经是 0 了

    # 3. 雅可比迭代
    for it in range(max_iter):
        u_old = u.copy()

        # 核心差分方程：u_ij = 1/4 * (u_i+1,j + u_i-1,j + u_i,j+1 + u_i,j-1)
        # 只更新内部节点 [1:-1, 1:-1]
        u[1:-1, 1:-1] = 0.25 * (u_old[2:, 1:-1] + u_old[:-2, 1:-1] +
                                u_old[1:-1, 2:] + u_old[1:-1, :-2])

        # 检查是否收敛
        if np.max(np.abs(u - u_old)) < tolerance:
            print(f"收敛于第 {it} 次迭代")
            break

    return u

# 执行求解
N = 500
u_final = solve_laplace(grid_size=N)

# 4. 绘图可视化
plt.figure(figsize=(8, 6))
plt.imshow(u_final, extent=[0, 1, 0, 1], origin='lower', cmap='plasma')
plt.colorbar(label='Potential u')
plt.title('Numerical Solution of Laplace Equation (Jacobi Method)')
plt.xlabel('x')
plt.ylabel('y')
plt.show()
```
